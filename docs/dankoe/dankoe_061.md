# 4 小时的专注工作将改变你的生活

> 原文：[`thedankoe.com/letters/4-hours-of-focused-work-will-change-your-life/`](https://thedankoe.com/letters/4-hours-of-focused-work-will-change-your-life/)

能够有意地随意引导你的注意力不仅能节省你的时间，还能节省金钱（见：[注意力经济](https://markmanson.net/attention)）。

+   主流新闻机构通过恐惧来货币化你的注意力。

+   社交媒体账户通过“有共鸣”的以及经常自我贬低的梗来窃取你的注意力。

+   Twitter、Instagram 和 Facebook 确切地知道哪些颜色、主题和内容传递风格能让你兴奋。

+   总是有东西可以填补无聊带来的空虚。

+   刚才，我点击过去查看我的 Twitter 通知。

从自上而下的角度来看，这是现代世界面临的大问题。

在心理健康方面，人们关注的是他们的想法，而不是想法背后的原因。

在身体健康方面，人们关注的是短期快乐，而不是长期能量。

在财务健康方面，人们关注的是快速致富的方案，而不是可持续的业务增长。

我们不是来谈论这些的。我们是来谈论专注的工作，但你也可以看到注意力经济所造成的问题。

专注是一种货币，我们必须明智地投资它。

## 风险最小，潜在回报

注意力经济只有在你是消费者端时才会是灾难性的。

机会像杂草一样涌现，为了利用它们，你需要产出。

你需要专注于建立风险最小但有可能带来巨大回报的东西。

这些被称为非对称赌注。

![非对称赌注插图](img/5496c897745f09a0ae36ccf1f9167995.png)

非对称赌注

在过去 5-10 年中出现的某些机会（你仍然可以利用）：

+   加密货币

+   建立社交媒体受众

+   提供专业化的自由职业服务

+   开始一家营销公司

+   将你的专业知识产品化并在网上销售

+   开始播客或博客

4 年前，我开始从事网页设计自由职业。这不需要一分钱，只需要在 YouTube 上花一些时间，但有可能取代你的收入。

1.5 年前，我创建了一个 Twitter 账号，它已经增长到 40,000 名关注者，并让我从为客户完成工作的业务中转型出来。

现在，我正处于博客、YouTube 频道和播客的起步阶段。鉴于我遵循了正确的公式，我毫不怀疑这些会带来巨大的回报。

> 公式：前期小额投资 + 持之以恒 + 迭代 = 指数级回报。

我的成功，无论大小，都是建立在非对称赌注之上的。

负责产生结果的载体是什么？专注的工作。

也就是说，通过为世界增加价值，一次投入一小时，将我的专注投入到注意力经济的股票市场。

## 4 小时比你想象的要多

我的挚友乔伊（[@heyjoeyjustice](https://twitter.com/psypreneur)）是一位心理表现教练。

他和许多其他人已经证明，通过专注，你可以将工作时间减半。

你在专注工作 4 小时内能完成的事情，比在分心工作 8 小时内能完成的还要多。

没错，即使是 1 小时的专注工作也能带来很好的回报，尤其是对于那些有其他耗时义务的人来说。

要开始你的赌注，有一个先决条件。

你必须有一些你想要追求的兴趣或好奇心。如果你不积极学习，找到可以从中获利的机会将会非常困难。

一旦你学会了，你必须将其应用于现实世界。

### 乔伊的进入专注工作 3 步法

在我们开始之前 - 拿一本物理笔记本和一支笔/铅笔。

**步骤 1）头脑风暴**

这有两个目的：

1.  清除内心的干扰（思想和想法）

1.  确定你为特定项目需要完成的精确任务

对此疯狂一点，写下所有想到的事情。这不仅仅局限于“工作”。

将家庭待办事项到本周购物清单（以及你必须为“工作”做的事情）都写下来。

**步骤 2）分类与优先级**

为任务创建分类：

+   家庭事务

+   社交成长

+   产品推广

+   寻找自由职业机会

+   客户工作

+   人际关系

+   健康

为了优先级，考虑使用艾森豪威尔矩阵

![艾森豪威尔矩阵](https://www.techtello.com/eisenhower-productivity-matrix/)

[艾森豪威尔矩阵](https://www.techtello.com/eisenhower-productivity-matrix/)

如果你不能决定某件事是否是优先事项，就抛个硬币。不知道是否是优先事项通常意味着它不是优先事项。抛个硬币可以节省你一些精神压力。

你应该带着每个类别中接下来几天内必须完成的 2-5 个任务离开。

**步骤 3）时间块与安排**

你知道你需要做什么，现在是时候将其固定下来。安排时间块让你的大脑为行动做好准备。

用 10 分钟将你的任务添加到你的日历或最喜欢的计划器中。

工作时使用番茄工作法 - 50 分钟专注，10 分钟积极休息（离开你的椅子）。

从小做起，逐渐增加工作量。从 1-2 小时开始，养成习惯，然后相应地增加。

安排你的时间块不应该超过每周 15 分钟。

### 乔伊的快速焦点保持技巧

**步骤 1）通过减少不确定性来增加清晰度**

+   战斗、逃跑或冻结 - 没有清晰度，你将无法战斗。

+   这可以通过再次执行上述步骤 1-3 来实现。

**提示 2）消除干扰**

+   将你的手机设置为勿扰模式，并将其从桌子上移开。

+   关闭不必要的浏览器标签页。

+   如果需要的话，出去走走（咖啡馆总是不错的选择）。

**提示 3）音乐**

+   听纯音乐而不是带歌词的音乐

+   电影或你最喜欢的视频游戏原声带

+   《星际穿越》、《黑暗骑士》或任何汉斯·季默的作品都能产生神奇的效果。

+   模拟波。

**提示 4）照顾好自己**

+   多运动（锻炼）。

+   吃得更好。

+   获得高质量的睡眠（绝对关键）。

+   优先考虑恢复（精神和身体）。

你无法强迫一个疲惫的大脑去工作。你的精神表现必须可持续。
