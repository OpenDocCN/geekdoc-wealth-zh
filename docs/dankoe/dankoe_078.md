# 我**必须**学习这些高收入技能（如果我想赚钱）

> 原文：[https://thedankoe.com/letters/i-had-to-learn-these-high-income-skills-if-i-wanted-to-make-money/](https://thedankoe.com/letters/i-had-to-learn-these-high-income-skills-if-i-wanted-to-make-money/)

我的商业之旅很简单。

我尝试了几乎所有事情，但都失败了。

但是，正如我们在这里都知道的，失败是一种祝福。

这是一个揭示可以改进之处的数据点。

那些没有失败的人不会成功。

当你想到它时，这根本说不通……期望没有失败的成功，而成功只存在于失败的基础上。

如果你从出生那天起就“成功”（你在很多方面都是），你会把它看作是成功吗？还是会把它看作是正常生活？

你必须设定一个新目标，将其设定为高目标，将巨大失败视为低目标，并在挣扎中舞动，以实现有意义的对比。

这提出了一个重要观点。

自立。

富家子弟并不自立，因为他们没有教会自己如何自立。成功对他们来说是常态。如此之甚，以至于他们没有新的潜力去发现。并非所有富家子弟都感到沮丧和社交无能，但许多许多都是，因为他们没有理由去培养一套技能来实现自我设定的目标。

没有人能教你**自我**自立。

否则，这就不算是自立。

要在任何领域（从金钱到关系再到健康）实现自立，你需要那些能够弥合你目前状态和想要达到的状态之间的差距的技能。大多数人不知道自己想要什么，甚至更多的人没有意识到他们目前的状态是一个问题。

注意上面段落中的“需要”这个词。

有一些技能你需要学习才能实现自立。

这些不是可选择的。

我重复一遍。它们**不是**可选择的。

这封信的其余部分不是个人方法。

我所展示的技能是每个成功人士都必须学习（或雇佣）以在商业、关系，甚至可以说是健康方面达到一定成功水平的技能——即使你可能还没有把它们看作是健康技能。

我在7种商业模式上失败了。

原因是我没有理解那些能真正让我赚钱的技能。

不，这并不是现代技能，比如网页设计、电子邮件营销或客户开发，这些都是人们告诉你要学习的。

我们需要剥开表面，深入探究这些技能之所以有效的原因。

一旦你真正通过实践理解了它们，那么你将能够使金钱流动得像你的水槽一样流畅。

## 永恒的技能

我正在构建一个软件业务，我现在感到迷茫。

商业结构、商标律师、普通律师、招聘、设计阶段、运营协议等。

但是，这些都是我可以即兴学习的事情。

每一个其他业务（除了[个人企业](https://thedankoe.com/letters/the-one-person-business-revisited-turn-yourself-into-a-business/))最终都需要学习如何开始一个“真正的”业务，因此网上有大量的资源可用。

在一家个人企业中，你可以建立一个社交媒体档案，创建一个数字产品，并在大多数初学者案例中免费推广它。

重点是，开始一项业务所需的技术技能是存在的，并且可以学习。

但我们不是来谈论那些的。

因为如果那些是商业成功的驱动力，那么大多数企业都不会失败。

**营销和销售**是不同的。

这些是永恒的技能。

他们需要创造力、经验和对你所在行业的整体理解才能发挥作用。

营销和销售是赚钱和亏钱之间的区别因素。

以科技初创公司为例。

他们有出色的开发人员和一个想法。

他们开发了软件，发布了，然后……就没有然后了。

他们没有信任他们的受众。他们没有吸引目标市场注意力的促销活动。他们没有解决一个具体的热点问题。他们没有计划随着时间的推移提高问题意识（他们不通过内容**创造**自己的客户）。他们没有任何可以让他们写出好内容、广告并将观众转化为客户的火力。

**营销和销售**与**写作和演讲**的结合是你创造价值的方式。

我之前讨论过价值创造，这是建立我的个人企业的单一技能。[这封信](https://thedankoe.com/letters/value-creation-the-skill-that-built-my-one-person-business/)将把你的任何行为转变为有利可图的业务。

为了总结你在写作和演讲中必须展示的营销和销售原则，你需要：

+   一个期望的结果

+   一个阻碍前进的热点问题

+   一个通往结果的既定路径（独特于你的经验）

+   你所针对的人的意识的水平（他们对问题的了解程度如何？）

+   基于八种人类欲望来谈论那个问题的角度

+   一份关于好处和解决问题如何影响他们生活的列表

+   来自你自己或前客户的证明，表明你可以解决这个问题

+   一个在互联网噪音中引起注意的大想法

+   一个风险逆转或保证，使他们跨过边缘

营销和销售是永恒的，因为它们基于机制、人类行为和心理。它们像所有事物一样从思想开始。

这些都是你内容、着陆页以及你与人们或客户沟通的任何其他地方的有力论点。

你**不仅**在推销一个产品或服务。

当你写作内容时，你正在推销你的想法。

你的内容没有得到参与或追随者，因为你不能将上述列表中的组件结合成有影响力的写作和演讲。

记得我们讨论过的自给自足吗？

其余的取决于你。

迷恋地研究营销和销售，但记住上述要点，这样你可以筛选出重要的内容。

## 永恒的容器

大多数人失败是因为他们不理解商业是如何运作的。

你需要一个好的产品（制作过程中涉及一系列技能）以及需要这个产品的人。

让我们通过一个场景来了解一下：

我想卖一个计划表。

因此，我先从自己身上进行实验。

为什么？因为你需要经验和结果。大多数企业失败是因为它们试图向那些从未经历过问题的客户销售。

我在亚马逊上购买了4-5本畅销的计划表。

我购买生产力书籍和课程。

我沉迷于播客、社交媒体账户和生产力技巧的视频。

我规划我的日子，规划我的工作，并将我所学的应用到实践中。

到这个阶段，我已经有了如何改进我想出售的计划表的想法——或者基于我的研究发现的全新事物。

我也让自己接触到使那个业务运作的全部内容。

+   我已经听过一些高绩效的社交媒体内容（我可以复制这些内容来推广我的计划表并建立追随者，而不需要广告）。

+   我已经从那些靠这些计划表谋生的人那里购买了计划表、书籍和课程（这样我可以逆向工程他们的营销和网页）。

+   我已经取得了结果，以至于我有直接的经验、期望的结果以及达到那里的系统（这构成了我的营销策略）。

它就在你面前，但你没有注意到。

如果你想在工作中赚钱之外赚钱，你需要**分销和产品**。

如果你想要作为一个没有地理位置限制的人来做这件事，你需要**代码和内容**。

代码是互联网的后端。

内容是互联网的前端。

代码是你托管内容的方式。

内容是你吸引注意力的方式。

你不需要学习如何编码，因为技术已经发展到我们现在这个阶段。

以前需要10名员工来建立会员网站的事情，现在一个人购买软件就能完成。

你可以点击“发布”，一个大型账户看到这个帖子，分享它，并且在不发布之前就能获得数十万的观看量。

你可以建立一个读者群，让你在生活中做任何你想做的事情。你不必局限于一个静态的细分市场。

总而言之，我想说的是，创造内容不再是可选的。

那是几年前的事情了，但市场在演变。

>

内容是你分发产品的方式。

营销和销售是你创造市场真正重视的产品和内容的方式。

这些是你无论学习什么其他技能都需要优先考虑的主要技能。

## 现代职业道路

![图片](../Images/b27c6bdc47b832784f89eb5c9c0f31fc.png)

即使大多数人看不到，我对创作者经济仍然充满信心。

我认为这是自然界纠正工业革命中出错的方式。

学校教育随着内容和课程而分散化。

学生可以追求他们的好奇心，并快速适应时代。

对于自学并掌握为其他创作者和最终为自己工作的技能的个人来说，8小时工作日和工资奴隶的时代已经结束。

这里是我预测的现在和未来的职业道路：

### 1) 代码与内容

> 代码和媒体是无许可杠杆。它们是新富背后的杠杆。你可以在睡觉时创建为你工作的软件和媒体。 —— 纳瓦尔

大多数人将不得不从代码或内容开始。

因此，你要么成为一名程序员，要么成为一名内容营销员。

你可以学习两者，但当然，随着时间的推移。（这是一套最强大的技能组合）。

程序员可以为创作者、初创公司以及任何现代工作工作。他们可以自由职业或被上述所有公司雇佣。

内容营销员可以做完全一样的事情。每个人都有内容、营销、销售或任何形式的媒体团队。

大多数人应该写内容来构建他们的公众简历（或个人品牌）。

那就是你将来找到客户、工作和吸引客户的地方。

你的个人品牌是你的在线形象。

如果你选择不学习编程，那完全没问题，因为现代软件使得你不必这样做。

### 2) 营销与销售

> 学习销售，学习构建，如果你能两者兼之，你将无所不能。 —— 纳瓦尔

这两条道路最终都会汇聚到学习营销和销售的*必要性*。

大多数软件公司和程序员失败，因为他们可以构建产品，但无法销售。

对于内容创作者来说，也是如此。

他们可以用良好的写作建立受众，但到了创造人们真正想要的产品时，他们就力不从心。

如果你想要在不可避免地失去光泽并把你束缚在工资上的工作之外生存，营销和销售就是你在创业狩猎中的武器。

### 3) 建立分销

> 先建立分销，然后构建你想要的任何东西。 —— 杰克·巴切

分销是在数字财富竞赛中你能理解的最重要概念。

分销方法在杠杆作用上有所不同。

自由职业者、代理机构、本地企业和甚至初创公司通常专注于低杠杆、即时满足的分销方法：

+   冷邮件

+   有偿广告

+   直接消息

现在，这些仍然是获取客户的可行方法，但如果你停止投入努力，你通常一无所获。

你最好建立长期的分发渠道（同时混合使用其他方法以快速看到收益）：

+   **建立观众群** - 就像拥有10万粉丝的X平台观众群。

+   **向你的网络推广** - 例如让你的10万粉丝互联网朋友推广你的作品。

+   **租赁他人的观众群** - 例如通过新闻通讯或YouTube赞助来面对数百万感兴趣的人。

你必须将社交媒体作为一种技能来利用全球分销渠道。

当你写内容并建立观众群时，你仍然可以通过电子邮件、消息或付费广告立即开始赚钱。

为了有参考，我仅在X上每月就有大约1200万的曝光量。在其他平台上，这个数字可能是四倍。

从谷歌搜索（我不知道这个有多准确），低CPM大约是3美元（1000次曝光）。

仅在X（以前是Twitter）上，我每月就能获得价值36000美元的曝光量。

如果你的内容/广告有利润 - 你应该能够从那每个月赚取超过10万美元。

### 4) 从可雇佣到不可雇佣

有那么一个时刻，你知道你永远不会再次被雇佣。

你的技能会累积成在任何市场条件下赚钱的能力。

我会遵循的步骤是：

+   开始学习所有品牌都需要掌握的技能 - 营销、销售、品牌、内容、设计等。

+   申请工作或为创作者工作以获得实际经验（并赚一些钱）。这可以是自由职业或雇佣。

+   每天至少投入1小时来建立自己的事业。写内容，迭代直到你开始增长，并推广你的产品。

+   随着你的观众群增长，开始过渡到自己做的事情。扩大你的产品栈以针对你观众中的更多人。

这样做，你几乎不可能不确保你的未来。

如果你没有良好的营销技能，一个庞大的观众群可以弥补这一点。

如果你没有良好的观众群建设技能，营销和销售知识可以弥补这一点。

一个多元化的技能栈，加上日常的努力，将解决你大部分的问题。

### 5) 重复性产品

大多数创作者失败是因为他们将货币化视为一次性的交易。

你的产品和服务*演变*与你一样。它们相互构建。如果你从未推出第一个产品，你就无法创造一个更好的产品。

停滞 = 死亡。

我的网页设计服务演变成了漏斗。我的漏斗知识演变成了3个数字产品。这些数字产品演变成了一个社区。我在此过程中积累了2个更多数字产品。我的10多个吸引点演变成了一个计划，现在将在9月底与我新书一起重新推出。所有这些都在演变成我正在构建的软件。这永远不会结束。这就是它有意义的原因。

显然，如果某样东西卖不出去，那问题出在你身上，而不是市场。

如果你持续学习和完善你的技能组合，你就会看到你产品改进的机会。

所有这些商业事务只是生活中的持续和迭代。

继续构建。

– 丹·科
