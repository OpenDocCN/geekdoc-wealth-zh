### 员工与企业家

大多数人梦想成为一家十亿美元公司的首席执行官，但甚至无法成为自己生活的首席执行官。

员工心态是很难打破的：

+   你不会追求除了你所知道的其他任何事物。

+   你不会继续发展你的技能集，超出分配给你的工作。

+   你等待别人告诉你工作和生活中的事情，而不是自己做出决定。

即使你已经搬出去，你的父母仍然是你的老板。编程仍然在拉扯你的心灵。

企业家具有与我们的祖先和自然相一致的独特特质：

+   他们勇敢地进入未知领域，开辟新的道路。

+   他们为推动人类进步的创造性问题构建解决方案。

+   他们从未停止获取知识并将其应用于现实。

在新经济中，你可以选择任何一条道路。

雇主根据你的社交媒体来雇佣你。

### 分配与创造的产品

“如今每个人都有自己的产品。”

我是在克里斯·威廉姆森推广他新推出的生产力饮料的帖子下看到的。

评论者将其视为一种负面说法。

仿佛他们不是在为拥有产品的其他人工作。

是的，如今每个人都有自己的产品。因为如果你不想被分配一个要卖的产品，这是必需的。

“我不想卖东西，这很俗气。”

那么你就不懂销售。你现在正在向我推销你的想法，以复制和维持你狭窄的身份。

*通过无所作为，你让比你意识差、智力低的人获得了比你更多的关注、影响力和金钱。*

*不采取行动，你将成为社会的木偶，被分配一个无意识的生活，那就是销售他人的产品，而自己却从未意识到这一点。*

创建一个你愿意购买、使用并从中受益的产品。

这就是商业中唯一的捷径。

你属于一个细分市场，你已经从这个细分市场购买了产品，如果你对商业有任何了解，你应该跳过50%的试错，并卖掉你购买的产品，但做得更好。

不要这么做。

从一个人开始。

利用你所能利用的技术。

部署数字地产，直到你的产品和服务有足够的分销。
