# 大多数人迷失了方向（如何快速毁掉你的生活）

> 原文：[https://thedankoe.com/letters/most-people-are-lost-how-to-ruin-your-life-fast/](https://thedankoe.com/letters/most-people-are-lost-how-to-ruin-your-life-fast/)

毁掉你生活的最快方式：

做别人希望你做的事情。

从不质疑你所被告知的事情。

选择表面上看似安全的路线，但实际上是最不安全的。

避免生命的本质——试错——并且关闭你对深刻、深邃和有意义的事物的接触。

最大的错误就是不做任何错误。

这就是大众的生活。

在表面上弹跳。

只学习他们在学校里学到的东西。

只是一天又一天地重复同样的经历。

上班打卡，下班打卡，然后在剩下的时间里麻木你的思维，不去想明天会发生什么。

脱离无意义生活的道路已经在你内心深处。

我们都可以使用自然指南针。

## 自然指南针

> 如果你不知道你想要什么，你会被告知你想要什么，你会相信它。[– 专注的艺术](https://theartoffocusbook.com)

如何做你想要做的事情：

做你想做的事情，但不要忽视你行动的后果。

你必须做你认为你想做的事情，才能意识到你不想做的事情。你必须去除你不想做的事情，才能做你真正想做的事情。

要绝对确定未来会发生什么是不可能的。

这就是为什么大众都涌向“安全”的工作和信仰体系。

避免斗争的确定性是一种错觉。你不能跳过犯错误。错误是你在黑暗中的光明。错误揭示了需要解决的问题。问题是你的思维的限制。

就像爱因斯坦说的，“你不能用创造问题的同一思维来解决它。”

大多数人都跳过了那部分。

他们认为他们想要一辆好车，一份舒适的工作，以及环游世界——但一旦他们做了这些事情，表面的光鲜很快就会消失。

他们继续说服自己，那就是他们想要的生活，从不进化，从不追求更深层次的东西，也从不得到他们真正想要的东西。

即，他们从未得到他们本性所想要的，而不是他们自我所想要的。

要从生活中得到你想要的东西：

+   追求你认为你想要的东西。

+   理解你不想做的事情。

+   意识到那个问题。

+   与其相伴，研究它，剖析它。

+   扩展你的思维，超越它。

+   解决问题以进化一个层次。

+   重复这个过程，直到你达到生命的本质。

没有人能给你一条明确的路径来实现这一点。

人们可以给你提供方向，给你建议和提示，但他们不能为你行动，思考或解决问题。

你不知道你从生活中想要什么，因为你没有尝试过任何事情。

你没有得到结果，因为你没有获得那些结果所需的技能。

你不知道什么是有可能的，因为你没有接触到让你觉得可能性的观点。

你现在想要的可能明天、下个月或明年完全不同。

没有什么是永恒的。意义在于无休止的解决问题，直到你挖掘到你所寻找的东西。

这是正常的。这是生活。

## 生活的宏观周期——为什么你总是感到迷茫

<picture fetchpriority="high" decoding="async" class="wp-image-1856">![](../Images/7955a53a9990577250becdff2f5ba82c.png)</picture>

你感到迷茫，因为你不知道接下来该做什么。

你不知道接下来该做什么，因为你没有目标。

你没有目标（你关心的目标），因为你没有意识到需要你关注的某个问题。

在你确定你生活中下一个想要解决的问题之前，不要担心设定目标。

生活以章节、阶段和周期展开。

你的生活是一个故事。

你的年份是一个故事。

你的日子是一个故事。

它们各自有一系列的目标、问题、高潮和低谷，这些可能随着时间的推移而重复。

生活中有一个宏观周期。

一旦你理解了它，你就可以确定你在故事中的哪个部分，了解其组成部分，并让你的心情平静下来，直到你进入下一个阶段。

这个周期由三个阶段组成：

1.  **你感到迷茫**。你不知道接下来该追求什么。你已经解决了问题，实现了目标，并认为这是终点。

1.  **你变得好奇**。你尝试不同的主题、兴趣和想法，直到你完全意识到你想要解决的问题。

1.  **你变得着迷**。你深入到你选择的现实裂缝中。你无法停止学习和朝着你的目标前进。

当你感到迷茫时，你必须强迫自己在两个方向上实验：

**1) 内向实验**

休息。小睡。写日记。散步。拥抱寂静。

观察并分解你的思想。

尝试捕捉机会的信号。

跟随你的思想到一个根本问题。

**2) 外向实验**

学习。阅读。滚动。构建。尝试新事物。

重新关注生活中唯一重要的事情。

健康、财富、关系、幸福。

尝试投资于任何目标的技巧。任何目标。

在实验阶段，你试图积累新的经验，直到你的下一个目标变得可见。

就像一款视频游戏，你处于一个任务尚未解锁但也没有其他任务可追求的点。你必须摸索在黑暗中，直到你在远处找到一支蜡烛。

注意：避免将此标签为闪亮物体综合症。

我今天的事业之所以可能，仅仅因为我尝试了7种不同的商业模式并“失败”。在反思中，那些失败是技能堆叠、解决问题和增加我对正确机会感知面积的阶段。

现在，理解这个阶段没有时间限制。

它可能持续几周、几个月或几年。

你必须坚持下去，直到你积累足够的视野来停止广泛探索并开始深入探索。

当你找到你的着迷之处，实验就不会停止。

**3) 向下实验**

秘诀在于尝试一切，直到找到那件让你无法自拔的事情。然后，一旦它成为你生活的一部分，就重复这个过程。

当你找到那件事时，深入挖掘。

尽可能多地了解那个领域的知识。

分析所有观点，避免对某一观点过于教条。

在营养方面，从素食主义、生酮饮食、食肉主义、雷·皮特、灵活饮食和其他意识形态中收集真理，直到你足够自信地创造出更接近真理的自己的理念。真理是一个过程，而不是一条命令。

在商业中，从电子商务、自由职业、软件和其他模式中收集真理，直到你注意到能为你自己的事业带来清晰度的原则。

自我实验是解决你问题的唯一方法。

人们可以诊断并为你提供解决方案，但这缺乏对那些提供解决方案的人的视角、目标和经验的考虑。

**4) 向上实验**

持之以恒和迭代。

我假设一旦你找到了你的痴迷，你希望让它成为你生活中的一致部分。

这意味着你必须从那个兴趣中获得创造性收入。

在当今世界，这意味着：

+   建立一个项目来帮助他人解决问题。

+   向公众教学以吸引有那些问题的人。

+   通过出售你的项目来继续追求你的痴迷。

+   在你学习只能通过反馈才能学到的东西的同时，改进你的项目。

+   当你达到只能通过商业达到的水平时，转向一个新的项目。

有很多种方法可以做到这一点。尝试。

但我可以自信地告诉你，通过我的实验，创作者经济是考虑到人类本质和未来工作的“最真实”模式。

## 数字文艺复兴

<picture decoding="async" class="wp-image-1857">![](../Images/0db151e65261fd3dbd55e50222bfd81b.png)</picture>

我们正在经历第二个黄金时代，它属于那些重视：

+   自我教育

+   自我实验

+   跨学科研究

+   数字杠杆

+   技能获取

+   创建自己的事物

如果你想要繁荣，成为一个数字文艺复兴人。

这是你要做的：

### 加入新社会

社交媒体不仅仅是你的手机上的一个应用程序。

它是一个去中心化的公立学校、市场和讲台，但只对那些重视个人责任的人开放。你不必被迫成为这个社会的一部分，这就是为什么它很重要。

你不再需要启动资金来创业。

你不再需要招聘网站来找到一份职业。

你不再需要出版社来出版书籍。

你不再需要唱片公司来制作音乐。

你不再需要正规教育来成为专家。

你不再需要宗教机构来赋予你意义。

对学习、意义和收入的集中式权威只属于那些仍在沉睡的人。

你所需要的只是一个互联网连接、社交媒体账户，以及教学、失败、改进和坚持的勇气。

我在[2小时作家](https://2hourwriter.com)和[数字经济学](https://digitaleconomics.school)中教授这一点。

### 变得有价值

每个人都告诉你变得有价值，但没有人告诉你价值是什么。

价值是你所做之事、你为什么这样做、你如何做以及谁从那个等式中受益之间的关系。

价值由4个部分组成。

+   **问题** – 当未解决时，会带来痛苦的限制或挑战。

+   **解决方案** – 一个有影响力的最终结果，使接收者能够超越问题。

+   **清晰度** – 一个创造性的系统，通过知识、技能和意识来弥合问题和解决方案之间的差距。

+   **人们** – 受问题困扰的人数，可以从解决方案中受益，并且准备好接受采取行动的清晰度。

价值是经验交换。

价值是你故事的影响力。

因此，为了创造收入，我们的工作是通过解决阻碍我们进化的问题来积累经验，并将浓缩的智慧传授给那些准备好接受的人。

**1) 用反愿景揭示问题**

从经验中知道你不想做什么比从想象中知道你想要什么更容易。

因此，为你的未来创造一个反愿景。

保持一个持续记录，随着你观察生活，你会添加内容：

+   你不希望看起来是什么样子？

+   你不希望感觉如何？

+   你的**最糟糕的一天**是什么样子？

+   你不希望在朋友、客户、合作伙伴和同事身上看到哪些品质？

+   多少钱**不足以**生存？

+   上述所有内容是如何渗透到你的生活的每个领域，并阻止你做你想做的事情的？

关注重要的事情。健康、财富、关系和幸福。这些都是商业中的永恒市场。如果你为这些领域内的一个问题构建一个创造性的解决方案，你将不会在从你的经验中谋生时遇到问题。

**2) 用愿景识别解决方案**

你现在有许多问题要解决，以逐渐增加你的价值。

选择一个。解决它。继续下一个。意义不在于目标的实现，而在于解决那些让你更深入现实本质的无尽问题。

现在，为了增加你解决它的愿望，为你的理想未来创造一个愿景。

为你的反愿景列出你所列出的所有内容的反面。

将其视为一个最小可行愿景。随着时间的推移，你会不断添加内容。你的愿望会随着你的成长而改变。

**3) 为了清晰度学习和构建**

先学会构建，再通过构建来学习。

你已经选择了你生活中想要解决的问题。这可能与你想要的生活相关，如健身、财务、自由或其他任何阻碍你过上想要的生活的事情。

在考虑这个问题时，集中精力在一个大的目标上。

但记住，目标一开始并不是有吸引力的。你可能并不关心目标。

你必须投入精力到目标中，直到你感觉如果不实现它就是在浪费资源。你的目标必须足够长时间地塑造你的视角，以便记录下继续追求它的理由。

你不可能在一天之内就发展出一种内在的哲学。

你通过对话、阅读、倾听和过滤现实来发展它，直到你堆叠足够的“为什么”来让你负责。

你会失败。这是唯一识别你需要学习下一步内容的方法。

当你失败时，立即将其重新定义为知识差距。

用失败作为新的框架来过滤你的经验以寻找答案。

**4) 用你的故事吸引人们**

成为失败者。

将项目推向一群蝈蝈。发布你的想法，然后被指责为傻瓜。开始创业，看看人们如何批评你的第一步。

每个人都这样做过。你不是特殊情况。

投资你的失败组合，直到你可以负担得起成功。

你的失败组合是你的故事。

就像电影一样，人们更感兴趣的是低谷而不是高峰。

当你生活中有一个问题的解决方案时，你会吸引人们和利润。

### 分享你的价值。

最大的技能是写作，因为：

+   它迫使你阐明你的价值。

+   它是所有媒介的基础。

+   它可以被改编成任何其他媒介。

+   你获得的任何其他技能都会增强它。

+   它带来了巨大的心理清晰度。

学习写作。

认识到写作是最具杠杆作用的工作需要时间。

你必须将你的价值综合成其他人头脑中免费存在并影响他们思想和行为的有影响力的想法。

你通过持续和迭代的努力撰写电子邮件、内容、指南和产品来做这件事。

写作是沟通的基础。

沟通是分享价值的方式。

分享价值是吸引人们的方式。

如果你不能吸引人们，你将永远不会有人参与价值交换过程。

这不是一份有人给你发工资来完成将被取代的机械任务的工作。

这是一个必须与他人共享的目标，以便赚取创意收入。
