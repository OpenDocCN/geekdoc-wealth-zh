## 3种杠杆形式

正如你们许多人所知，Naval Ravikant对我的人生产生了重大影响。

他完美地体现了我所追求的哲学理解和商业洞察力。

我一直在回顾他的一些最佳想法，并发现了他认为的3种杠杆形式。

第一点是劳动力。

第二点是资本。

第三点是复制边际成本为零的产品。

首先，为什么你应该关心在改变生活时杠杆的作用？

因为“*致富在于知道做什么，和谁一起做，以及何时去做*”。

如果你每天只有1小时的时间，这变得尤为重要。你需要确切知道应该做什么工作，与谁合作，以及这两者如何融入充满机会的数字世界的拼图之中。

### 1) 劳动作为杠杆

劳动在过去是有用的，但在今天的世界里，依赖它来成功是不明智的。

劳动是一个巨大的问题，它对我们社会的演变贡献最大。

没有人*喜欢*工作。尤其是那些他们觉得不感兴趣、不愉快，或者对任何有意义的目标没有贡献的任务。

因此，几乎是无意识地，技术发展到大多数人担心他们会因为自动化和人工智能而失去劳动工作。劳动的巨大问题几乎已经解决。感谢宇宙。我们很快就能作为一个集体超越到精神和物质层面。正如我们将在下周的信中讨论的，思想是新的石油。

技术使得一人的百万美元企业成为可能，创作者可以完全控制他们所做的工作以及他们工作的时间。

劳动作为杠杆意味着你雇佣他人为你工作。这有几个问题。

一个原因是每个人都想成为“顶尖玩家”。无论他们有多大的技能，每个人都想要股权。这全是一场充满变数的地位游戏。

两者是互补的，因为大多数人知道他们可以学习任何东西，建造任何东西，并利用他们培养的技能使数字业务盈利。他们不再需要依赖雇主来提供让他们致富的资源。所有这些信息都在网上，包括免费和具体的信息，如内容、课程、无代码工具，以及与任何人建立联系的能力。

劳动仍然可以工作，但还有更好的选择。

### 2) 资本作为杠杆

劳动自从第一个生物进入这场伟大的生存游戏以来就存在了。

资本是较新的发明。

从本质上讲，资本作为杠杆意味着你可以花钱来赚更多的钱。

你可以投资企业、房地产、股票或加密货币。

你可以投资那些能帮你分担工作负担的资源。

你可以投资信息，这些信息能让你避免代价高昂的错误，做出更好的决策，并更有效地利用你的时间。

你花钱是为了赚更多的钱。

问题在于，大多数人不仅在财务上破产，在其他领域也是如此。

他们没有足够的现金流来进行能带来显著收益的投资。

更糟糕的是，他们陷入了一种思维定势，认为每个月存几百美元，把每一分丢失的钱都预算进去，就能使他们变得富有。

更糟糕的是，他们认为他们可以利用像加密货币这样的新机会迅速致富，这是他们妈妈给他们的津贴。

当然，它们有自己的位置，但在我看来，它们是相当愚蠢和无知的选择。

为什么我要为了父母在我小时候给我的稀缺心态，每个月节省几百美元，在40年后赚100万，因为我很清楚普通人开始互联网业务，并在1-3年的专注努力中赚那么多？

然后，只有当我有充足的现金流时，我才能使用资本作为杠杆来增加我的收入。

我不想等待40年股票价格上涨（并让通货膨胀赶上），我想通过一个我认为是我一生工作的业务每年赚几百万，并在我40岁时将其变成数百万。

总之，资本作为杠杆是很好的，但不适合初学者。

这引出了我们的最后一个选择。一个你不应该轻视的选择。一个你不应该视为“只是一个阶段”的选择。

### 3) 数字产品作为杠杆

> 这是一个以数字为先的世界。物理产品现在是一种高端产品。 —— 巴拉吉
> 
> 物理产品，然后是数字产品，最后是原生数字产品。 —— 巴拉吉

首先，什么是产品？

产品是人们可以消费的商品、物品或创造物。

产品是消费者和创造者之间的桥梁。

物理产品有其局限性，如运输成本、制造成本和熵成本（如易腐商品和被磨损的书籍）。

相反，数字产品符合纳瓦尔对“复制边际成本为零”的定义。

数字产品是“一次构建，两次销售”的典范。

数字产品包括但不限于：

+   代码

+   电影

+   文章

+   播客

+   内容

+   通讯

+   课程

+   电子书

简而言之，它是媒体和代码。

所有这些都可以货币化，甚至像付费播客、博客或通讯这样的东西，但那些往往不是最好的货币化方式。

我的建议：

选择你最感兴趣的路线，但了解媒体通常比代码更重要。我认为这就是我和纳瓦尔的不同之处，他站在代码这边。

代码是赚钱的最后手段，但如果没有媒体和[未来保障技能栈](https://thedankoe.com/letters/the-future-proof-skill-stack-learn-in-this-order/)，这笔钱是无法赚到的。如果你不是控制媒体和代码愿景的企业家，你的收入仍然依赖于他人。

一个程序员可以在互联网上构建最好的应用程序，但如果他们不通过媒体（内容、播客、电子邮件、课程等）吸引人们，他们将如何获得用户？那代码将消失在虚无中，除了可能对你自己有价值之外，对其他人都没有价值。

### 1) 当心理熵低时构建

你每天只有1个小时。

嗯，大多数有责任的人只有这么多。如果你年轻，你有一个最大的优势：时间。不要像同龄人一样浪费它。

对我来说，把那个小时留到下班后或深夜，当心理熵高的时候是愚蠢的。你精力不足，你的头脑是明天任务和今天夜间责任的混乱。

成功是关于熵的管理。

你需要知道如何在你的日子里释放和约束熵，无论是你的日子、周、月还是年。当生活变得无聊和重复时，你需要通过设定和追求新的目标来释放熵。当生活变得压倒性时，你需要通过将这些目标分解成更小、更清晰、更易管理的目标来约束熵。

提前一个小时起床。

专注于1-3个能够推动你设定目标的事情。

我假设这个目标将是赚更多钱，因为这是在当今世界获得更多能量、健康、时间和联系的前提。这可能不公平，但现实并不关心你的感受。

你可以走任何你想走的路，但如果你想跟随数字杠杆的道路，做你喜欢的事情，以下是如何填充这段时间的方法。

### 2) 生产力与创造力时段

我喜欢在早上在两种宏观意识状态之间波动：生产力和创造力。

我发现，当熵值高的时候，我晚上根本无法完成任何事情。在那个时间点，我做的任何工作都不会是我能做的最高质量的工作。我需要专注于在场、休息和其他活动，这些活动允许我在早上做最高质量的工作。

因此，我这样安排我的日子：

+   早上醒来后，为了生物钟健康、一般健康，并在脑海中设定我的工作意图，我会先去散步。

+   我第一个工作时段是专门用于推动杠杆的任务。大多数时候，这是为新产品和我的通讯（因为那封通讯后来变成了YouTube视频和播客，所以这些很重要）写作。

+   第二次散步是为了更多的晨光。我一直在重新学习阳光的绝对重要性以及我们被告知的关于其危害的谎言。我会将这些内容留到未来的信中。我喜欢听这些讲座的*长*有声书或YouTube视频，因为大多数人讨厌的“废话”中，你可以找到大多数人都不谈论的罕见想法。这是即时满足的反面。

+   我的第二个工作时段是专门用于当前项目，比如Kortex或继续我之前已经开始的写作。

+   我还有一个更短的午间散步。这是我开始释放熵的时候。我检查我的手机，回复消息，并“打开自己面对世界。”我让我的日常变得宽松。

+   这通常是我感觉有点不专注的时候，所以我开始读书或寻找好的视频或文章来阅读。当灵感迸发时，这就是我一周大部分想法生成和计划发生的地方。

+   然后，我就做我想做的任何事情。如果我想的话，我可以工作得更久，或者我可以吃午餐，去健身房，多散步，和朋友见面，或者去吃晚餐。

在我的工作时段，我不会拖延。我直接进入写作。如果我等待太久并四处浏览，我最终会分心，浏览时间过长。

在我的创造力时段，我为当前和未来的工作时段生成想法。我每天做的每一件事都专注于我的工作目标。作为副产品，我过着相当愉快的生活。这并不意味着它总是愉快的，但缺乏愉快让你更加珍惜愉快的部分。

我认为大多数人的工作之所以有问题，是因为他们工作太多，然后继续为自己辩解，说他们永远不可能工作得少而完成同样多的工作。

我的饮食、散步、阅读、训练和休闲时间是工作的核心。由于我在思想空间工作，如果没有好的想法，我的工作就会受到影响。

即使你不从事思想空间的工作，正确的想法也是帮助你做出决策，让你在那些操作容易找到想法的人中脱颖而出。

当我工作过多而忽视创造力时，我会感到压力、愤怒和思想狭隘。

+   **产品** – 打造一个产品，让人们可以用金钱交换对他们生活有益的东西。这是掌控你的收入并停止依赖他人赚钱的唯一方式。这就是我在[Mental Monetization](https://mentalmonetization.com)中要教授的内容。

我觉得所有初学者都会问的一个问题是“我怎样才能赚钱？”

当他们得到唯一一个能真正让他们赚钱的答案时，他们会忽略它，转而去追求“更快”的赚钱方法，这些方法是为那些 *with* 钱的人保留的，比如投资、加密货币、初创公司等等。

你通过创业来赚钱。

你通过分发带有价格标签的产品来赚钱，这样人们就可以为你支付它。

认为不需要打造产品是愚蠢的。我知道你们中的一些人只有在我严厉对待你们时才会理解这一点……所以，停止做那个该死的白痴，最终意识到你需要一个 *产品* 和以 *观众* 形式存在的 *人* 来为你这个 *产品* 付费。

然后，只有在你每月至少有20,000至50,000美元的情况下，你应该考虑将资本投资作为杠杆来从中赚取更多利润。这不是一条硬性规则，但明智的做法是将其视为一条规则，这样你可以在这一旅程中保持一些心理秩序。不要分心。

### 4) 选择2-3个交集兴趣作为你的细分市场

我以前多次讨论过[缩小领域](https://thedankoe.com/letters/the-anti-niche-why-being-nicheless-makes-you-irreplaceable/)和[建立广泛的观众](https://thedankoe.com/letters/the-age-of-the-generalist-how-to-thrive-with-multiple-interests/)，所以在这里我就不赘述了。如果你需要复习，可以免费下载我的[个人业务迷你课程](https://theone-personbusiness.com)。

简而言之，你需要吸引人们朝着你引导他们的未来愿景前进。你的内容和产品帮助他们实现与该愿景一致的生活目标。

你必须教育他们实现这一目标所需的技能集和心态。

+   选择2-3个你想要围绕其创建媒体（社交媒体上的内容）并通过数字产品进行货币化的广泛兴趣。

+   将它们分解为子主题，这样你就可以更有效地进行研究以产生想法。

+   将它们进一步分解为帖子、通讯、播客和其他媒体想法。

然后，开始写作以吸引观众。

当然，创建内容或写作只是谜题的一部分。你需要了解如何使用[如何在没有粉丝的情况下建立观众](https://thedankoe.com/letters/how-to-actually-grow-on-social-media-what-they-dont-tell-you/)来控制你在社交媒体上的增长。

首先，开始教育自己如何在社交媒体上建立观众作为杠杆。

观看教程，购买课程，研究你想要成为的人，这样你就可以借鉴他们的想法并模仿他们的工作。

如果你每天只有一小时，你必须学习和一起构建。稍微教育自己一下，但不要犹豫开始写作和建立你的品牌。

现在：

你不需要销售产品的经验。

产品是帮助你获得经验的东西。

不，你不需要创造夸张的承诺，让自己变成一个不道德的骗子。

创建数字产品与创建实体产品或提供自由职业服务没有不同。

假设你想卖一本计划本。

你需要经验来销售计划本才能销售计划本吗？不，这是不可能的。

而不是这样：

+   买几本计划本，开始使用它们，看看哪些方法对你来说效果最好。

+   生产、销售和向通过内容建立起来的观众群体分销。
