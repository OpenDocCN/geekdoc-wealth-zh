## 掌握方法 - 如何快速学习任何事物

![图片](../Images/ef1008f25b3f1d5cdc1b25954ee58d52.png)

可以说，你将不得不学习……很多。

教育扩展了你的思维。

它向你介绍了新颖的视角。

它作为持续的能量来源，增加大脑中的多巴胺。

它给你提供了清晰地向目标行动的知识。

它让你接触到你尚未意识到的潜力。

一致的教育流动增加了你遇到有意义事件的机会。

有意义的事件发生在已知事物的边缘，当你的神经系统信号你应该注意时。

当你有一只脚在未知中时，你只能勉强代谢现实的新方面并将这些信息用于实际。

成为你想成为的人是你能奉献给你一生的最痛苦和最有回报的过程。

当你意识到你现在所处的痛苦和快乐比你得不到看到你所能做到的奖励的痛苦要小得多时，你就开始了这条道路。

基于此，让我们讨论如何掌握几乎所有事情，在你朝着成为新自己的道路上：

### 1) 扩展你的心智

所有真正的变化都是身份的变化。

你的心智水平决定了你的身份可以拥有的价值观、信仰和标准。

你不在乎全球问题，因为你还没有解决那些限制你心智将其视为重要问题的个人问题。

人类的目的是将你的心智水平扩展到宇宙的水平。

你必须给自己留出空间，通过在未知中抛锚来发现新的目标。

创建一个巨大的目标，在未知中充当焦点。

你不是为实用性或成就设定这个巨大的目标。

你设定它是为了视野、方向和机会的过滤。

专注于让它尽可能有吸引力。

+   如果你拥有世界上所有的钱，你平均的一天会是什么样子？

+   你希望生活在什么样的环境中？是否有特定的地点？你想要旅行吗？

+   你想要一个家庭吗？你希望那种生活是什么样子？想象一下家庭时间的平均一天。

+   你希望工作日有多长？如果你可以做任何事情，你会在工作上做什么？

+   你想要看起来和感觉如何？描述你的身体、能量水平以及你想要向世界展示的方式。

+   你的理想一天是什么样的？规划出每一小时。

+   列出任何其他你想要为自己构建的特定未来的想法。

为了使这一点更加有效，将这个愿景转化为一个反愿景，以完善你理想自我的视角：

1.  你存在的痛苦是什么？

1.  写出关于你视野的每个问题的对立面。

为了增加更多动力，创建一个愿景板。将图片添加到剪贴簿、软件或墙上，使那个未来更加具体。

记住，没有什么是一成不变的。

在这条路上行走时，你会发现你对愿景的启示。当你发现你不想做的事情时，要开放地改变你想要的事情。

达不到你愿景的痛苦应该超过平庸的愉悦。

不仅仅是一个巨大的目标提供了视野，它还允许你在需要做出决定时采取一个视角。

通过你的视野过滤你收到的每一个机会。

对除与你想要成为的人一致之外的所有事情说“不”。

通过有意识的选择，开始短路你大脑运行的错误编程。

### 2) 目标层次结构

这就是为什么我创建了[FOCI计划者](https://thedankoe.com/planner)：

大多数人不需要动力，他们需要清晰。

有序的心智是快乐的，自我生成的目标层次结构使得令人沮丧的干扰难以渗透你的意识。

大目标是用于视野和过滤，而小目标是用于实用性和进步。

现在，将你的视野分解成你生活中每个领域的目标——心智、身体、精神、商业。在这里，关系等同于精神。

是的，这将会很耗时。把这一点写在一个安全的地方，这样你可以在进行过程中随时更改。

对于生活的所有4个领域，创建：

+   10年目标

+   1年目标

+   月目标

+   周目标

然后，每天，你将写下3-5个优先任务，从底层推动这些目标。

早上第一件事就是完成这些任务，在责任和干扰有机会醒来之前。

我在我的信中谈到了我的深度工作结构：[在6个月内改变你的生活（我的深度工作常规）](https://thedankoe.com/letters/change-your-life-in-6-months-my-deep-work-routine/)。

### 3) 如何学习

学习来自斗争，而不是记忆。

你不会通过整天研究教程来学习。

你通过构建项目并面对现实来学习。

项目是可以衡量和迭代的目标。

因此，将你的一些1年或月目标变成一个项目。

+   构思一种构建有形事物的办法（你必须做工作）

+   记录下你可以达到的里程碑。

+   在笔记应用或笔记本中创建一个大纲。

+   将任何突然想到的想法都记录下来。

为精神、身体、财务和精神发展这样做。

这为你的大脑提供了模式识别引起的有意义的多巴胺。

你的项目给你的情境感知增加了额外的“规则”。

你现在有一个地方可以写下你在教育自己、构建过程中接收到的信息以及解释反馈时的高信号信息。

先开始，然后学习。你必须边学习边构建，才能真正地学习。

当你构建某物时，你会遇到需要更高层次思维来解决的真实问题。

只有这样，你才能搜索信息来扩展你的思维并解决这些问题。整天看教程是无法做到这一点的。

### 4) 技能是技巧的组合

构建一个项目并不是学习的终点。

要构建一个项目，你必须在这个过程中获得技能。

但你并不是“学习一项技能”。

你学习一项技巧，对其进行实验，并继续添加技巧，直到你可以以某种方式将它们结合起来，达到你想要的结果。

你不会通过学习Photoshop来学习技能。

你创建一个项目，寻找一个技巧来完成其某个方面，并继续学习技巧，直到完成项目。

试图学习软件本身几乎毫无用处，而且会花费更长的时间。

有10多种不同的技巧可以去除图像的背景。

知道多种技巧将允许你几乎无缝地从任何场景中去除背景。

你不会通过整天研究教程来学习文案写作技能。

你会学习以下技巧：

+   吸引注意力

+   保持注意力

+   提高你的产品或想法的感知价值

+   等等

每个目标都有数十种技巧，你学得越多，作为一项技能的文案写作就会越好。

在[2小时作家](https://2hourwriter.com)中，每个模块都是一项让你在新经济中成为高薪作家的练习技巧。

一切都是技能。

生活是一种技能，它将健康、财富和关系作为技能封装起来，直到分解成你可以在一生中练习的技术，以掌握生活本身。

所有技能都是心理的。

随着练习变得更加高效，心理系统技能将帮助你实现预期的结果。

### 5) 重新塑造自己

把你的思维看作是现实的消化系统。

如果你吃得过多或运动不足，你不会代谢那个经验。

如果你吃得过少或运动过多，你会感到饥饿和不安。

当你吃得正好时，你创造了一个有利于心理肌肉生长的环境。你穿越现实，朝着目标前进。你进入了一个纯粹的进步季节，感觉难以置信。

你已经创造了必要的边界、限制或清晰度，以在未知领域取得相当大的进展。

有了一个愿景、目标和项目，你为全新的生活做好了准备。

你必须采取飞跃。

将自己投入到一个新的物理或数字环境中。

+   搬到另一个国家。

+   在线取消关注所有人。

+   关注与你未来目标一致的人。

+   改变你的穿着、购物地点和见的人。

+   沉浸在你一直推迟阅读的书籍中。

将所有信息或现实与你想成为的人对齐。

在你扩展到新的身份之前，你可能会感到受到威胁。

你的旧身份将死去，因为它缺乏强化它的信息。

在这个极其困难但极其有意义的转折点中坚持下去。

### 6) 自然指南针

人们如此害怕犯错，以至于他们犯下了最大的错误：不犯错。

你无法避免犯错。

它们是自然的指南针。

不犯错的人没有给自己实现成功的机会。

想象一辆自动驾驶汽车。

几年来，它收到了许多负面反馈，这些反馈精炼了塑造汽车思维的系统。

它可以轻松地导航道路，可能比人类驾驶汽车更安全。

尽管我们经常不这样想，但自动驾驶汽车在真正到达有意义的目的地之前，犯了几百万甚至数十亿次的错误。

你必须与你的大目标保持一致，通过错误的负面反馈来形成新的系统，你的思维将在其中运作。

+   养成强迫自己采用最高版本自己的观点的习惯。

+   允许这个视角揭示你可以采取行动的新机会。

+   当你犯错时，因为你会犯错，将其用作你下一个选择的指导。

随着时间的推移，你甚至不会意识到你已经成为了你选择发展的任何技能中的艺术家。

你将能够实现大多数人认为难以实现的目标，就像你能够走路（而大多数人仍在爬行）一样。

### 7) 自我实验解决问题

你的生活和构成它的项目应该被视为科学项目。

为了解决实现你目标的问题：

+   从你的目标中假设一个结果。

+   尝试不同的技术（用于技能获取）

+   记录这个过程（作为一个项目）

+   加倍努力以获得结果（作为解决方案）

现在，你有了经验，技能，一个你可以将其转化为商业产品的项目，以及一个你可以在创作者经济中收费的解决方案。

我在[《反细分市场》（为什么成为无细分市场的人让你无可替代）](https://thedankoe.com/letters/the-anti-niche-why-being-nicheless-makes-you-irreplaceable/)中讨论了这一点。

自我实验是解决你问题的唯一方法。

假设你的目标是获得六块腹肌。

你在寻找解决方案，他们告诉你素食主义将解决你所有的问题，并且从一个未开化的状态中，你：

+   清理你的饮食

+   将他们的建议视为法律来遵循

+   看到他们提到的结果

然后，你依附于那种饮食意识形态，成为一个先知。你将其他人通过不同方式获得的结果归因于素食主义，并诋毁任何质疑你的人。

这显然是危险的，这是低意识的定义。

事实上，发生的事情与素食主义无关：

+   你吃了更多营养丰富的食物

+   你的行动帮助你生存你的素食身份

+   通过遵循有纪律的营养计划，你有了清晰（而不是混乱）的思路。

在这种情况下，素食主义仅仅是一种整理你思维的方式（持续的流畅状态），并让你遵循健康的原则——但你将其归因于一种方法。

现在，如果你尝试素食主义一个月，食肉主义一个月，生酮饮食一个月，以及灵活饮食一个月，你会：

+   在饮食之间建立联系，揭示健康的基本原理（模式识别=多巴胺）。

+   选择并选择你喜欢的某些方法，这意味着它们会带来可持续的结果。

+   精炼一个适合你个性的系统，达到完美。

然后，一旦健康生活变得毫不费力，你也可以为你的财务、社交生活、浪漫关系、精神追求或生活的任何其他领域做同样的事情。

如果你正在经历关系中的问题，可以雇佣一个治疗师，观看YouTube大师的视频，参加一次静修，并尝试不同的选项，直到找到正确的解决方案。

如果你的业务没有增长，购买一个课程，雇佣一个教练，测试新的软件，退后一步，制定新的策略。

总是有解决问题的方法，但当你紧紧抓住一个解决方案（可能无法真正解决问题）时，你就剥夺了自己这种能力。

将你生活的每个领域都视为一个项目和一项技能，在其中你收集技术以掌握该领域。

### 掌握方法：总结

这就是人们如何快速实现目标，以至于在生活的游戏中人们会向你索要作弊码：

1.  **通过愿景生成会议扩展你的思维。**这种观点将允许你创造新的目标，发现你生活的新的潜力（这样你就可以突破狭隘的思维，实现更多）。

1.  **创建一个目标层级，以使你的生活更加清晰**。你需要通过实际的目标来弥合你现在所在的位置和你想要达到的位置之间的差距。

1.  **将你的目标转化为项目，这样你就可以在构建和学习中同步进行**。项目是可以衡量、跟踪和改进的目标。

1.  不要学习技能，获取技巧。你的技能水平取决于你工具箱中可以解决挑战性问题的技巧数量。

1.  **用信息过载来重新发明自己** 改变你的物理和数字环境，以强迫你保持开放的心态，发现新的潜力。

1.  **将错误视为你系统中的一种负面反馈**。你没有其他方法来测试那些能带来更好结果的事情。

1.  **将你的生活视为一个科学项目**。在健身、商业、知识和人际关系方面，对实验变得着迷。
