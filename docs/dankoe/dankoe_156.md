想象你在18岁时就应该知道自己一生的职业规划是疯狂的想法。

你想学习什么。

一份工作。一个配偶。一笔抵押贷款。几次假期。几次美好的晚餐。也许养一只猫或狗。也许有几个孩子？

也许不是…你知道…经济和所有那些。

至少这是你“应该”做的事情。

但你与众不同。

你有多种兴趣。

专注于一件事情是你存在的痛苦。

他们把你的思想灌输到一个小盒子里，他们可以理解，但你讨厌它。

他们几乎不知道，拥有多重兴趣是进入工作新未来的超级能力。

曾经被认为是安全路线的现在变得最不安全。

## 三个差异化因素 – 代理权、品味和通才

> 人类应该能够换尿布、策划侵略、屠宰猪、驾驶船只、设计建筑、写十四行诗、平衡账户、建造墙壁、接骨、安慰垂死之人、接受命令、下达命令、合作、独立行动、解决方程、分析新问题、扔粪肥、编程计算机、烹饪美味的饭菜、高效战斗、英勇地死去。专业化是昆虫的专利。
> 
> 罗伯特·A·海因莱因

在19世纪中叶，当美国工业化并需要教育大量移民儿童时，像霍勒斯·曼这样的教育家前往普鲁士研究他们的体系。

这个普鲁士教育体系旨在培养顺从的士兵、顺从的公民、公务员和表现良好的工人。

它通过以下方式实现了这一点：

+   强制性出勤

+   教师培训

+   国家课程和学生的考试

+   按年龄划分学生

+   年级层次的概念

如果你和我一样，那么你的一部分在读完那之后就已经死了。

*学生被教导如何工作，而不是如何思考*。

被它的效率和标准化所打动，曼和其他人将这些方法中的许多应用于美国学校，尤其是在马萨诸塞州，这成为其他州的典范。

由于你没有资助这个学校体系，你没有控制*教什么*。

你被告知要读这个，记住那个，如果你不做，你的工业时代父母会责骂你，并威胁要让你 grounded（不能和朋友交谈或追求爱好或兴趣）。

更进一步，标准化教育必须偏袒最底层的五分之一。

换句话说，你被迫以最愚蠢的20%的人的速度学习。

这是非常具有破坏性的。

因为如果你理解了大脑是如何工作的，你就会明白你只能吸收那些框架你思维的目标所允许的信息。

如果你目标是获得学位和找到工作，你的命运就是变得可替代，因为你没有学习那些允许一个人自由的多重兴趣。

这里是关键点：

社会希望你成为专家，因为有用的工人服务于社会。

但你钦佩的每一个人*都不是*专家。专家是工具。工具会被取代。AI正在做取代的工作。

每一个自由且成功的人都是一个通才。一个博学者。他们明白，为了实现宏伟的愿景，他们必须在生活中许多未定义的领域学习和做很多事情。

但普及性只是方程式的一部分。

我们正在经历第二次文艺复兴。

我们口袋里有爱因斯坦和达芬奇的能量。

个人——而不是机构——比以往任何时候都有更多的力量，可以通过多种兴趣来繁荣。

但尽管如此，人们并没有做什么。

这并不是因为他们懒惰。

这并不是因为他们缺乏创造力。

这是因为他们的思维已经被一个从他们的愚蠢中受益的社会编程了20多年。

在工作的未来，你需要彻底改变你对未来的思考方式。

普及性是必须的，让我们谈谈*品味*。

### 如何培养品味

我本想总结这句话，但我不想做不公正的事情：

> 没有人告诉初学者，我希望有人告诉我。我们所有从事创造性工作的人，我们之所以进入这个行业，是因为我们有很好的品味。但存在这个差距。在最初的几年里，你制作的东西并不好。它试图变得好，它有潜力，但并不是。但你的品味，让你进入这个游戏的东西，仍然是杀手。而且你的品味是为什么你的作品让你失望。很多人从未越过这个阶段，他们放弃了。我所知道的大部分从事有趣、创造性工作的人，都经历了多年的这种状态。我们知道我们的作品没有我们想要的那种特殊的东西。我们所有人都会经历这个过程。如果你刚开始或者你仍然处于这个阶段，你必须知道这是正常的，最重要的事情是你必须做大量的工作。给自己设定一个最后期限，这样你每周就会完成一个故事。只有通过大量的工作，你才能填补这个差距，你的作品才会像你的抱负一样好。而且我比我所遇到的任何人花更长的时间来弄清楚如何做到这一点。这需要一段时间。花时间是很正常的。你只需要努力克服困难。
> 
> 爱拉·格拉斯

让我看看我是否能表达清楚。

跟随我，仔细阅读：

智力不再是区分资源。

随着时间的推移，人工智能模型越来越好，而这些模型的价格越来越接近于零。

每个人都有他们可能需要的所有答案（然而，99%的人都没有意识到或利用它……因为智力在孤立状态下已被证明是一种糟糕的资源。成功的背后还有更多故事）。

这就是事情变得棘手的地方。

人工智能正在取代专业角色，因为可预测的输出来源于可预测的输入。

由于许多工作是一系列重复的任务，导致非创造性输出，AI可以做得足够好。

但是当你添加另一个抽象层，或者多个，可能出现的输出数量是无限的。除非你细化它或告诉AI自己细化，否则AI只会给你一个。问题是，这可能是最好的输出。**而且绝对不是那个会赢得支持者**（他们**消费**输出）的喜爱和关注的输出。

大多数人使用的人工智能生成的文本都是垃圾。[死互联网](https://thedankoe.com/letters/ai-will-end-the-internet/)正被这种无人关心的垃圾填满。

例如，让我们假设你想写一本书。

你也可以考虑构建软件、设计网站或任何其他类型的真正创造性工作。

目前，你只需用几个精心编写的提示，就可以让AI写出一本关于“如何集中注意力”的不错的书。

+   打开Claude Sonnet 3.5。

+   给它一个目的（幕后作者）。

+   用你的写作风格、语气和声音来表达。

+   给定目标受众。

+   让它根据该受众的痛点创建目录。

+   对于每一章，让它包含一个相关的故事、主要见解、可执行的步骤，以及像“时间块”这样的新概念。

+   给它之前你关于专注写过的想法

+   为每一章创建和细化大纲

+   完成这本书并按照你的喜好进行润色

现在拿1000个真实的人（使用AI或不使用）让他们就同一主题写一本书。

*每个最终输出都将彼此不同*。

一本书可以以1000种不同的方式来写。

1000个不同的人可以用1000种不同的方式*解读*那本书。

重点是这样的：

无论AI能否为你写一本书、为你构建软件、设计一个网站、写一百万条社交媒体帖子，这都无关紧要。

为什么？因为那不是使创造性工作成功的原因。

这也不是使人们*关心你的创造性工作*的原因。

用AI写所有你想写的书。看看它能带你走多远。大多数人喜欢空谈AI将取代一切，作为他们不采取任何行动的借口，就像他们一生中抓住的任何借口一样。

行动者行动。大多数人不会。我们下次再谈。

现在，认识到多种兴趣与*品味*相结合。

任何人都可以创造任何东西。如果他们选择，他们可以拥抱自己作为通才的本质。

但使一本书、一个品牌、一个软件以及任何其他创造性作品成为独特之处的，不是创造那个东西的*能力*，而是*你如何将各种学科中的想法综合起来，以一种只有你才能想到的方式*。

当你从哲学、心理学、形而上学和精神领域引入概念，而不是仅仅从自我帮助的角度来看，“[如何专注](https://theartoffocusbook.com)”这本书将呈现出全新的形态。

由于大多数人被学校和工作的口味钝化，你处于巨大的优势，因为你没有让这些阻止你超越定义去学习。你喜欢探索未知。

入门级将会灭绝，因为初学者有品味，但不能使之成为现实。他们还没有像Glass在上面的引语中指出的那样练习和失败足够多。

在工作的未来，高薪就业将仅限于技能高超的人。那些能做很多事情并且有独特品味的人。

但坦白说，大多数人不会得到那些工作。

它们将会很少见。

就像进入NBA一样，只为0.1%的精英保留。

一个空想。

所以如果你有多个兴趣，你有品味，这就完成了拼图的一部分。

### 代理：就业 VS 创业

很遗憾，“商业”和“创业”已经变成了脏话。

对于大多数人来说，当他们听到这些时，他们的脑海中充满了怀疑和恐慌：

+   “那需要很多钱才能开始。”

+   “我没有足够的才华去做那件事。”

+   “收费？听起来像是个骗局。”

+   “我不知道如何创建有限责任公司或做我的税收，所以我永远不会学习。听起来很无聊。”

我不是来谈论创业作为一种角色或头衔，或者它只为有才华的人保留的。

我在这里谈论的是它实际上是什么：

*一种心态*。

员工与企业家之间的区别，就是低自主性与高自主性的区别。在这个背景下，*高自主性的人是指那些设定自己的目标并主动追求它们，而不需要他人许可的人*。低自主性的人是指那些被分配目标并追求它们的人，因为他们没有足够的思想去看到其他选择。真正的自主性只有在当你为每一个问题承担责任，即使你并非过错方时，才能得到发展。

企业家通常也是员工，但员工很少成为企业家。

在顶级公司，最被寻求的员工是那些无需许可就能解决问题的人。他们只是*做*。他们有自主性。他们不需要别人的帮助去做有价值的事情。

但再次强调，未来的工作将仅限于精英阶层。

所以，在我看来，对于有高抱负和多个兴趣的人来说，主要的选项是*创造自己的道路*，而唯一的方法就是成为企业家（无论你的梦想需要多长时间才能获得动力）。 

![图片](../Images/05d4150db2c87632b168713ba6948a31.png)

现在，很容易把我归类到“所有9-5的工作都是不好的！”这一阵营，但这根本不是我的信念。

然而，我确实对心理学有一定的理解。

作为孩子，生活中有一种“魔法”，因为一切都是新的。你不断地进步，进化，接受新的挑战，使生活变得值得过。

学校通过提供类似视频游戏的进步学习方式来模拟这种“狩猎”好多巴胺的欲望。

当你接受第一份工作时，一开始是具有挑战性的。你发展技能来治愈你的焦虑。然后，当你达到那个职业信函所允许的极限时，挑战和新鲜感就消失了。你的日子变成了重复的模糊。

这对你的心理造成了极大的伤害。

你陷入了既定的、已知的，而且你无法逃脱，因为月薪是上瘾的，你将你的生活方式最大化，以适应那份薪水所允许的范围。

对于那些想要从生活中得到更多的人来说，唯一的出路就是创业。

要深入未知。拥抱不确定性。接受那些带来孩子般魔法的极高和极低挑战的无穷序列，让这些挑战重新回到你的生活中。

如果你拥有多个兴趣，你正处于一个绝佳的位置。

## 如何将你的兴趣转化为收入

> 我会专注于内容创作者这一方面，因为我相信那是“冰球将要去的地方”。这是一个更加重要的转变……关于我们所有人如何消费信息。
> 
> 查马斯·帕利哈皮蒂亚

最后的问题是这个：

你拥有多个兴趣。

你正在培养品味。

你已经准备好增加你的自主性。

但你不知道*从哪里开始*。

如果你年轻，你唯一真正知道去哪里寻找的地方就是招聘网站，也许还有自由职业市场。

所以，而不是给你一个单一的地方去寻找，因为一个已知的单一路径对你没有任何好处，我会引导你创造自己的路径，将你的兴趣转化为收入。

### 1) 选择一个元路径

你能犯的最大错误就是将自己局限于一个特定的模式。

就像开始一个代理业务或电子商务业务或像亚马逊按需印刷这样的其他随机利基模式。

这些都是值得学习的好东西，但只是为了获得帮助实现你愿景的相关技能。就像一份工作一样。

一个更好的方法是退后一步，发现那些*使所有这些模型都有效*的原则。

因为一旦你理解了这一点，你就可以在任何你选择的事情上创造成功。

所以，如果你想做自己想做的事情，你需要三样东西：

+   一个有价值的产品或服务

+   一种吸引关心这件事的人的方法

+   让人们关心的技能

多亏了互联网，现在这些对任何有互联网连接的人来说都是可访问的。

你可以相对容易地构建一个产品或服务：

+   你可以写一本电子书

+   你可以围绕你的兴趣开设一个基于团队的课程

+   你可以使用AI辅助编码和学习来构建软件

我们在[*如何包装和营销你的知识*](https://thedankoe.com/letters/the-1-million-product-how-to-package-market-your-knowledge/)信件中深入讨论了这一点。

这些是对于不想在建立传统业务上投入过多时间和金钱的初学者来说的最佳选择。

一旦你推出了一些这样的产品，你现在就有了关于什么有效的工作数据，并且你有一些现金可以投资。

在那个阶段，你可以选择Justin Welsh的道路，作为一个个体，享受一个价值数百万美元的信息产品业务，或者你可以选择我的道路，将这些作为另一个需要更多资源的产品的起点，比如[Kortex](https://kortex.co)。

这里的问题在于，同样，任何人都可以构建任何东西。

但并不是每个人都能*吸引关心这件事的人*。

大多数程序员、作家或创作者失败的地方就在这里。他们构建了一个酷炫的应用程序，或写了一本书，或创造了艺术，但他们无法吸引客户，所以失败了。

幸运的是，你不再需要付费广告、广告牌或上电视谈话节目来为你的业务引流。

你只需开始在网上发布关于你的知识和兴趣的内容。

*你出现在受众面前*。

你可以建立、借用或购买一个受众。

由于我们希望产品易于获取、价格合理且具有高杠杆效应，现在你最需要关注的是*建立*一个受众。这是在社交媒体上完成的，直到社会的下一个社交层出现。

“但Dan，每个人都在成为创作者，谈论同样的事情。”

错了。你正处于一个回音室中。

成为创作者与兴趣无关。只要你能够说服人，你就可以谈论任何你想要的事情。

你**不必**谈论自我发展、哲学或商业。你只是认为你必须这样做，因为你和我有相似的兴趣（这就是你为什么读这些），所以也许你应该围绕这些兴趣建立听众。并不是每个人都有这些兴趣。

每个人都在成为创作者，因为这是你作为个体恢复主权的方式。这里有企业和消费者。创作者既是企业也是消费者。所以，是的，这个市场将绝对巨大，并构成新经济的大部分。

问题是，这一切都不容易。

你需要正确的技能来吸引听众，并且让他们关心你是谁以及你在做什么。

换句话说，你需要一个**媒介**和一个**信息**。

写作和演讲是你的媒介。

你要么写帖子、线程、通讯、产品、广告、电子邮件、文章等，要么对着摄像机说话，把你的信息传达给可能从中受益的人。

心理学、营销、销售和说服力有助于形成你的信息。你需要以吸引注意、阐述价值和建立信任的方式写作或演讲。

如果你需要帮助，下一届[作家训练营](https://bootcamp.kortex.co)将于3月4日开始。它将帮助你创建你的个人领域，写关于你的兴趣，并学习建立听众的技能，作为你未来的“护城河”。

### 2) 交叉点是你的优势

如果每个人都建立了一个听众，你如何脱颖而出？

嗯，这只是一个困难的问题，因为你不是你真实的自我。

如果你就是，而且这个星球上的每个人都一样，那么每个人都会是独特的。

换句话说，如果每个人都成为创作者，并且这些创作者都是他们真实的自我，这意味着他们解决了自己的问题并出售解决方案，那么我们将进入某种形式的数字部落时代。小型自给自足的部落将会形成，而部落中的每个人都能用自己的独特性帮助部落中的其他人。这已经在发生了。

你不需要很多追随者。你需要几百个人支持你的工作，并需要你提供的东西。而且，随着互联网的发展，这几百个人每个月或两个月就会进出一次，让你能够维持你所做的事情。你进化，部落进化，互联网是注意力不断流动的源泉。

但我们却趋同。我们模仿他人。我们对未来的愿景不足，也没有实现它的能力。

这是我想让你做的，以发现你的独特性：

+   **写下你的故事** – 几年前你在哪里？那时的你是什么心态？发生了什么重大转折？你学习了哪些技能？你现在在哪里？你能帮助他人做到同样的事情吗？

+   **列出10-20个兴趣** – 拿出笔记本，写下10-20件你热爱学习、想要学习或稍微感兴趣的事情。

+   **为每个问题列出3个痛点** – 在每个兴趣下，写下你或其他人面临或相关的3个问题或痛点。

+   **连接点** – 在你的故事、兴趣和痛点之间画线。你会惊讶于有多少事物是相互关联的。

我的写作无法用AI复制（除非我是操作AI的那个人），因为我的“优势”是我个人的经验与哲学和商业的结合。

我可以和大家一样谈论相同的话题，因为那里是关注的焦点，但我的内容之所以突出，是因为我*如何框架*它。

其次，我专注于吸引有相似兴趣的人。

为什么？因为我知道我能在某种程度上帮助他们。

如果你想做同样的事情，就采取你上面写的内容。

为你列出的每一个痛点写一篇文章或帖子。

在你的写作中穿插你的个人故事。

将相关兴趣融入其中，以综合独特想法。

当然，要有毅力。

记住，你有品味，但除非你练习多年，否则它不会反映在你的工作中。

不，几年时间不算多，考虑到你花了18+年去做你不想做的事情。几年的努力绝对微不足道。

你不想承诺，因为你“有更好的事情可以做”，而这一切只是用廉价的快乐来麻痹痛苦，逃避当前的情况。

你唯一想做的事情就是走出去。

### 3) 融合艺术与商业

> 要构建产品，“制作人们想要的东西。” >  > 要创作艺术，制作*你*想要的东西。 >  > （最好的两者都做）。
> 
> 纳瓦尔·拉维坎特

很遗憾，大多数人无法接受商业可以是一件非常令人满足的事情这一事实。

那种快乐是一枚双面硬币：

需要的创造力以取得进步并为超越自己的事物做出贡献。

换句话说，变得有价值并分享这种价值。

在互联网上，任何人都可以这样做，并从中获得回报。

当你制作出你想要的东西并与有相似兴趣的人分享时，你就将艺术与商业结合在一起，因为他们可能也会想要。

为自己写内容。

把互联网当作你的公共日记。

付出努力让更多的人知道。

建立一个你愿意购买、使用并从中受益的产品。

给予时间，你会为自己做得很好。

– 丹
