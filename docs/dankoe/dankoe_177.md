# 如何变得更加不同意（创造一个现实扭曲场）

> 原文：[https://thedankoe.com/letters/how-to-be-more-disagreeable-create-a-reality-distortion-field/](https://thedankoe.com/letters/how-to-be-more-disagreeable-create-a-reality-distortion-field/)

我在过去的一年里对自己有了很多了解。

尤其是现在我意识到我通常让人难以接近。

我更喜欢独处。我讨厌超过2-3天打破常规。我要么在思考毫无意义的事情，要么在思考所有事情，中间几乎没有其他事情。思考想法。制定项目策略。沉思问题。我一天的质量取决于我独自一人的质量，这可能会伤害关系，因为我不可预测，并且会拒绝大多数外出邀请，随着时间的推移导致邀请减少。

起初，我以为这是性格缺陷。

我想要改变它。

我们被培养在所有情况下都要善良，永远不要露出牙齿。我们经常被告知要“放松一点”，用我们的版本代替他们的乐趣。我们害怕看起来像坏人，因为我们持有一种错觉的自我形象。一种既高尚又善良，但我们永远无法达到的形象。

当我过于善良时，我受到了利用。当我试图放松时，我的生活只会变得更糟。我觉得我戴上了面具。我的注意力会分散，进步会放缓，那些曾经给我带来快乐的目标现在却给我带来了痛苦。

经过一番思考，我意识到**不愉快是你在正确道路上的一个迹象**。

当我观察我尊敬的人时，他们的使命排在第一位。这就是生活愉快的原因，如果这种愉快被剥夺，它就会蔓延到他们的工作、他们的亲人以及他们的友谊中。

如果你知道自己想要什么，并且不屈服于他人认为你应该想要的东西，你周围的人就会觉得不舒服，**因为知道你想要什么的副产品就是不同意**。

事实上，你会发现人们更愿意围绕在你身边，因为他们信任你。

他们可以感觉到你并没有像其他人那样装模作样。

### 过于和善的陷阱

非常和善的人讨厌冲突。

他们会做任何事情来平息风波。

正是这里出现了问题：**他们会为了不伤害他人的感情而牺牲自己的价值观、目标和愿景**。大多数人确实很和善，因为他们并不在乎某些事情，以至于他们别无选择，只能不反对。

他们是否同意每个周末都出去聚会并不重要。

他们没有朝着受派对影响的重要目标取得进步。他们的健康目标不受影响。他们的创造力没有被利用，所以这也不重要。感觉糟糕是他们基本的操作状态，因为他们无法鼓起勇气拒绝快乐，因为这会让他们的朋友开始推他们出去。

如果你对自己的收入或人际关系中的待遇不满意，你会避免为自己辩护。

这里会发生三件事：

+   **问题无声地积累成环境焦虑**，直到你进行艰难的对话才能摆脱

+   **别人的梦想优先于你自己的梦想**，让你觉得一切都不满足和凄凉

+   **你无法从生活中得到你想要的东西** – 你会带着遗憾死去

好事是，你不需要成为一个讨厌的人来改变一切。

### 如何变得难以相处

![](../Images/f4764356fa13ca3f32df11d73d1c875d.png)

高自主性的人难以相处。

他们必须这样做。因为如果他们想要追求自己的目标，他们必须首先不同意社会希望他们走的默认路径，然后他们必须对所有试图将他们拉向不同方向的干扰说“不”。

史蒂夫·乔布斯臭名昭著地难以相处。

他有一个愿景，并且为了实现它而不妥协，这通常通过他强烈的完美主义和残酷的诚实表现出来。

现在，我们不需要在公司会议上称我们员工的工作为“垃圾”，或者没有理由地停在残疾人停车位上，**但我们确实需要一个现实扭曲场**：一个用来描述乔布斯说服自己和他人深信他的愿景的概念。

大多数人无法复制约伯的才华或个性，我们必须小心明显的操纵，但每个人都可以创造一个人们无法抗拒的魅力，帮助你更快地实现你的愿景。

### 对你想要的东西着迷

回答会被给那些对问题着迷的人。

解决方案会被给那些对问题着迷的人。

这就是你怎么获得新知识。

你会与问题坐下来。与之搏斗。考虑放弃。达到疯狂边缘。最终，它会屈服。一个小问题可能需要一周，一个生活问题可能需要十年，一个宇宙问题可能需要千年，但没有任何问题是无法解决的。

你生活中最重要的任务是弄清楚你想要什么。

因为如果你不这样做（大多数人都会陷入这个阵营），你会在不知不觉中被操纵和利用。

你不会因为一个不存在的目标而分心。

如果你没有学习的目标，你就学不到任何东西。

人类是目标导向的生物。我们需要目标。我们追求的目标塑造了我们是谁。当别人为我们设定默认路径上的目标时，我们会毫不犹豫地接受，然后我们慢慢地变成一个克隆体，朝着我们未选择的生活前进。

为了弄清楚你想要什么，你需要对它着迷。

不应该有一天你不去思考你想要的生活，你应该追求的目标，你应该采取的行动。这个简单的行为开始重新连接你的思维。你开始注意到你不想的东西，比如工作、财务不稳定、缺乏满足感。你的思维开始纠正错误。你尝试新事物。慢慢地，你想要的东西的图景变得更加清晰。

不愿意妥协是你知道你想要什么的表现。

### 相信你比大多数人更优秀

这可能不会让你感到舒服。

但你需要相信自己可以做得比其他人更好。

不是因为你作为一个人在某种程度上优于他们，而是因为你可以创造一些足够好的东西，这样你就有机会竞争。

事实上，大多数人创造的东西并不好。世界上99%是消费者，1%是创造者。创造者并不是一些拥有你不知道的秘密的神奇生物。他们只是知道他们必须创造以获得他们想要的东西，他们不想要那种在创造他们不关心的东西中的齿轮般的生活。

如果你认为任何产品、写作、设计或其他方面都不能在某些主要或次要方面变得更好，那么你是在欺骗自己。再次，为了避免冲突。

**当你相信某件事可以改进并据此行动时，你就有潜力为世界贡献有价值的东西**。

如果你没有有价值的东西可以提供，没有人会足够关心以至于愿意付钱给你，支持你，或加入你的事业。所有这些都是你得到你想要的东西所必需的。

### 制定一个超越可能的计划

你知道你想要什么。

你有需要改进的东西 – 有需要创造的东西 – 这推动你朝着那个方向前进。

现在，你需要一个计划，这个计划既能为你的思维带来秩序和清晰，也能为你将需要说服以获得资源（时间、注意力、金钱、劳动力、知识）以构建你的愿景的人们的思维带来秩序和清晰。

但不仅仅是任何计划。

**成功是衡量那些关心你所做事情的人的标准**。投资者、朋友、员工、潜在客户不会关心一个他们已经听过的无聊的计划。

史蒂夫·乔布斯因要求产品按时交付而闻名，这些时间框架工程师最初称之为不可能。

这做了一些事情：

+   **增加心流状态的可能性** – 当任务“刚好”在不可能的边缘时，你与任务合为一体，干扰消失了。

+   **强迫创造性思维** – 当我有一个4小时的截止日期来完成我的工作时，几乎神奇地，我找到了一种方法在规定时间内完成最高质量的工作。

+   **要求无情的优先级排序** – 压力消除了思考“想要的东西”的能力，并迫使你专注于必要的事情。

这是一个令人不悦的配方。如果你甚至考虑让你的愿景分心，一切都会崩溃，因为你的愿景与你的身份交织在一起，你会做任何事情来避免这种现实（如果你对你的愿景有真正的信念）。

### 掌握说服性沟通

说服和操纵之间有一个明显的区别。

+   **说服**：激励人们采取对双方都有益的行动

+   **操纵**：误导人们采取对操纵者有利的行动

如果你给予*任何东西*你的关注，你既是在被说服也是在被操纵。你大部分的信念和个性都是他人想法在你脑海中生根发芽的结果。

除非你生活在虚空之中，否则你既是说服者也是被说服者。你最好研究它，以便有意识地追求互利。

如果你想要得到你想要的东西，你不能独自一人去实现。你需要投资者或员工。即使你因为走在“独行”的道路上而不需要他们，你也需要观众、客户和资源。唯一能够控制这些的方法就是掌握说服性沟通。

要有说服力，你需要两样东西：

+   **足够的研究来阐述你的论点** – 这样你就可以支持你为什么认为你的方式是“更好”的原因。

+   **一个展示变革的引人入胜的故事** – 大脑通过故事来理解世界，而故事让人们关心你所做的事情。

如果我们要把所有东西都结合起来创造一个现实扭曲场：

你知道你生活中想要什么，并且不愿意在这个问题上妥协。

你相信有些事情需要改进。换句话说，你有一个需要解决的问题，这个问题*引导你走向你想要的目标*。

你有一个清晰且战略上可行的计划来实现你的目标。

当需要时，你可以激励他人帮助你完成使命，移除可能阻碍你进步的潜在障碍。

现在，你不需要开始下一个苹果公司来变得不悦。

你只需要有一个有意义的、你无法摆脱的目标，并且有知识去行动，不受干扰地朝着这个目标前进。

无论是在健康、职业还是人际关系方面 – 你得不到你想要的东西的根本原因是因为你避免冲突，你没有设定使这种冲突值得解决的条件。
