媒体让人们接触到他们的潜力。

你正在看的电脑？屏幕上的所有东西都是媒体。

你开车经过的广告牌？路边的商店？旁边的汽车品牌？收音机里的歌曲？媒体。

我们正在进行一场精神上的思想战争。

媒体与内容同义。

并且内容可能比媒体更全面，所以我们从现在开始使用这个词。

你意识中持有的每一件事都是内容。**你注意力的内容**。

有意识的内容帮助人们成长、发展和扩展他们的自我感。

无意识的内容会鼓励衰退、停滞和捍卫他们低级的发展自我。

那是我们正在玩的游戏。

## 7种数字职业道路

再次强调，你是在建立受众，以便为你的产品提供流量来源。

这是第一步。

注意我并没有说“你在写内容”，我说的是你在“建立受众”。

这里有一点不同。

写作是必要的，但大多数人就止步于此。他们不努力让他们的内容被分享，让他们的名字在领域内为人所知，也不努力结交帮助他们成长的朋友。我在这里讨论[增长机制](https://thedankoe.com/letters/how-to-actually-grow-on-social-media-what-they-dont-tell-you/)，但我们还提供了一个想要帮助你成长的网络，在[Kortex大学](https://university.kortex.co)。

现在，你是如何利用你正在建立的受众来创造职业生涯？多元化收入来源？为自由未来的到来做好准备？

我将按从入门级到高级的顺序排列这些：

### 1) 为另一位创作者或品牌工作

你的个人资料是你的数字简历。

你的内容是你吸引机会和工作的方式。

知道你做什么的人越多 = 可能雇佣你的人就越多。

现在，理解网络效应……

如果你“只有”1000名关注者，但每个关注者都有200-500名关注者，而这些关注者每人又都有200-500名关注者……

然后，你有一个间接的网络，有4000万至2.5亿人可以传播你的名字。

当然，这是假设你的每个关注者都分享你的帖子（不可能）。我们还必须考虑参与度、内容质量、这些关注者是否甚至登录社交媒体，以及他们是否认识某个需要你在该经验水平上工作的机会的人。

所以让我们假设每个月有1万人多少了解你是做什么的。

将此乘以6-12个月的努力，我发现在这个游戏中失败非常困难。

**所需技能：**

如果你想要为另一位创作者或品牌工作，你需要练习任何有助于你个人品牌成长的技能。

+   社交媒体增长

+   内容写作

+   邮件文案写作

+   管道建设（着陆页等）

+   品牌设计

+   领导生成或营销

选择一个，学习它，实施它，选择下一个，学习它，实施它。

然后，写下来。提供建议。教学。一些创作者可能会联系你。你可能需要联系一些。这一切都不是保证的，但通过扩大你的受众并联系更多创作者以帮助他们使用你的技能组合，你可以增加成功的机会。

你无论如何都需要学习它们。

你不需要所有这些才能取得一些成功，但继续提升自己和你的业务是有意义的。

### 2) 开始服务业务

如果你有一套技能组合并取得了成果，你可以直接进入这个领域。

或者，你可以将你在与另一位创作者合作中学到的知识转化为业务，并作为客户帮助其他创作者。

服务业务是客户业务。你通过你的技能组合向客户收费1000至50,000美元（在非常高端的情况下）。

我们在刚开始在线时关注的几种服务业务类型：

+   家教——类似于咨询，但你一对一地教授你的技能。比如，如果我要每周进行电话教学，教授某人网页设计。

+   自由职业——很明显，你帮助客户做任何你的技能组合，比如网页设计，这是一个很好的起点——但你应该学习如何创建更好的报价，因为你不会长时间销售这样简单的东西。

+   咨询或顾问——这是你帮助人们实施你的技能或系统的地方。你不是像自由职业者那样为他们做，而是与他们一起做并指导他们。

+   代理——将代理视为一个拥有员工或承包商的自由职业业务，他们提供服务。如果你没有员工或承包商，你就不是代理。

你从服务业务开始，因为你的听众还不够大，无法支持另一种类型的业务。

你只需要每月2-4个客户，每个客户2000美元左右，就可以替代你的收入并赚得不错的钱。

你将不得不学习如何通过直接接触或目标内容等策略来生成潜在客户。

### 3) 联盟营销

我不是特别喜欢将联盟营销作为你唯一的收入来源。

为软件和产品做联盟营销以链接到你的产品内容？当然可以。但如果你能卖出去，这意味着你理解它，你可能应该只创建你自己的版本。

联盟营销是指你将流量发送到他人的产品以获得佣金。

所以，如果你推广了我的[2小时作家](https://2hourwriter.com)课程，有人购买了，你将获得那笔销售的一部分（顺便说一句，我不提供联盟营销）。

我之所以提到它在这个列表中，是因为它对初学者练习很有帮助。

你可以用这个选项从头开始建立整个业务，而无需任何经验：

+   围绕联盟产品的主题写内容

+   投入努力建立听众

+   推广联盟产品

你不需要技能，也不需要产品。

因此，这可以在你完善技能并创建产品的同时成为一种良好的练习。

不要陷入这个阶段。建立你自己的东西。

### 4) 数字产品

一旦你有一个足够大的听众并且有能力获得成果，你就可以通过数字产品赚取不错的收入。

我建议尽快建立一个，这样你可以快速迭代（因为大多数第一个产品都会很糟糕）。

但是，遵循以下步骤是有意义的：

+   通过服务业务进行货币化

+   扩大你的听众

+   通过服务业务获得成果

+   扩大你的听众

+   将你在服务业务中提供的东西变成数字产品

+   扩大你的听众

+   获得数字产品买家并向上销售你的服务

+   扩大你的听众

+   做你想做的，放弃客户工作，这样你就有更多空闲时间（并且减少初学者的市场饱和度）

为了简单起见，我们将考虑数字产品**课程、小组或会员资格**。

如果你每天能卖出**两个**价值150美元的数字产品，你每年的收入将达到六位数。

如果我们进行计算，这意味着你需要将80人带到着陆页，并将转化率提高到2.5%。

如果你写一个X帖子并在底部推广你的产品，你需要大约10000次曝光才能实现这一点，或多或少。

如果我有1000个追随者，但我的一个朋友有10000个，他们分享我的帖子，如果帖子有说服力，我可以几乎保证我那天能卖出2笔销售——而且这是在一个小追随者群体中。

最酷的是，即使我不做那两笔销售，我正在建立一个可能会后来购买我产品的观众群体。

### 5) 赞助商和广告收入

YouTube、时事通讯和播客赞助可以带来不错的收入。

但是你需要一个庞大的积极参与的观众群体来做这件事。

对于作家来说，时事通讯赞助是最可行的——但为什么要在你的电子邮件中为别人的产品浪费促销空间呢？

这也是与联盟营销相同的问题。

只需创建自己的产品，就能比赞助商多赚2倍到10倍。

与YouTube广告收入一样。不要关心它。创建自己的产品或服务，并控制自己的收入。

### 6) 实体产品

如果你想要超级盈利，从数字产品开始，这样你就可以投资于实体产品收入流。

对于实体产品，我更喜欢销售可以补充我的其他产品或是我自己使用的产品。

这就是为什么我创建了FOCI计划。

它补充了我的商业产品，也是我使用的东西。

像SolBrah或Greg Doucette这样的人会销售符合他们训练和营养理念的补充品。

这是一个不错的步骤，尽管它不像数字产品和服务那样简单或有利可图。

### 7) 软件或服务扩展

如上所述，你可以创建一个数字产品并放弃客户工作。

这对于许多不想满足不断增长的公司需求（招聘、系统等）的人来说是一个稳固的举措。

仅凭不断增长的观众和数字产品，一个人一年可以赚取100万至500万美元（在高端，经过多年的努力和迭代）。

但是，另一个选择是更进一步。

你可以扩展你的服务业务或创建类似软件的东西。

到目前为止，你拥有如此庞大的观众群体，你可以创建任何你想要的东西。
