# “细分市场”是糟糕的建议（对聪明人来说）

> 原文：[https://thedankoe.com/letters/niche-down-is-terrible-advice-for-smart-people/](https://thedankoe.com/letters/niche-down-is-terrible-advice-for-smart-people/)

当我刚开始创业时，我根本不知道我在做什么。

我认为这是显而易见的，尤其是在自由职业方面：

如果我有技能，人们需要这个技能，所以他们将会雇佣我。

我不知道任何人都可以学会这项技能（而且在线自由职业领域开始充斥着像网页设计这样的单一技能商品）。

那是我要解决的第一个问题。

简而言之，我购买了关于：

+   自由职业

+   Facebook广告

+   e-Commerce品牌建设

+   营销

+   销售和客户获取

+   至少15门与网页开发无关但值得注意的课程

当我尝试建立他们在那些课程中教授的业务时，我没有看到多少成功（这是我的错，不是他们的错）...

但我注意到它们之间有一些共同的模式。

其中一个非常明显：

我必须非常具体地确定我的目标人群。

*我必须进一步细分市场。*

我必须确定一个饥渴的市场（如健康）。

将其分解为子市场（如健美）。

瞄准一个细分市场（比如极简主义培训）。

研究该市场内的人并收集他们的数据。这样，我就能击中他们的痛点并直接与他们交流。

这一切都有很多道理！

这就是了。

我知道我必须找到特定的人，并以产品或服务的形式调整我的技能集来满足他们。

这是我一直缺少的秘密！

只需细分市场，兄弟！这就是全部！

这仍然是你在网络空间中听到的最常见的建议。

而且它仍然在初学者中引起最多的困惑、压倒性和焦虑。

他们觉得如果他们一开始没有最赚钱的细分市场，他们就会失败。

因此，他们花费时间：

+   在细分市场之间摇摆不定

+   学习该细分市场的技能（不是坏事）

+   最终一无所获，没有钱或可以利用的东西（这是件坏事）

我喜欢挑战常见的观念（不仅仅是商业），因为大多数时候它们都被无异议地接受为法律。

他们得到了一些——但不是最好的——结果，因此人们纷纷涌向他们，躲在后面，并对任何说有更好方法的人进行抨击。

商业教育很像政治或宗教，如果人们能意识到与意识形态相关的危险，他们就会过得更好。

随着社交媒体的兴起，商业世界发生了巨大变化。

依赖于冷门的外部接触而不是建立随着时间的推移而变得温暖并成为客户的受众的商业模式之间存在显著差异。

你在网上学到的大部分东西都是过时的，并不是新经济中做生意的最佳方式。

这封信将会很长。

我们将深入探讨现实的形而上学解释。

这很重要，但如果你想跳到实际步骤，我们谈论的是：

+   将你品牌的演变视为一个实现层次（这样你就不必细分领域）

+   如何从入门到高级构建你品牌的每个层次

+   在每个层次上要构建的实用事物——这样你知道现在该做什么

在最后一节，*你是无限细分领域*。

### 导致你阅读这封信的心态转变

我不知道为什么，但我正在刷Twitter。

我刚刚重新下载了它，因为它从我手机上被删除了8年。

一条推文引起了我的注意。

有个人同时在做网页设计服务和提供自我发展建议。

我被训练成认为，我必须进行冷电话、冷邮件和投放广告，才能看到任何形式的商业成功。

经过多年的尝试和错误，我从未看到我想要看到的那种成功。我可以坚持下去，但我永远不会作为一个人的成功。

当我看到人们发布内容（关于我知道我也能够谈论的事情）并从中吸引客户时……事情开始变得明朗。

通过冷邮件和私信：

+   没有人知道我是谁

+   没有任何先前的信任

+   我没有一种方式来展示我的权威或结果——我必须证明自己

+   如果你想让信息个性化，作为一个人来说，这是一项大量的手动工作

+   我发送的每个提案要么导致客户，要么什么也没有。我在过程中没有建立杠杆。

通过社交媒体内容、向观众的热门推广和促销：

+   我随着时间的推移建立起了声誉

+   我通过广度和深度（短篇和长篇内容）建立信任

+   我可以在不向他们解释自己的情况下在观众中建立信任

+   我可以在不尝试的情况下激发我对产品的兴趣

+   没有东西会浪费。所有内容要么为数据（失败）做出贡献，要么为增长（成功）做出贡献

+   随着我规模的扩大，我可以将我的品牌发展成我喜欢做的事情。随着时间的推移，我获得了自由，而不是失去了自由。

因此，我开始在Twitter上写内容，以吸引我的营销服务客户。

**另一个问题：**

人们总觉得他们只能谈论与他们销售相关的一个话题。

他们没有意识到：

1) 你显然对与你销售相关的话题感兴趣。

2) 你还关注了200-500个其他有不同兴趣的人。

你怎么能相信你必须坚持一个兴趣，当你**正在跟随那些谈论10-20个不同兴趣的人**时呢？

并非人们不会对你的其他兴趣感兴趣。

是因为你不知道如何让你的兴趣对其他人来说有趣。

人们总觉得，如果他们选择广泛的领域而不是细分领域，他们就会稀释自己，以至于无法赚到钱。

我很早就对此提出了质疑。

每一个大型的YouTube红人、播客或其他伪名人都是广泛的。去看看伊曼·加兹或哈姆扎。

他们专注于社交媒体漏斗顶部的入门级教育。

他们专注于中级到高级的产品和服务，从低票价到高票价。

他们谈论的事情在某种程度上是整体性的。他们理解商业或自我提升需要谈论健康和生活方式。

他们因为理解人性而远远领先于其他人。

想象一下：

大多数**品牌**在社交媒体的世界中都是社区内的一个建筑。

我希望我的品牌在社交媒体的世界中成为一个社区。我的通讯、吸引物和产品就是建筑，或数字房地产。

但是，你首先必须建立一座建筑，然后才能扩展。

人类本性是我们现在将尝试打开的虫茧。

## 发展阶段，自然等级，以及它们如何在创作者经济中反映

让我们从理解**等级**和**发展阶段**开始。

**等级是存在的基石**。

是的……即使那些认为等级不好的人，也仍然将“等级不好”视为一种等级，这本身就是一种等级。

现实的结构由**心灵单位**组成。

这是我新书《聚焦的艺术》（即将更新）中的一个基本概念。

其他人为这个术语命名为“全息”或“思维单位”，就像艾伦·瓦茨喜欢说的那样。

我更喜欢心灵单位，因为这符合心灵主义哲学——或者说一切都是心灵。

在动物生命或人类意识出现之前，没有心灵来标记、解释和创造语言，以便每个人都能处于同一页面上。

我们必须理解现实。

这些“单位”的心灵是全部。

你能标记或感知的每一件事。

它们有几个独特的特征：

**1) 心灵单位是等级化的，并且** **与整个世界紧密相连**。

就像原子到细胞到分子。

或者说字母到句子到段落到书籍。

或者内容到产品到品牌到社区到社交媒体。

或者说短篇内容到中篇内容到长篇内容。

此外，任何这些单位都通过近缘关系垂直或水平地与一切事物相连。

就像关于健身主题的短篇内容如何与健身房、你的营养、你的厨房、你手中的叉子以及你肠道中的微小生态系统相连。

**2) 心灵单位是完整的部分。它们自身就是一个整体，也是更大事物的一部分。**

一个个人品牌的整体如果没有其各个部分，就什么也不是，而大多数人忽视了每个部分在进化中的必要性。

**支配性等级**——那些给“等级”带来坏名声并让人们想要“摧毁”父权制等级的——当等级内的一个单位试图支配时，就会被分类。

就像癌细胞在身体中复制……直到它通过摧毁允许整个等级存在的较低级别来杀死宿主，或等级。

癌细胞必须被治愈并重新整合到等级体系中，以便它能够继续健康地增长。

**3) 因此，思维单位出现、超越并包含。**

从本质上讲，只有少数创作者能够达到等级的最高层。拥有数百万粉丝的创作者数量是有限的。

但是，拥有10万粉丝的创作者数量——这已经足够让你赚到难以想象的钱——每天都在增长。

随着创作者经济的持续发展，构成它的等级体系的新层次将作为新的整体出现，超越其部分。

我们还必须考虑到，你不需要数字产品业务或追随者就能赚取收入。

人们可以把你推荐到工作中，你可以创建一个实物产品，或者你可以抓住大机会。

但是，如果你想让自己实现这一可能性，基本原理仍然存在：品牌、内容、价值（或提供）和营销（或推广）。

如果你不对创作者经济做出贡献，你就无法从中获得回报。

你是你网络的一部分，而你的网络是社区的一部分，社区又是创作者经济的一部分，创作者经济又是互联网乃至更广阔领域的一部分。

如果你想要生存，你必须交换价值。

### 理解你的个人进化

所有事物都会经历发展阶段。

即使是你的个人品牌。

我在[《一人企业路线图》（The One Person Business roadmap）](https://thedankoe.com/letters/the-one-person-business-roadmap-99-of-creators-make-this-mistake/)中深入探讨了这一点。

不想过于复杂化并写一本书来讲述自我发展阶段的螺旋动力学，让我们用两个例子来简化这一点：

### 1) 初级、中级、高级

在创作者旅程的每个阶段，你都必须优先考虑不同的事情。

初学者不会有经过验证的想法。

他们不会有发展完善的品牌（或个人）哲学，因为那只能通过多年的写作和反思来找到。

他们也不会有必要的杠杆来专注于带来指数级结果的事情（如通讯、YouTube、数字产品等）。

重点是，你将不得不像其他人一样培养社交媒体技能。你必须优先考虑你的学习和行动，无论你在游戏中的哪个层次。

### 2) 预理性、理性和后理性

这种发展进程可以通过臭名昭著的钟形曲线梗最好地说明：

<picture fetchpriority="high" decoding="async" class="wp-image-1314">![](../Images/b1bd4b4ff236c5bf0b7bc234ca704114.png)</picture>

当人们在生活的任何领域开始自我发展时，他们会依附于特定教义和相关的意识形态原则。

如果他们不继续进化，他们可能会陷入那个阶段。

就像虔诚的宗教徒成为“圣经敲击者”，不加质疑地重复他们所学的所有东西。

然后，他们进入中级或理性阶段。

这就是人们开始质疑和反抗初学者阶段的时候。

在发展的精神领域，这是无神论者发挥作用的地方。

在创作者发展的领域，这是人们在这个游戏中变得痛苦的地方（我最近在推特上看到很多。市场正在经历一个周期，指责人们并抱怨游戏不公平）。

“每个人都喜欢模仿。”

“教练向教练销售是在进行金字塔骗局。”（对等级制度缺乏理解）。

“高价值销售是最好的商业模式，因为其他人都是基本的，没有像我这么聪明。这也是我生存的方式，所以我与它深深地联系在一起，否则我会感到威胁”

再次强调，这可能会变得危险。

最后，你进化到高级或后理性阶段。

在这个阶段，你重新发现了原则，并可以从经验的角度来接近它们。

就像大多数发达的精神人士都同意并体现了圣经（以及所有其他指向真理的信仰体系中的模式）中的原则，但他们看到了作为唯一意识形态的传教是徒劳的（否则你会在地狱里燃烧！）。

在创作者经济中，我们可以注意几点。

许多中间阶段的人厌恶简单的陈词滥调或俗语。他们进入了一个痛苦的阶段，试图反抗商业的本质。

（这很好，但就像一个复制的癌细胞，你必须将你的发现重新整合到等级制度中，以进入下一个阶段——否则你就像我一次又一次看到的那样，摧毁你的业务）。

高级创作者将它们视为他们成长的一个支柱，但从经验和权威的角度来看，或者简单地说，像引用一样的内容没有影响力。

在有了这个漫长的背景之后，我们现在可以从形而上的理解来探讨“创造你的细分市场”。

## 你是无限的细分市场

我过去讨论过“你是最有利可图的细分市场”几次，[特别是在这封信中。](https://thedankoe.com/letters/the-most-profitable-niche-is-you-how-to-create-your-niche/)

因此，这封信不会写成5000字（可能还是会的）：

+   你的品牌愿景是你对未来的愿景，这是你引导追随者走向的方向。

+   你的品牌身份是你的故事。你的起源以及你是如何转变的。

+   你的内容通过教育你所学的技能和兴趣来展示这一点，这些技能和兴趣让你达到了现在的位置。

+   你的思想、观点和世界观是你以独特方式定位你内容的方式。

+   你的产品是你为帮助那些在你之后的人而创建的系统。

让我们通过深入探讨这个想法来继续。

希望这能提供一些清晰：

### 不要细分市场——将你的品牌视为一个实现等级制度

<picture decoding="async" class="wp-image-1315">![](../Images/5ec1fbea6b4d016ecb8e8cee364426d4.png)</picture>

你是复杂的。

好吧，至少这应该是你的目标。

宇宙在向复杂性进化，正如你在学习、实施和理解的过程中，你的自我意识也在进化。

你的角色复杂性是由你克服的目标或挑战决定的。

你达到的每一个目标都需要一定的技能水平和相应的视角。

一旦你达到这个目标，就会开启一条新的道路，让你可以朝着下一个目标努力。

你的技能水平（跨多个领域）越高，你的自我复杂性就越高。

随着你变得更加复杂，你可以在你自己的生活和他人的生活以及他人生活中解决更深、更有意义的问题。

如果你是一个细分市场，你的品牌应该随着你的技能和兴趣在追求目标的过程中不断发展和复杂化。

随着你学习和获得技能，你有责任建立能够为他人带来成果的现实世界项目。

这既是你在商业中创造盈利产品的方式，也是你履行精神责任的方式。

随着你达到每一个发展阶段，你将建立一个有形的产品来反映这一点。这是你教育你的追随者并建立权威的方式。

在我解决了我生活中的一个生产力问题后，我创建了一个Power Planner，现在它已经成为我的一个吸引注意力的工具。

在我通过网页设计和自由职业获得成果后，我为每个领域都创建了一门课程（这进一步加深了我对这些领域的理解，教学是学习更多知识的方式）。

在我获得技能栈后，我创建了一个品牌咨询服务，开始帮助他人。

现在，当人们谈到细分市场时，很多人会错过的是这一点：

他们没有意识到，你创建的任何通讯、吸引注意力的工具、产品或服务都是一个受众。

你正在*平台之外*建立一个受众。

问题：

大多数人都害怕创造和发布任何东西。

他们没有意识到，长形式内容、产品和服务是你整个品牌的支柱。

我不想听起来自负，但这个例子很好地说明了这一点。

“丹，你怎么能保持这么高的参与度，而我的所有推特客户都趋于零？”

嗯，我想知道为什么……因为我不认为我的品牌可以被别人建立，或者可以被模板化内容系统化到极致。

因为我在内容中培养了深度，拥有数千名学生，以及数十万人阅读了我的通讯并下载了我的免费产品。

人们阅读我的内容，是因为他们理解我是谁以及我的哲学。

我已经谈论了这个话题好几年了，但人们只是匆匆而过，因为短形式内容和浅层次产品是容易走的路。

我的品牌是我人生工作的载体。

我的品牌是一个品牌，*不是一个细分市场受众*。

这是一个存在于社交媒体中的城市。一个拥有数百万人的社区。

我的Power Planner拥有围绕生产力的70,000名下载者的细分市场受众。这是我正在占据的数字地产。

我的通讯拥有围绕深度、哲学和商业的120,000名订阅者的细分市场受众。

我的写作、品牌和商业产品拥有超过15,000名学生的细分市场受众。

当你将你的品牌细分时，我看到人们在3年后可能达到10万的总粉丝数。

我的Power Planner的受众更偏向于小众，那里有70,000人。

你看懂我的意思了吗？

这就是如何打破你对自己的界限，并随着时间的推移构建你想要的任何东西（f*ck）。

### 每个人都是从第一级开始的

现在，当然，你不可能在眨眼间就构建所有这些事物。

我们在这里谈论的是发展的**阶段**。

你必须下定决心构建你金字塔的第一层。

一旦这个项目出现，超越到下一个层次，并包括之前的层次。

大多数人从未进化，因为他们停止了构建。

由于他们停止了构建，他们也停止了学习。

如果你没有东西可以应用你所学的知识，那么你就是在进行精神上的自慰。

课：

如果你不在你的品牌下建立数字资产，你将陷入那种浅层次的发展水平。

你将继续与那个水平的人竞争。

对于大多数人来说，这意味着撰写内容、吸引客户，并陷入 feast or famine 服务业务周期。

**2) 练习顶部漏斗写作和增长**

你在社交媒体上的首要任务是持续的品牌增长。

在Twitter、Instagram、LinkedIn和其他短形式平台上，你可以更好地控制你的增长。

你想要建立一个基础受众，随着时间的推移，你可以利用你的数字资产、产品和服务。

随着你的受众增长，你构建和讨论任何你想要的内容的能力也随之增长。

+   购买你主题的畅销书并阅读它们。

+   关注那些讨论这些主题的社交媒体账户，并从创作者的角度模仿他们，而不是消费者的角度。

+   注意哪些写作、思想和结构表现最好。

这就是练习写作的方式。

你的起点可能会很糟糕。

在没有人观看的时候把它完成。

**3) 以建立项目为载体开始创建通讯**

我总是推荐开始创建通讯。

你**不必**持续不断地写作。

你可以建立一个订阅者基础，直到你准备好保持一致性。

人们推迟建立通讯，因为它是一场长期的游戏。你不会看到立即的结果。

开始通讯的好处是：

+   你可以练习在你自己的想法背后发展深度。这导致原创思想和写作的挑战。你进步得更快。

+   你可以用它作为一个地方来写你的吸引人的内容或产品的模块或部分。不，即使你最初免费发送它也没关系。

+   你表明你作为一个初学者在平台之外提供价值，这有助于建立感知权威（“销售”某物有助于你的品牌，不销售某物会伤害它）。

在接下来的步骤中，每当你创建一些内容时，都使用你的通讯来撰写这些部分的第一个版本。

**4) 创建一个吸引人的内容作为你的第一个“细分市场”**

我已经建立并测试了8-10个吸引人的内容。

从如何创建一个基本网站到营销的基础，再到力量规划师，再到我的想法生成系统。

有“细分市场”品牌的人必须持续讨论他们细分下来的一个主题。

让我们以视频剪辑为例（这适用于其他任何兴趣。健康、营养、营销技能、哲学，任何东西。）

如果我创建了一个关于视频剪辑基础吸引人的内容，我是否必须总是担心只写关于那个内容？

或者我可以写任何我想写的东西，然后只是插入我的吸引人的内容？

尤其当我的吸引人的内容包含我对主题的最佳思考和阐述时。

并且，我在旅途中建立了我的通讯。

吸引人的内容想法的例子：

+   获取结果的清单（如网页设计清单）

+   用于系统的Notion模板（如写作或习惯养成）

+   一本入门级电子书，概述基础知识，让人们跟上你谈论的更高级内容。

+   视频教程或屏幕录制相同的基础知识。

创建一个吸引人的内容来在你想要货币化的主题上建立权威，或者至少你认为你想要货币化。你总是可以创建另一个。

随着时间的推移，你将建立一个下载者的细分市场受众。

**5) 以最小可行服务提供销售以获得经验**

你需要一个“下一步”来吸引下载者。

希望能有一些你可以货币化的东西。

对于初学者，我建议一个最小可行产品。

这意味着单一技能的自由职业服务（如网页设计）或单一兴趣的辅导服务（如生产力或健康）。

我之前已经讨论过这个问题[在其自己的信件中（最佳在线商业模式）](https://thedankoe.com/letters/the-best-online-business-model-to-make-1-million-in-2023/)

收费500-1000美元，直到你了解什么会产生真正的结果。

系统化它。

收费更高。

用每月3-4个客户替换你的收入，并全职工作。

**6) 脱离细分市场，进化，超越，包容**

到这个时候，你已经在某个细分市场建立了权威。

你应该花一些时间在这个领域取得成果，同时测试你内容中的不同兴趣。

为了加深你的权威性并开拓探索新兴趣的空间，你应该完成你的品牌实现金字塔。

+   将你的自由职业或咨询服务转变为基于团队的课程。在这个层面上获得成果。

+   将这个团队产品化成一个数字产品。在这个层面上获得成果。

+   重复吸引物、服务、团队和产品的过程，直到你有一个内容、产品和服务的全面综合。

如果你想谈论某些兴趣，你需要有基础内容（如吸引物或通讯）来让人们跟上进度。

如果你谈论一个新兴趣时没有得到参与，那是因为你还没有在这个兴趣下发展出深度。

你需要让你的观众对理解达成一致。

你现在可能看不到，但你正在创造社会。

你的工作是注意兴趣和行业之间的模式。

在你的创作中将它们联系起来。
