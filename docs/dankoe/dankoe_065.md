# 数字黄金时代，通才正在接管世界

> [原文链接](https://thedankoe.com/letters/generalists-are-taking-over-the-world-in-the-digital-golden-ages/)

我不禁要 wonder，过去的艺术家、思想家和先知会对今天的世界有何看法。

阿兰·沃茨会有播客吗？

马库斯·奥勒留斯会成为推特名人吗？

尼采会做关于他《纽约时报》畅销书的公开演讲吗？

许多人认为他们会“拒绝世界”并隐居山林。

我会争辩说，正好相反。

他们会意识到互联网传播他们思想的原始力量，因为人类有在比物理存在更多的层面上生存的天生驱动力。

但那不是我心中主要的问题。

我对今天的创作者以及他们将如何被记住更加好奇。

在我之前与[Justin C Scott](https://twitter.com/iamjustincscott)在南卡罗来纳州格林维尔冒险的聊天中，他说了一句话，让我印象深刻：

> 一个有影响力的人物离他们历史上的时代越远，他们就越显得神话般。
> 
> 人们不再将他们视为一个人，而是开始将他们看作是某种更重要的东西——某些思想、概念和哲学的仲裁者。
> 
> 他们不再被看作是曾经活过的人，而是一个感觉似乎始终存在的原型概念。

在沃茨或奥勒留斯的时代，他们的语言是正常的。

这可能并没有像我们经常认为的那样被视为“深刻”和“神秘”。

这只是正常的语言。

随着语言随着文化和社会的演变，我们对古老教诲的认识也发生了变化。

一百年后，语言将再次经历重大变化，今天的创作者将被视为古代文物。

乔尔丹·彼得森、乔·罗根和Mr. Beast都为我们在继续创造的文化织锦增添了他们自己的线索。

但下一代创作者将是谁？

我会让你的想象力自由驰骋来回答这个问题。

但我相信公众知识分子的格局将整体改变。

我们在心中高度重视的人将根据社区和自身利益而多样化。

对于每一个马库斯·奥勒留斯，都会有十个乔·罗根。

但乔·罗根们将根据他们独特的视角和个性吸引不同类型的追随者。

他们将通过互联网内容谈论同样的事情，但由于个人经历的不同，他们的教诲将大相径庭。

我在[《未来创作者经济》](https://thedankoe.com/the-future-of-the-creator-economy-my-bold-prediction/)中谈到了这个问题的另一面。

每个创作者都会提炼他们的专业知识，吸引一群与他们的教学风格产生共鸣的学生。

创作者将进化，他们的追随者也会随之进化。

这消除了创作者经济中的饱和，因为生活并不饱和——我们都在过渡到数字社会。

## 新兴的数字社会

在历史上被记住的每一个人都有一个共同点：

他们是巨大的价值创造者。

他们是知识、观点和洞察的源泉。

他们没有试图将他们的思想限制在有限的大脑空间中。

他们写作、演讲、营销和销售那些编织他们世界观网络的有力思想。

这就是为什么我建议学习这4种永恒的技能，并将它们与你的个人兴趣相结合，以在新的文艺复兴中取得成功（参见：[一百万美元的技能堆栈](https://thedankoe.com/the-1-million-dollar-skill-stack-learn-in-this-order/))。

今天的世界是不同的。

你可以访问互联网。

互联网。

人们视为生活中正常部分的那项革命性发明。

那些分散人们注意力的东西，但却是逃离表面生活的确切工具。

正在出现一个数字化的社会。

互联网继续以不可预见的方式发展，我认为这是人类进化的下一步。

你要么在网上提供价值，要么推迟到被迫为了生存而做。

在过去的十年里，数字工具和辅助工具已经成为我们生活的一部分，以至于我们没有意识到我们已经一半是机器人。

当然，这伴随着古代大师们警告过的后果。

机械和无意识的生活是美好生活的对立面。

互联网必须被使用，但它不能利用你。

权力已经重新回到了个人的手中。你拥有以下能力：

1.  在3-6个月内成为任何（有价值）主题的高手。

1.  形成一个高级网络，为你提供机会。这在之前的物理位置障碍中是不可能的。

1.  吸引观众关注你的人生工作，这样你就可以得到报酬来做你喜欢的事情。

人类通过解决问题来进化，我真诚地相信我们已经通过解决问题找到了几乎在一天中的任何时间都能做自己喜欢的事情的机会（如果你投入工作去做的话）。

## 在新世界中开始构建你的遗产

<picture fetchpriority="high" decoding="async" class="wp-image-1169">![](../Images/9cd8a383bcf7ca3b26dd3fb7c3258ef2.png)</picture>

我们正处于第二次文艺复兴的中间，未来的思想领袖正在左右涌现。

这不能被轻视。

我以前认为“思想领袖”这个词相当令人尴尬。但可以说，这是一个人可以承担的最重要的事业。

要成为思想领袖，你必须首先学会自我领导。

通过引导自己进入未知领域，你将暴露自己于：

+   新颖的想法和观点

+   你生活的新的潜力（以及他人的）

+   你必须获得的技能来进一步推进

+   你必须用这些技能获得的结果来保持进步

如果一个思想领袖不构建出能够为社会的居民带来结果的解决方案，那么他不会长期成为思想领袖。

他们必须拓宽思维，以便让思考有更广阔的边界。

真正的思考只能达到你所拥有的经验的边界。

这就是你在新世界中开始建立你的遗产的方法：

### 1) 确定你必须做的事情

有无数条道路可以通往你想要的生活。

但第一步是确定你想要的生活，这样你才能朝着正确的方向前进。

在你想要的未来和讨厌的未来之间跳舞。

随着你经历生活：

+   对你周围的人保持警觉和观察。

+   记住你不想过的生活。

+   研究成功人士的习惯、模式和想法。

+   随着时间的推移，为你的理想未来添加笔触。

最初可能不会完全清楚，这是正常的。你有时间。

### 2) 巩固实现该现实的容器

这就是大多数人犯错误的地方。

他们梦想着做一些了不起的事情，但让它只是一个梦想。为什么？

因为他们没有一条清晰明确的路径来实现它。

他们没有意识到没有什么确定的东西是值得拥有的，无论你做什么，未来都是不确定的（这应该是显而易见的）。

你需要可以练习和应用于一生的技能和兴趣。

真的，你选择的技能或兴趣是什么并不重要。

如果你意识到了阻碍你执行计划的难题，下一步应该很明确了。不要回避它。

如果你想要的生活是通过创业实现的，但你“不知道如何”，那么就通过网上无穷无尽的信息来学习如何。

哎呀……这需要一些时间。

如果你意识到你需要钱来开始你选择的企业，但你没有，那么你或者要从[低成本商业模式](https://2hourwriter.com)开始，或者找到一份更高薪的工作（或者尝试预算技巧）。

如果你意识到你不能得到更好的工作或者没有技能来启动低成本企业……自己回答这个问题。

太多人避免做他们“不想”做的事情，即使这阻碍了他们想要的事情。

改变你的视角。

如果它阻碍了你想要的事情，那么你应该想要去做它。

思考。

不要分心。

学习并应用你最感兴趣的技能或兴趣，使其与你的理想未来相一致。

当问题出现时，获取克服它们所需的技能和兴趣。

非常简单。

不需要任何自我帮助的处方。

你有一个自我纠正的指南针，那就是你的思想。

使用它。

### 3) 强大的习惯形成启发式方法

*如果你不知道你为什么要做某事，你为什么要做它？*

大多数人对他们做出的99%的选择的原因都是无意识的。

开始质疑你所做的每一件事。

一切。

如果行动背后没有有意识的意图，就停止做它。

你为什么出去喝酒和派对？

我不是要你停止，我是要你有一个理由。

我喜欢偶尔参加派对，并将我成功的大部分归因于那些时候我遇到的对话和人物。

但它通常是在一段长时间的紧张之后用作战术重置。这是一个被迫的低谷，平衡了被迫的高峰。

回到质疑：

你为什么要把辣味麦辣鸡塞进喉咙？

再次强调，我不是要你停止。

我喜欢去麦当劳，但这仍然不会让我完全偏离我的精神、身体和财务健康目标。

你为什么躺在床上而不是去健身房？

你为什么要在抱怨你讨厌它之后去工作？

因为你必须吗？

这不是工资奴隶制吗？

你甚至在乎吗？

你是在欺骗自己说这是另一个原因吗？

你打算为此做些什么？

或者你只是会一直做同样的事情，直到你死去的那一天？

你什么时候会让自己完全意识到你所处的状况？

这些问题的答案可能会让你震惊（如果你对自己诚实的话）。

没有错误的行为，只有没有意图的行为。

领导。你的。思想。

### 4) 个人分销

建立观众、读者或分销不仅仅是为了在线业务。

这是新的生活方式。

建立个人分销是你如何：

+   没有工作板就找工作

+   没有出版商就卖书

+   没有唱片公司就制作音乐

做你自己，学会说服，你就能跳过那些依赖他人成功的人。

我正在自我出版我的书，《专注的艺术》，它有新的计划比预期更早地推出……

平均自出版书籍销售250本（谷歌统计数字）。

他们每年的收入不到1000美元。

我几乎保证了我的书会成为畅销书，仅仅因为我过去3-4年里建立了一个观众群体。

*并且*所有的想法都通过我的通讯和社交媒体互动得到了验证。我早就知道人们会喜欢它，因为他们已经证明了这一点。

这同样适用于我建立的任何与我所谈论的略有关系的产品。

由于我已经建立了分销渠道，我可以建立我想要的任何东西。

我可以开一家健身房，补充品线，服装店，甚至开始制作音乐并登上大舞台，而大多数制作人将永远不会看到超过100人的观众。

公平吗？也许吧，也许不是。

或者这只是这样一个现实：人们在尝试赢得比赛之前没有学会游戏。

### 5) 专注于视角发展

对我来说，最大的“转变”是意识到建立观众群体并不关于最新的技巧和策略。

这关乎新颖的视角——这是AI工具在人类的名义下永远无法做到的。

我不明白AI如何能够将像肯·威尔伯的万物理论这样的东西，与创作者经济联系起来，并从经验[像我这里](https://thedankoe.com/the-one-person-business-roadmap-99-of-creators-make-this-mistake/)的角度提炼出“创作者发展阶段”。

我假设AI无法有洞察力，无法将其技能或兴趣与抽象模式联系起来。

这场游戏有一个进步的过程，但你希望达到无需个人努力就能传播的感染性想法的点。

创作者经济是一场心理战。

你传播的越多，形成他人身份和世界观的想法，你将获得更多的成功。

如果这对你来说没有意义，那么直到你真正开始构建，它都不会有意义。

祝你在创作者的道路上好运，亲爱的创作者。
