# 你们大约有36个月的时间去实现目标

> 原文：[https://thedankoe.com/letters/you-have-about-36-months-to-make-it/](https://thedankoe.com/letters/you-have-about-36-months-to-make-it/)

我在网上看到这个说法在流传：

*“你们大约有36个月的时间去实现目标。”*

这是有道理的：无论人们如何反对，AI都会继续取代工作。我们所知道的货币将会改变，甚至可能消失，因为数以百万计的ASI（人工超级智能）将迅速执行超出人类理解范围的任务，决定人类的未来（参见：[AI 2027](https://ai-2027.com/)）。抛开这一点，这将是你们最后一次看到加密货币的荒谬收益，因为这项技术正在被采用和监管。一旦被采用，它就变成了像股票一样的东西。

我写这封信的意图并不是要吓唬你们或听起来耸人听闻。所有这些都是预测和推测。这完全可能是错误的，我预计其中大部分都是。我只是在设定互联网某些小众领域正在发生的讨论场景。我的希望是给你们提供一个非教条主义的视角，这样你们就能注意到机会，做出不会让你们陷入困境的决定，因为很明显，*某种东西*正在改变，未来将会迅速到来。

我认为你们没有36个月的时间去实现目标。

相反，我认为你们有36个月的时间，在这段时间内，“成功”的定义将会发生根本性的转变。

幸运的是，人性保持不变，我不禁想到，这个快速增长时期正在明确区分机器和人类。

那些拥抱自己人性的人将会繁荣昌盛。

另一方面，那些无法停止将自己认同为机械生活的人，也不会停止。

### 行动派与指挥者

> 我们很快就会看到10人1亿美元的初创公司……在我的技术CEO朋友们的一个群聊中，有一个赌注池，赌第一年会出现一家一人1亿美元的初创公司。
> 
> – 山姆·奥特曼

一件事是肯定的：

个人今天拥有的权力远远超过过去任何时刻，而且这种权力转移只会继续增加。这种权力转移发生在三个层次上。

**互联网**让人们能够接触到任何和所有的知识。权力从学校和机构转移出去。

**社交媒体**让人们有了吸引自己受众的杠杆。权力从雇主、出版商、集中式媒体甚至唱片公司转移出去。人们现在可以学习、建设和赚钱，而不依赖于他人的权威。

**人工智能**正在赋予人们创造、自动化和外包几乎任何事物的能力。权力正在从传统的守门人和中间人手中转移出去。人们很快就能以少量的人力达到大型团队的水平。

简而言之，你们现在拥有三种超级力量来掌控你们的未来。

+   学习——适应并找出必须采取哪些行动才能获得特定结果的能力。

+   说服力——建立信任并吸引人们到一个互利共赢的愿景或叙事的能力。

+   执行力——将想法通过自动化、创造和委派转化为现实的能力。

这在那些将繁荣昌盛的人和那些不会的人之间划出了一条清晰的界限。

做事者与指挥者。雇员与企业家。低自主性者与高自主性者。那些主动分配工作而不是等待工作分配给自己的人。那些不需要许可就能解决问题的人，因为随着现实变得复杂，新问题不断出现，而人工智能正在加速向无限复杂性的发展，问题无穷无尽，因此以解决方案形式存在的价值也是无穷无尽的。金钱，不仅仅是美元，作为价值交换，实际上也是无限的。

那些为解决未来问题做好准备的人，即使没有人可能知道这些问题是什么，也会获胜。简单明了。[一人企业启动平台可以帮助你做到这一点。](https://letters.thedankoe.com/p/full-course-the-one-person-business?lli=1)

### 味道是新的智慧。

反对人工智能的群体将会失败。

将会有许多情绪激动的人试图诋毁那些试图在工作（尤其是他们的创造性工作）中利用人工智能的人。忽略这些人，并且不要成为其中的一员，因为在未来几年里保持悲观的心态只会导致具有破坏性的确认偏差。你关注什么，就会成为什么。通过从山顶底部试图阻止雪崩，你只会被压垮。

这些人还没有理解的是，我们创造的方式正在我们眼前演变。摄影和视频在人类历史的时间尺度上是非常近期的发展。照片编辑软件和计算机生成的图形让我们能够通过几点击就能创造出如此不切实际的场景。是的，创造这些事物是困难的，但与几年前相比要容易得多。仅仅因为屏幕上的几个点击被自然语言所取代，并不意味着创造我们视为艺术的东西变得更容易，尤其是当我们视为艺术的东西随着文化而变化，而文化目前正在迅速转变。**我们不考虑人们是艺术家因为他们使用了什么特定的工具**。*我们考虑人们是艺术家因为他们* ***为什么*** *和* ***如何*** *使用工具*。我们根据最终产品来评价他们。故事。情感。愿景。灵魂。而不是投入创造它的劳动时间。你可以花上10年或10秒钟写一本书，并称自己为作家，但这并不意味着这本书是好的。

这就是反对人工智能的群体所犯的错误。

他们把所有使用AI的人归为同一类别。他们把AI看作是一个用于短句的聊天框，你可以在里面输入“为我写这封新闻稿”或“为我生成这个图像”，尽管这些请求的输出可能看起来视觉上吸引人，但这并不意味着它们是艺术。*这仅仅暴露了大多数人过于强调创造视觉上吸引人的东西所投入的劳动，以至于这已经成为他们的身份，人们会不惜一切代价避免限制自己*。我不认为艺术应该与人们按下的工具按钮相关联，而应该与*为什么*按下这些特定按钮的原因相关联。

我想我们都可以同意，将所有的思考和控制都外包给AI不会让你有任何进展，但专注于这一点的人并不理解AI的本质。他们看不到AI已经成为语言本身的编程语言。他们看不到你可以给AI提供非常具体的指令，以至于可以完全控制输出（这本身就是大多数所谓的“创造性工作”，你是在执行你未设定的项目指令——这不是艺术，而是披着“艺术家”头衔的工业工作）。

他们不理解一个*词*是新的画笔，是新的低通滤波器，是新的选择和遮罩。他们不理解艺术家做出的每一个小决定，将个人风格融入作品，仍然可用，他们可以在不花费数小时的情况下创造出比他们最狂野的梦想还要多的东西，但他们却看不到随机文本框生成的局限。

**区分艺术与垃圾的唯一标准是品味。**

换句话说，什么都没有改变。人们只是讨厌新事物，而这个新事物正是照亮了最初值得关注的东西。如果你不能用AI创作艺术，那么你从一开始就不是艺术家。你只是擅长使用工具。工具会被取代。

### 实用性与意义

硅解决了实用性，碳才能超越到意义。

我们讨厌在DMV（美国州务卿办公室）排长队。我们讨厌服务员弄错我们的订单。我们讨厌本可以用5个要点邮件解决的问题的会议。AI和自动化解决了人类讨厌的必要工作。另一方面，我们渴望失败的可能性。我们喜欢第9局的最后击球手。我们跨越世界只为享受五星级用餐体验。当新娘和新郎朗读手写的誓言时，我们会流泪。

机器是用来提高速度、重复和必要性的。

人类是用来讲述故事、创新、神话和意义的。

大多数人认同机器的特质——教育、工作、日常任务——这正是为什么普遍的满足感如此低。在这个变革中将会出现很多动荡，那些没有掌握自己心智的人不会对此感到高兴，但我们已经集体对工业生活厌恶了几十年。你没有理由抱怨，因为你有99%的几率你的生活会比现在好得多。

放大来看很可怕，缩小来看很美好。

随着机械工作的取代，剩下的就是意义。

为了弄清楚你生活中想要什么，选择你深感兴趣的事情，无情地剖析它，并毫无顾忌地在公共场合分享你的故事、价值观、愿景和相关的生命工作，与一小群支持你所做的事情的人形成工匠般的生活方式。与AI结合，你就能为休闲腾出空间：这是对创造性工作的最高赞誉。

### 如何准备

需要36个月来实现。

这不是一成不变的，但这是你能得到的准确数字。

你可以选择坐下来继续你一直在做的事情，或者利用这个紧急的时刻来改变你的生活。就我个人而言，我不明白为什么你不会选择第二个选项。改变你的生活并为一个令人难以置信的未来做好准备，并不需要你放弃一切并搬到另一个国家。这需要你选择你的工作。这需要你思考。这需要你追求你关心的事情。这需要你承担风险，创造一个值得讲述的故事，并与那些从中受益的人分享。

过去创造“美好生活”的一切现在都成为了一种需求。

**1) 成为哲学家-建造者**

在当今世界，我们倾向于尊重两种人。

程序员与市场营销人员。宏观与细节。精神与实用。哲学家与科学家。

未来要求我们将这些极端合并。一个建造者始终需要一个思想家。一个没有与市场营销人员配对的程序员几乎总是导致用户数量令人失望。一个困在天空中的嬉皮士和一个困在原子中的科学家都错过了真理。那些能够体现这种双重性的人将变得不可阻挡，而且我们已经在看到这种情况发生。

职位名称正在从“设计师”和“工程师”合并为“设计工程师”。工具，无论是否由AI驱动，都达到了人们可以在不增加额外劳动时间的情况下做更多事情的程度。

创作者在创立和推动公司。编码者在社交媒体上产生数百万的观看量。

人类能力的巅峰在于成为一个专业通才。既不要太专业化以至于瞬间被取代或因为与证书相关而被利用，也不要太泛化以至于在实际上无所作为。

**2) 成为思想的过滤器**

现在想法比执行更重要。

如果品味是新的智慧，而在信息泛滥、任何人都可以创造任何事物的世界里，选择*哪些想法值得执行*应该排在首位。

你的大脑必须成为一个精细调校的信号与噪声过滤器。

为了实现这一点，你必须理解你的大脑是根据你的身份和世界观来解释现实的。你的价值观和信仰。你生活的准则。你的发展水平。你试图实现的愿景。当信息对你的主要和总体目标有用时，你的大脑会释放多巴胺。

由于大多数人没有目标，或者他们的目标是由学校和职业分配的，这是一个极其痛苦的过程，只有那些在生活中[承诺反抗默认路径](https://letters.thedankoe.com/p/this-single-decision-will-determine?lli=1)的人才会开始调整他们的本能，以便捕捉和利用面向未来的想法。

唯一知道哪些想法值得执行的方法是在试错之火的考验中锻造你自己的哲学。

**3) 成为人工智能编排者**

在采用人工智能时，首先要做的决定是何时利用它，何时手动完成。

通常，这个决定在于你喜欢做什么。对我来说，那就是写作。虽然我使用各种提示进行研究和开发想法（我最喜欢的提示之一是接受我给出的主题，并问我观察性的问题，直到我亲自达到一个我可以用于写作的“啊哈”时刻），但我的所有信件都是手写的。人工智能是一个*增强*我写作的工具，而不是取代它。

在你的技艺之外，我看到人工智能提示就像高度专业化的员工。我讨厌营销，所以我训练AI使用我尊重的营销人员的信息，给它我的框架、例子或灵感来源，然后让它输出一个我可以根据直觉和品味进行改进的初稿。到最后，它质量更高，但与我亲自完成的作品没有太大区别。

专注于你的技艺，编排一系列与你的价值观和愿景相一致的有品味的提示。

如果你还没有开始，学习人工智能的最好方法就是尝试自动化自己从工作中解放出来。

是的，这增加了你当前工作被取代的可能性，但如果你能自动化它，你本来就会被取代，而且至少这会提升你的技能，达到晋升或能够为自己工作的水平。
