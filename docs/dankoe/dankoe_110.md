# 消失并重新出现，无法辨认（12条改变你生活的规则）

> [https://thedankoe.com/letters/disappear-and-come-back-unrecognizable-12-rules-to-change-your-life/](https://thedankoe.com/letters/disappear-and-come-back-unrecognizable-12-rules-to-change-your-life/)

![图片](../Images/f7d5112739a4a4e1d1bda0ec933216b8.png)

我这么说并不是为了显得傲慢。

但我从未在知道未来想要什么方面遇到过问题。

当人们说“我不知道自己想要什么”时，他们真正想说的是“我不想做得到我想要的东西所需要付出的工作。”

你并不是不知道自己想要什么。而是你知道自己不想要什么——这意味着你知道自己想要什么——并且正在逃避它，逃避它需要付出什么代价才能摆脱它。

我一直都知道自己想要什么，因为观察社会并知道我不想要什么非常简单：

+   我讨厌的工作

+   我讨厌的工作

+   一个我讨厌的身体

+   我讨厌的伴侣

+   一个讨厌我的心灵

仅从这一点来看，很容易就能想出我必须做什么：

+   成为一名企业家，无论我失败多少次（我失败了7次）

+   获得摆脱我不愿意做的工作（在社交媒体上写文章以获得影响力）的力量

+   去健身房锻炼并关注我的营养

+   让这三件事在每个生活领域的每个领域都为你打开更多的机会

+   让那条道路从责任的渐进式超负荷中创造一个平静的心灵。随着你变得更强大，重量会感觉更轻。

每个人都知道，某种形式的这条道路是他们注定要做的事情。

你的心理渴望实现和超越。你存在的深度想要这些事物，但你的自我被它认为想要的事物所分散。

这就是问题所在。

你没有一种方法来集中你的思想。

你没有为你的未来制定一个比生活中分散你注意力的东西更有重量的计划。

你努力维持长期的时间视野，却陷入了永无止境的短期压力产生的工作中。

拥有一个单一的框架，它将决定你全部的成功，你会感觉如何？一个你可以在感到迷茫时随时参考的框架？

这就是我们接下来要讨论的内容。

## 《创造的12条规则》

> 你需要一个计划。因为如果你没有，社会就有，他们已经为你规划了几十年。 —— [《专注的艺术》](https://theartoffocusbook.com)

![图片](../Images/e44594f73ffa45c445760447b39d62c6.png)

你从出生那天起就被社会分配了目标。

这些目标继续塑造了你如何看待世界。

你学会了实现这些目标所需的技能。你注册了与这些目标一致的机会。你生活中所经历的一切都是通过引导你头脑中形成的系统中的有意识或无意识的目标的视角来体验的。

如果我们想要建立一个有意义、有钱财和影响力的生活，我们必须追求我们理想的未来，创造一个值得讲述的故事，并将这条道路传递给那些准备好接受它的人。

在经历了十年的创造性和失败之间的舞蹈之后，我发现了一个大局观的框架，它将使你在任何你所做的事情上都能取得成功。

这些模式可以在营销、销售、人类行为、巅峰表现、心理学、故事、游戏、价值创造和分配的结构、成功的产品设计以及任何涉及创造和分配价值的过程中发现。

你将使用它作为你生活的指导之光，但我鼓励你冥想如何将其应用于你的业务、关系和对话。

### **反愿景**

我们从反愿景开始我们的故事。

你存在的祸害。

你将要培养的世界观的第一极。

一种积极的恐惧机制，它会推动你采取行动。

*你的反愿景是你不希望生活的未来。*

开始记录你不想重复的经历。

+   你不愿意学习的材料。

+   你不愿意完成的工作。

+   你不希望发生的争论。

你不可能立刻摆脱它们。

你应该把它们识别为需要解决的问题。

### **愿景**

如果你没有愿景，你就会迷失。

你不能创造结果，所以你注定要过着机械的生活，追求确定的结果。

在你生活的任何领域做出的每一个决定都必须通过你的愿景来过滤。

这就是你怎么给你的行动带来意义，并最小化干扰。

写下你从生活中得到的确切东西。

*不要错过任何细节，但要知道这是一个迭代的过程。*

你不可能一开始就做对，你可能永远也不会。这不是重点。花30分钟生成一个最小可行愿景。经常回到它那里，随着你的愿望不可避免地随着你的失败而改变，添加、删除和改进。

### **使命**

你的使命是你生活中最重要的事情。

它是你所做和所不想要的之间的桥梁。

你正在为之奋斗的通往愿景的道路。

*任何与你的使命不一致的东西都应该被视为一种干扰。*

你的使命随着对新信念、机会和知识的认识而发展。

你的使命需要信念。除非你采取第一步，否则你看不见下一步。一旦你做到了，第二步可能与你所想象的一切完全不同。

### **标准**

你之所以没有达到你想要的地方，是因为你对现状感到满意。

*你不屈服于你的处境，你接受它，这样你才能前进。*

标准是从你的环境中吸收的。

+   你一起出去玩的朋友。

+   你所读的书。

+   你所消费的媒体。

+   抚养你的父母。

+   知道一切的老师。

可能是时候做出改变，并坐下来思考其后果。

### **目标**

大目标是方向。小目标是清晰。

当你面前的任务如此简单以至于你不得不完成它时，你不需要动力。

将你的愿景分解为十年、一年、一个月、一周和一天的目标。

它们是你的指南，而不是你的主人。

对愿景要有固执，对细节要宽松。

你的目标会随着你的变化而变化，对此感到满意吧。

### **项目**

学习来自斗争，而不是记忆。

你需要一系列有形的项目来构建，以实现你的愿景。

+   将你的目标转化为项目。

+   构建一个大纲。

+   确定里程碑

+   设定截止日期

+   规划研究区域。

构建，然后学习。开始项目，暴露你的知识和技能不足，并以此作为你教育的参考点。

### **教育**

你没有达到你想要的地方，因为你没有像你想象的那样聪明。

每个项目都需要一套技能和心态来完成。

每日自我教育必须成为你生活中的基石习惯。如果你停止学习，你就停止了进化。机会停止出现，你陷入了当前的发展阶段。

教育是实验的燃料。

### **限制**

一个傻瓜以生活中所有美好的东西为代价变得富有。

一个创造者以他的选择为代价变得富有。

目标上的限制迫使创造力。

问题是，你不愿意为了实现目标而牺牲什么？

当你试图在不背叛你的愿景的情况下实现目标时，创造性挑战就会出现。

你可以在不牺牲家庭的情况下变得富有。

你可以在不牺牲工作的情况下变得健康。

你可以在不牺牲使生活值得活着的因素的情况下变得有价值。

### **杠杆**

每天你需要优先任务，这些任务从底层推动你的项目、目标和愿景。

这些通常被视为没有培养掌握感的枯燥基础。

做你需要做的事情，但把你的愿景作为锚定未知的力量。如果你没有取得进展，那是因为你没有移动杠杆，即使你认为你在移动。

写作是我创作者业务中的主要杠杆（以及几乎所有其他现代业务）。这是我在[2小时作家](https://2hourwriter.com)中教授的，也是我们在[Kortex大学](https://university.kortex.co)帮助那些需要个性化帮助和反馈的人所提供的。

### **挑战**

当新手与大师对决时，双方都不会感到快乐。

新手会感到焦虑，而大师会感到无聊。

当你的技能与情境的挑战完美匹配时，世界会变得安静，你将优雅地前进。

挑战是乐趣的源泉。

乐趣存在于无聊和焦虑之间的平衡木上。在能力的边缘。

有意义的生活的道路通常在于视角的简单转变。

### **好奇心**

愿意偏离航线，发现新的潜力。

我们很容易陷入我们试图逃离的机械常规中。

保持好奇心。

深入你的兴趣。

让少数问题悬而未决。

避免陷入使你的思想局限于一个理想化的路径的范式和信念。

你的愿景就像一块电池。你必须用经验、教育和误导来给它充电。

### **实验**

如果你只是像其他人一样做同样的事情，你注定要受到其他人结果的影响。

+   研究别人教授的过程，他们看到成功的那些。

+   尝试多种技术，看看哪一种对你最有帮助。

+   创建你自己的过程，你可以保持一致性。

在健身方面，尝试不同的训练和营养计划。在关系方面，尝试治疗或自助书籍。在商业方面，尝试冷接触和有机内容。

避免对单一正确的方式变得教条。没有正确的方式，只有你的方式，而且这并不一定保持不变，因为游戏的不同级别需要不同的方法来完成。

当你迷失方向时，运行这个过程。

当你的关系失败时，运行这个过程。当你的业务无法起飞时，运行这个过程。

每一次与现实的成功互动都是从对需求的清晰图像、如何实现它的清晰认识以及创造性地执行以获得罕见结果开始的。

## 做出更好的决定

![图片](../Images/5747c3a446fab8a0880df2104bda392f.png)

你还没有达到你想要的地方，因为你：

+   你没有做出导致有目的职业生涯的选择。

+   你没有做出导致满足关系的决定。

+   你没有做出导致健康和美观身体的决定。

现在，你就是你过去选择的体现。

因此，如果你想掌握你将成为的人，你的选择至关重要。

这里有两件事：

1.  你想成为什么样的人——视角和放大。

1.  带你到达那里的选择——感知和聚焦。

好的生活是通过不断提醒你的愿景，并通过一致的行动来实际化你的身份来创造的。

更好的决定来自视角和感知。

每天都要放大视野，提醒自己你不想做的事情。你不需要专注于你想要的事情，因为那将通过你的选择自然而然地显现出来。

在你的脑海中保持这个框架在顶部。

不要让分心的事情渗透进来。

任何时候出现选择，放大视野并保持一致。

“这会对我试图创造的未来有益吗？”

然后，果断决定。做出决定。

允许失败进入你的生活，这样你就可以在下一次纠正你的行为。

意识是良药。

你不必放弃所有的坏习惯。你只需要通过你愿景的视角来看待它们及其后果，足够长的时间。
