# 如何在社交媒体上真正成长（即使你一个粉丝都没有）

> [https://thedankoe.com/letters/how-to-actually-grow-on-social-media-what-they-dont-tell-you/](https://thedankoe.com/letters/how-to-actually-grow-on-social-media-what-they-dont-tell-you/)

我有很多粉丝。

（一点自恋能帮助这个开始，对吧？）

严肃地说，我这么说是为了设定这个对话的框架。这是给那些想在社交媒体上成长的人。

我并不总是有这么多读者。

这很明显，但你了解人性的运作方式。

人们会根据他们对你的了解来塑造你的形象，并围绕这个形象做出判断。

人们看到我有280万粉丝，不会想到我曾经有过500个粉丝，而且有一个令人尴尬的资料。

找到一个从未根据封面判断一本书的人是非常罕见的。

我第一次尝试社交媒体是作为一个摄影师。

我热爱拍照，但从未能通过几次价值50美元的毕业照（图片很糟糕，抱歉那些付钱给我的人）来从中获利。

我会在Instagram上发布我的图片，使用我从其他账号复制的一大堆标签，并希望有一天我能够以某种方式获得10万粉丝。我没有盈利计划。

编辑开始越来越吸引我的注意。

我开始拍照只是为了对它们进行一些疯狂的编辑。

我把白天都花在狂热地观看YouTube视频上，学习Photoshop和Lightroom。

最终，我遇到了一个名叫Visuals Of Julius的人。

他会发布这些超现实的图片。就像我以前从未见过的一样。

我想做他做的事。

所以，我的YouTube狂热继续。

我从室友那里借了一些Adderall，把自己锁在房间里，听着重摇滚，创作我心中所想。

这是我最喜欢的一个：

![](../Images/bcba586a247f6d4d7ba36d61494cf316.png)

这就是事情开始变得真实的时候。

我开始揭开社交媒体增长的“秘密”。

我美丽的图片本身并没有什么意义。我无法只是发布并希望我们的救世主，算法，会拯救我。

*我必须让我的内容出现在人们面前。*

*我必须让我的内容被比我受众更大的人分享。*

（当你放大视野时，这是社交媒体增长的唯一原则，即使你在主持X Spaces或播客，而别人在主持你分享你的口头内容）。

在数字艺术界，有很多编辑账号。

这些账号每天都会发布他们最喜欢的数字艺术作品，并标记创作者。

你只有通过给他们标记、使用他们的标签、与他们互动，甚至给他们发消息来建立联系，才能有机会让这些账号看到你。

我连续坚持了两个月。

我知道我必须创造其他账号都愿意**分享**的优秀内容。

如果我记得正确的话，我的帖子在大小不同的账号上被分享了30-40次。

像是“A Game Of Tones”这样的账号就是大奖。如果你能被他们分享，你的关注者数量就会激增。从10万到50万粉丝账号的分享带来的多巴胺冲击是疯狂的。

在我的前两个月里，我将我的Instagram从0增长到2500名关注者。

根据我自身和我的学生的经验，前10,000个总是最难的。

到这个时候，我已经筋疲力尽并放弃了。

我没有看到自己能够持续使用兴奋剂和每天工作8小时来制作一张Photoshop合成图。

而且，我并不知道营销或销售。

如果我围绕我如何创作艺术（注意，艺术家们，这就是为什么每个大艺术家都有一个课程或教程数据库的原因。卖掉正在销售的东西）创建一个教育产品，我本可以赚一大笔钱。

这种短暂的社交媒体品牌增长失败教会了我真正需要增长的东西：

将优质内容展示给其他人——这是你获得他人关注的唯一机会。

但做这件事有很多正确的方法。

让我们讨论你可以利用的每一个增长机制，按优先级排序。

## 大图景——如何在社交媒体上增长

我不再做Photoshop艺术。

我现在是一名作家，这是[数字作家系列](https://thedankoe.com/blog/category/digital-writing/)的一部分，所以我们将从这个角度来讨论这个问题——如何让你的作品被分享。（如果你想要了解如何写和写什么，请[阅读这篇关于撰写原创内容的信件](https://thedankoe.com/letters/dont-get-replaced-by-ai-how-to-write-authentic-content/))。

我所说的写作，是指社交媒体上的一切文字，这几乎是一切。

标题，带有图像文字的Instagram帖子，推文，LinkedIn帖子，reels脚本，YouTube脚本等。

社交媒体的增长如此直接，以至于每个人都过于复杂化它，或者根本不重视它。

有些人不断发帖，只是为了对算法不传播他们的内容感到愤怒。

别人知道如何增长，他们是聪明的人，但他们的帖子就像永远不会被购买的艺术品。艺术家认为他们是世界上最棒的东西，但市场对此漠不关心。

然后，还有一些人增长了一点点，但不知道如何将他们的最佳想法加倍以实现增长。

**1) 写出有说服力的内容**

这里的关键词是说服力。

说服力意味着几件事：

+   **一个钩子**。注意你的第一句话是否会吸引他们的注意力。这不仅仅取决于你使用的词语。帖子的结构和设计也能做到这一点，甚至在X上也是如此。

+   **一个重大问题**。暗示或陈述人们能够相关联的问题。不要让它过于抽象。

+   **一个清晰的解决方案**。一种可行的克服问题的方法，最好是结合个人经验的一种独特方式。

+   **一个巨大的好处**。暗示或陈述人们将从你的建议中获得的好处。

+   **自信或极化**。坚定你的信念，但愿意改变它们。没有人愿意跟随一个听起来不自信或不能选择立场的人。说出你想说的话。你将像其他人一样受到仇恨。

+   **一个新颖的视角或大想法**。这就是为什么阅读或消费内容很重要。如果你能找到一种解释不寻常想法的方法，这会给你的读者带来多巴胺的刺激，并提高他们的理解。

在你的脑海中巩固这些想法的最好方法是将它们保存在一个安全的地方，并尝试在网上看到的内容中找到它们。

在下一封信中，我将介绍10种策略来让你的写作更具吸引力（附有示例）。

我在[2小时作家](https://2hourwriter.com)教授所有这些（如何围绕你的兴趣写作和建立品牌）。

**2) 让你的内容受到关注**

我们已经讨论了这一点，将在下一节深入讨论。

至少现在，要牢记如果你的内容没有人关注，它就是无意义的。这一点应该是显而易见的。

为了练习，将内容写入空白处是可以的，但不要让它成为借口，因为你害怕成长和面对批评。

**3) 用数据迭代你的想法**

你只能达到这一步，如果你有人关注你的内容。

如果你没有对这些想法进行任何参与，你就不知道哪些想法最适合你。

当你这样做时，每个月在你的分析中回顾你表现最好的内容。将它们复制粘贴到一个笔记文档中。

当你准备写更多内容时，加入那些想法，并从不同的角度来写它们。

这就是如何通过时间来增加你的参与度。

**4) 为你的最佳想法创造深度和广度**

你可以在像X这样的一个平台上做得很好，但最终你将变成这个平台的奴隶。

你将不得不继续使用增长策略来让你的内容始终出现在人们面前。它不会存活很长时间。

现代力量属于那些在其一生中拥有最多注意力的人。

一个只能吸引10秒注意力的简短帖子，与一个能吸引15-60分钟注意力的视频、播客或时事通讯相比，微不足道。

你必须将你最好的想法（来自最后一部分）转化为更长的内容形式。

这就是如何在你不尝试的情况下让你的名字传播开来。

这就是如何将你品牌的根基深深植入集体心理。

这就是如何建立比只发布短内容的人高得多倍数的信任和权威。

这就是为什么你几乎总是将你生活中的结果归因于一本书或一位教你这样做的作者。

即使有人在社交媒体上谈论相同的思想，当你在阅读时，作者会突然出现在你的脑海中（吸引你的注意力）。

## 交通机制 – 如何让你的写作被分享

现在你可能正在写作和发布内容，了解社交媒体游戏的机制会有所帮助。

当你理解了这些如何帮助你成长时，你可以创造性地思考如何让你的内容被分享。

这些没有特定的顺序。

尝试它们所有，看看哪个效果最好。

不要被社交媒体大师们告诉你说一个比另一个更好的陷阱所迷惑，因为算法已经改变了！！！

**1) 回复**

这是最明显的一个，每个人都在做。

如果你回复了别人的帖子，读者进入评论，他们有机会看到你的回复。

如果你的回复不错，他们可能会点击你的个人资料，阅读你的内容，如果内容好，他们可能会关注你。

这里有一些人们容易忽视的事情：

+   **无聊的内容不会让人们阅读回复**。我看到人们整天回复无聊的帖子，他们的回复只有几次曝光。然后他们想知道出了什么问题。

+   **无聊的回复不会让人们点击你的个人资料**。停止重复第一条帖子中的内容。在这里谈谈你自己。告诉人们你在生活中是如何经历同样的事情的。谈论自己会导致最多的个人资料点击。

+   **小号的整体曝光率较低**。与规模相当的账号回复和建立人脉是好的，但不要认为这会帮助你大幅增长（这没关系，你仍然应该这样做）。只是确保你是在回复有有趣内容的较大账号。

**2) 引用帖子**

引用帖子是指某人用他们自己的评论转发你的内容。这可以在X上，或者如果有人用他们自己的评论分享你的帖子到他们的Instagram故事中。

在X平台上，如果你知道如何写有说服力的内容，这可以非常有效。

如果你引用了某人的有价值的建议或有趣的内容，原始创作者可能会将其转发给他们的观众。这意味着你的内容会得到更多人的关注，这增加了人们点击你的个人资料并关注你的可能性。

在这里要小心，不要像回复那样把目标定得太高。

大型创作者有如此多的通知，他们可能会错过你的帖子。

能够从你试图利用的创作者观众的视角来审视你的成长努力。

**3) 转发**

你的内容必须足够好，足够有共鸣，以至于人们愿意在自己的动态中分享。

在点击发布之前，一定要仔细考虑。

人们会分享它吗？为什么他们要分享它？他们是否认同我所说的任何内容？这足以引发讨论并获得评论吗？

**4) 手动私信**

这一个更难且更慢，但这对建立人脉和让大号分享你的帖子（如果你是一个普通人，而不是LinkedIn公司机器人在他们的消息中）是有效的。

没有人*想要*收到这样的消息：“嘿！很高兴认识你！我喜欢你的东西！我们应该喝杯虚拟咖啡，谈谈Zapier自动化复杂性的问题，以提升你忠实电子邮件订阅者的整体体验，这样你就可以建立一个令人难以置信的数字形象！！！！”

你可以在线交朋友，直到他们愿意分享你发送的任何内容。

这是我写的关于我的[非依赖型网络过程](https://thedankoe.com/letters/how-to-network-with-millionaires-and-build-a-name-for-yourself/)的信件。明智地使用它，不要照搬我说的话。

手动直接消息方法也可以转化为控制你增长的方法。

发送一条直接消息可能只会让你的帖子获得一次曝光，但如果你有一个真诚的对话，他们分享这些帖子的可能性更高。

## 如何在任何平台上*控制*你的增长

如果你足够聪明，你可以以创造性的方式使用我们讨论过的任何流量机制来控制你的增长。

这些是你如果不想要成为算法的奴隶，可以用来增长的唯一工具。

**1) 短内容到长内容**

短内容平台如X、Instagram和TikTok在增长上比YouTube或播客等平台更容易。

你可以写更多内容，这些内容会通过我们讨论的流量机制被分享。

我无论如何都无法在YouTube上增长，但我知道如何在X上控制我的增长。

所以我同时在两个平台上发布，并随着时间的推移将我的观众引导到YouTube。

你不必做我说的一切，但考虑到每个人都想成为YouTube博主，这条路线是有意义的。

你可能需要走一条更聪明的路，而这其中之一就是。

**2) 精选、引用和混搭**

DJ和制作人通过混搭流行歌曲来增长，这样他们可以从那首流行歌曲中“窃取”流量。

有时候，他们会在Instagram上发布混搭的reels，以及原始艺术家（通常拥有很多粉丝和影响力）会在他们的页面或故事中分享。

这与我做数字艺术的情况类似。

我可以混搭别人的艺术作品，标记他们，并可能被分享。

对于作家来说也是如此。

你可以：

+   **在帖子或通讯中引用他们**。在你的长帖子中使用他们的想法来说明一个观点。然后通过直接消息给他们帖子，告诉他们你提到了他们。他们可能会不经过你的请求就重新发布。

+   **评论他们的内容**。类似于YouTube博主制作反应视频来利用大创作者获得的流量。

+   **邀请人们参与播客**。人们喜欢参与播客。这样你可以接触到更大创作者的受众。他们会分享播客，让他们的受众观看或收听。

+   **谈论他们的想法并给予他们认可**。一篇简单的帖子，你说“本周看到的最棒的想法是：”，然后继续解释它如何影响你，并认可原始作者。

这强调了长内容和短内容结合的重要性。在短帖子中实现这一点将很困难。

你当然可以在X这样的平台上写更长的帖子来利用这种策略。

**3) 付费增长**

这可能会引起争议。

“超级真实”、道德高尚（同时在社会媒体上毫无经验）的人喜欢说他们多么真实和道德，因为他们没有使用付费增长策略。

我已经观察了这些人多年，4年后他们要么一点都没成长，要么失去了粉丝。

人们不喜欢那些停止关注提供价值并推动更全面世界观的人。

现在，我并不是在谈论通过广告来增加粉丝。那是最糟糕的事情。不要这样做。我重复一遍，不要这样做。

我也不是在谈论为Instagram的推广付费，你支付给一个大主题账户发布你的内容，并让他们大声呼吁人们关注你。

那些是拥有50万粉丝但互动为零的账户。

我在谈论一个真正真实的帖子，并使用一点钱来提高你的结果。

优质内容是前提条件。

我在谈论的是支付给那些与你产生共鸣的大账户，让他们在上面花费时间使用上述流量机制。

大账户很忙，除非你付钱让他们关注你，否则他们可能不会这样做。这不是不公平。这只是现实。你不懂他们正在建造什么占据了他们的注意力。

**你可以请他们转发你的内容。**

在X上，这就像让他们分享你的帖子一样简单。

在Instagram上，这就是让他们在你的故事中分享你的帖子。

不要让他们说任何话或给你@。

让读者看到你的内容，如果他们想的话，就跟随你。

**你可以请他们回复你。**

当平台开始青睐评论时，这在对X这样的平台上效果很好。

这对你要支付的人来说通常更容易，因为他们没有在他们的动态中分享任何东西。

你真的可以请他们做任何事情。

给他们发私信，询问他们是否提供增长帮助，将对话转移到别处，并谈判一个交易。

在我看来，这比付费广告或任何其他货币化产品或服务的方法都要好得多。

通过真实的分享建立你的受众，让人们因为喜欢你的内容而跟随你，然后随着受众的增长，向他们推广你的产品或服务。

如果你认为广告商在你的面前随机推广产品是可以接受的，那么我不明白为什么你不认为创作者接受金钱来分享内容（你可能会喜欢的内容）而不要求你购买任何东西也是可以接受的。

**4) 智囊团**

坚硬的事实：

你认识的每个创作者都在与其他创作者的群聊中。

他们讨论策略，并一起享受建立企业的乐趣。

他们还会互相分享帖子并帮助他们成长。

他们利用流量机制制定一个可以融入他们日常生活的策略。

有些群组每天早上都会评论彼此的帖子。

其他人在群组中分享的帖子也会被转发。

最重要的是，他们形成了一个部落，并开始将自己视为一个群体的一部分。

你可以注意到有5-10个创作者的部落，他们都在网上互相交流。这就是原因。

是的，你将不得不发私信给人们，交朋友，并让这些关系几乎自然地形成。

这是一个长期的游戏。

如果您坚持这样做，不到一年时间内您不会看到疯狂的结果。

即使如此，随着您的成长，您仍然可以使用其他策略来货币化。
