# 这3个决定将决定你是否能发财

> 原文：[https://thedankoe.com/letters/these-3-decisions-will-determine-if-you-get-rich/](https://thedankoe.com/letters/these-3-decisions-will-determine-if-you-get-rich/)

发财的唯一途径是：(1) 将你的时间价值评估为一个幻想的美元数额，(2) 迷恋于发财，(3) 给自己没有其他选择，只能发财。

现在，我知道有一些人在后面大喊“钱不是一切！”或者重复那些他们从不懂金钱或幸福的人那里得到的肤浅的“钱不能买到幸福”，但这封信是给那些不仅仅点击这个链接来发泄愤怒，完全浪费时间的人。

这封信是给那些想要无怨无悔地发财的人。

不是因为你是一个糟糕和邪恶的人，而是因为在你脑海中，唯一的其他选择是按照市场认为你的价值来衡量，这是令人恐惧的。

对于我自己来说，我从小就意识到，我不愿意每天辛苦工作8-10小时，只为在一份我一无所有、做我不关心的工作中每年赚80-150,000美元，*尤其是*当我可以清清楚楚地看到，有些人做有意义的工作，他们的收入是我的100倍，而工作时间却少得多。

显然，当我年轻的时候，我没有那么大的价值去实现这一点，但我知道这是可能的。

我知道我可以做出一系列的决定，这将使我对我的金钱拥有如此大的控制权，以至于它不再主宰我的思想、情感或时间。那些不断道德化和精神化金钱的人无法想象一个他们不再受金钱束缚的生活。

虽然我不能告诉你具体该做什么（我的意思是，我可以，但现实的结果因人而异），但我*可以*给你3种方法来重新连接你的大脑，使其能够*做出正确的决定*，从而让你发财。

换句话说，我可以在你的心中植入信念，帮助你测试那些更有可能让现实吐出财富的输入。

### 励志的时薪

> 制定并执行一个励志的时薪。如果解决问题将节省的金额少于你的时薪，就忽略它。如果外包一项任务的成本低于你的时薪，就外包它……我的励志时薪是每小时5000美元。
> 
> —— 纳瓦尔·拉维坎特

这会让大多数人感到困惑（这肯定会让我的过去自我感到困惑），但到了一个有人愿意出价5000美元进行一对一通话，而我却不会接受的地步。不是因为我不需要这笔钱或者我不想帮助别人，而是因为这里有时间、专注和交付价值5000美元的义务。这是一个重大的责任，要求我将它置于其他可能更有回报的事情之上。

当我知道一个小时内可以赚得更多的时候，分散我的注意力是愚蠢的。

设定理想时薪的目的就是：做出更好的决定，让你的思维注意到那些让你处于更高杠杆位置的机会。

让我们假设你购买了一件价值 50 美元的损坏或破损的商品。你会花上一个小时去退货吗？如果是这样，你每小时的价值就是 50 美元，如果不是，那就更少了。这并不是“坏事”，但它照亮了一个更重要的事实：你没有意识到一个你可以着手解决的问题，这个问题会在同一小时内让你赚得 500 美元——或者 5000 美元。如果你意识到了，那么花一个小时去退货就没有意义了。

如果你的理想时薪感觉并不难以实现，那么它就太低了。

为什么？

幻想的目标会重新连接你的大脑。

你的大脑现在正根据你所学习的内容和思考的内容重塑自己，包括这封信。当你想象一个更好的未来时，它会激活与你正在经历相同的脑网络。每一个专注的注意力瞬间都是重新连接你的大脑，使其与你想成为的人一致的机会。这就是为什么末日滚动者和那些不管理他们的输入（心理或物理）的人，正在一点一点地挖掘自己的坟墓。

除此之外，在追求目标的过程中，多巴胺的峰值比实现目标时更高。但这就是危险所在。整天可视化你的目标可以增强学习，但只有当你将其与来自现实场景的最佳挑战水平相结合时。置身于游戏中。否则，它只会变成心理自慰：告诉每个人你的目标以获得廉价的 dopamine 冲击，但实际上从未采取任何行动。

挑战的黄金区域大约是 85% 的难度。不要太高以至于让你感到焦虑，也不要太低以至于让你感到无聊。挑战的渐进式超负荷带来的压力释放去甲肾上腺素，这进一步增强了学习，并帮助你适应新的心态。

所以，你不能只是想着你的理想时薪并祈祷你能赚更多钱。

你需要带着你的时薪来做出艰难的决定。

从不退货开始。

你会没事的。

### 力量的集中是致富的唯一途径

如果你想要致富，这需要成为你压倒一切的第一大愿望。

至少直到致富的行动在你的心理上深深地根植，你不需要花费太多的精力去执行它们，就像一个不需要*思考*就能命中三分的专业篮球运动员。

这适用于你想要在任何事情上取得成功：力量的集中是成为实现大目标的人的唯一途径。你需要着迷。你需要尽可能多地将能量投入到那个目标中。

但关于外出？享受生活？社交？追求西方版本的幸福（一种不可能持续的高潮状态）呢？

好吧，你不会创造财富，因为这不是你的**优先事项**。这完全没问题。你**不需要**设定致富的目标，但很多人都有这样的目标，他们关于生活的意义和你一样错误，而没有将此作为优先事项的人也不可能意外地致富。在接下来的40年里投资一些钱到股市，这是享受晚年生活的一定程度财富的一种方式，但谁会理智地不想在30年前就有这样的知识并去实现它呢？

哦，顺便说一下，将致富作为优先事项并不会让生活变得更糟，反而会让它变得无限美好。

职业运动员的生活很糟糕吗？可能不是，他们大多数时间都在做自己热爱的事情。那么，为什么你不将同样的逻辑应用到商业上？难道在所选领域努力达到顶峰的人就不能过上好生活吗？

有人说，幸福是进步和对你自己以外的事情做出贡献的结合，所以当你不被这种进步分散注意力，并且在大规模上解决有价值的问题（这就是创业）时，生活比平均水平要有趣一千倍。

是的，你仍然需要做生活中要求你做的事情，但这样做会给你带来痛苦。这种痛苦是你做出决策的方向。因为你能容忍的越多，你采取行动去改变它的就越少。

为了使这一点具体化，首先列出你**不**做的事情。

真正不值得你花时间的事情。咖啡会议。退回愚蠢的商品。无聊时滑动。因为缺乏激情和适当的刺激而聚会。及时回复所有人的短信（这一条尤其愚蠢，因为没有人愿意在任何时候都能被别人打扰到他们的注意力）。可以外包或忽略的小任务。当这些决定出现时，坚定地说，“我不会做那件事。”

然后，列出你**将要**做的事情。

将你的实际时薪与你的理想时薪合并的事情。学习能大幅提高你收入潜力的信息。练习让你自给自足的技能。创业。加入初创公司。争取高管职位。学习和构建。

你越思考你的行为，就越容易看到为什么你还不富有。

### 只给自己一个成功的选择

我做出的最佳决策总是涉及利用我的钱作为推动力来创造更多价值。

换句话说，我投资了我买不起的东西，并且学会了相信自己能够更快地赚回来。

我收入的第一次显著增长是在我购买了一款价值1500美元的电子邮件营销和产品托管软件之后。当时，那笔钱对我来说非常宝贵。花掉那笔钱让我深感痛苦。当我点击“购买”时，我感到一阵焦虑。但那种痛苦驱使我试图像我的生命依赖于它一样尽快赚回来。这迫使我学习如何通过营销、文案和漏斗知识充分利用它。通过这次购买，我不再需要猜测该学习哪些“高价值”技能。很明显，如果我学会了它们，我就能提高我的收入潜力。

当我持续搬进各种公寓时，也发生了同样的事情。每一个都比下一个更贵。每一个都超出了我舒适的范围。表面上，这看起来就像我只是一个生活方式膨胀的受害者。实际上，这是一种在现实世界中给自己设定挑战截止日期的方式，我知道我可以接受。我实际上是在把自己置于未来自我的环境中，我没有选择留在我不想要的那个生活中。

这三个决定——一个有抱负的小时费率、集中力量以及给自己没有其他选择，只能成功——创造了比我的年轻自我认为可能更多的赚钱条件。

那些决定都没有涉及到对“学习什么技能”或“选择什么商业模式”的痴迷。我没有时间去想这些。我必须学习所有这些。

当然，这需要很多失败和迭代，但这是社交媒体，谁会谈论那些东西呢？；）而且虽然我不是快速致富的倡导者，但你可能会惊讶，如果你设置了这封信中概述的正确条件，你可以多快地致富。事实上，我会争辩说，你可以在两周内学会任何技能（并且比99%的人做得更好）。

我会把那留到另一封信里再说。
