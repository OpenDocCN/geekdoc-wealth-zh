# 如果你想要改变生活，就必须极端

> 原文：[https://thedankoe.com/letters/you-need-to-be-extreme-if-you-want-your-life-to-change/](https://thedankoe.com/letters/you-need-to-be-extreme-if-you-want-your-life-to-change/)

我清晰地记得2021年的生日周末。

我的生意开始变得非常好。我在减重（在COVID期间体重增加了20-30磅）的尾声。我正在我的第一本书的第一稿中间。

在我的生日之前，感觉每个周末都有一个“我不能错过”的活动。

音乐会。和朋友聚会。这类事情。正常生活。

我已经取得了很大的进步，稍微休息一下不会有什么坏处。

所以我出门了。我喝了酒。熬夜。打破了饮食。正如你所猜到的，我在各个方面取得的进步不仅停滞不前，而且迅速下滑（感谢熵）。

我的心理和体能水平恢复大约需要4-5天。到那时，又有另一个活动要参加。我无法清晰地思考我的书或生意，保持健身习惯变得更加困难。

这很痛苦，但那种痛苦还不是无法忍受的。

在几个月前，我和我的朋友群计划在我生日时去西雅图。

我已经厌倦了没有进步，但今天是我的生日。我为什么不放松一下，找点乐子呢？似乎我找到了一种为我在这种环境下做出的每一个糟糕决定找借口的方法。

经过一个周末的晚起，在任何事情上都没有取得任何进展，并且远离家，我已经受够了。

我对自己非常愤怒。

因为当你有目标是你世界中最重要的东西时，它们会照亮你正在做的每一件错事。世界上最痛苦的事情是看着这些目标逐渐消失在背景中，被肤浅的快乐目标所取代。

一回到家，我就消失了。

我对任何人都没有回应。我变得不可用。任何形式的邀请都是自动拒绝。我采取的每一个行动都感觉无缝。我*想要*自律。这让我感到快乐。我的思绪回来了。我的身体回来了。我的生活回来了。

我的整个世界在一瞬间改变了。

但为什么呢？

### 1) 极端改变你的大脑

“一起放电的神经元，会建立联系。”

这是赫布定律，一句著名的格言，总结了与神经可塑性相关的神经心理学理论。

如你所知，神经可塑性是指你的大脑通过在一生中形成新的神经连接来重新布线自己的能力。你的大脑不是固定和僵化的。它是一个动态的网络，可以根据你的经验、思想和行动进行适应、学习和改变。

对改变生活极端可以帮助加快这个过程。

人们大声疾呼“一致性是关键”，这是真的，因为重复通过持续的努力强化了神经通路，但我们还可以更进一步。

新奇和挑战更能刺激神经可塑性。

因此，当你全力以赴追求目标时，你让你的大脑处于一个快速适应并帮助你将其成为*新标准*的环境。

### 2) 强度和痴迷创造了一种神经化学鸡尾酒

大多数人陷入困境，因为他们寻求*外在动机*。

但在这次讨论的背景下，你被*内在动机*所驱动。

每一种都相互叠加并加强，以维持某种程度的流动（最佳体验，或最令人愉悦的心态之一）：

+   **好奇心** – 探索未知、学习如何改变和填补知识空白的需求。这会产生来自新奇和去甲肾上腺素的好多巴胺，这提高了注意力，为你学习做准备。

+   **激情** – 对改变你生活的道路产生强烈的热情。这会产生更多的好多巴胺和去甲肾上腺素。

+   **目标** – 感觉到你的行动对比你自身更大的事物有所贡献。实现目标会导致更多多巴胺的产生，从而强化行为。血清素源于重要性和归属感。催产素源于联系。

+   **自主性** – 指导自己的生活和工作，控制你的选择、行动和环境。这又会导致多巴胺的增加和皮质醇（感觉被困在框中的压力）的减少，从而允许进行创造性的决策。

+   **精通** – 学习和成长的过程本身就是一种奖励。这会导致可持续的好多巴胺，让你保持参与。

当我们谈到如何度过接下来的6个月，而你又有不可否认的改变欲望时，你将理解所有这些是如何发挥作用的。

### 3) 你的大脑根据你所痴迷的事物过滤现实

> 那个自认为是“失败型人物”的人，尽管他有好意，或者有意志力，甚至如果机会直接摆在他面前，也会找到某种方式失败。那个自认为是“不公正的受害者”，一个“注定要受苦”的人，必然会找到证实他观点的情况。
> 
> – 马克斯韦尔·马尔茨

我们的大脑是为生存而设计的。

但人类有趣的地方在于，我们不仅试图像动物一样保护和复制我们DNA中的信息，我们还试图保护和复制我们意识中的信息。

也就是说，当我们的身体或我们的*身份*受到威胁时，我们会感到受到威胁。

在负面方面，我们会注意到那些加强我们信念和世界观的信息，这往往导致我们猛烈抨击质疑我们宗教或政治信仰的人。我们的身份感到受到威胁，大脑的压力反应发出信号，所以我们感到需要“生存”。

在积极的一面，当你与你所痴迷的目标认同时，你会注意到环境中的机会来加强你正在成为的更高版本。由于你的身份更高，它试图在这个层面上生存，这导致你收集知识并采取行动来实现这一点。

由于我们的祖先生活在资源匮乏的环境中，当他们注意到有助于他们生存的东西，比如灌木上的浆果或用于生火的木材时，多巴胺就会涌入他们的大脑，提高他们的注意力。

如果我们将控制论应用于心灵（心灵/灵魂），我们可以看到我们的心灵有一种向我们的目标归巢的机制。我们的心灵接受有助于它的信息，拒绝不利于它的信息。你几乎不费吹灰之力就会被引导向你的目标。

总结来说，*如果你痴迷于一个目标，宇宙会通过给你提供实现它的资源来支持你*。

## 进化从混乱中创造秩序

热力学第二定律表明，所有自然过程都会随着时间的推移向更加无序的状态发展。

换句话说，熵随时间增加，熵是衡量系统混乱、无序或随机性的度量。

这导致了“时间之箭”、“不可逆性”和“决定论”的概念。

时间是不可逆的，正朝着增加熵的方向发展。

然而，这并不是全部。宇宙并不是朝着死胡同的方向发展，世界在火中终结。实际上恰恰相反。

在他的著作《不确定性的终结》中，伊利亚·普里高津提出了这样的观点：“时间之箭”不仅仅是一种幻觉，而且是现实本身的根本特征。当一个系统被推离平衡状态足够远时，它可能会变得不稳定。

关键点在于：这种不稳定性可能导致“耗散结构”这种新的、复杂的、有序模式的自发出现。

想象大量的水从水龙头流出。当它流进下水道时，它会达到形成美丽漩涡的点。这是从混乱到秩序。

对于那些热衷于探索的人来说，让我们深入探讨一下。

我喜欢用具有实际应用的宏观模式进行绘制，因为它们在理论上可以接近真相，而不至于陷入某种方法或范式。

当我们以一般的方式思考宇宙时，我们可以看到它是由整体和部分组成的。

不仅是有部分，不仅是整体，而是整体-部分。每一件事物本身都是一个整体，也是另一个整体的一部分。从上到下，从下到上。

原子是一个整体，也是分子的一部分。

分子是一个整体，也是细胞的一部分。

细胞是生物体的一部分。

生物体是环境的一部分。

字母 → 词 → 句子 → 段落

物质 → 生命 → 心灵 → 灵魂 → 精神

捡拾 → 园艺 → 农业社会 → 工业社会 → 信息社会

无限嵌套的层次结构，无限互联的链接。

任意选择你能看到、听到、触摸、品尝、闻到或思考的事物，你就会意识到这一点。

现在，这些整体-部分有各种原则，描述了它们的特征和行为，比如它们的基本能力（自我保存）、交流（自我适应）、自我超越和自我消解。如果你想了解更多关于这些原则，我推荐阅读肯·威尔伯的《万物简史》。这是我最喜欢的书之一，还有《整体心理学》，它现在正在震撼我的思想。

我想要关注这些整体-部分的一个特征，它与普里戈金的论点相联系：

*进化从混乱中创造秩序*。

随着物质的进步，生命出现。随着生命的进步，心灵出现。

随着任何事物的发展或复杂化，一个新的整体随之出现，其中包含着复杂的部分。

请耐心等待，因为我试图将这些全部联系起来。

当你将熵的概念应用于心理时，你得到的是心理熵。

换句话说，心灵倾向于无序、随机、混乱等。

在你的个人生活中，当你感到迷茫，或者觉得你的生活正在陷入混乱时，*你将有机会出现一个更高版本的自己*。

正如我在生日时对我在进步上缺乏进展感到厌倦的故事中所展示的那样，你需要达到*失调与洞察力*的十字路口。

一旦你充分尝到了你所在的位置，你将开始感觉像被撕成两半。

你的一部分想要保持不变。

你的一部分想要改变。

当你趋向更高层次时，你会遇到洞察力。

推你进入极端的临界点。

## 消失3-6个月，专注于4个习惯

> 当你仅仅记住你需要知道的事情时，新的知识就会出现。继续将问题保持在心中，它就会解决。人类的历史无疑是这一事实的证明。一个人遇到一个问题，就会简单地痴迷于这个问题，直到他或她解决它……这可能需要一周、一个月、一年、十年、一百年或一千年，但宇宙似乎总是这样，解决方案总是会出现。
> 
> – 肯·威尔伯

僧侣模式。

我相信你很熟悉它。

这个概念在自我提升领域已经讨论了十多年，但最近由于它的普及化，它已经变成了其前身的一个糟糕版本。每个人都谈论过它，既然每个人都了解它，它通常被简化为“切断所有人并赚更多钱”的肤浅定义。

我们不会这样做。

事实上，我们甚至不会遵循僧侣模式的原始含义。

当马拉犁被发明时，自由的人不需要花太多时间寻找食物，因为食物很丰富。这使得他们（历史上第一次）能够追求专业兴趣，如写作、数学、帝国建设和我们的讨论，*冥想*——思考他们自己的存在。

人们很快意识到，神圣的不是“在那里”或“在那里”与神话中的神，而是在“这里”。不久之后，出现了轴心时代，在多个文明中涌现出革命性的精神和哲学思想。古希腊的苏格拉底，印度的乔达摩·悉达多（佛陀），中国的老子。

精神上的这一突破是“纯粹的提升”。

提升精神，或禁欲主义，是一种认为显现世界是邪恶的观点。因为天堂“在这里”而不是“在那里”，所以金钱、性、食物和快乐都被妖魔化了。

*真正的*僧侣模式就是这样。一段致力于提升之路和自我实现的时间。

问题是，大多数非教条主义者意识到，精神只是谜题的一部分。它非常重要，但将解决所有问题的方案简化为冥想，与将其简化为原子、金钱或*仅仅部分*一样糟糕。

相反，我们将消失一段时间，专注于追求更好的生活。

一个你沉迷于问题直到它得到解决的时间段。在这种情况下，我们的问题是想要改变我们生活的全部。

我们需要一种全面的僧侣模式，如果可以这样说的話。

这分为两个部分：防御和进攻。

### 防御 – 消除干扰

专注不仅适用于工作。

干扰不仅仅是让你远离工作的通知。

“消失”的全部意义在于在矩阵中制造一个故障。一夜之间改变你的身份。摆脱那些拉你下水并消耗你能量（这些能量必须用于创造更好未来的选择）的螃蟹和寄生虫。

为了极端，一次改变一切比一次改变一件事要容易。

剥掉创可贴。

停止回应那些只索取而不给予的人。

任何将你从目标拉开的邀请都应该是一个自动而坚定的“不”。

每个对你无益的坏习惯明天都将停止，没有例外。

选择自己的局限性的感觉是难以置信的。

为了摆脱那些分散你注意力、让你无法做最高善事的障碍。

贫乏的干扰——如思想、担忧、饥饿或与工作相关的视觉杂乱——将在下一步解决，并且当你致力于这条道路时。纯粹承诺的标志是干扰甚至没有机会在你的注意力中登记。

### 进攻 – 4个专注习惯

好的生活源于对善、真和美的追求。

追求真理。

**善**是主观真理，对人类或社区有益——做正确的事。

**真实**是客观真理，存在于科学研究中（科学推导数据，数据是经验，经验是物理的、心理的，*以及*精神的）——学习和理解事物的真实状态。这是一个过程。

**美丽**是主观真理，是美学的领域——注意到并*创造*你周围的美。

当我们将这种追求应用于我们自己的个人生活，而不是社会变革或其他系统性问题时，我们的注意力转向了我们的日常习惯。

你可以采用许多不同的习惯来检查这些框，但我想要提供一个清晰简单的建议。

这是你接下来6个月每天都要消失去做的事情。

一个项目。

一本书。

一次冥想。

一次锻炼。

**1) 一个项目**

工作是生活的一个必要部分，它为休息和休闲带来了对比。

但大多数人都在做着他们讨厌的工作，因此他们把婴儿连同洗澡水一起倒掉，希望永远不再工作。这是一种错觉。你不会在永久的假期和休息状态下感到快乐。而且，普通人对于休息的看法是以快乐的形式自我毁灭，以减轻为他人设定的目标而工作的生活的痛苦。

*选择一个作为你想要生活的生命基石的项目。*

这可以是任何东西。

一个能够导致独立工作的创意项目。

一个能够导致职业转变的文件夹项目。

大多数时候，考虑到AI的状态，我建议至少尝试以某种形式进行创业，因为你不能被替代，如果你是那个能够规避替代的适应性[高代理](https://thedankoe.substack.com/p/main-character-energy-mce)个体。

因为你在尝试将美好、真实和美丽融入你的工作中。当你被分配工作时，你无法控制这一点。

你的工作和推动你走向这项工作的项目，是*美好*的体现。创业是解决他人生活中问题的艺术。这是你通过履行改善他们生活的角色，为一个小社区的人们做出贡献的方式。如果你能通过基于直接经验构建解决方案来融入*真实*，通过表达你个性的深度来融入*美丽*，你就走上了正确的道路。

**每天在一个可以改变你职业生涯的项目上投入一小时深度工作**。

把它放在你的日历上。

那些不觉得像工作的工作，是这个世界需要的东西（美好）、你经验中的东西（真实）以及你深深关心的东西（美丽）的交汇点。

当正确完成时，工作与生活同义。

这是你所有其他行动的核心支柱。

**2) 一本书**

你肯定会在6个月前完成一本书。

你可以读更多，但说“两本书”会剥夺了让一切从“一个”开始带来的乐趣。

如果你想要改变你的生活，你需要改变生活的知识。

你需要学校无法提供，或没有提供的教育。

你需要让你的心灵沉浸在你想要过的生活的人的思想中。

这很简单。

一本书。

每天锻炼30分钟。

因为你好奇而学习新事物是*真实*的最好体现。

并且，正如你会发现的那样，简单的探索行为会带来新颖的想法，从而带来进步。

**3) 一项冥想**

工作可以是美丽和灵性的，但它不能完全取代一个保持你与现实联系的习惯。

我所说的冥想，并不是指坐在黑暗的房间里让思绪流逝，尽管这也是一条非常可行的路线。

在我看来，冥想是一种生活方式。

通过惊奇和感激的视角看待世界，注意你周围那些在我们压力和狭隘时往往被忽视的奇妙事物。

在这种意义上，冥想的目的是注意到*深度*。

为了逃离浅薄生活的陷阱。

至少每天10分钟，尝试通过你的感官细致地注意到现实。

去散步，并且只专注于你的脚踩在地上的感觉。

直视一棵树，注意你以前从未注意过的复杂细节。

当与心爱的人交谈时，要深入倾听。不要只注意词语。注意语气、表情，一切。

直视你的手10分钟（是的，我是认真的）并让你的心灵深入到细节中。

在洗碗时感受手上的水。感受你一生中接触到的每一件事。聆听寂静的声音或远处汽车的声音。分辨出即使是无聊食物的复杂气味。

不要过于复杂化这个问题。

**4) 一项锻炼**

> 没有人有权利在体能训练方面成为业余爱好者。一个男子在未看到他身体能够达到的美丽和力量之前变老是一种耻辱。
> 
> – 苏格拉底

回到现实由整体和部分组成的主题：

另一个重要的原则是，低级事物的破坏对高级事物具有自我毁灭性。

如果我们破坏地球，人类也会随之毁灭。

如果癌细胞侵入生物体，它就会生病。

如果你从一段话中删除一句话，它往往会失去其意义。

你生活在你的身体里。

应该把它视为你的全职工作，去了解它、训练它、尊重它，因为其他任何事都是字面上的自杀。

不仅如此，健康缺失的影响还会向上蔓延到你的创造力、工作质量以及人际关系中的深度。这一切都是相互关联的。

承诺一项锻炼。

无论你做什么都感到最有吸引力。

找一个跑步计划、举重计划、普拉提练习，无论是什么。

关键是要关注让你每天都能在这里的身体。

4个简单的习惯可以彻底改变你的生活。

4个简单的习惯可以使你与美好、真实、美丽的事物保持一致。

4个简单的习惯可以通过好奇心、激情、目标、自主性和精通带来更高的享受状态。
