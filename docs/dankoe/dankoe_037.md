## 学习的重要性

我对现代教育体系有一些问题。

请不要误解……它确实有一些很好的方面——*每个人都应该在社会运作方式上保持一致的观点*。

但从自然本意生活的角度来看呢？

这对于帮助人们找到、了解并从他们关心的事情中找到职业是一个糟糕的解决方案。

现代教育体系与我们的生存本能心理作对。

它推动我们去“学习”，记住，并训练进入一个与我们学习和记忆无关的工作。

另一件事……大规模教育永远无法跟上以指数速度持续发展的数字经济。

只有那些知道如何追求好奇心、快速学习并成为“工具独立”的人才能最终站在顶峰。

通过顺应你的天性——你的生存本能的大脑——你将开始收获这样做的益处。

### **学习是你如何在生活中变得更清醒、更有意识、更有技能的方法。**

当你第一次开始玩游戏时，你会学到：

+   游戏规则

+   机械和控制系统

+   如何赢得游戏

你好奇这个游戏会有多有趣。

你会通过教程视频或阅读规则手册。

你作为一个一级玩家测试你的极限，并很快意识到你还有很多工作要做。

你开始问自己问题：

+   这个游戏适合我吗？

+   这真的值得我花时间吗？

+   我甚至会在玩这个游戏时感到乐趣吗？

+   我的朋友们在玩游戏吗？

+   世界上有多少人正在玩游戏？

+   这将消耗多少时间、金钱和其他资源？

提问是确定是否值得将这项新活动纳入你生活的方法。

你还有其他事情要做，不是吗？

你有常规、生活方式、社交活动、习惯、差事和其他可能会因持续玩游戏而受到影响的事情。

如果玩那个特定的游戏没有被看作比其他你正在玩的游戏更有趣、更有价值或是一项活动，那你为什么要把这变成你生活中的常态呢？

这就是很多人掉队的地方。

他们还在第1级时就停止玩游戏了。

### 学习是你揭示生活所提供深度的方法。

当你刚开始学习一个游戏（或在生活中尝试新事物）时，你可能不喜欢它。

这可能对你来说现在还没有意义。

也许你“讨厌”在周日和妻子一起去古董店购物。

并不是你讨厌它。而是你不理解它。

对你来说，这不仅仅是一个游戏，而是一种麻烦。

你看到的只是无聊的旧古董，在你的生活中没有任何作用。

然而，对你妻子来说，古董店购物是一种有趣的游戏，她在其中是高级玩家。

+   她喜欢尝试在古董店找到“隐藏的宝石”。

+   她对历史的理解到了一个程度，可以欣赏古董。

+   她的父亲是一位木匠，当她看到摇椅时，这让她想起了他。

她对深入探索游戏的内部运作感到**好奇**。

她通过经验对如何玩游戏有**清晰**的认识。

她将游戏呈现的其他过去或潜在经历联系起来。

她在心中虚构了一个**挑战**，以便在那天参与游戏。

她提高自己的**意识**，看到游戏呈现的相互关联的网络。

但你……你甚至还没有开始玩。你不把它当作游戏。你专注于冰山一角，并假设水下的一切都没有作用。

你没有给自己一个机会去找到你的“为什么”。

做得足够多，你就不会看到生活的冰山一角。这是一种肤浅的存在。

*在你离开教程之前，你可能会发现自己想要放弃。*

在第1级，你只能假设或预测在你之上的层级中存在什么。

在第100级，你对下面的层级有鸟瞰般的视野。
