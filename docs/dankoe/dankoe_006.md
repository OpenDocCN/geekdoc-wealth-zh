# 当生活变得艰难时，这是你如何坚持下去的方法

> 原文：[https://thedankoe.com/letters/when-life-gets-hard-heres-how-you-push-through/](https://thedankoe.com/letters/when-life-gets-hard-heres-how-you-push-through/)

我们生活在一个幻想中，我们想要进步，但又不希望它艰难。

我们成为了“应该是什么”的奴隶，而我们的“应该”是“容易”。

人们会因在炎热天气中行走而发疯，但可以毫无怨言地进入桑拿房。

差别在于期望和意图。

当你在寒冷中时，你的自动反应就是颤抖。

为了战斗。

这听起来合理，对吧？

寒冷让人不舒服。

因此，我们专注于由期望产生的强迫性负面思想，并成为我们称之为家的心理结构的奴隶。

但是，如果你能与之共存，顺应它，并意识到不适是人类经历的一部分……

你会发现，你对思想的解释是你痛苦的根源。

我已经多次学到这个教训。

第一个是在因持有大麻被捕（他们在大学校园里不会对此轻视）之后。

第二个是在我从父亲那里借了2000美元，刷爆了我的第一张信用卡，并为大学第五年贷款之后。我为了使我的小电子商务业务运转而财务上自我毁灭……结果却惨败。

第三个是在哥斯达黎加。在我达到业务收入的高峰后，我冲动地和我当时的女朋友一起买了飞往CR的机票。

我们关系的真正本质被揭示，我们分手了，第二天就飞了回去。

所有这些经历都有一个共同点：

心理动荡。

不仅仅是压力的轻微增加，而是那种让你感到被困住的压力。

最小的工作也变成了不可能完成的巨山。

每个思想都变成了一个敌人，它将你困在心理游戏中。

隧道尽头的光明消失了，你忍不住想知道你是否能出去。

这些生活的阶段显然并不有趣。

但是，这些时刻正是生活提供只能通过改变方向才能找到的教训的时候。

所有这些重大斗争有几个共同点：

1.  所有由进步引起的快乐都被剥夺了，就像在高潮之后突然切换到新电影场景一样。

1.  我感到迷茫，几乎没有欲望去维持我之前目标的进步——我已经实现了它，必须继续前进。

1.  我内外都进行了实验。我尝试了内在平静的技术，同时设定新的目标，并伴随着教育和实践。

1.  随着时间的推移，拼图的碎片拼在一起，带来了清晰，并把我带入了新的强度季节。

这就是我们在这里要解开的东西。

这为什么很重要？

所以你可以确定你目前生活中的位置，看到隧道尽头的光明，并在你筋疲力尽之前避免在水中挣扎。

## 平衡和平与进步

![图片](../Images/d15dc8e06a7887927898fecdd61ccf9c.png)

幸福是一种技能。

它不是被给予的，而是被创造的。

不是应得的，而是通过努力获得的。

就像所有技能一样，它必须被练习、提炼并成为你生活的一部分。

如果它不属于你现在的每一天、每周或每月，以及你未来的每一天、每周或每月，那你为什么要做它？

如果你想要成为一名作家、创作者或营销人员——为什么你不每天写作？即使一开始你得不到报酬，你仍然需要建立杠杆、声誉和权威，直到你得到报酬。

如果你想要学习高影响力的写作（在一个有助于你为未来工作建立杠杆的实际环境中）[查看2小时作家](https://2hourwriter.com)。

当被孤立时，快乐失去了它的意义，因为它不能在没有不快乐的先前经验参照点的情况下存在。

*这就是新时代积极运动的问题所在。*

你可以整天想些积极的思想。但人们没有意识到的是，每一个积极的思想都包含一个消极的种子。**这就是为什么几乎每一次巨大的成功都发生在巨大的失败之后。**

你可以通过思考走向一个积极的未来，但那个未来的形象是通过一个相等且往往更痛苦的过去锻造的。

没有负面经验作为锚点，你没有什么有意义的目标可以朝着它前进。

有两种心态需要意识到：

存在和行动。

和平与进步。

静止与运动。

当你的注意力集中，没有干扰，专注于这些纯粹生活的任何一种模式时，你就能进入当下的力量。

*每个人都想要金钱、肌肉和理想的伴侣。*

实现那些目标的感觉在开始时是难以置信的……

如果你没有在物质追求背后一个哲学上的掌握感，你将面临一个粗鲁的觉醒。

即使你的行为没有改变，它们背后的“为什么”可以改变。

一个在艰难分手后开始健身的人，如果他们让心灵超越对物质世界的认同，可以在健身这个微观世界中找到生活的意义。

那些获得物质欲望的人在外表上发生了变化，但内心并没有。

许多人虽然身体已经60岁，但心灵却像12岁的孩子经历情感青春期。

*佛教的主要教义是无常。*

也就是说，没有任何东西可以被心灵孤立并保持静止。

这包括思想、感受、情绪、人、地点以及我们贴上标签并期望永远保持不变的一切。

与意识形态、教条或其他静态的心理结构认同，是我们痛苦的根源。**当它们受到威胁时，我们的幸福也会受到影响。**

思考在关键时刻并不坏，而且是必要的。

但当没有受到控制时，强迫性的思想可以创造一个无法逃脱的混乱的个人现实。

*当人静止时，他们的心灵却想要移动。*

*当人移动时，他们的心灵却想要静止。*

没有有意识的练习，心灵会抓住过去或未来，试图使不永恒的事物永恒。

这是导致不快乐的原因，但不应被妖魔化，因为这违背了宗旨。

当你感到不快乐时，观察那种感觉。深入其中。让它指向“月亮”并呈现生活试图教给你的教训。从那个低谷，深入即将到来的高峰。

创造这种快乐心态的活动是私人的。你必须观察、假设、实验、失败、学习，即使你已经达到了客观的成功，也要重复这个永无止境的过程。

因为即使那样也不是永恒的。

## 实验与强度季节

![图片](../Images/0a526f6123f82a6a9e30b17aadb7fc4d.png)

秘诀是让心灵与生活的无常相结合，*而不是*试图让生活与心灵想要使之永恒的欲望相结合。

当然，我们大多数人都有经历过强度季节：

+   “宇宙对齐”，一切顺利，你行动在一个完美的想法上。

+   你对自己的目标有清晰的认识，达到新的技能水平，并享受这个过程。

+   你有一种使命感。感觉一天的时间似乎不够用，但你不会为此改变任何事情。

纯粹进步的感觉是无与伦比的。

没有干扰，绝对清晰，放下一块、两块、三块砖来构建你对未来的愿景。

用更多导致这种体验的活动来填满我们的日子是有意义的。

生活的强度季节并不是问题。

这些事情在实验季节之后自然发生。

*这就是大多数人问题所在。*

他们没有把感觉迷失的可怕、消极和绝望的时期视为一个机会。

如果你能在这些斗争时期掌握你的心灵，那么消极就不再引起痛苦。

相反，你让自己准备好进入一个新的进步季节。

### 实验的目的

自我实验是解决你问题的唯一方法。

人们可以诊断并为你提供解决方案，但这忽视了那些提供解决方案的人的视角、目标和经验的差异。

让我们想象你感到精力低落。

你在社交媒体上看到一篇关于他们提供的饮食建议充满信心和信念的帖子（这也可以应用于资金少和特定商业模式的问题）。

你在寻找解决方案，他们告诉你素食主义将解决你所有的问题，从一个未开化的状态，你：

+   改善你的饮食

+   像法律一样遵循他们的建议

+   看到他们提到的结果

然后，你依附于那种饮食意识形态，成为一个先知。你将其他人通过不同方式获得的结果归因于素食主义，并谴责任何质疑你的人。

这显然是危险的，这是低意识的定义。

事实上，发生的事情与素食主义无关：

+   你吃了更多富含营养的食物

+   你的行动帮助你生存下来，保持你的素食身份

+   你通过遵循有纪律的营养计划获得了清晰（而不是混乱）。

在这种情况下，素食主义仅仅是一种整理你思维（持续的状态流）并让你遵循健康原则的方式（但你将其归因于一种方法）。

现在，如果你尝试一个月的素食主义，一个月的肉食主义，一个月的生酮饮食，一个月的灵活饮食，你会：

+   在饮食之间建立联系，揭示健康的基本原则（模式识别=多巴胺）。

+   选择并选择你喜欢的某些方法，这意味着它们将带来可持续的结果。

+   精炼一个适合你个性的系统，达到完美。

然后，一旦健康的生活方式变得毫不费力，你就可以为你的财务、社交生活、浪漫关系、精神追求或生活的任何其他领域做同样的事情。

如果你正在经历关系中的问题，可以雇佣治疗师、观看YouTube大师的视频、参加静修，并尝试不同的选择，直到找到正确的解决方案。

如果你的业务没有增长，你可以购买课程、雇佣教练、测试新的软件、放宽视野，并制定新的策略...

总是有解决问题的方法，而当你抓住一个解决方案（可能无法长期解决问题）时，你就剥夺了自己这种能力。

自我实验是开放心态的过程，是创造而不是寻找你生活中无尽问题的解决方案。

**对创作者和独立创业者的话：**这就是你如何在竞争激烈的市场中创造一个独特、整体的产品或服务，并使其自我推广。

我们不需要更多的专家，我们需要的是能够创建新系统并伴随细致教育的整体综合者。

### 向内实验

你感到迷茫，因为你没有系统来维持一个基线水平的清晰。

当你进入生活的新篇章时，清晰度的创造应该是首要任务。

仅此一项就能使意识有序到能够感觉到对未来的控制感。

要向内探索，你将主要尝试非物质追求的技术：

哲学、精神或情商。

你可以尝试的技术包括但不限于：

+   阅读

+   冥想

+   观察

+   自我探究

+   思考

+   长时间的宁静散步

这些活动让你专注于当下。

就像所有技能一样，你通过渐进式超负荷来提高。

也就是说，通过逐渐增加挑战的难度，这样你不会感到无聊或焦虑。无聊是因为挑战太低，焦虑是因为挑战太高。

向内实验的目标是将这些活动之一纳入你的日常或每周常规。

如果你想保持这些追求带来的好处，实践必须是持续的、终身的，并且是在掌握的道路上。

### 向外实验

你感到迷茫的原因是你失去了与一个目的、目标或日常习惯的联系，这些习惯为你的生活带来了秩序。

当你结束生活中的一个章节时，你可能不再想要之前的东西。这是好事。这意味着你在成长。

但是，没有人愿意长时间陷入迷茫的时期，而且只有一条出路：

+   保持对新的目的的警觉，不要让生活的教训从你身边溜走。

+   设定一个目标，这样你就可以通过这个目标来感知生活中的情况（再次强调，模式识别=多巴胺=收集拼图碎片直到图像清晰）

+   识别你与目标之间的差距，并注意阻碍你的问题。

+   与目标一致地学习、构建和实验。

你现在生活中最紧迫的问题是什么？

是自尊心？金钱？缺乏满足感？

坐下来思考这个想法，进行沉思。

现在，你能否现在就着手解决那个问题，还是你必须解决另一个问题？

如果你缺乏资金，没有时间开始创业，那么显然你不应该开始创业。你需要尝试预算或找一份新工作。

如果你正在寻找男女朋友，你是否在财务、身体和精神上能够维持一段关系？如果不能，先解决其他问题。

一旦你确定了一个可以追求的目标，就可以研究并实验那些帮助你实现它的技术。

如果你需要关于建立高性能生活方式、开始基于兴趣的创业和加入一个支持性社区（你可能会在那里遇到你的下一个商业伙伴）的指导和教育，请查看[现代精通](https://modernmastery.com/letter) – 读者可以以5美元的价格加入。

### 维持新的基准

随着你通过实验的季节的进展，清晰度开始达到高潮。

不久，最后一块拼图就会到位，你将进入一个充满活力的季节，在生活的相应领域达到新的高度。

就像在商业收入中达到月度新高一样。

但是，保持警觉，这个季节将会结束。

就像健身中的增肌和减脂一样，你只能“增肌”一段时间，否则你会积累过多的脂肪。你开始感到昏昏欲睡、行动缓慢，不确定是否应该继续努力。

当到了“减脂”的时候，如果没有一个合适的系统，你可能会失去你获得的大部分肌肉。

我们不希望这样。

当你翻到最后一页时，努力区分信号和噪音。

加倍努力于那些将维持你获得的新“肌肉”基准的技术、任务和行动。

整个这个过程：

+   通过实验来解决你的问题

+   在生活中新的一章中获得清晰

+   向新的高度努力，并系统化过程以维持结果

这是生活的本质。

事实上，这只是科学方法的一个另一个视角。
