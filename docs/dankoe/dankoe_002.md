# 我如何寻找病毒性想法（5 分钟阅读习惯）

> 原文：[`thedankoe.com/letters/how-i-read-books-for-maximum-intelligence-5-minute-habit/`](https://thedankoe.com/letters/how-i-read-books-for-maximum-intelligence-5-minute-habit/)

书籍让你探索别人解读过的世界。

书籍让你尝试不同的世界观，以新的视角看待情况。

书籍呈现了巨大、强大且经得起时间考验的思想，这些思想可以改变你的思维方式。通过改变你的思维方式，你改变你的行为方式，这决定了你未来的全部。

很长一段时间，我会强迫自己读完整本书。

当然，我从未完成过，因为它们在一段时间后无法吸引我的注意力。

这些年，我变得越来越忙。

如果我不说“不”，商业项目可能会无限期地进行。

我有健身和自我实现的目标，这要求我每天都要抽出一段时间。

我正在投入更多时间于约会和人际关系，因为我推迟了它们以构建我生活的其他领域。

我理解阅读的力量，但现在它只是一个事后考虑。

当然，我可以为好书安排 30 分钟的时间，但我似乎从未能保持这个习惯。这很遗憾，因为阅读习惯能让你思维活跃，充满想法。当无数解决方案在脑海中时，很难保持停滞不前。

在这封信中，我想向您介绍我用来：

+   将新颖的信息融入我的日常生活中（以扩展我的思维并保持我对生活的热情）

+   捕捉我可以用来激发我自己的写作、表达和理解的想法。

+   每天花费不到 5 分钟的时间，这样我才能保持我的阅读习惯

让我们深入探讨。

## 设定一个目标

人类是目标导向的生物。

我们从个人的角度解读信息、对话和事件。

在那个观点的根源处，有一系列围绕那些目标或问题的目标、问题和信念。

对于大多数人来说，这种观点是无意识的且坚定不移的。

也就是说，这种观点是通过社会和文化条件反射形成的。

大多数人都是自动化的，目标是上大学、找工作，并在 65 岁退休。

因此，他们通过那个视角看待世界。

他们只*看到*有助于他们实现那些目标的事物。

任何与他们观点不一致的信息都视而不见。

意味着 99%的生活本身都未被看见。

如果你想要体验超越表面生活的深度和机会：

+   为你的未来设定一个有意义的目标

+   提醒自己那个目标，让它留在你的脑海中

+   注意你是如何以新的视角解读生活的，你的决定开始与你的目标一致

当你读书时，你将开始注意到解决你问题的潜在方案，无论书籍的主题是什么。

## 寻找想法

我们的祖先会寻找食物和资源。

但我们并不生活在一个那样的世界。

我们不必担心他们所面临的威胁，我们的生存需求已经超越了存在的物理层面，进入了精神层面。

相反，伟大而难忘的人会寻找想法。

“狩猎”意味着从我们在线上线下可获取的信息网络中获取有意义的多巴胺。

你不会在已知中狩猎。你探索未知以进行新颖的发现。

+   阅读那些挑战你的书籍。

+   阅读那些激发你好奇心书籍。

+   阅读那些你一直推迟阅读的书籍。

那些让你将一只脚踏入未知领域的事情。

这样，词语就会穿过大脑左右半球的中间。

右半球控制你所知道的东西，而左半球控制未知的东西。

当信息穿过“中间”，即你理解边缘时，你的神经系统会向你发出信号，这是一个有意义的事件。一个正在形成的创新发现。一个反映在你心理上的宇宙冲突，当它得到解决时，会孕育出创造性的涌现。在我们的例子中，是一个新的想法、洞察或认识。

有些书籍之所以引人入胜，是因为你意识到你生活的潜力。这些想法作为原材料，你可以用来雕塑你自编故事的音乐插图。

## 创造力是玩心理乐高。

你会忘记。

我们都是。

因此，在方便的笔记本或手机应用中创建一个常见的地方。

你需要一个地方来捕捉那些最吸引你的想法。

这是第一步。

第二步是停止阅读。

严肃地说，一旦你找到一个新颖的想法，就放下这本书。

你读书不是为了记住每一个单词。你读书是为了找到一个改变生活的想法，并将其付诸实践。

一旦你找到并记录下新颖的想法：

+   **思考这个想法的实用性。** 它以多少种方式与你的人生相关？

+   **将其应用于你的目标。** 这个想法如何帮助你改善心智、身体或业务？

+   **尝试将其转化为你自己的。** 你如何从你的角度重新表述这个想法？

如果你没有想法发展系统，[你可以在这里下载我的 7 天创意挑战，以供参考。](https://7daystogeniusideas.com)

同时，这里有你可以尝试将其转化为你自己的想法的方法：

思考你的目标。

思考你当前的问题。

思考你的个人经历。

思考你是谁，你将成为谁。

然后，从你对上述任何一点的思考中构建这个想法。

我前几天从约翰·沃尔夫冈·冯·歌德那里得到了这个绝妙的主意：

> 音乐是液态建筑；建筑是凝固的音乐。

如果我思考我的写作和出版书籍的目标，我的重新措辞的想法可能是这样的：

> 书籍是纸雕塑。写作是粘土，而表达是凿子。

我确信它可以变得更好或更完善，但你会惊讶于随着时间的推移，你存储的想法有多么强大。

**教训：**

专注于一个想法比被成千上万的想法淹没要好。深入思考。沉浸其中。沉思。并避免陷入收集没有实际应用到你生活中表面想法的精神自慰循环。

阅读五分钟。

*如果某个想法没有吸引你，就选择另一本书或跳到另一个章节。*

## 使用法则

![](img/f32b44ab4223d6e0198c6ec32352ee0c.png)

大多数人不需要更多的建议。

他们需要更强大的想法来塑造他们的世界观和框架他们的决策。

这样他们就能做出更好的选择，自己弄清楚，并从试错引起的进步中获得多巴胺。

问题在于，要达到强大的想法，你必须扩展你的思维，直到理解它们。

要理解，想法必须与现实相遇。

你必须将你收集到的想法付诸实践，并通过直接经验来巩固它们。

只有这样，你才能[提升你的思维水平](https://thedankoe.com/life-is-hard-get-used-to-it-the-way-to-mental-mastery/)，进入下一个想法阶段。

就像健身但使用心理能力。

渐进式超负荷仍然适用。

沉浸在表面层的“如何做”建议中就像一生都在举 5 磅的哑铃，并期望建立心理肌肉。

为了让你的想法与现实接触：

+   写作或日记，深入一个想法的深处

+   从一个常见问题、克服它的步骤或故事开始，以及这样做所发生的转变（这是人类行为、讲故事和内容营销的结构）

+   与你写作产生的想法坐下来，并对其进行精炼

当然，我最喜欢的将这种心理扩展过程融入其中的方式是通过[通过创作者经济为人类做出贡献](https://thedankoe.com/the-future-of-the-creator-economy-my-bold-prediction/)。

人类在概念层面上生存，这种现象不仅限于当有人挑战你的信仰时感到受到威胁。

我们通过思想的传递来寻求复制。

在负面方面，意识形态者和狂热者强迫你接受他们的信仰。

在积极方面，创作者分享他们的个人经历，提高集体的福祉（并且通过这样做谋生）。

通过撰写长篇内容，如通讯，你实际上：

+   解构你在寻找中收集到的想法

+   通过原创写作重新连接这些点

+   将你的写作分解成众多想法

+   重复这个过程，并提高你的心理能力

在超实用术语中：写一个通讯或 Twitter 帖子，将其分解成推文，然后将你的推文转换回通讯或帖子。

这就是我的内容策略的精髓。

我将我的通讯、帖子、推文发布到所有平台（并使用我的[2 小时作家](https://2hourwriter.com)系统来完成）。

不仅这能防止你的思维堵塞并停留在某个琐碎的问题上，而且在这个过程中你还会影响到数百万人。

你一生中会认识 1000 个人。

他们一生中会知道 1000 个人。

10 亿人就是两个能将你的想法传播开去的人。

是的，这个数学是正确的（即使我在数学上失败了）。

1000 x 1000 = 1,000,000 人，每个人在他们的一生中都有机会接触到 1000 个人 = 1,000,000,000

（顺便说一句，这个想法是从彼得森那里得到的）。

不要低估你的潜在影响。

设定目标。

寻找灵感。

用心理积木来玩耍。

分享你的想法。
