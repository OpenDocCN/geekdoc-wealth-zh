# 如果你感到迷茫——了解生活的这三个阶段

> [如果你感到迷茫——了解生活的这三个阶段](https://thedankoe.com/letters/if-you-feel-lost-understand-these-3-phases-of-life/)

我总觉得我的生活是分阶段的。在一个阶段，我感到迷茫。在另一个阶段，我被钉在办公桌前。充满能量。深深陷入心流状态。构建我的未来。在另一个阶段，我只是在例行公事，但感到快乐。

愚蠢的是，我在高中或大学时从未有过这种感觉。当老师和雇主引导我的注意力时。主权生活的周期性只有在我要管理我的目标时才会发挥作用。当我必须掌握我的注意力，以确定我想要在生活中去往何方时。

迷茫的阶段是最具破坏性的，但也是最富有启发性的。在黑暗中摸索，希望能找到一根蜡烛。找到了。看着它熄灭。找到另一根，希望它能引导你走向下一个，在黑暗再次降临之前。

通过意识和改进的意图，这些周期变得不那么具有破坏性。

我只能想象其他人必须感到多么迷茫。不知道如何利用那些感到迷茫的时期——实际上，你只是在为下一轮的热潮做准备。

就像桑拿房一样。不要与之抗争。*让它净化你。*

## 进步的三个驱动力

进步的三个驱动力

当你承诺根据你的目标来构建生活——而不是别人的——有三个进步的驱动力。这意味着如果你参与其中——或者是有意识的——这些阶段，你正在朝着更好的生活迈进。

这些阶段对每个人来说都会发生，但对那些没有做自己喜欢的事情的人来说最具破坏性。对那些跟随别人的道路而不是开辟自己的道路的人来说。从童年起就被训练去重复一个未来的生活。

理解这些阶段将帮助你确定你在旅程中的位置，你可以做什么，以及如何做出准确的决策。

**进步的三个驱动力是：**

1.  强度

1.  一致性

1.  好奇心

这些周期循环出现。阶段。一个导致另一个。它们是生活的必然周期。起伏。无为。推拉。秘诀是放手。让这些周期按照它们的方式发生。随波逐流。与之抗争只会让你陷入那个生活阶段，直到你学会它试图教授你的教训。

强度用于构建。一致性用于维持。好奇心用于探索。

这些不仅是生活周期，还有年、月、周和日的周期。我们将在本文中深入探讨宏观视角。同时，你能否开始意识到你日常生活中的这些周期？

*这是你如何导航它们的：*

### 探索未知

> “当你走到你所有光亮的边缘，并迈出第一步进入未知的黑暗中，你必须相信两件事之一会发生。会有一个坚固的东西让你站立，或者你会被教导如何飞翔。” – 帕特里克·奥弗顿

我们将从好奇阶段开始，因为许多人对此有误解。这是人们一生中都会陷入的阶段。有无限的改进空间，*但需要保持与舒适区边缘50英尺的距离*。

**生活是一系列实验**。没有人——没有单个灵魂——已经或将会完全弄清楚。未知是一个充满无限机会的土地。然而，大多数人将其视为一个充满巨大风险的地方。

旁白：风险，就像其他一切一样，是一个标签。一个虚构的概念，暗示着回报。顶尖的1%的表演者将风险视为它本来的样子，*生活的一部分*。而99%的低表现者将风险视为它所做的事情，摧毁了他们熟悉的小气泡。*令人恐惧*。

好奇阶段很棘手。它看起来不像探索。它看起来像你在太空中漂浮。没有日常的例行公事来让你扎根。生活中没有方向感。介于“我想做这件事直到生命的尽头！”和“谁会做这件事直到生命的尽头？”之间。

我们必须采取探险者的心态。在丛林中砍伐，寻找改变生命的新发现。探索未知是“向墙上扔东西，看看什么会粘住”的真实世界体现。

要轻松地导航这个阶段，需要直觉、屈服和接受——但有一些更有效地导航的方法。试试这个：

+   列出你拥有的20个兴趣

+   记录相关的兴趣（交叉点）

+   沉浸在你所感兴趣的信息中

+   阅读书籍，收听播客，关注该领域的领导者

+   连接这些点，理解现实世界的实用性

+   教授你所学的知识，以更好地理解它

到最后，你将更深入地理解你必须前进的方向。你的神经生物学——兴奋的感觉——将暗示激情。在未知之旅中不断教育自己。教育将意识之光投射在未知地图上。在你的时间上投资免费内容。[在你的金钱上投资你渴望成为的人的教诲](https://join.modernmastery.co)（课程或辅导）。

好奇阶段是为了探索、实验和发现新事物。这些新发现将引导你走向你想要走的道路——而不是别人希望你走的道路。这需要你与你的直觉——你的神经生物学——保持联系，让好奇心成为你的内在驱动力。这是一个测试新兴趣、评估它们的实用性，并最终连接点的时期。这导致巨大的清晰度，将你带入下一个阶段。

### 快速构建一个更好的未来

强度阶段是构建与你的好奇心相关的项目。当清晰度到来时——早晨、夜晚和白天都模糊不清，试图尽快找到时间完成任务。没有什么能阻止你开辟自己的道路。专注是无缝的。心流似乎持续一整天。你甚至不需要你宝贵的待办事项应用来保持你的进度。

这个阶段发生在好奇心阶段之后。它发生在一切变得清晰的时候。你有了新的发现，有了清晰的洞察力，并制定了一个将你推向新高度的策略。你已经在你的未来角色中感受到了自己。你是你，你想要成为的人，合并为一个人。你就是那个人。你只是在等待时间赶上你。

> 强度可以澄清。它不仅创造了动力，还创造了你需要感受到的摩擦或满足感的压力。 —— 马库斯·巴克敏斯特

建设阶段是将好奇心转化为狂热的过程。它是为了酿造让你进入心流状态的神经化学鸡尾酒。它是为了在培养自主性和精通的同时，培养激情和目标。这是你构建未来的地方。个人自由。自我实现。这个阶段最适合用于具有高潜在盈利能力的项目。它可以开始作为一个爱好——并且可以保持为一个爱好——但如果你想继续这条路，自主性是必要的。

在这个阶段的后期，如果你想保持进度，你必须系统化。

### 一致性累积

在强度阶段的结束时，你将达到一个新的基准。无论是心理的、身体的、精神的还是财务的，无论是什么。如果你不维持这个基准，你将再次达到一个新的低点。再次，这是一个循环。你不必再次陷入好奇心阶段，只是为了再次感到迷茫。

一致性最好通过系统来导航。这些系统应该围绕基本原理旋转。产生增长的杠杆式行动。将这些基本原理融入你的日常惯例中，如果你能的话，自动化它们，并将你不喜欢做的事情委托出去。

再次，拿出笔和纸，或者下载我的计划：

+   记录你的杠杆式目标导向任务

+   为执行这些任务安排时间块

+   避免分心，进入深度工作模式

+   在你精力最旺盛的时候做这件事

+   经常反思，将不喜欢的技术或VA工作委托出去

通过建立适当的生活和商业系统，你可以通过2-4小时的专注工作来维持全职收入。无代码技术进步已经减少了独立创业者的工作量。数字产品和服务创造者拥有许多人都无法理解的奢侈。你只能通过自动化或委托任务来减少工作时间。员工是那些被委托任务的人。

一旦你感到稳定，你就可以保持你的进度，并跳入另一个好奇心阶段。探索未知，自我教育找到你的道路，让新的发现将你带入强度循环。（如果你在已经建立的基础上构建，那将是一个加分项。）

### 宏观与微观阶段

一旦你意识到了这些阶段，它们就会变得更加可预测。你可以更有效地导航它们。你可以预测将要发生的事情，减轻你对未来的担忧。

事实上，*这些阶段不仅存在于宏观层面。它们也存在于微观层面。你可以利用对这些阶段的理解来制定一个可持续的日常安排。例如，我： 

+   早晨前两小时进行建设（强度）

+   之后保持2小时的规律（一致性）

+   在下午进行探索（好奇心）

[这已经融入了我的日程安排中。](https://thedankoe.com/the-3-habit-morning-routine-to-end-procrastination/) 一个精心设计的生活方式。在宏观阶段中包含微观阶段。这是在分割你的时间以及知道何时转向平衡。
