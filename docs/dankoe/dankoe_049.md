# 如何通过模仿走向成功（而不是平庸）

> [https://thedankoe.com/letters/how-to-copy-your-way-to-success-instead-of-mediocrity/](https://thedankoe.com/letters/how-to-copy-your-way-to-success-instead-of-mediocrity/)

当我阅读一本新书时，我知道我是否想继续阅读它，这取决于**一件事**：

我能感觉到我大脑的某些区域亮了起来。页面上的文字深入我的灵魂，给了我一直在寻找的答案。我的好奇心将我拉得更深，思想如洪水般涌入我的脑海，让我熬夜到很晚（现在通常是晚上9点左右）。我**必须**继续阅读。我**必须**继续深入这个兔子洞。我**必须**了解这个话题的任何和所有信息。我必须获取所有观点，以便我能够巧妙地引导未来的对话。

如果一本书不能让我产生这种感觉，我就把它放在书架上，留待以后阅读。我为什么要浪费时间阅读那些不感兴趣的东西呢？我们的大脑里有一个奖励机制，对吧？是为了帮助我们朝着积极的生活方向前进？是为了给我们信号和噪音的反馈？许多人用它来沉溺于多巴胺？电子游戏、梗图、自慰，以及任何其他追求好奇心的替代品。

真正的问题是；这有什么用？追求我的兴趣能带来现实世界的价值吗？我的好奇心探索会有目的吗？除了让我的大脑感觉良好和扩展我的知识深度之外？

让我们探索一下。

## 你是一个天生的模仿者（这是设计好的）

从时间的开始（对你来说，就是出生）你就一直在模仿——或者说复制——你周围的人。否则你怎么能生存下来？

作为孩子，你模仿你的父母。你学会了说话、走路、吃饭，以及（希望）正确地使用洗手间。随着你年龄的增长，文化开始发挥作用。学校系统为每个人提供了相同的课程。你的朋友们将他们的观点投射到你身上（他们从父母那里模仿来的）。随着你在学校体系中前进，这种神奇化学物质——多巴胺——开始帮助区分信号和噪音。你会模仿那些点亮你大脑一部分的事情。

这不是一个问题。问题在于当**依恋**加入进来时，问题才会显现。这些我们无意识地采纳的特征、偏见、信仰、价值观和观点构成了我们的身份。

大脑讨厌混乱。它需要一个目标层次来获得清晰和繁荣。无论你是否意识到，你的大脑都会构建一个目标层次。文化和企业生活方式都有大和小目标，这些目标指导着人们的行为。上学（目标1），找工作（目标2），退休（目标3）——但这要付出什么代价？

大脑需要抓住一个故事来理解并使其有意义。大脑通过了解过去和未来发生的事情来理解世界。我们必须了解心理上的过去发生的事情，以便对未来的确定性有一个*感觉*。没有确定性——焦虑、压力和熵的产物就会接管。

> 为了克服当代生活的焦虑和抑郁，个人必须达到独立于社会环境的程度，以至于他们不再仅仅根据其奖励和惩罚来做出反应。为了实现这种自主性，一个人必须学会为自己提供奖励。她必须发展出在无论外部环境如何的情况下找到乐趣和目标的能力。 —— 米哈伊·契克森米哈伊

如果有人没有意识到这个模仿过程，他们就会附着在自己的身份上——那一层帮助他们的心灵理解他们是谁的故事。当他们的身份看到威胁时，他们的生存雷达就会启动。他们的世界观受到了威胁。心灵将这种身份的威胁等同于潜在的死亡。人类是唯一会这样做的人。如果没有人类来称呼它为电话，那它还是电话吗？如果没有人类来称呼我为丹，我还是丹吗？如果没有信念来理解这一切——我甚至存在吗？

这是我推荐写下并完善你的愿景的另一个原因。拥有一个与你想过的生活相一致、由你创建的目标层级很重要。清晰的愿景使你的日常行动变得清晰。它给你的心灵带来秩序。如果这个目标层级是内在激励的（参见：[好奇心、激情、目标](https://youtu.be/3kSMxMlbktc)、价值观等），它将消除你生活中的所有压力和焦虑。一开始你可能不会做对。这是一个终身的过程，包括实验、测试和迭代。写下来。即使你认为它第一次不会奏效。

这就是如何使无意识变得有意识。

> 能够随意集中注意力、对干扰视而不见、集中精力直到实现目标，而不是更长的时间，这是意识掌控者的标志。能够这样做的人通常享受着正常的生活。 —— 米哈伊·契克森米哈伊

**成功是一场智能模仿的游戏**。模仿可能会对我们的未来造成伤害——但模仿本身是无法逃避的。我们在模仿我们周围的一切。我们处理我们接触到的信息，并且它被储存在我们大脑的某个地方。

为了最大限度地利用这种生存机制，我们需要意识到我们在模仿什么，以及我们在模仿谁。如果我们暴露自己给一个人，他们的信息是什么？他们是在推广消极和限制性的信念，你将接受吗？或者他们是在推广成长、教育和向你展示如何朝着积极的生活方向前进？他们的目标层级是什么，它与我们的目标层级是否一致？

再次强调，这是好奇心发挥作用的地方。追求你的好奇心是你智能模仿的方式。如何？好奇心的第一步是“倾听”——或者感受——你头脑中的多巴胺水平。当你分心时，这几乎是不可能的。沉浸在廉价刺激的多巴胺中。遵循别人为你指定的道路。等等。

## 你是一个视角容器

![图片](../Images/e41ef2bb2d6add241b6fd074f8383470.png)

智能模仿

一切都是视角。现实是视角。*把视角想象成相机，把感知想象成镜头*。两者结合构成了你的世界观。最终，每个人都在试图捍卫他们的视角，基于他们的世界观。你在评论区、新闻中、与伴侣的争论中、到处都能看到这一点。这是一种必要的实践；能够放大你的相机，看到大局，并在世界上移动你的相机。*这就是你如何停止对无关紧要的事情做出反应*。你改变你的视角，以便为自己设定有利的感觉。找到有利感觉（对情况的解释）的观察点，这是你在生活中做出积极决策的方式。

### **关于做出更好选择的辅助课程：**

特伦斯在社交媒体上滚动浏览。他看到了一条让他生气的帖子。这种反应是由多年来的模仿、条件反射和编程造成的，这些信念和观点来自他人的视角。

特伦斯留下了一条刻薄的评论，感受到了多巴胺的喷涌，并享受了那种会反过来咬他的即时满足感。评论在那里，让更多的人生气，并引发了一连串的多巴胺成瘾者强化他们头脑中糟糕的视角。

现在，*想象一下简单暂停和重新定位的力量*。如果特伦斯能够转换到一个能够激发好奇心的视角，他就能从最初让他生气的事情中汲取积极的教训。他不是假设整个故事，而是接受对方有不同的视角。这个视角由他们的自身愿景、目标、经验、价值观以及所有其他不可能假设的东西组成。

通过简单的暂停，特伦斯本可以打开他的心扉，抓住一个将把他推向全新生活方向的机会。

### **回到人类作为视角容器（以及它与提高你的价值的关系）：**

注意力是新的货币。新奇性是吸引注意力的东西。新颖的视角是让人们愿意听你而不是别人的东西。我们在《科伊信件》中经常谈论创业，但这远远超出了那个范畴。新颖、积极的视角是让人们记住你的东西。帮助人们理解——在他们脑海中连接点——会在他们的脑海中留下你的话语。

新颖的视角使人们愿意雇佣你而不是其他人。它们是真实性的基础——给出你独特、内在且未经筛选的视角。新颖、深思熟虑的视角是看清事物本质的关键——这样你就可以利用它们来获得优势。

***如果你掌握了视角的艺术，你就掌握了心智游戏。***

## 智能模仿过程

这不仅适用于创业。追求好奇心、在线发布内容、建立数字杠杆是每个人都能做的事情。雇主正在积极招聘那些拥有磨砺过的数字简历的人。个人品牌是一个你可以应用和测试你所学一切的项目（让你学得更快）。而且正如我们上周讨论的，世界正在回归信任、简单性和个性。 

通过保持真实，你建立**信任**。通过以使信息有意义的方式提炼信息，你表达**简单性**。通过对你生活的结果承担全部责任，你实践**个性**。

你通过追求好奇心，提炼你所学的知识，并通过你独特的视角在网上发表内容来做所有这些事情。（这就是我现在正在做的事情）。

如果你想要开始建立忠实的观众群，数字杠杆，并开始有机会主动来找你，这里是我的智能模仿过程，用于浓缩和理解知识。

### **步骤1) 选择3-5个你渴望成为的“导师”。**

*你成为你所消费的东西，所以按照你想要成为的人去消费。*

通过写下3-5个改变你**观点**的人的名字，使这个过程变得简单。

你推荐给每个人的最喜欢的书是什么？写下作者的名字。

有哪些YouTube红人、播客或博主让你无法停止观看？每次他们发布内容时都让你“惊叹不已”的那个人？

你关注哪些社交媒体账号，让你想，“他们是怎么说得这么完美的？”

因为……不是每个人都这么想。只有你。这些“导师”的组合和**联系**是你创造自己独特领域、视角和内容的方式，这些内容使你与众不同。

再次阅读你最喜欢的书的第一章——我保证你的大脑会发光。

### **步骤2) 不要消费。研究。**

消费内容意味着你没有做任何事情。你没有**使用**它（参见：普遍的“使用”法则）。

当你采用研究者的**视角**时，你会把一切都视为一个实验。你所研究的每一件事都变得有用。你测试想法，通过连接这些想法来制定理论，确定这些理论在现实世界中的效果，然后将它们转化为对他人有结果的复现过程（这就是人们以个人品牌出售的东西，也是我提出这个过程的由来）；)

[我有一个名为“如果你想赚更多钱就停止学习”的播客，它更深入地探讨了这一点。](https://open.spotify.com/episode/5DdgoAufTrFF6tzd4B6zXy?si=f1e8f435c0004c1a)

### **步骤 3) 与自己接触并写下它。**

如果你想让这一切发挥作用，你不能被现代的干扰所淹没。你必须为多巴胺提供空间，以便它能够准确地辨别信号与噪音。

当你根据你的好奇心进行研究时，不要做笔记。等待吸引你注意力的新颖观点。*感受*兴奋。*感受*你的注意力集中，并更加仔细地倾听（或阅读）。

这是你笔记中写下的内容。

### **步骤 4) 解构与提炼。**

这是发展独特视角的关键。在实验中，你分解某物是为了了解其组成部分。如果你不这样做，你将只有对该主题的肤浅理解（社交媒体上的大多数人都是这样做的）。肤浅的理解阻止你做出更深层次的联系，这些联系让你与众不同。

[我的免费 7 天创意挑战为你提供了这个过程的一个模板（以及如何深入使用它的指南）。](https://7daystogeniusideas.com/)

*以下是你根据笔记中所写的内容将要执行的操作：*

+   它相关的主题

+   简要总结你对它的理解。

+   与该主题相关联的常见问题。

+   克服这些问题的常见好处。

+   人们解决这些问题时可能遇到的反对意见或限制性信念。

+   你与该主题相关的个人经历和故事

+   克服该问题的逐步过程

现在，在写下这些之后，你将总结*所有*这些内容，形成一个一句话的大想法。这将表明你可以从你独特的视角创造出深刻、引用式的说法。

“好吧，丹，但我能做些什么来处理所有这些信息？”

从本质上讲，你刚刚从一个新颖、有说服力的视角为文章、视频、帖子、多条推文、产品以及着陆页创建了一个大纲。是的。我真的把所有这些都浓缩成了一个过程。

如果你具备营销、销售或内容心理学的初步教育——我知道你能看到它的力量。

当你这样做几个月后，你会构建一个充满天才想法的网络，这些想法具有现实世界的实用性。吸引他人的想法。可以货币化的想法。为新的想法开辟空间——加速人类进化。是的，想法正是推动人类进化前进的精确事物。它们层层叠叠。记住……故事。一件事必须发生在另一件事之前——一个绝妙的主意可以真正改变人类的方向……或者只是你生活的方向。
