# 如何在20多岁时成为百万富翁

> 原文：[https://thedankoe.com/letters/how-to-become-a-millionaire-in-your-20s/](https://thedankoe.com/letters/how-to-become-a-millionaire-in-your-20s/)

我一直讨厌为这么少的事情付出这么多。

在凌晨3点醒来，为了在最低工资的工作上上早班。

每天上学8小时，心里清楚自己花这么多时间只是为了在将来和普通人一样多或少地赚钱。

为别人的公司全力以赴，不管我多么努力工作，收入都一样。

我很早就有了这种心态。

当我向别人解释这件事时，他们告诉我我小时候非常观察入微。

我可以看到，大多数人都不快乐，懒散，对几乎他们所做的每一个决定都感到压力。尤其是财务决定。

因此，我的注意力转向了相反的方向。我在青少年晚期的大部分时间都在剖析是什么让成功人士成功。

我一直梦想着成为“百万富翁”。

这听起来如此诱人。

那个标签……“百万富翁”。

地位。自由。压力的减少。

但现在，作为一个已经实现这一目标的人，我可以告诉你，你关于成为百万富翁的所有想法都是真实的。

我不会轻视这一点。当我回想起我为了支付在7个室友合租的房子中我那份微不足道的300美元租金而挣扎的时候，我很感激。不必担心你在99%的购买上花费的事情，这是一件既幸运又不幸的事情。

当然，仅仅因为你成为了一个百万富翁，并不意味着你的生活整天都是完美的。发展你生活的财务领域并不意味着你已经发展了心理、身体、社会和精神领域。

事实上，如果你环顾四周，对于只关注金钱的大多数人来说，生活相当糟糕。

所以，在我们开始之前，请记住这一点。

我将给你确切的方法，让你在20多岁时成为百万富翁。

但要明白，赚钱只会解决你的金钱问题。

而且你大多数的金钱问题实际上并不是金钱问题。它们是心理问题。除非你致力于成为最高版本的自己，否则金钱不会像你想象的那样对你产生很大影响。

## 学校里没有教给你的事

如果你正在阅读这篇文章，你是由那些生活在旧经济中并受到其条件影响的人抚养长大的。

“把你的储蓄投资到401k和股市中，”妈妈说。

“房地产是99%的人成为百万富翁的方式，”斯科茨代尔的一个随意的富有老人说。

“要节俭。节省每一分钱。收集优惠券。不要做出冲动的决定。”一个无聊的生活没有人愿意过的人说。

“去学校学习一些你不在乎的东西以获得工作，”几乎所有不理解我们的公立学校系统是从普鲁士军事国家模式中采纳的——反映了给予奴隶的教育——并且是为了让人们的思想狭隘，依赖主导的社会系统，同时阻止一般化、探索和构建独特的愿景。

不，谢谢。

真的，我们正在经历第二次文艺复兴。

新富们明白，在一个任何人都可以学习和创造任何东西的世界里，思想是新的石油（如果你一直在关注独立黑客，你就知道非编码者可以用新的AI工具在周末构建MVP应用程序）。

他们明白，新一代人在思想空间中积累财富，而不是在物理空间中（物理空间仍然很重要，但现在它只对那些已经建立现金流的人开放）。

数字空间是无限的。物理空间是有限的。

关于这个时代，人们通过发布表情包和知识成为百万富翁，你需要了解以下3件事：

### 1) 数字景观

对于我来说，这需要一段时间才能理解。

我被培养成认为我应该预算每一分钱，并且“慢慢致富”。

但这在我心中从未感到舒服。

多亏了互联网，我残酷地意识到确实有人们在做他们喜欢的事情来赚钱。我每天都消费他们的内容。这对我来说已经足够证明，我可以通过一切可能的方式尝试实现那种现实。

在我们深入细节之前，这是你需要理解的东西：

**1) 如果你想在除了工作之外赚钱，*你*需要创建和分发产品。**

我会一直重复这一点，直到每个人和他们的母亲都意识到他们有多么依赖主导范式来生存，因为他们没有在离开学校系统后（或之前）继续学习。

如果你不去创造一个可以销售的产品，你将一直为别人销售产品。

你现在之所以能获得任何形式的收入，仅仅是因为你在一家公司中扮演了一个**特定**的角色，该公司通过销售产品或服务来赚取这些收入。

换句话说，如果你想获得自由，首先，你需要一个企业，其次，你需要一个产品，人们可以自愿用他们的钱来交换该产品中封装的价值。

简而言之，创建一些你可以出售或生存的东西，那些压力和情绪，将像由公司决定的无控制的海浪。

**2) 互联网使任何拥有wifi连接的人都能接触到商业和教育。**

如果你正在阅读这篇文章：

+   你可以在3-6个月内学习任何东西（如果你着迷的话，我用了2-3周的时间学会了制作[这个视频](https://youtu.be/hN-ITerTpLg)）。

+   你可以写关于你的兴趣，并吸引观众。

+   你可以用零钱作为产品或服务来教授你的兴趣

+   或者，你可以用一项有市场价值的技能自由职业，直到你感到需要将产品化和教学。

如果你想要成为百万富翁，你可以走你父母和老师教你的漫长道路，但他们中的大多数并不是百万富翁，所以你为什么要听他们的话？

我今天使用的每一个技能都是从课程、随机的一连串文章、好奇心驱使的探索和试错中学到的。

我赚的每一分钱都直接受我所创造的东西、看到它的人数以及这些人是否关心的影响。

如果你不在互联网上学习和构建，我恳求你看看你实际上在做什么。

将你的一天细分到每一分钟。

你是否在做那些能帮助你实现目标的事情？

或者你是在等待，希望如果足够地滚动社交媒体，它某天会神奇地出现？

### 2) 劳动价值论不起作用

你可以在任何事情上努力工作，但这并不意味着这对人类进步有用。

这是辛勤工作的错觉。

你可以投入4年的工作去获得学位。

你可以投入10年的工作去攀登企业阶梯。

而且你得到的报酬远远低于你想要的。

因此，与其将你的未来掌握在自己手中，你却抱怨和哀叹。

“我应得的工资更多！”

“我辛勤工作了14年，这就是我得到的回报？”

“我几乎没有时间陪伴家人。我没有足够的钱去度假。我在黑暗的隧道中辛勤劳作，看不到任何希望。”

世界上的抱怨者缺少一个关键的拼图：

劳动价值论是说你应该得到你做的工作的报酬。

感觉自己像跳过了圈圈。

感觉到自己应得某些东西。

但现实并非如此。

货币是价值的单位。

价值是衡量人们对你所做的事情关心程度的一个指标。

你的价值 = 你解决的问题的规模，你创造的解决方案的结果，以及你让人们关心你的创作的能力。

如果你对自己的收入不满意，可能到了残酷地诚实地审视你对世界贡献的时候了。

根据一个人工作的数量来支付报酬是没有意义的。根据一个人解决问题的水平来支付报酬是有意义的。

为什么？

我可以努力工作一年去写作成为作家，但一年并不足以证明10万美元的报酬是合理的。很多时候，这种写作可能被认为比做汉堡的最低工资工人价值更低。这不是*你做什么*，而是*你怎么做，以及你为谁做*。

你能赚多少钱取决于其他人对你那一年所做的事情的关心程度。也就是说，你的创作如何帮助他们解决问题，改变他们的生活，并为人类进步带来进步。

抱怨收入不足不会让你得到报酬。事实上，这可能会让你被解雇或忽视。或者被你接触到的任何人所憎恨，因为他们会想，“好吧……你打算*做什么*来解决它？”

唯一的选择是亲自处理事情。

### 3) 被动收入 VS 现金流

对于99%的人来说，被动收入只是一个空想。

被动收入是通过投资来自现金流业务的超额和丰富活跃收入来建立的，就像我在[数字经济学](https://digitaleconomics.school)中教授的那样。

大多数人不可能通过每年在股市或退休基金中投资几万来超越通货膨胀。

你需要超额。

你要么需要一份顶尖1%的工作（这就像进入NBA一样，普通人做到这样的可能性几乎为零），要么你需要建立一个你的收入完全依赖于你的技能获取、个人成长和持续迭代能力的企业。

现金流业务归结为简单的控制论和数学。

**控制论**是指一个系统自我调节以实现目标。

换句话说，当你下定决心追求一个目标，听取反馈并根据这些反馈调整你的行动时，你将实现这个目标。控制论说明了低智能系统（或心智）是那些选择放弃而不是调整方向的人。它们不够聪明，无法意识到任何问题都可以通过调整他们追求的目标来解决或转移。

这样，要大胆思考。要考虑数字化。要考虑无限。给你的思维留出连接点子的空间。

你每天的时间是有限的。

因此，你将花费与建立任何互联网业务（几乎零启动成本和收入上限仅取决于你的创造力）一样多的时间来建立一个实体咖啡店（限于几十万美元）。

**数学**是对系统本身及其所需行动的简单理解。

如果你想在1年内赚100万美元，那就把它分解开来。

$1,000,000 / 12 = $83,333 每月

$83,333 / 30 = $2,777 每天一

现在有几个方法可以做到这一点。

+   每天卖出18个价值$150的产品

+   每天卖出111个价值$25的产品

+   每隔一天吸引一个价值$5000的客户

+   每隔4天吸引一个价值$10,000的客户

+   结合两者。每周1-2个客户和每天几件产品销售。

这本身就能决定你大部分的决策。

如果你选择客户路线，你可能需要一个团队（或者将服务产品化到你可以自己处理履行的程度）。

如果你选择产品路线，你需要大量的流量。

假设你擅长社交媒体。

你可以通过YouTube视频获得50-100,000次观看，或者每月在社交媒体上获得1-5百万次曝光。

当然，偶尔会有观看次数或曝光量异常高的现象。

要在价值$150的产品上每天卖出18件，你需要将720人引导到产品页面，转化率为2.5%。

如果你不能通过每月 4-8 个视频或一个活跃的社交媒体账户做到这一点，那就是技能问题。变得更好。你控制着你能获得多少观看次数以及有多少点击来自那。

但“如何”做这件事呢？

## 在 20 岁时拉动 2 个杠杆，赚取 100 万美元

你需要一个产品（或服务）。

你需要分销（一个受众）。

除此之外，你的全部成功都取决于实验和迭代。试错。

你可以通过像 [2 Hour Writer](https://2hourwriter.com) 这样的课程来补充分销，以及 [Mental Monetization](https://mentalmonetization.com) 这样的课程来补充产品，但课程只是你在更少出错的方向上实验的其他过程。除非你是完美的客户形象，否则课程不会考虑你的意识状态和先前经验，因为你是你唯一能接触到自己思想的人。

### 1) 产品：赚取独立收入的唯一途径

为了避免混淆，我使用“产品”这个词来定义你出售的任何东西。那也可以是一个服务。

最近在 Kortex 社区，有人提出了一个好问题：

*“我对自己的产品销售能力没有信心。我有结果，但我没有使用框架或特定的步骤，我只是做了，所以我不确定我如何能帮助别人。”*

这可以适用于健身、心理健康、编程、任何东西。

许多人认为他们不能卖产品，因为他们已经取得了结果，但他们没有使用特定的过程、程序或框架来做那件事。

你没有意识到的是，这正是开始产品的完美地方。

任何人如何创建一个产品？

你不只是抄袭别人的框架。

你自己创建一个。自己创建需要从实验中获得客户的真实世界反馈。

你创建一个 MVP（最小可行产品）。

你获得第一批客户或免费用户。

你听取反馈并衡量结果。

你实施反馈并尝试使你的产品变得更好。

然后，一旦你有了稳固的结果，你可以更努力地营销，并自信你能做到这一点。

这与初学者自由职业者获得客户是一样的。他们以相对较低的价格做，直到他们对能力非常有信心。他们获得了*真实世界的经验*，而不是他们学到的理论和他们自己建立的个人项目。

我的产品哲学是：

+   建立一个你之前使用过但更好的产品。

+   解决你自己的问题并出售解决方案。

+   建立一个你想要但不存在的产品。

大多数人应该从第一个开始，进步到第二个，一旦他们有了现金流，就转向第三个。

但你必须意识到你可以创建任何你想要的产品。

你在亚马逊上能找到多少个计划表？一个亿。

你也可以通过以下方式创建一个：

+   购买 3-5 个计划表

+   使用它们并注意哪些地方不顺畅

+   制定你自己的计划表

+   在一周或两周内使用它并改进它

然后，将其打包并出售。

倾听客户反馈，让它变得更好。

简单。现在你有了**能力**赚钱，但你还需要一件东西：

### 2) 分销：如何在未来生存

> 建立分销，然后建立你想要的任何东西。 – 杰克·巴彻

分销是你将你**建立的内容**展示给**关心的人**的潜力。

现在我们已经建立了一个产品，我们需要把它展示给人们。

最容易让人们看到你的方式是在互联网上。

互联网的前端是媒体。

传统媒体正在消亡，所以你有几个选择：

**有三种类型的分销：**

*建立、借用和购买*。

你可以通过互联网内容扩大受众来**建立**分销渠道。然后，你可以（并且应该）通过“去平台化”你的受众来更进一步。

你通过将你的受众引导到一个社区或新闻通讯来**去平台化**。这些可以是免费的或付费的。

社交媒体平台可以随时消失。你的账户可能会被暂停，你可能会被取消，或者你可能只是足够讨厌社交媒体以至于想要离开。

但是，他们不能从你那里拿走离线社区或电子邮件列表。而且，这些是你最忠实的读者，所以无论如何，你都应该给他们提供最多的价值和关注。

你可以通过利用其他人的受众、社区或新闻通讯**借用**分销。

你可以成为他们的播客的嘉宾并推广你的产品。

或者，你的想法可以如此独特，以至于他们在播客中提到你，即使你不在那里。他们可以在他们的帖子、新闻通讯中给予你的想法认可，有时甚至是一本完整的书籍。

这是你最终想要玩的游戏。让你的想法如此具有传染性，以至于你的分销渠道无需任何人工努力就能持续增长。

让你的想法在人们的脑海中免费存在。

你可以通过许多不同的方式**购买**分销。

付费广告的各种选项是显而易见的，但不是最有力的。

你可以在播客、新闻通讯和YouTube视频上购买赞助位置。

你还可以购买付费分享，如转发、Instagram故事，甚至在YouTube视频上的“观看下一个”位置。对于那些想知道的人，我最近的YouTube增长是突如其来的（但我并不反对尝试它）。

最后一个，付费增长，是我最喜欢的，因为它通常会导致大量积极参与的人跟随你。你不需要告诉他们跟随你，他们是因为你的内容好而跟随。这会随着时间的推移而累积，并与“建立”的分销交叉。

对此的常见反对意见是“作弊”，这通常是由那些不在创作者游戏中的人提出的。如果你的品牌、内容或产品很糟糕，你不会增长、销售或被视为任何类型的权威。

购买的分销会放大你在游戏中已经拥有的结果、想法和时间。

有一样东西太多人忘记了：

你的受众不仅仅是**你的**受众。

*网络效应*非常强大。

我不仅有权访问我的追随者。

我可以间接接触到每个跟随我的人的追随者——其中许多人本身就是创作者或拥有10万+受众的人。例如，戈登·拉姆齐和约翰·塞纳在关注我。我不认为会有什么结果，但谁知道呢。

我还可以接触到那些通过数字或口头分享我内容的人。

此外，我可以接触到所有跟随我的人的“追随者”。当然，程度较小。

重点是，如果（1）我保持一致（2）我优先考虑那些能留下深刻印象的创意想法，我的想法就有可能触及互联网上的45亿人中的任何一个。

我有320万粉丝，我可以间接接触到10倍于潜在流量的流量。同样，对于较小的粉丝群也是如此，他们只是没有意识到这一点。一个“微不足道”的1000粉丝受众群实际上可以彻底改变你的生活，因为那1000人中的任何一个人可能有100万的粉丝。

### 我最大的商业教训：模仿，然后创新

无论你是构建产品还是编写内容以建立分销，你只需要一个启发式方法来指导你的决策：

*模仿，然后创新*。

售卖已经热销的产品。

写出已经表现良好的内容。

添加一些个人经验和观点，使其独特。

把你的手伸入已经流动的注意力中。

学习什么有效。学习原则。在构建或写作时，打开另一个产品或写作。注意产品的*结构*而不是内容。你并没有抄袭任何一个单词。你是在模仿有效的东西的本质。你从这些事物中抽象出教训，并试图复制它。

一旦你看到结果——如果你正确地、有意识地这样做，你将会——然后尝试创造你自己的东西。

这就是这一点的全部。
