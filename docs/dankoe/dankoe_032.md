# 你为什么总是感到不知所措

> 原文：[https://thedankoe.com/letters/why-you-always-feel-overwhelmed/](https://thedankoe.com/letters/why-you-always-feel-overwhelmed/)

我注意到每个都存在**一个模式**：

+   成功的企业家

+   开明的灵性大师

+   幸福的已婚夫妇

+   破纪录的运动员

什么模式？

**专注**。

我听说每个人——从上面的每个类别中——都提到“专注”是他们成功、幸福和进步的关键。

在阅读了关于性能、幸福、商业、灵性、哲学以及与生活享受相关的任何书籍之后……我认为我找到了答案。

享受最大生活乐趣的关键（附带你在任何事业上的成功。）

> 一个人可以让自己快乐，或者痛苦，无论“外面”实际上发生了什么，只需改变意识的内涵。 —— 米哈伊·契克森米哈伊

*专注*是受控的意识。

或者，关注点在于控制“意识的内涵”。

当你的意识内容，你关注的那些事物，可以以积极的方式理解时；你的生活就会变得积极。

意味着，如果你努力从任何给定情况的有利角度寻求清晰、无偏见的认识，你会感到难以置信。

这也引出了Leo from Actualized.org提出的观点，即*一切都是意识的幻象*。例如：一个原子是意识的幻象。这是我们意识到了它时为了理解它而创建的标签。这有点形而上学，但有趣的是建立这种联系。

通过专注于单个“意识的幻象”——而不是分散在10个幻象之间——我们可以过上快乐、健康和充实的生活。

## 意识的内涵

我想让你把你的大脑想象成一个超级计算机。

你处理信息的能力取决于一些因素，但我想把注意力引到你的RAM上。

RAM——或“随机存取存储器”——是计算机最重要的部分之一，它决定了性能。

你运行的不同程序、打开的浏览器标签以及你正在运行的性能要求，使用的RAM越多，你的性能就会越慢。

这与你的关注点，你意识中的注意力所持有的东西，没有区别。

人类每秒可以处理126位信息。这加起来在你的 lifetime 中是1850亿位信息。

大多数人同时运行着多个高需求程序，这正在消耗他们有限的创造性能量。

+   对后悔的过去错误的思考

+   对有压力的未来事件的思考

+   想要逃避那些想法的饥饿和娱乐欲望

+   一种内在的呼喊，要打破他们习惯的生活方式

+   需要完成的多优先级任务清单

+   他们本应完成但忘记的任务的开放循环

列表可以一直继续下去。

现代人的注意力默认分散到无限的方向。我们生活在压力和接近疾病的状态中。我们不是以专注和无忧无虑的心态活在当下，而是相反。生活在由分散的注意力创造的虚假现实中。

当我们在意识的 内容中持有太多的过去或未来时，就会产生混乱。心智倾向于无序。如果不加以控制，我们将失去对生活的控制感，这会导致快乐。这是通过有控制的意识实现的。

即，专注。

## 8步重获你的专注力

要打破我们机械化的生活方式，我们首先必须意识到它对我们生活的影响。

从混乱到有序，我们必须将注意力回归中心，并选择值得我们关注的事物。

我们使用的RAM越少——我们的专注力越集中——生活就越愉快。秩序回归，对我们生活的控制感带来对生活的深刻满足。

### 1) 选择你生活中一个落后的领域

我们的力量取决于我们最薄弱的环节。落后的最远领域拥有最易摘取的果实。这些是造成最多问题的领域。

思考，我生活的哪个领域一直持续地带来问题，如果这些问题得到解决，你的生活质量将飞跃。

金钱？约会？心理健康？自尊？社交生活？健康和能量水平？散步，沉思，并选择一个领域，当它得到解决时，结果会在你生活的其他领域产生连锁反应。

为了本文的目的，以及许多人的共同问题，让我们谈谈***金钱***作为一个落后的领域。

### 2) 对基本原理有清晰的了解

我想要区分：

+   基础，原则或基本原理

+   战术，技巧和技术细节。

基本原理驱动结果。技术细节可以进一步推动结果，但只有在基本原理被掌握、系统化和习惯化之后。

基本原理移动杠杆。技术细节提供了陷阱、压倒和创造不存在问题的机会。

对于我们的关注领域——金钱，你需要流量和一个提议。这是得到你想要的东西的普遍原则。

如果你没有流量：

+   一个社交媒体受众

+   支付广告或推广

+   一个社区或团体

+   有机搜索结果

+   直接接触（例如，冷电子邮件）

没有人会看到你的提议。

如果你没有一份令人信服的提议：

+   数字或实体产品

+   自由职业或咨询服务

+   群体课程或在线辅导

+   会员提议或社区

你将没有地方发送流量。

就像你在约会市场上。

如果你是一个高价值的“提议”，但将自己置于“交通”繁忙的环境中，祝你好运。

### 3) 创建一个基于能力的目标

专注于看起来更好、获得更多关注者、获得最多的互动、拥有最热门的合作伙伴等，并没有什么错。这些都有它们的使用场景，并且可能有益。我建议你关注这些，但不要把它们作为你的主要焦点。

**问题：**基于虚荣的目标比基于能力的目标更容易超出我们的控制。

这些步骤不是线性的，除非你了解你每天可以采取的杠杆移动行动，否则很难创建一个基于能力的目标。

在我们这个例子中，如果我想建立受众（流量）并销售数字产品（提供）来赚钱——你需要执行以下杠杆：

+   每天写3篇短篇帖子（比如推文）

+   每周写一篇长篇帖子（新闻通讯/博客）

+   将那篇长帖子浓缩成一篇中等形式的帖子（比如LinkedIn帖子的线程）

+   用30-60分钟为你的数字产品制定课程

在这种情况下，你可以设定一个目标，每天写1000个高质量的字，并评估你的写作（而不是由此产生的结果）。

过程胜于结果。

说到写作，这是我们将在[《2小时作家》](https://2hourwriter.com)中讨论的一切。

### 4) 开始后自我教育

基础知识会帮助你开始，但你需要学习如何将这些基础知识应用到你的具体情况中。

太多人陷入了“教程地狱”，试图尽可能多地吸收信息，却从未开始。

而不是：

+   快速开始并快速失败

+   确定导致你失败的原因

+   寻找具体信息来克服它

+   再次失败，并让经验随着时间的推移而累积

在开始之前不要试图弄清楚所有的事情。这被称为心理自慰。

### 5) 每天留出30-60分钟

深度工作不仅适用于工作，也适用于你生活的每个领域。

当企业家想要在商业上取得进步时，他们会投入时间和精力来实现这一进步。

当已婚夫妇想要在他们的关系中取得进步时，他们会为对方留出时间。

当个人想要用他们的心智取得进步时，他们会花时间进行独处和意识工作——比如冥想。

*无干扰的时间必须被创造出来，这样在需要投入“深度工作”时，你的注意力就不会分散，无论生活领域的哪个方面。*

### 6) 沉浸在那个目标的内容中

对于有精神追求的人来说，这是“在场”。

对于其他人来说，这被称为“参与”并进入了一定程度的流动状态。

想想当你刷社交媒体时，你如何全神贯注地看手机。营销人员和科技公司有专门的团队致力于通过注意力捕捉来触发一种伪流程状态。

当你被手机吸引时，你的注意力就没有其他东西了。大多数人用这个来逃避现实，而不是挖掘现实本身。

如果你的目标与人际关系相关，练习让你的注意力完全集中在对方身上。他们的身体、动作、言语。使用你所有的感官去感受当下，并让它成为你意识的焦点。

### 7) 识别时刻

这些都不是轻而易举就能做到的。你会失败。事情就是这样。

一个干扰会进入你的意识，你的注意力会分散，你可能会被干扰引发的负面情绪所吸引。

只需一个识别时刻，你就能从这种状态中摆脱出来。

不论是抱怨艰苦的训练，还是因为你在与他们交谈之前为他们设定的身份而害怕接近新人。

暂停，认识到你在抗拒现实，然后让它流动。

你对现实的期望并不等同于现实。

它“不应该”是你“认为”它应该是的样子。

“它应该就是它现在的样子。”

### 8) 进行比喻性的练习

任何事物都是肌肉。

你的关注点、人际关系、商业、健身、精神实践以及任何类型的习惯。它们随着关注、充足的能量来支持表现和恢复，以及直接经验而成长，这样你就可以加倍关注对你**有效**的事情。

不要期望快速的结果。即使有类固醇（如课程、辅导、方法和系统），你仍然需要付出努力。你不能注射一些测试，坐在沙发上，就期望成为下一个罗尼·科尔曼。

将你生活的每个领域都当作一个科学项目来对待。

+   识别问题

+   设定期望的结果

+   测试不同的解决方案

+   找到一个适合你的方法

+   复制结果

从那里，继续构建肌肉，朝着你期望的结果前进。然后，你将经验丰富到足以知道你是否想要增重、减重或保持现状。

— 丹·科
