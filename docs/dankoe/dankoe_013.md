# 非需求型社交过程（如何建立高价值联系）

> [原文链接](https://thedankoe.com/letters/how-to-network-with-millionaires-and-build-a-name-for-yourself/)

我本质上是一个内向者。

当我去上大学的时候，我从来不是第一个开始交谈的人。我的朋友圈（我至今仍然有）偶然间遇到了我，然后我们相处得很好。

更别提在课堂上发言了。

或者参加“Networking”聚会去认识有价值的人。

我什么都不想要。

我只想坐在我的宿舍里，玩电子游戏，并经营我的副业。当时我的副业是一个健身 YouTube 频道。

这是在我意识到内向并不是反社会之前。社交是一种技能，而不是性格特征，然而人们却躲在“内向”的身份背后，以避免看到任何形式成功所必需的不可避免的不适。

那是大学时代，但现在我们处于一个新派对的中心……社交媒体。

我在过去尝试过多次为自己树立名声。

第一个是一个 YouTube 频道。

第二个是 Instagram 摄影和艺术页面（如果你一直滚动到我的 Instagram 底部，你可以找到这个页面）。

第三个是通过社交媒体上的多种营销策略建立多个机构和电子商务品牌。

然后我偶然发现了 Twitter。

是的，Twitter，所有地方中，我正是在这里进入了社交媒体的领域。

我之前在 Twitter 上活跃过，但我认为那是一个政治和半成品模因的地方，我可以在 Reddit 上找到这些。

但是，有一天，一个来自 Jose Rosado 的帖子出现在我的时间轴上。

我不记得具体是什么了，但那是一些关于自我提升的残酷真相。

我非常喜欢。

我关注了 Jose，并意识到他也谈论在线业务，这也是我感兴趣的事情。（这是[将你的兴趣融入更多内容](https://thedankoe.com/the-most-profitable-niche-is-you-how-to-create-your-niche/)而不是成为一个超细分市场机器人的另一个原因！）

通过关注他，其他账户开始出现在我的时间轴上，我还没意识到，我已经被高价值的“Money Twitter”帖子淹没。

我潜伏了大约 6 个月，没有参与其中。当时我在自由职业业务上取得了一些成功，所以这些账户提供的建议非常有帮助。

突然间，一个想法闪过我的脑海：

*“我可以写所有这些相同的推文。为什么我只是在坐着刷屏，而其他人显然在获得线索、销售，并享受写关于他们其他兴趣的事情？”*

那个问题改变了我生活的方向。

## 数字世界没有障碍

![](img/97704700e7ddea480870675ba40e0703.png)

在过去，通往顶峰的唯一途径是通过你的人脉。

你不能仅仅通过一条巧妙的信息私信一个名人，并让他们支付你四位数的服务费（这是一个过于理想化的情况）。

你必须去上学。

取得好成绩。

申请一所可能需要你改变地理位置的著名大学。

并且要足够“外向”以通过建立人脉的方式向上爬，直到你遇到一个完美的机会。

大机会仅限于那些努力工作以获得它们的人。

现在，机会无处不在。

你刚刚滑过的那个推文？他们需要一个视频编辑。

你在评论中找到的那个人？他们想学习如何快乐。

你从未看到的那个视频，因为它被 YouTube 的算法埋没了？它提供了如果坚持不懈就能成功的自由职业业务的详细步骤。

互联网让你能够接触到 4.9 亿使用它的人。

只需一篇内容被传播到正确的角落，让正确的人看到，他们就会联系你，并为你提供的价值付费。

当然，随着你建立起一个分享你内容的观众群体，这会变得更容易，所以我们不希望在开始时就依赖这一点。

人脉——尽管我讨厌这个词——是我数字成功的基础。

Jose Rosado 是我最早联系的人之一。

我们相处得很好，他帮助分享了我的内容，最终我投资了他的辅导计划（这是大约 3 年前的事情）。

现在，Jose 和我对在线业务有不同的看法，但这就是重点。我们每个人都有独特的路径可以分享，而且两者都包含了其他人可以从中学习的宝贵经验。

另一个例子是我的好朋友 Joey Justice，他也是我的最佳朋友之一（今年我还要飞越全国去看他一次）。

我当时和一群人一起试图扩大他们的品牌。我们制定了策略，分享彼此的内容，并共同努力提高我们想要收费的技能。

去年，我和 Dakota Robertson 以及 JK Molina 一起搬到了德克萨斯州。

我还遇到了像 Justin Welsh、Sahil Bloom 和 Dickie Bush 这样的重量级人物。

总的来说，可以说，与喜欢的人进行一致性和有针对性的建立人脉是进入社交媒体游戏（并维持增长）的方式。

如果你没有在建立人脉，那么你已经失败了。

问问任何真正有持续账户增长的创作者（并且没有陷入讨厌游戏的恶性循环），人脉是否重要。

如果他们不认为这是帮助他们增长的第一要务，那么不信任他们是明智的。

## 私信的力量

私信是你出现在互联网上任何人的面前的方式。

尤其是你还没有观众的时候。

自由职业者理解这一点是为了吸引客户，但在你决定转型时，你如何建立可利用的受众群体，以免你一无所有？

我看到过人们通过这种方式找到工作，推文因为很少的粉丝而走红，并从数百甚至数千名粉丝中获益。

人们通常分为两个阵营：

+   他们不理解持续直接消息给人们带来的纯粹力量和网络效应。

+   他们不理解如果你足够策略，你可以让大账户分享你的帖子。

直接消息是你作为新手（或你想要为内容吸引更多流量时）建立人脉、成长和吸引内容关注的方式。

但是，大多数人都不擅长直接消息。

严肃地说。这是一个问题。

永远不要，我再说一遍，永远不要期望有回应，如果你发送的只是：

+   你好或嘿

+   你是如何

+   你能回关我吗

+   没有先前联系的直接销售方案

+   一段太长，需要花费太多时间阅读的文字（尤其是如果句子没有间隔）

你必须策略性地处理直接消息。特别是如果你正在联系一个每小时收到 10+条直接消息的大账户，而且他们可能不会检查（他们通常只是快速浏览以查看是否有重要内容）。

再次，设身处地为对方着想，思考一下。你如何能在没有提供任何价值就要求回报的糟糕直接消息中脱颖而出？

## 7 步骤非依赖型人脉网络

这个过程最初是一个病毒性的推文线程，后来成为数字经济学小组的一个模块，然后被用来帮助人们在我的写作课程中吸引人们的注意（[**2 小时作家**](https://2hourwriter.com/)）。

这是基础性的，我们将在即将到来的独立创业者冲刺中使用这一点来保证增长。

知道如何与你想联系的人沟通和发送直接消息将为你打开有偿工作、社会杠杆和结识高层人士的机会。你可以使用这个策略来：

+   获取一个拥有 50,000+粉丝的账户来转发你的帖子，这将导致巨大的增长（任何其他社交平台分享你的帖子也是如此）。

+   有与你分享新增长策略的联系（迪基·布什前几天突然给我发了一个领英提示，我现在感到有必要回报这份好意）。

+   加入或创建智囊团，开始形成你的部落（你将在社交媒体旅程的大部分时间里与之共同成长）。

+   把你的名字告诉更多的人。了解你的人越多，送来的工作和潜在的联系就越多。**尽可能让更多的人看到你的名字。**

这里是非依赖型人脉网络的 7 个步骤：

### **1) 找到你想直接消息的人**

这不仅限于建立联系。对于有偿工作来说，这也是至关重要的。

在服务行业（如自由职业者和代理机构）中，一个普遍的主题是，如果他们与正在合作客户不产生共鸣，他们的生活就像在地狱里。

与以下这些人取得联系：

+   你受到

+   你愿意与他们合作

+   你愿意与他们策略性地合作

+   你看到了互惠互利的机会

当你刚开始时，你将不得不一步步攀登。首先从 DM 那些在你关注范围内的人开始。

如果你不知道该向谁 DM，我假设你还没有开始。消费者从与创作者截然不同的角度感知社交媒体内容。

一旦你有了受众和影响力，你就可以真正地向任何人伸出援手并获得积极的回应。

*你可以在哪里找到可以 DM 的人？*

你喜欢的账户的“关注”列表。

或者，在你的导师之一的回复区域（如果有人在回复，这意味着他们正在尝试成长和建立人脉）。

这些是两个主要领域，你可以找到人而不使用任何花哨的寻找工具。

### **2) 发送他们一份鼓舞人心的赞美**

找到一些你真正受到启发的他们的内容、作品或当前项目。发送给他们，并告诉他们这如何与你产生共鸣。

这需要你真正喜欢他们的工作。

人们喜欢赞美——根据互惠原则——会感到有义务回报这份小小的恩惠。

下面是一个关于管理情绪的推文示例：

> 丹，这条推文对我影响很大。我过去几天一直在经历这件事，这立刻给了我一些缓解。谢谢你。[添加你喜欢的帖子的链接]

简单到这种程度。

### **3) 对他们表现出兴趣**

如果他们没有回应，你可以尝试以同样的方式再次联系他们。

表现出兴趣是沟通 101——表现出兴趣让你变得有趣。

询问他们：

+   他们的目标

+   他们正在构建什么

+   他们从事的工作

这为你提供了给予价值的机会（即使你现在没有可以给予的价值）。

假设他们会用通用的“非常感谢”来回应。

现在，你可以从他们的个人资料中找到他们正在做什么，或者直接询问他们。

**如果你找到了他们正在做的事情：**

> 你对现代精通的下一步计划是什么？看到它疯狂增长，我很好奇你有什么计划。

**如果你找不到他们正在做什么：**

> 你现在在构建什么？有了那种内容，你肯定有更大的计划在筹备中。

### **4) 以价值为起点**

这是大多数人都会感到困惑的部分。

你的第一个选择是：

+   看看你能提供帮助的地方

+   发送可操作的技巧

+   发送你创建或记忆中的资源、系统或视频

在没有先给予任何东西的情况下，不要直接寻求交易。

如果你必须提出更多问题以便发送更好的可操作技巧或资源，那就做吧。

我所说的资源是指 YouTube 视频、文章或任何可能帮助他们——或者引起他们兴趣的东西。

如果你没有可以提供的东西，你可以：

+   发送可以帮助他们实现目标的资源

+   将他们与你网络中的其他人联系起来

+   只需继续进行愉快的对话并表现出兴趣

表明你愿意帮助他们。

### **5) 通过电话进行深入联系**

这一部分是可选的，但推荐。几天后，你可以直接跳到第 7 步，但我建议你最终还是要练习所有这些步骤。

进行“面对面”的互动是无价的。没有人能通过文字看到你，你的个性或你的举止。

当我开始的时候，我一直在与我想更好地了解的人进行 zoom 会议。这比你想象的要正常——而且当人们清楚你只是想了解他们时，他们通常很开放。

你也可以将对话转移到 Telegram 等地方，发送语音消息，让自己看起来更像真人。大多数社交账户都使用 Telegram 和 WhatsApp 群组来沟通和制定策略。

### **6) 跟进并提供价值**

记住他们的目标，并留意任何你可以发送给他们内容、资源或人。当你找到某物或某人时，就发送给他们。

> 我记得你告诉我你的[项目或目标]计划。我今天找到了这个，觉得你会觉得有帮助。

“这”可以是一个你找到的 YouTube 视频、文章、吸引物，或者一个你可能帮助他们的人。

### **7) 跟进并提出请求**

到目前为止，你已经建立了一个相当稳固的联系。你已经提供了价值，以至于他们准备好回报。从这里你可以：

+   请求他们加入一个智囊团

+   发送你投入了大量时间的其中一篇文章

+   向他们提出你任何具体的问题，而不必支付咨询费

如果你的计划是利用他们的受众来增长，确保写一篇他们愿意分享的文章。然后，你可以把它发给他们，并提到他们激发了这篇文章。

> 嘿，刚刚写完这个，觉得你会喜欢。这是受我们之前的对话启发的。

不要要求他们分享它。让他们以他们认为合适的方式与之互动。

即使你没有收到转发或分享，任何互动都会帮助更多人看到它——帮助你增长。（这不仅仅限于 Twitter。我们谈论的一切都可以转移到任何社交媒体上，它们都有相同的结构。）

你应该几乎每天都在联系人。如果你沉浸在你兴趣的环境中，你应该每天都能看到你想联系的有启发性的人。

**如果你想增加大号回复你的私信的机会——在他们联系你之前，先关注他们并与他们的内容互动几天，这样他们就会熟悉你的面孔和头像。**

### 8) 最后的提示

“丹，这不会花费很多时间吗？”

是的，显然。

如果要花 4 年时间去一个与你学位无关的工作，再花 10 年才能达到合理的薪水——建立自己的梦想需要时间。

这很讽刺，因为“不确定”的道路是解决方法中最快的，只是人们缺乏视角。

如果你明白，当你要对未来负责的时候，私信将是你生活的一部分，那么你现在就应该养成这个习惯。

写作也是一样。

如果你计划未来每天都写作，为什么现在不每天都写呢？是因为你不赚钱吗？所以你通过不免费写作来推迟获得更多的报酬？你真的对这一切认真吗？

把你的事情整理好。

正如你所见，我今天感觉很有活力。

太多咖啡了。

我退出。
