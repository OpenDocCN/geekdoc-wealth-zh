# 为什么选择一个细分市场是愚蠢的（对于聪明人来说）

> 原文：[https://thedankoe.com/letters/why-choosing-a-niche-is-stupid-for-intelligent-people/](https://thedankoe.com/letters/why-choosing-a-niche-is-stupid-for-intelligent-people/)

人类每秒可以处理126比特的信息（根据我的研究，谷歌说不同，哦好吧）。这加起来在我们的一生中总共是1850亿比特的信息（平均而言）。这些信息比特存储在我们的大脑的某个地方。通过意识，我们整理这些信息。也就是说，我们所意识到的信息会被存储。我们意识到的就是我们关注的东西。我们专注的东西。还有更多，但重点是我们在一生中能意识到的信息量是有限的。

我认为这是人类潜力中的一个关键因素——如果不是人类潜力的主要指标。虽然大多数人将他们15%的注意力用于洗澡、洗澡、打扮、烹饪和其他必需品——但大多数人也会闲逛、看电视、滚动表情包，用无用的活动麻木他们的思想。把他们的潜力浪费掉。

[上周我们讨论了肯·威尔伯的蜂巢理论](https://www.modernmastery.co/post/how-to-not-end-up-mediocre)。我们是如何在特定领域发展自己并经历3个发展阶段——在那个领域的技能变成艺术。肯提到了这一点，但上面的统计数据巩固了这一点。人类可以通过有意识的开发处理1850亿比特的信息。这限制了我们可以真正发展的领域数量。这也意味着其他人以同样的方式受到限制。每个人都会在生活的不同领域发展自己——根据许多哲学家和古代教师的说法，这就是幸福。不这样做会导致不幸福。

也就是说，拥有一个愿景、目标，并自觉地努力实现这些事情是有效的——“在过程中找到快乐。”

来自尼采：

> *“幸福是力量增加的感觉——阻力正在被克服。”*

来自齐克森米哈伊：

> *“一个能够控制意识的人的标志是能够随意集中注意力，对干扰视而不见，集中精力直到实现目标，而不是更长。而且能够这样做的人通常享受日常生活的正常过程。”*

如果我们将这个想法放在游戏（比如《魔兽世界》）的背景下——你会在最高等级时退出。你只能获得这么多经验。你只能在技能树上的特定特质中给予这么多“点”。如果你不玩故事或参与游戏的各个部分，你就无法达到最高等级（或潜力）。如果你把点分散得太薄，你最终会发现你的力量比那些明智分配点的人要小。

心态、专注、整理你的思绪、避免分心，以及其他暗示使用你有限的意识来发展意识和追求基于兴趣的目标层级的格言，是整体生活享受的秘诀之一。

这本身就是一个教训，我将在未来的信中谈到。你永远无法了解另一个人的心态。你永远无法理解他们是如何控制自己的意识的。你永远无法理解他们追求的目标层级，这些目标带给他们快乐。仅仅因为某人不锻炼、不创业、不改善健康，以及我谈论的所有这些事情，并不意味着他们不彻底快乐。那些跟随他们多巴胺诱导的素食主义探索兔子洞的人，可能因为他们的有序意识比我更快乐。判断是无用的。假设是无用的。期望是无用的。做自己认为最好的事情，让其他人做他们认为最好的事情。只有当其他人想要被触及时，才去教导。“智慧之口除了理解之耳外，对其他一无所言。”

如果我们要更进一步，我未来书中提到的三个支柱是专注、能量和经验。在我的看法中，专注就是有意识的注意。能量，无论你是否相信，都存在于你周围的每一件事物中。无论是积极的还是消极的。你关注的对象会赋予生命/激发你“关注”的任何事物。这种能量在你身上得到反映，决定了你的经历。根据齐克森米哈伊的观点，经验是基于你所意识到的信息存储。这种经历将成为未来一切的基础。

## “创作者经济”如何加速我们的发展

我前几天晚上和JK Molina聊天，我们讨论了在商业发展中创造力的必要性。可以这样想……通过具体知识——即课程、辅导、学徒制和其他在线教育形式——人们正在将大约10年的信息、经验和失败简化成一个简单且可操作的路线图，用于在生活的特定领域取得成功。书籍、文章、视频、播客等。[消费它们，但要有项目将想法联系起来，并引发模式识别。](https://open.spotify.com/episode/5DdgoAufTrFF6tzd4B6zXy?si=9b98ded372ce4bf8)

你可以说，这些信息已经被压缩或紧凑化。就像一个压缩文件。现在我们的头脑可以更快地处理过去需要10年的信息。不再是这些信息占据我们大约50亿位的意识注意力，而是只需要大约1000万位。这意味着我们正在“创造”解决问题的方案，突破个人和职业发展的限制，为他们腾出空间去做同样的事情。

我花了5年时间才在自由职业中取得成功。在我把我的第一本电子书打包并出售之后——其他人几个月内就看到了成功。现在他们可以在更短的时间内看到更多的成功，将1-3年的学习成果打包，并将他们的教诲传授下去。这导致特定领域的指数级发展。同时也允许在多个领域快速进步。计算机编程曾经非常专业化。现在有框架、教育和学习方法，可以在6个月内让你获得高薪的网页开发工作，而不需要学位。这只会继续加速——特别是对于科技行业的人和那些敢于追求创造性工作的人。

我的观点是，我们在压缩需要处理的信息量。得益于创作者和技术，我们可以比前辈们学到更多。我们不必专门研究一个领域。实际上，我们通过专注于一个领域极大地限制了自身。一个人通过专业化变得有价值，通过成为现代文艺复兴人变得不可替代。通过增强他们的专业化，通过学习互补领域来为创造力腾出空间。一个独立创业者可以繁荣发展，因为他们可以学习市场营销、销售、一项可销售的技能、电子邮件营销、创建内容、编辑视频、运行广告、构建产品、生成潜在客户等。通过在整个创作者生涯中压缩、系统化和产品化这一过程——你可以作为一个人的运营赚取100万至500万美元。当然，一个小团队可以增加更多的火力。

你看到了成为创作者的力量和必要性吗？你实际上是在“提升集体人类意识”的同时做着你热爱的事情。通过创造新颖的解决方案，迅速提高个人的特定领域发展。同时，由于教师比学生学到更多，你的自身发展甚至更加迅速。如果你持续地生产/创造内容、产品、服务和简化理解的方式，你将为进一步的创新腾出空间。

顺便说一句：我真心惊讶这份通讯中创意“融合”的多少。如果不是作为一个创作者，我无法做出这些联系，这正是我的观点。你必须经历发展阶段，利用可用的具体知识（自我教育），并成为一个创作者，通过价值生产来进一步增强、连接和简化这些知识。

## 专业化是一种虚假的神

记得上周我们讨论了“前理性谬误”吗？从前理性、理性和超理性阶段的发展阶段？本质上，这个梗对生活的每个方面都是真的吗？

在古代，"万能手"或文艺复兴时期的全才曾经带来成功。然后进入了专业化的时代。现在，对于那些看到机会并抓住它的人来说，文艺复兴时代正在回归——主要是在网络世界。

让我问你一个问题。我的细分市场是什么？我的专业是什么？你能给出一个直接的答案吗？如果你告诉我“在线商业”或“自我提升”或“哲学和精神”甚至“健身和健康”，我不会同意。我的品牌是我选择发展的领域的综合——一种相互关联。我的细分市场是“所有这些”，这是创造力的先导。不限制自己让我能够与那些在类似领域发展自己的人建立联系。不限制自己为模式识别打开了空间，让我能够连接点，对任何给定主题带来新的、新颖的视角。这让我和我的产品能够从商品的大海中脱颖而出。

现在，专业化在开始时很重要，并且可以长期发挥非常有益的作用。这是变得有价值与变得不可替代之间的区别。万能手不如精通一技之长。专注于一个领域，同时利用现代创新，让你能够快速在多个领域发展自己，这为你提供了*杠杆*。你在子领域的发展将只会增强你在主要领域发展的有效性。

## 现代商业的体验模型

世界正在转变。随着技术的进步，我们逐渐变得更加人性化（如果你明智地使用它）。这为更多的创造性追求和通过这种方式赚取巨额收入的能力打开了空间。

我的体验模型概念应运而生，以在创作者/注意力经济中繁荣发展。

你一生中处理的信息量决定了你的条件反射、编程、信念、偏见、感知、身份、个性、技能水平以及其他方面。你的注意力放在哪里以及如何放置（或操纵）决定了你的体验。你的体验将成为你创作的基石。那就是你的内容、产品和服务。你的潜力将成为你的品牌。那就是你的愿景、目标和价值观。没有人可以复制这个“细分市场”，你也不局限于你创造的产品。只要你能理解[常青技能](https://www.modernmastery.co/post/learn-these-skills-if-you-want-to-thrive-in-2022)，它们就会盈利。

你是否看到一切都在回归原点？个人成长 = 商业成长。

在内容创作方面，有一点需要思考：你能通过新奇的事物吸引人们的注意力，让他们意识到他们想要追求的特定兴趣，用紧凑而有价值的信息填满他们有限的意识，引导他们走向更好的生活吗？最重要的是，这是个人品牌或创作者的关键。

你塑造自己的愿景、目标和价值观——朝着它们努力，吸引追随者，引领他们走向更美好的生活。你的愿景是你和你追随者所期望的结果。你的目标是你在努力实现的目标。你的价值观是你和你追随者正在发展的领域。所有这些都构成了你的内容支柱。当是时候创造内容时，你依靠你的经验。随着你在这些领域的发展，你学到的教训会越来越有创意。你从你的错误、洞察和成功中学习，在你实现愿景的道路上开辟道路。这是一条非常充实的工作道路。人们跟随你，因为你是一个自信、真诚的领导者，引领他们走向更美好的生活。

创造力随之而来。我们如何将其融入生活和事业，以实现最大化和快速的发展？通过利用已经创造出的想法和具体知识。当你跟随和模仿其他领导者或励志原型时，你为你的愿景带来了清晰。就像那句俗语，“每个优秀的教练都有自己的教练”。
