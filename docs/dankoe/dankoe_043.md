## 创造或被创造

意识是现实的织物。[对意识进行排序的能力是21世纪最伟大的技能](https://thedankoe.com/how-to-create-a-life-of-meaning-money-impact-as-fast-as-humanly-possible/)。也就是说，能够：

+   从不确定性中创造确定性

+   从无意义中创造意义

+   从混乱中创造清晰

+   从分心中创造专注

+   从噪声中创造信号

+   从无中创造有

+   从失败中创造成功

+   从斗争中创造意义

+   从死亡中创造生命

+   从负面中创造正面

*那个项目符号列表是你版本成功的路径。*

自然是富有创造力的。你是富有创造力的。上帝是富有创造力的。宇宙是富有创造力的。

我们的潜意识存储着基于我们通过感官接触到的信息处理的信息。就像在用你的意识将点连接起来之前收集点一样。

创造是塑造、控制和安排意识（可以通过潜意识来挖掘）的过程。到那时，我们可以通过感官将这份礼物带入世界。

视觉、听觉、味觉、触觉、嗅觉，当然还有很少有人触及的第六感：

*意识。*

我们的潜意识储存了我们生命中某个时刻意识到的所有事物。

这是创造力的基础——从无限深处汲取、连接和解决问题的能力。

*创造力是从混乱中创造清晰*。

*创造 + 清晰 = 创造力*。

现在生活或“活在当下”并不局限于精神人士。它是过上快乐、有创造力和充满惊奇的生活的必要条件。当下是我们进入无限（或上帝之源、宇宙或任何其他我们无法用传统五感把握的智能）的方式。

当你的注意力是无意识和狭窄的，你就无法发挥你的创造力。

## 创造力并非天生就有价值

你可以花费六位数来获得艺术编织博士学位——但如果你不理解人类认为有价值的东西，你将永远无法依靠这个职业谋生。

有一些技能需要学习（如何扩展你的意识和觉悟），以便提高你创造力的感知价值。

而且这正是问题所在，价值是由感知决定的。你认为有价值的东西可能对下一个人来说只是一堆垃圾。

你必须研究人类理解的各个方面是绝对至关重要的：

+   讲故事

+   心理学

+   精神

+   哲学

+   营销

+   销售

+   品牌建设

+   说服

+   设计或视觉叙事

+   色彩理论

+   吸引注意力

+   自我（自我意识）

通过研究**其中之一**，你让自己接触到**所有**这些，并开辟通往未知领域的独特路径。它们都是同一件事，只是从不同的感知角度和通过独特的感官组合来感知。

人类理解是一块每个人都必须掌握的冰山。为什么？因为你需要理解如何传达你创造力的价值。如果你想赚钱，你需要玩金钱游戏。不要让你的自负阻碍你发展沟通技巧。如果你想要建立一个创造性的收入，这些技巧是必要的。

你越深入探索这个掌握的冰山，你的生活就会越顺利。从那些最实用、能帮助你传播价值的地方开始——**营销、品牌、说服和销售**。

这些都是将任何技能集货币化的燃料。如果你想能够销售一个创造性解决方案，你需要能够吸引人们到这个解决方案。换句话说，你需要流量和分发。你可以通过[建立受众、建立杠杆和建立其他分发网络](https://thedankoe.com/the-future-of-work-will-be-play/)，比如建立一个电子邮件列表来实现这一点。还有其他方法，比如付费广告和媒体购买——但一旦你有一个正在销售的创造性解决方案，这些方法才有意义。

*你如何学习这些技能？*

就像你对待其他事情一样——但大多数人都不擅长学习：

**1) 沉浸在你试图学习的技能相关的信息回声室中。**

在一个月内，只消费与你试图学习的相关内容。

购买一本概述该主题所有方面的书籍。购买一位在该领域是专家并且已经将信息浓缩到你需要知道的内容的课程。关注博客、播客、社交媒体账户以及你能接触到的一切。

无论你去哪里，你都希望接触到这个感兴趣主题的所有方面。*你想要获得意识——收集点——在你正在潜入的掌握冰山中。*

**2) 开始一个具有潜在盈利能力的个人项目。**

当我学习编码时，每一位教育者都会强调的一条建议是：

*摆脱教程地狱。*

当然，没有人会听这些建议。他们害怕跳入真正的未知。

当一个人开始后不知道接下来要做什么（有时甚至不知道如何开始）时，他们为什么要开始个人项目？

这正是你应该开始一个项目的原因。

因为它让你的*无意识*问题*变得明显*——**迅速**。

**3) 接受混乱，并理解任何值得拥有的东西都是不可预测的。**

> 通过追求你真正的好奇心和激情，而不是当前热门的东西，来找到特定知识。
> 
> — Naval (@naval) [2018年5月31日](https://twitter.com/naval/status/1002105081855016961?ref_src=twsrc%5Etfw)

通过探索未知来排序意识是找到特定知识的方法。在本文的语境中，特定知识是通过自我教育和构建个人项目收集和连接点而找到的创造性解决方案。

通过构建你自己的[与你要实现的愿景相关的目标层级](https://thedankoe.com/how-to-create-a-life-of-meaning-money-impact-as-fast-as-humanly-possible/)，你将经历[生活的阶段](https://thedankoe.com/if-you-feel-lost-understand-these-3-phases-of-life/)，这些阶段会教授99%的人无法接触到的教训。

从头开始是完全可以的。

如果你完全放弃非传统道路，这是不行的。

当你不确定时，请抽出一些时间来反思你通过未知探索有意义的生活。

## 3点商业模型

你将开始的第一项具有高潜在盈利能力的个人项目是在社交媒体上建立个人品牌。

这不再是可选的。你天生就是一个创造者。雇主是根据个人的公开简历（个人品牌）来招聘的。市场永远不会饱和。我们回到了公元前5世纪，权力回到了个人手中，而不是组织或公司手中。随着想法和信息的大量涌入，导致不确定性和焦虑 — 个人有责任构建自己的生活。与宗教、企业文化或其他组织为你生活中创造确定性相反。

> 最好的工作既不是由命令决定的，也不是由学位决定的。它们是自由市场中持续学习者的创造性表达。
> 
> — Naval (@naval) [2019年1月29日](https://twitter.com/naval/status/1090363813721862144?ref_src=twsrc%5Etfw)

为什么我如此坚决地要开始一个个人品牌？因为建立受众和利用是我为自己做的最伟大的事情。

*你可以在现实世界中练习任何有创造性的或可销售的技能。*

想学习文案和说服技巧吗？遵循上述快速学习3步，并通过撰写电子邮件和推文来练习。

想学习电子邮件营销、网页设计、销售或其他技能吗？建立一个电子邮件列表，建立一个个人网站，并通过DM与人联系进行销售通话，以增加你个人品牌的价值（并在过程中赚一些钱）。

我有点啰嗦，让我们来点实际的。以下是一个3点商业模型（针对个人品牌）：

**1) 开始一个每周通讯。**

为什么？因为大多数人都在玩短篇内容游戏，没有任何权威可以展示。

权威来自深度和专业知识。还有什么比将高质量通讯作为你业务的主要支柱来建立这种权威更好的方法呢？

我95%的销售额来自我的长篇内容。它建立了其他人无法竞争的信任。

**2) 开始一个Twitter账户。**

为什么？

+   如果你不想的话，不必展示你的脸。

+   你不必创造复杂的设计。

+   你不必花费数小时来创建内容。

你只需要一半的头脑和打字的能力。

任何人都可以写少于280个字符并按下“发送推文”。

Twitter的其他好处：

+   转发可能是最有力的分享功能——如果你能与他人建立联系，并通过转发利用他们的受众，你可以快速成长。

+   你可以在任何地方发布链接——这意味着你有更多的创意潜力来推广你的产品和通讯（以创造更多销售）。

+   你可以将你的每周通讯压缩成一个帖子——帖子是你快速增加粉丝的方式。

+   没有人关心专业性。如果我要赌一个将成为创作者经济中心的平台，那将是Twitter（只需看看人们如何将推文复制粘贴到其他平台，它们通常比花费数小时创作的帖子传播得更广）。

+   由于创建Twitter内容花费的时间非常少，你可以快速测试想法。如果一条推文表现良好，这意味着YouTube视频、产品、服务、播客剧集、文章或Instagram帖子也会表现良好。

进入这个领域没有任何障碍。

**3) 一个可以迭代的创意解决方案。**

开始赚钱最简单的方式就是开设与你专长相关的服务型业务。

你可以在一开始免费帮助人们，取得一些成果，然后开始为4次咨询通话收取500-1000美元。

如果你在网页开发方面很擅长？为开发者写文章，与他们建立联系，并收取指导他们构建项目的费用。

如果你在营销方面很擅长？为创作者写文章，与他们建立联系，并收取咨询费，帮助他们解决营销的任何方面。

列表还在继续——你可以用任何东西做这件事。我知道有些人做的是大麻疗法，有些人做的是在线泰拳教练。我也见过人们提供冲浪课程和评论。

这有一个进步的过程。

你开始收集将变成创意解决方案或系统的点，你可以以溢价的价格出售。

这就是大多数人出错的地方——他们想在没有任何创建独特系统以获得结果（并让你在人群中脱颖而出）的方法时，就直接进入销售5000美元的辅导套餐或30000美元/月的营销保留金。

我自己发现的一个独特解决方案的例子（我可以每月收取5000美元以上的费用）：

当每个人都试图销售销售漏斗、线索生成和其他现在极其常见的东西时——我在所有这些方面都有足够的经验和实践。

我最近创建了一个内容生态系统（我在[数字经济学](https://digitaleconomics.school)中教授），它能够吸引销售而不显得像是在销售。它允许你专注于你热爱的事情（制作高质量的内容），并简化了促销、社交媒体增长和权威建立。这比“以转化为中心的网站”、销售漏斗或线索生成要吸引人10倍——我已经将它们都融入了我的内容生态系统中。

（我并不是说以转化为中心的网站或销售漏斗是坏的。它们是阶梯的一部分。只是没有实践和迭代，你将无法收取你想要的溢价价格）。

逐步攀登你的职业阶梯。

+   专注于免费帮助人们理解他们的问题。

+   开始研究、撰写关于以及创造解决方案来解决那些问题。

+   使用这个解决方案开始以较低的价格提供咨询或自由职业服务。

+   继续改进你的解决方案，并取得更多成果

+   提高你的价格

+   在做所有这些的同时，你正在扩大你的听众群体——客户会开始向你靠近，你也有潜力推出一个产品

+   你可以将你创建的系统打包成一个数字产品，好好推广它，并开始完全掌控你的时间和精力

我可以继续说——但如果你还没有开始，这对你来说就没有意义。从比你落后2-3步的人的角度进行教学是一种常见的做法。如果我告诉你我现在做什么，而你甚至还没有开始——那只会让你一头雾水。

在游戏中，你必须达到特定的等级才能学习特定的特性，获得高级装备，并承担更具挑战性的任务。

*停止试图在等级10的时候获得等级100的结果。*

从今天开始深入未知。开始在世界真实环境中积累经验值。
