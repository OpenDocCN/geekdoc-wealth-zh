# 结束拖延的3习惯早晨例行程序

> 原文：[https://thedankoe.com/letters/the-3-habit-morning-routine-to-end-procrastination/](https://thedankoe.com/letters/the-3-habit-morning-routine-to-end-procrastination/)

我尝试过所有的早晨例行程序。

我已经慢慢地做了，冥想，阅读了30分钟，以及其他所有类似的事情。

我总是感觉我落后了。

对等待我的任务感到紧张和担忧。

这种能量会持续到我的整个白天——他们总是感觉不对劲。

经过多年的实验，我发现只需要三个简单的习惯。

有三件事情在脑海中，减少了起床的摩擦。

一旦我在早晨有了意图和目的——这产生了巨大的差异。

## 3习惯早晨例行程序

早晨是神圣的。它们为一天剩下的时间定下了基调。你的早晨例行程序应该具有产生能量的目的。

我们希望在整个白天都能保持这种能量。

### 去散步

我们都知道身体跟随心灵，但心灵也可以跟随身体。

它们相互影响。

如果你想要为你的剩余时间产生不可阻挡的动力，就从活动你的身体开始。

更好的是，在外面做。

> “清晨散步是一天中的祝福。” ——亨利·大卫·梭罗。

亨利确实发现了些什么。

不仅你完成了你的一些日常步数，而且你还让你的皮肤暴露在直接阳光下。

当你持续这样做时，这对健康的昼夜节律生物学至关重要。

（这甚至可以帮助你建立更一致的睡眠计划——也许你根本不是夜猫子！）

你可以走到咖啡馆，公寓楼周围，或者你喜欢的任何地方。

在前一天晚上在你的计划中写下这些，这样在早晨就会记住。

### 10分钟创意

让我们让动力的雪球变得更大。让我们让一天被认为是一个损失变得不可能。

我最喜欢的早晨习惯是我多年来一直在培养的。

它是日记、笔记和写作的结合。

这有助于你更快地学习，训练你的创造力肌肉，并拥有巨大的心理清晰度。

（如果你是创作者，这有助于你为你的工作提供无限的想法。）

这是你要做的。

**步骤1）获得灵感**

阅读一篇文章，眨眼，喜欢的引言，或书的一页。

你也可以在早晨散步时听有声书！

注意：这些应该与你的兴趣相符。

这将为你的剩余时间提供灵感。

**步骤2）注意你的兴奋**

多巴胺将信号与噪音分开。

当你在阅读或倾听时，写下吸引你注意并让你有点兴奋的内容。

我在手机上有一个运行笔记应用，我把所有这些写下来。

**步骤3）写一个3-5点的摘要**

你应该在这个总结中包含什么？

+   你如何理解这个主题的描述。

+   个人经验和故事

+   其他想到的想法

创作者的额外步骤

+   列出与主题相关的一个常见问题

+   列出解决那个问题的好处

+   列出克服它的步骤

现在，你已经有了一个博客、帖子、通讯或多条推文的提纲。

（这就是我写这篇文章的方式。）

**步骤4）一句话的大想法**

这就是创意发挥作用的地方。

从你之前写下的所有内容中，写出一个总结性的句子。

发挥创意。尽量用你最喜欢的引言的质量来写。

我们为什么要这样做？

+   这能让创意源泉涌动

+   它让你的大脑为模式识别做好准备

+   这会向你展示你能够原创且深刻

随着你写下更多的大想法，这只会越来越明显。

### 力量小时

到目前为止，我们已经花费了我们早晨大约20分钟的生产力时间。

我们的身体和心灵都准备好继续前进。接受挑战。进入心流状态。

接下来是我喜欢称之为的力量小时（不，不是那个饮酒游戏）。

> “幸福是力量增强的感觉——阻力正在被克服。”
> 
> —— 尼采

力量小时是一个1小时的时间块，专门用于构建。

没有干扰。一个优先事项。激光般的专注。

把它想象成每天自动提高1%。

你在构建什么？

个人自由。

+   经营你的副业

+   写一篇文章、帖子或书籍

+   创建社交媒体帖子

+   学习一项可销售的技能

+   联系潜在客户

你知道，那些你说要开始做的事情，但你还没有做…

我怎么知道该做什么？

+   列出你对未来的愿景

+   反向工程到10年目标

+   然后设定一个1年目标

+   然后设定月度目标

现在，将你的力量小时任务与那个月度目标对齐。

移动杠杆、关注基础、导致每天自动提高1%的行动。这就是我创建[The Power Planner](https://shop.thedankoe.com/planer)的原因。

你将结束你的力量小时，对未来感到积极。

在1小时内，你就在大多数人醒来之前赢得了这一天。

这种感觉是无与伦比的。

**总结：**

1.  去散步以活动身体并直接接受阳光。

1.  用10分钟训练你的创意肌肉。

1.  用1小时为自己构建一个更美好的未来。

如果持续这样做，6个月后你将回顾自己的进步并沉浸在其中。
