## 价值创造者的崛起

简而言之，价值创造者是这样的：

+   用他们的头脑而不是时间来赚钱（这样他们就可以作为一个[个人企业](https://thedankoe.com/letters/the-one-person-business-revisited-turn-yourself-into-a-business/)存在，而不限制他们的收入）。一个人的劳动无法扩展，但他们的创造力可以。

+   研究他们的痴迷，并通过写作、视频和其他互联网内容提炼他们的学习成果。

+   以数字产品（或如果你愿意，也可以是实体产品）的形式获得聚合知识和经验的报酬。

+   重视教育他们的受众（而不是发布梗图、开始戏剧或其他类似的事情）。

价值创造者是那些致力于他们的兴趣，将其作为他们一生的使命去探索，并以教育的形式提炼信息的人。

这就是我教授的这种个人企业的风格。

创建社交媒体内容，以你的名字建立分发。

[将你自己定位为细分市场](https://thedankoe.com/letters/the-most-profitable-niche-is-you-how-to-create-your-niche/)。这就是你销售的对象，也是你为谁创造内容。

建立一个你的过去自我可能会使用的[教育产品或服务](https://thedankoe.com/letters/you-have-100000-of-knowledge-trapped-in-your-brain/)（比如课程、辅导或家教提供）。

然后，当你达到你想要的任何数额（每年10万美元至300万美元……随着时间的推移）时，你可以建立你热衷于的其他任何事情。

通过成为你热衷于的技能和兴趣的专家，你积极地创建一个不断发展的教育体系。

政府无法，也不会培养一个课程，为个人提供导致旧经济解体的工具。

### 创作者是带着想法的DJ。

DJ们播放音乐集。

他们将他们自己的歌曲、适合的其他人的歌曲以及混音结合起来，为正在收听的听众编织一个*故事*。

技术的发展使得电子DJ能够创造出几乎任何他们想要的声音（受限于他们的创造力、思维和视角）。

通过软件和技能，他们可以将声音混合在一起，编辑它们的声音，并创造出人们愿意给予关注的音乐。

创作者并无不同。

他们收集自己的、他人的以及他们最喜欢的混音的*想法*。然后，他们利用永恒的技能：写作、演讲、营销和销售，将它们编织在一起。

随着时间的推移，它们吸引了一个将注意力投资在这些想法上的观众。有了这种关注，他们可以造福人类，并在享受他们喜欢的事情的同时获得收入。

**创作者常犯的一个错误：**

人们希望一遍又一遍地听到他们最喜欢的艺术家演奏的同一首曲子。

如果我最喜欢的dubstep艺术家转向乡村音乐，他会失去90%的观众。

即使他想创造一种新的dubstep风格，他也必须随着时间的推移让听众的大脑适应享受它。就像你不喜欢一个流派，直到你听了足够多（这同样适用于你追求的任何兴趣、目标或技能——激情来自投入的能量）。

这同样适用于价值创造者。

如果我最喜欢的创作者停止制作让我成长到这一点的个人商业内容，我的观众就会流失。

你可以从我的YouTube观看次数中看到这一点。有几个精选的大想法让我获得了吸引力。

作为创作者，你的工作是找到这个角度，并永远地利用它。

注意我如何能够“混音”我所说的内容，涵盖了我所有的兴趣。从哲学到灵性到进化心理学。

发挥创意。

人们不在乎你是否重复你说的话。

对你作品感兴趣的人更喜欢它。

你不可能，也不会在一次阅读中就记住我在这封信中说的每一句话。内容创作是编程。如果你想被人记住，重复是必要的。

人们作为价值创造者失败的第一大原因：

他们害怕将自己推向市场（因为他们认为自己不独特）。

你有多少双鞋？

你阅读了多少本关于同一主题的书？

你听了多少同类型的艺术家？

写下你想要阅读的内容。

卖出你想要购买的产品。

制作你想要观看的视频。

将你想要的市场上的产品版本。

只有这样，你的运气才会从0增加到1%。

### 现实挖掘以提取价值

我们如何开始作为价值创造者的旅程？

通过成为一个思想锻造者。

锻工（至少在视频游戏中）会挖掘矿石来为他们的创造提供燃料。他们将矿石熔炼并锻造为盔甲和武器。

创造者挖掘他们感兴趣的现实的裂缝。

他们研究、调查，并对主题着迷，以收集他们可以向观众传播的价值。

带着这些想法，他们创造了他们可以自己穿戴并给予他人的盔甲。

“盔甲”是更好地导航世界的视角。

就像一本书可以“改变你的生活”，因为它给了你如何做到这一点的意识。

“武器”是战斗的工具。

这些是人们可以采取行动的建议、技巧和策略，以取得进步。

进步意味着挑战。个人在生活中必须克服的战斗。你将你的教训传授下去，以便他人能更有效地导航生活。

这就是通过教育和世界观编程来对抗社会教化，从而提高人类进化的速度。

当你收集信息、想法和价值时，你需要一种方法来应用这些信息以测试现实。

你需要一个过滤器来使你的思想原创。一个渠道你的创造能量的容器。

换句话说，你需要建立一个与目标一致的项目。一个能引导你实现愿景的项目。

为什么？

这就是两个人阅读同一本书、文章或内容，却得出完全不同想法的原因。

如果你正在努力改善健康，你的思想会吸引帮助实现这一目标的想法。

正在努力改善财务状况的人会从他们所消费的信息中产生不同的想法。

**2) 追求好奇心以收集素材**

不要消费，要研究。

最好的创造者不会停止消费。相反，他们为了学习和传播思想而消费。

这种认知转变本身就会让你成为一个想法生成机器。不要只是滚动互联网来娱乐自己。滚动来教育自己。这比你想象的要强大得多。

你必须重新爱上学习，这种学习是在使你讨厌传统教育系统的教育体系之外进行的。

+   阅读书籍、收听播客、阅读博客文章和通讯，并沉浸与你目标一致的信息中。

+   拥有一个笔记本、应用程序或想法发展系统来保护你的想法。不，你不会记得它们。这一步绝对至关重要。我并不是夸张。去做吧。

+   记下那些能在大脑中引起多巴胺上升的想法。那些吸引你注意力的想法。那些改变你的观点并为你带来清晰度的想法。

+   思考你记下的想法的不同角度以及它们如何适用于你的愿景、目标和生活中的问题。

*这就是你在内容中写的内容。*

*不要将自己限制在特定的主题上（尽管写下来是有好处的，这样你可以进行研究并收集想法）**。*

*你是在按照一个目标分发想法。*

*无论什么主题都无关紧要，因为很可能它只是属于你自己的主题。这就是你如何使你的内容原创**。

如果你不知道要学习什么，请阅读[《一个人业务模型2.0信件》](https://thedankoe.com/letters/the-one-person-business-revisited-turn-yourself-into-a-business/)。

如果你想要提供确切步骤和策略的课程，请查看[2小时作家](https://2hourwriter.com)或[数字经济学](https://digitaleconomics.school)。

**3) 创建盔甲与武器**

你收集的想法将开始塑造你的世界观、基线观点和身份。

通过编程你的思维，你的行为将随之而来。

这就是编程的工作方式。

即使你很懒惰，对已知存在的机遇不采取行动的痛苦也将超过无所作为的痛苦。

**盔甲创造：** 给予人们实现目标的想法、动力和灵感。这塑造了他们的观点和身份。他们可以在生活中更加快乐，并实现他们的目标。

**武器创造：** 提供可操作的步骤、解决方案和建议。

写作、演讲、营销和销售是你将想法转化为有影响力的创造的方式。

从那里，你将它们分布到社交媒体平台，以在数字社会中建立一个社区。

如果我们生活在信息时代…

而普通人感到不知所措、焦虑和不确定…

而教育是塑造那些继续创造更美好未来的人的自我…

价值创造是理解。

你的工作是理解你的生活、世界，并为其他人应该采取的行动带来清晰性。那些跟随你的愿景并希望将其变为现实的人。

这需要时间和行动。

如果你从未让他人看到你的作品，你将无法知道它们是否有价值。

你必须发布帖子。你必须获取数据。你必须吸引读者关注你的作品，因为如果你不这样做，这是你无法提供价值的第一个迹象。或者你只是不知道如何玩游戏。

市场营销和销售需要人。你需要让你的作品出现在市场上。

如果你不去努力让你的内容出现在人们面前，你就不会知道你的想法是否有价值。

如果你将内容展示给人们，但它表现不佳，那么它并不具有价值。你需要写得更好。

如果你不去努力让你的产品出现在人们面前，你就不会赚钱。

如果你将产品展示给人们，但没有人购买，那么你的产品被认为没有价值。你需要更好地销售。

“感知”是那里的关键词。

市场营销和销售是感知。

你必须写作、演讲、营销和销售，以与你所针对的身份产生共鸣。

一个山谷女孩不会对健美建议产生共鸣。

一个足球妈妈（可能）不会对商业的哲学沉思产生共鸣。

你的语言决定了你吸引谁。

当你停止被你的思想所分散，并深思熟虑时，一切都很简单。

随着时间的推移建立受众群体，这是你减少将内容产品展示给人们所需的手动工作的方法。

好吧，我们从哪里开始？

**1) 从写作开始**

写作迫使你在想要成功的情况下，通过写作、销售和说服来补充你的学习。

它澄清了你的思想。

这使你更加善于表达。

它可以改编成任何其他媒介（演讲、视频或视觉）。

这也是一个无缝、非要求性、杠杆移动的任务，你可以在每个早晨的第一块时间安排它。

因此，我们在写作平台上优先考虑增长和内容。

**从时事通讯开始**

深度是你与表面市场的区别。

不，你不需要订阅者。你也可以将其作为博客文章上传，用它来写吸引点击的磁铁或产品模块，或者像我一样用它作为YouTube脚本。

订阅者会来的。

**将复杂性浓缩成推文（或帖子）**

在我看来，Twitter是一个被低估的写作平台。

它仍然是这样的，但Instagram的新Threads应用让我想起了Twitter的初期。

有一个充满机会的世界，社区的感觉令人难以置信。

Threads有很大的潜力。

为什么？

+   你可以发布你想要的任何内容（在你将它们发布到其他平台之前测试你的想法）。

+   它是基于写作的，这意味着你不需要是一个住在泰国、在海滩上啜饮椰子的10/10模特。

+   它就像一个论坛，你可以在你最喜欢的账户的回复中更快地找到志同道合的人。

+   转发或重新发布按钮是一个低摩擦的分享功能，这就是你获得关注者的方式（其他平台作为初学者更难增长）

+   你可以直接截图你的推文来在其他平台上增长（这就是我不花几个小时创造新内容就能增长的方式）。

在Twitter上成长的基石很简单。

+   每天发布1-3次帖子（并开始处理你的“失败”）

+   如果你拥有不到1000名关注者，专注于在回复和私信中与其他人建立联系，以利用他们的受众。

+   让你的个人资料图片可点击，这样人们就知道你是一个创作者

+   使用你生活中的大目标或你谈论的兴趣作为你的个人简介（你将带领人们走向何方，他们将通过你独特的兴趣如何到达那里？）

+   根据你的故事和个人经验撰写帖子或长篇帖子（但确保你有一个会分享它的网络，这样它就会半病毒式传播）

从初学者级别的教育内容开始。

让它变得可操作。

这就是每个人在开始成长（在任何平台上）时都会做的事情，市场中的95%都是初学者。

在建立自己的名声之前，不要试图表现得过于聪明或睿智。

**2)** **优先考虑** **想法与注意力吸引**

注意力是21世纪的货币。

有影响力的想法是你吸引注意力的方式。

我能给你的最好建议是，以创作者的视角阅读他人的内容。

为什么他们的帖子做得好？是什么吸引了你的注意力？为什么它有如此高的参与度？

答案并不总是显而易见的。

作者通常有很高的参与度，因为人们已经知道他们的想法。但当然，他们仍然是伟大的作家，值得学习。

在[数字经济学](https://digitaleconomics.school/)中，我们深入探讨了吸引注意力的心理学，但这里有一些原则，让你可以现在就开始。

**问题** - 每一个好故事都是通过开始讲述读者面临的问题来吸引注意力的。它开启了一个好奇心循环，使他们想要找到解决方案。问题是人类行为的基础。

**数字** - 在社交媒体上滚动时，具体的数字会打断你的模式。“我读了347本自助书籍，所以你不必”是一个例子。

**统计数据** - 类似于数字，统计数据可以很好地设定场景。“97%的人口注意力有所下降，这是如何解决你自己的方法”是一个例子，可能需要一些改进，但你看到了它的力量。

**3) 将自己视为理想读者**

当有人正在为所创造的价值建立读者群或受众时，常见的建议是针对你的理想客户进行写作。

在单人企业中，*你*是你的细分市场。你是你的客户形象。你是你的品牌、内容和产品。

我让人们在数字经济学中参加[迈尔斯-布里格斯个性测试](https://www.16personalities.com/)，这样他们就能更好地了解自己。当你了解自己时，你可以给自己写信。测试结果是你的客户形象。

现在，你该写些什么呢？

+   如果你正经历困难时期，给自己提供建议，并在你的“公开日记”中写下来。

+   给过去的自己提供克服你所面临问题的建议，但要更快地做到。

+   给未来的自己以鼓励，以实现你试图实现的目标。

这些只是你可以创造内容的许多方法中的一部分，但它们是一个很好的（且易于访问的）起点。

**4) 将个人项目转化为产品**

如果你没有朝着目标努力（在任何领域，即使建立品牌是你的目标，也要写下来），你就没有什么可写的。

目标 = 你正在工作的项目。

随着你取得成果，你开始理解是什么导致了这些成果。

这是盈利性产品或服务的基础。

如果你已经建立了一个企业，就通过咨询服务或数字产品教人们如何做。

如果你已经打造了自己的身体，那就做同样的事情。

如果你已经打造了自己的思想，那就做同样的事情。

教人们如何获得他们想要的结果。这并不困难。

你将不得不研究他人的产品或服务，看看他们是如何构建的。这是在新社会中购买课程的一个额外好处。

再次强调，如果你想找到一份工作，就购买那些培训你进入某个领域的昂贵教育。

如果你想要成为一名创作者，就购买那些构成创作者经济的人的教育。
