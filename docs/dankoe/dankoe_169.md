# 7 个阻碍你过上理想生活的坏习惯

> 原文：[`thedankoe.com/letters/7-bad-habits-preventing-you-from-the-life-you-want/`](https://thedankoe.com/letters/7-bad-habits-preventing-you-from-the-life-you-want/)

这 7 个习惯是心理上的。

（我实际上会争论说所有习惯都是心理上的。）

这些不是你试图停止做的物理行为。

这些是无意识的思维过程，导致人们保持现状而不是采取未来想要的生活。

而这需要一些工作来重新连接你的思维。

所以请跟我一起坚持下去。

### 习惯 1) 不够自私

因为氧气面罩原则适用于现实生活：

如果你想帮助他人，你必须首先确保自己的福祉。

一个窒息的救世主对任何人都没有帮助。

我们被训练去认为在关心自己之前需要先关心别人，而那些训练我们这样思考的人是无意识地自私的。大多数人并不希望你得到最好的。他们希望的是对他们最好的，他们会说服你以一种让你看起来自私的方式为他们服务。避免这个陷阱。它把你定位为一个可利用的资源。

你生活中最重要的人和事值得你最好的版本。

而当你忽视自己来迎合他人的需求时，他们得到了所有的好处，而你的潜力却慢慢流失。

停止把自私看作是负面的事情。

相反，要变得**策略性自私**。

把自己想象成一个手机电池或肌肉。当你完全充电或恢复时，你表现得最好。当你筋疲力尽时，你就关闭了。大多数人做任何事情都是在零电池电量下运作。

+   **确定不可协商的事项** – 一份简短的清单，列出你绝对需要来良好运作的事情。锻炼、独处时间、睡眠。要具体。这些都是必需品，不是奢侈品。

+   **在说“不”时不要道歉** – 这很困难，但练习说“不”而不说“对不起”。说明你不可用，让反应暴露出他们对你的福祉缺乏关心。

+   **根据资源过滤请求** – 如果你没有精力，如果它不重要，如果你必须放弃一些重要的事情，或者如果你会讨厌它，*拒绝*。句号。

+   **当你富足时给予** – 知道你需要什么来正常运作，知道你需要什么来发挥最佳状态，并且知道在优先考虑这些之后你可以自由地给予什么。

自私和无私是同一枚硬币的两面。你不能有其中之一而没有另一个，就像你不能有快乐而不与悲伤形成对比一样。

这可能听起来像基本建议，但你真的做到了吗？

如果不是，你需要这个提醒，而且可能不应该因为之前听过就把它排除在外。

如果你练习变得更加自私，你的生活质量将在一周内显著提高。

### 习惯 2) 询问“我如何开始？”

请原谅我在这里发泄一下。

为什么人们会对错误的事情如此着迷？

感觉好像没有一天不有人问：

“需要多长时间？”

“我该如何开始？”

这两个问题都源于一个更深层次的问题，“如果我失败了怎么办？”

我明白了。

你想开始副业来赚更多的钱。你想去健身房来恢复你的自信心。你想做困难的事情，以便生活可以更容易。

对于这一点，我说：

无论需要多少年都无关紧要。

你还能做什么？

你不会学习、建立、实验以及所有使生活有意义的事情，因为你还有更好的事情要做？

让它有意义。

你想要生活更容易，因为它是艰难的。但是，当你追求梦想变得艰难时，你就放弃并接受你生活的艰难？

真的吗？

问题在于你的优先级是谎言。

你说你想要成功，但你显然并不想，因为你的成功欲望并不超过你躺在床上吃多力多滋的欲望。

你必须意识到，无论需要你的一生，因为至少你会在成长，而不是死亡。你的生命工作不是一个结果。它是一个过程。

接下来是下一个问题：

“我该如何开始？”

你从开始做起。

简单明了。

想象一下，你是世界上第一个将要做那件事的人。开始业务，保持体型，学习新技能，无论是什么。

你认为那个人有一个可以观看的教程吗？一本书可以阅读？

或者他们犯了一个错误？然后又一个。然后又一个。他们没有将其称为失败，因为他们明白他们正在积累一个“不适用”的集合，而且从某种意义上说，考虑到他们所获得的所有经验，他们最终不可能不成功。

这里是开始真正的秘诀：

假设没有人会确切地告诉你该做什么。

假设你必须自己弄清楚。

我不会给你提供学习或做任何事情的最好方法。

而不是专注于这一点。

对它着迷。学习一切。尝试一切。让它失败。让它变得令人尴尬和羞愧。这是你在生活中实现任何有价值的事情的唯一方式。

### 习惯 3) 从不发布

当我意识到传统的工作是存在的祸害时，我开始尝试任何我能找到的“在线赚钱”方法。

我尝试了几乎所有商业模式，但都以失败告终，直到我不得不找工作来维持生计。

所有那些在线业务失败的唯一原因是，我从未发布过。我会花过多的时间在标志、网站和建立一份免费作品集上，以便我有一些权威感。

正如我所学，你不需要任何这些来成功。

而不是这样，你需要有一系列公开的作品，潜在的客户、雇主和追随者可以通过这些作品深入了解你所做的事情以及你做得如何。这样你才能提高你的运气百分比。

从不发布会导致危险的事情。

这被称为进步的隐形性。

换句话说，你没有*现实世界*的反馈，也没有给自己一个*机会*去承担一些形式的责任来坚持下去。

换句话说，你实际上并没有学到任何东西。这就像你上了大学，申请了一份工作，然后又上了大学。你从未真正开始工作……这是你*真正学习如何做这份工作*的地方。你大部分的学位知识都被抛到窗外了。

我没有不每周写时事通讯的选择。我没有不构建产品并推广它们的选择。有人在等待它们，我的整个生存都依赖于这一点。我*必须*这样做（并且*想要*这样做），因为我拒绝了其他所有事情，包括工作。对你来说这是可选的，因为你有一个 B 计划——也就是你试图逃离的当前生活方式，因为你不喜欢它，这没有意义。

最后，永远不公开你的工作是一种通过孤立来扭曲现实的形式。

你完全独立工作，没有社区、客户、反馈或问责制。在这种环境中，维持你的自我欺骗很容易。

由于你没有现实检查，所以很容易为你的谎言辩护，比如你“多么努力”地工作以及你的缺乏成果。

点击发布。

如果你不知道要发布什么，这篇文章可能是我的最不受欢迎的付费文章，因为最不耸人听闻的事情通常是最有效的，但得到的关注最少。在其中，我教授了 3 种练习在 Substack 笔记、LinkedIn、X 或其他任何平台上写短篇想法的方法。[查看它。](https://thedankoe.substack.com/p/3-drills-to-become-a-master-at-short)

### 习惯 4）创造人为的复杂性

简而言之：

比你笨的人比你在进步和赚钱上多 10 倍，因为他们不会过度思考而行动不足。

我在跟那些阅读我的商业内容并想“哦，那很基本”的人说话。

就好像我在向你撒谎，告诉你需要什么。

问题在于你潜意识中在商业基础周围维持着人为的复杂性，以证明不采取行动是合理的。你相信商业成功需要一些神秘的秘密，其他人正在向你隐瞒，所以你堆积了越来越多的无用知识，这只会让你更慢。

我会尽量简短，以避免复杂性。

你需要持续执行基本原理。

如果你需要超过入门级课程才能采取行动或看到结果，*你做错了。*

学习基础知识。

开始。犯一个错误。从错误中学习。

这就是蓝图：根本没有蓝图。

一个冲浪者不需要蓝图来回应波浪的反馈。他们已经失败并练习得足够多，以至于这已经成为第二本能。

所有这些习惯的共同主题是，你实际上从未坚持不懈地迭代地做那件事。

### 习惯 5）过于节俭

![图片](img/9dc28d49c122822b09a27685defb0bfe.png)

大多数人都与金钱作斗争。

这是一个事实。

金钱与现代生存紧密相连。

它影响你生活的每一个方面。即使你没有在考虑金钱，你也在考虑金钱。你的健康取决于你的金钱。你关系的理智取决于一定数量的金钱。

你可以说这是不公平的，并抱怨它应该如何不同，但我强烈不同意。

我不是来告诉你金钱的真相是如何“隐藏”在你面前的。

处于这种状况，既有你的错，也有你的不对。

你没有质疑你的成长方式，与金钱发展了糟糕的心理关系，因此你未能理解赚钱是一项技能。你可以赚到你想要的任何数额，但你没有花时间去学习、练习和掌握这项技能。

金钱问题不存在。心理问题存在。

大多数人无法赚钱、保留或明智地使用金钱，这归结于他们的信念，因为你的信念决定了你能注意到去做那些事情的机会。

这里有一份关于普通人金钱观念的清单：

+   你认为赚钱很难。

+   你不相信金钱是一个像健康、关系或你的创造性工作一样可以掌握的领域。

+   你可能认为金钱是邪恶的（因为你被先前的负面经历所编程）。

+   你对创业不感兴趣，因为你只看到了那些聒噪的销售员和每周 80 小时的工作周。

+   你认为每个人都试图欺骗你，或者认为任何非大型企业都是骗子（当他们通常是你说你支持的自营企业）。

+   你陷入了生存模式，因此你看不到自我发展和商业的重要性，并感到有必要将它们一笔勾销，以便对自己的行为感到更好。

+   你认为自己在你所在的领域赚不到很多钱，这是错误的，如果你只是观察你所在领域的顶尖收入者（提示：他们有自己的企业）。

这些限制了你所能赚到的每一分钱。

如果你认为金钱不是一项技能，你就不会学习这项技能。

如果你认为金钱是邪恶的，你会对赚取更多金钱的想法感到厌恶（即使这是你摆脱自我制造的混乱的唯一途径）。

你对金钱的看法取决于你的发展水平。

解决你的金钱问题（心理问题）的方法是通过每个级别的发展。

随着你的自我发展，金钱的目的也在改变。它最初是生存的方式，变成了获取地位的方法，这进而导致了对自主性的渴望，然后变成了创造力和表达的燃料，最终转变为帮助他人通过类似阶段——无论与金钱是否相关——的方式。

这里有一些级别：

+   **生存** – 你现在在财务领域的发展正处于生存阶段（你可能在健康或智力方面更高）。

+   **地位** – 当你（1）对金钱有安全感，并且（2）意识到为了更好地生存，需要别人认为你有价值时，你就达到了地位阶段。

+   **创造力** – 你意识到地位游戏的缺陷，开始渴望自主性，并在你的技艺周围发展一种哲学上的掌握感。在跟随他人的方法如此之久之后，你创造了自己的做事方式。

+   **贡献** – 你发展到无法被雇佣的程度。工作、休息和娱乐变得难以区分。

当你发现前一个阶段变得难以忍受时，你将超越到每个阶段。你会知道的。

### 习惯 6) 最小化预期的遗憾

> 你所寻找的魔法就在你正在避免的工作中。

虽然我相信你真正寻找的魔法就在你正在避免但看不到的实验中，因为你的心思全在避免工作，但这确实包含了一丝真理。

几乎每天都有人问：

+   “我应该从哪个平台开始？”

+   “我应该创建什么样的数字产品？”

+   “我应该开始一个服务型业务吗？电子商务？还是其他什么？”

然后，当他们最终决定并经历一个过早优化的阶段。

就像我在之前的企业尝试中做的那样，我专注于网站、标志，并在私下里做作品集，直到我觉得“准备好了”，但那永远不会到来。

你做决定主要是为了避免未来的负面情绪，而不是为了最大化积极的结果。你无意识地是在玩输而不是赢，因为在你内心深处，你真的不想赢。

解决方案是视角。

你会后悔什么？

走错路——这是找到正确道路的唯一方式——并且获得比你一生中任何时候都多的经验……

或者保持不变。

这不是一个困难的决定。

做出能带来结果的工作，其余的都忘掉。

如果你认为你需要一个网站、标志和有限责任公司才能开始吸引客户，或者你认为你需要完美的数字产品或商业模式才能开始赚钱，那么你是在欺骗自己。

我会在未来的信中讨论这一点。

### 习惯 7) 保留一个不符合你目标身份

你没有达到你想要的地方，因为你不是那个应该在那里的人。

当人们想要改变生活时，他们试图在不做太多改变的情况下进行改变。

+   他们想要在不放弃他们最喜欢的食物的情况下减肥。

+   他们希望在周末仍然外出游玩的同时赚更多的钱。

+   他们想要在不成为邋遢的人的情况下进入一段关系。

我并不是说你不能做这些事情并看到结果。

我说的是，你没有取得结果的原因是，你的当前生活方式无法产生那些结果。

问题是，所有这些事情都是可选的。

你之所以喜欢甜食或油炸食品，仅仅是因为它们已经成为习惯。你已经做了这些事情，以至于感觉没有它们你的生活会很糟糕。

如果你真的想要改变，你需要撕下创可贴。

你需要一夜之间跳入一个新的身份。

你需要极其意识到，你现在所在的位置，正是因为你现在所做的事情。

如果你放弃当前的习惯，你不会死。

你可能会经历一段短暂的痛苦时期，但当你尝试翻新房屋时就是这样。它不会感觉一样，但如果你继续翻新，你将创造一个身份，使这一切都变得值得。
