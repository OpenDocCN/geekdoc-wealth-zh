# 赚钱是精神上的（如果你学习这些不道德的技能）

> [https://thedankoe.com/letters/making-money-is-spiritual-if-you-learn-these-unethical-skills/](https://thedankoe.com/letters/making-money-is-spiritual-if-you-learn-these-unethical-skills/)

在当今世界，你所能做的最糟糕的事情之一就是陷入“高尚的失败者”的思维模式。

我认识到的许多聪明人讨厌在线业务、营销和销售，因为他们认为这“低于”他们。

他们陷入了自己认为的“更高”的视角，这种视角妖魔化了卑鄙的销售员。实际上，这是一个错觉。这种“更高”的视角充满了误解、缺乏同理心和仇恨。

它与“卑鄙的销售员”本身相比，意识水平可能相当，甚至更低。

这与民主党和共和党为了捍卫他们认同的意识形态而争论的方式相似。类似于无神论者和有神论者。当他们进行这些狭隘的争论时，根本不可能发现情况的整体真理。

让我们开始分解这个整体真理：

如果你不开设业务来提供信息、教育和商品，以改善人类——那么不道德的企业就会在没有竞争的情况下上升到顶端，使人类变得更糟。

不道德的企业之所以充斥市场，是因为你的意识或智慧并没有你想象的那么高。

你没有开始一项业务来提升集体意识，这是直接对邪恶的贡献。

通过仅仅妖魔化金钱和作为社会生命线的商业，你被分配了一个雇主很可能在你看来是不道德的工作。

你不会开始一项业务，是因为你的幻觉性**假设**和条件反射，认为赚钱是不道德的——但你却在为一家使人口生病、向全世界投掷炸弹并使员工陷入机器人般常规的公司工作，使他们不敢实现自己的潜力。

人们批评新的创作者教育企业是“骗局”和“诈骗”，因为它们解决了唯一能够提升人类基准意识的真实问题。

在生活的健康、财富、关系和幸福领域销售教育、课程和指导的人，正在攻击积极行为改变的根本。我在[你大脑中有10万美元的知识被困](https://thedankoe.com/letters/you-have-100000-of-knowledge-trapped-in-your-brain/)一文中讨论了教育的重要性。

如果人类的强大程度仅取决于最薄弱的环节，99%的人遭受着相同的生存问题，而教育是允许**个人**解决他们自己的问题的（而不是提供暂时的缓解），问题是什么阻止了个人到集体的进化，如果教育体系专注于为自身利益培训人们进入工作，那么价值创造者的新数字社会是走出这一混乱的一个现实途径。

回到主要论点…

*不采取行动，你让比你意识低、智力低的人拥有比你更多的关注、影响和金钱。*

金钱产生涟漪效应，使他们能够传播他们的信息更远，雇佣员工（这些员工贡献了不道德的行为），并增加他们在社会中的关注度。

唯一阻止人们关注一件事的方法是给他们提供更有说服力的东西来关注。

这个问题并不像“只是开始创业赚钱”那么简单。

一切都是相互联系的，而你却不知不觉地让整个宇宙在邪恶的手中倾斜，因为你没有成为一个有意识的商业主，影响其客户、员工、读者、经济以及其他无限的事物。

*精神是理解你在整个宇宙中的部分。*

*精神不是脱离那个整体去森林生活，让宇宙陷入混乱。*

*当你通过投入努力和能量来推动人类进步和进化，试图逆转熵时，精神就可以被考虑和感受到。*

> 幸福是——力量增加的感觉——阻力正在被克服。——尼采

如果你恨金钱，难道你不恨你的生活吗？

你现在周围的一切，从你手中的电话到你驾驶的道路，到你坐的桌子，到你维持生命的食物，都是为那些为文明进步做出贡献而赚钱的商业产品。

你正淹没在一个由金钱驱动的世界中，这种金钱源于生存。

## 金钱的重要性与不可忽视性

*“最好的赚钱方式是教别人如何赚钱。”*

我经常看到人们这么说。

有些人把它当作智慧。

有些人把它当作一句拆散任何商业建议的话，这样他们就不必面对现实。

当金钱统治人们的生活时，有些人将这句话视为坏事。这是阻碍他们达到个人发展下一个层次的一件事。*它几乎决定了一个人采取的每一个行动，即使是“精神”最重的人，因为金钱与现代生存紧密相连。*

**你为什么每天工作8小时，工作45年？** 因为你需要支付账单并支持你的家庭。

**你为什么像修道士一样离开去森林生活？** 因为你想消除金钱的需求以追求个人发展。（认为每个人都应该这样做比让社会腐烂还要愚蠢。“一切皆是一体”，直到你实际上必须对世界的健康负责，所以你选择退出）。

**你为什么去健身房吃健康食品？** 为了健康，是的，但还有更多。你想要提高你的感知地位，以便吸引更好的机会，提升你的职业生涯，赚更多的钱。

我为什么现在在键盘上打字？为了帮助你，是的，但任何人的行动背后不仅仅只有一个原因。写作是我的事业。我每周写商业文章，因为我为此得到报酬，这给我带来了许多其他的好处。

你为什么在读这篇文章？为了获得新的视角，是的，但这将打开你的思维，减少围绕赚钱的局限信念。

几乎你采取的每一个行动都有金钱作为附加的理由。

金钱深深植根于现代生存之中。

如果你看看马斯洛的需求层次理论，每个阶段在现代世界中都需要金钱。

你生活的财务领域就像其他任何领域一样。

就像视频游戏中的技能树一样。

你生活的所有领域都需要随着时间的推移播种种子，以便花园能够繁荣。

为了获得健康，你需要金钱。

为了获得满意的关系，你需要解决摧毁99%关系的那个问题…金钱。

为了快乐（更确切地说，为了享受生活，因为快乐是一个易变的指标），你需要朝着有意义的目标前进。你需要建设。建设需要资源。资源需要金钱。

当你忽视对金钱的需求时，你限制了你在心理、身体甚至精神实践上的发展空间。

精神实践？是的。如果你与物质世界的生命之源关系不佳，那么你的精神就会受到阻碍。你无法获得让你能够为人类精神进化做出贡献的资源。

无论是从关系、健康和健身，还是学习新技能如数字艺术的角度提出的建议，一切都在“教人们如何赚钱”和提升他们的职业生涯。

恐怕，你为什么要在你生命中花费12到24年的时间去上学？当然是为了提高你的赚钱能力。是的，你学习如何在社会中运作，但社会的生命之源是什么？

学校里教的是什么？仅仅是“如何赚钱？”还是一种你可以采纳的兴趣，最终帮助你赚钱？

回到创作者经济的主题…

你没有意识到，在新的经济中赚钱的唯一方式是从那些**创造**了在该经济中赚钱的职位和机会的人那里吗？

就算你不理解它，并不意味着它没有用。

你没有意识到，在旧经济中赚钱的唯一方式是从创造了那些职位的机构和结构那里吗？

一切都在进化。放下你十年前认为有效的东西。

学校教授的是过时且不那么赚钱的赚钱方式，因为随着互联网的创建，更有利可图的赚钱方式出现了。就像在拖拉机和其他一切被创造出来之后，更有利可图的职位出现了。

通过把一切都视为骗局，你可能会推迟你的发展数年，甚至是一生。

这是我们正在经历的进化阶段，你错过了将成为新常态的东西。

## 你无法避免学习“不道德”的技能。

最成功的人是那些研究不道德策略并在道德方式中使用它们的人。

+   销售

+   营销

+   说服

+   催眠

+   影响

+   游戏

他们的工作、业务和对崇高目标的追求是他们整合阴影的方式。（我在[数字经济学](https://digitaleconomics.school)中从哲学商业的角度教授所有这些，除了催眠）

当你放大到看到原则时，它们都是同一件事。

所有这些“不道德”的技能只是理解人性、心理学和宇宙的模式，如讲故事（甚至建筑和音乐）：

> 音乐是流动的建筑；建筑是冻结的音乐。 —— 歌德

每个人都有阴暗面。

就像地板地毯的丑陋背面或一首歌的低音。

宇宙是创造与毁灭、统一与分裂、出生与死亡的平衡。建筑物被摧毁，以便新的建筑物可以建造。你的过去自我死去，以便你的理想自我和新生。

你并不纯粹是好的。

当“好”是相对的，并且没有邪恶就不会存在时，不可能做到纯粹的好。

那些试图压制它而不是整合它的人，他们的阴暗面会在无意识中以某种方式显现出来。通常，那些声称自己不是操纵性的人是最具操纵性的，他们只是没有意识到这一点。

获得销售工作或开始营销业务是整合你的阴影的绝佳方式。你被赋予了选择使用技能为善或为恶的选择。你可以看到结果以及它对你和你的客户的影响。

精神大师并不都是“全好”，他们只是学会了整合不良，不让他们的生命之歌被写成一首令人沮丧的歌。你的故事中会有低谷，但当你放大视野时，它会如何发展？

思考你的技能栈的一个好方法就像一个共同基金。

如果一支股票下跌，共同基金仍然可以上涨。

如果一项技能是“不道德”的，那可能就是你的业务盈利、道德和有影响力的解锁。

## 作为一个人类，就是要超越、扩展和创造。

世界正在变得疯狂。

由于技术、科学和计算能力的进步，作为一个物种，我们正在超越物质。

我们作为一个物种正在统一——意识的统一——通过技术。这没有金钱可能吗？

当大多数人认为社交媒体对人类是一种净负面时，我完全不同意。

社交媒体是我们记录集体意识的地方。

我们分享思想、观点、想法、创作和我们的身份。我们在将我们的思想上传到互联网。不仅是创作者。内容是集体意识。

社交媒体消除了我们沟通的界限。

就像人们都在排队进入他们的屏幕一样。他们整天盯着它。就像有一种原始的吸引力，想要与每一个心灵合为一体。

你的整个身份已经出现在屏幕上了，但仍然有一个屏幕构成的障碍。

想象一个你可以向另一个人发送一个思想信号的这个世界。想象一个你阅读我的通讯时，它们会直接进入你的大脑而不需要阅读的世界。想象一个你观看我的视频时，我就在你身边的世界。

如果你还没有看过Lex Fridman与马克·扎克伯格的播客（[《元宇宙中的第一次访谈》](https://youtu.be/MVYrJJNdrEg)），那么那个未来并不遥远。他们在虚拟现实中有一个播客，他们的化身非常逼真。这令人震惊。

虽然我现在确信这是真的，但未来会怎样呢？如果虚拟活动像实体活动一样真实，但信息流动是瞬间的，而体验又完全不同于我们现在能想象的那样，会怎样呢？

另一个问题：

你可以触手可及地拥有整个世界的知识。这让你感觉如何？

你有学习的能力，扩展你的自我意识，并创办一家能为世界提供价值的公司。

当大多数人认为这是幻想和不可能的时候，我认为这是自然的，而且在未来可能是唯一的选择。也许不是特别像“YouTube红人”，但像宇宙一样，作为一个用你的心灵创造的存在。

我们对关心、欲望和需求的背后有多个“为什么”。

## 为什么你不成功的一个简要总结

你之所以不成功，并不是因为你声称你想帮助他人，但你不会学习那些让你能够帮助他人的技能。

你之所以不成功，是因为你过于专注于艺术，然后在市场看不到你所提供的东西的价值时（因为你无法用营销和销售来阐述它），你就责怪市场。

你之所以不成功，是因为你“专注于你的技艺”而不是让你的技艺出现在潜在客户面前。你的网站、设计、写作和其他项目，除非其他人看到它们，否则都不算数。你正在延迟反馈的时间。没有反馈，你就无法使其有价值。你的第一次迭代不会是有价值的。

你之所以不成功，是因为你对商业的理解不完整。它是通过创造一个能够提升集体意识的产品来参与人类进步的方式。

你之所以不成功，是因为你把这些观点当作个人攻击，关闭了你的潜能，让你的条件反射控制了你。

你之所以不成功，是因为你没有创造出市场愿意为之付费的有价值的东西。人们不想要你自我意识想要的东西，他们想要的是大自然想要的东西，而你有什么资格去判断那应该是什么？

你之所以不成功，是因为你沉溺于那些你认为高贵的信念中。你将自己锁定在一个阻止你个人成长的范式之中。你开始应对而不是创造。

> 自我发展是通往创业的入门药，因为你意识到提高他人是提高自己的下一个层次。

要成为成功的人，有一条黄金法则：

向他人提供价值。

这意味着两件事：

1.  **你必须培养你的价值**。你必须扩展你的思维，识别问题，获取解决问题的技能，并在现实中取得实际的进步。（你不能整天坐着学习理论，你必须实践。

1.  **你必须分配你的价值**。你需要一个载体——也就是一个企业——来分配市场*想要*购买的产品或服务。如果他们不想要，那么它就不像你想象的那样有价值。

换句话说，你需要自我发展和他人发展。

个人成长和商业成长。

商业是你以规模向他人提供价值的方式。

你如何在现代世界中培养和分配价值？

社交媒体。虚拟现实。

社交媒体是你的公立学校、公共简历和公共作品集。

它包含了美好生活的神圣三合一：学习、建设和销售。

你通过学习新知识来培养你的价值。

你通过建立新的项目来巩固你的价值。

你通过出售那些项目来分配你的价值。

所有这些都会导致更宝贵的技能集和心态。
