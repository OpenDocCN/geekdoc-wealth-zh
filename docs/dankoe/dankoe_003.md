# 价值创造：构建我单人企业的技能

> 原文：[https://thedankoe.com/letters/value-creation-the-skill-that-built-my-one-person-business/](https://thedankoe.com/letters/value-creation-the-skill-that-built-my-one-person-business/)

我最喜欢的失败业务之一是使用Photoshop创作数字艺术。

这是我最喜欢的一件作品（如果你滚动到我的Instagram底部，你可以找到它们全部）。

<picture fetchpriority="high" decoding="async" class="wp-image-1111">![](../Images/5cdd4df0d1f400e99535ba0f696884a0.png)</picture>

这次尝试是我第一次意识到社交媒体的力量。

我模仿了大型账户如何增长他们的追随者：

+   标记分享数字艺术的社区页面

+   创造我能做到的最好的合成图像，以便它们能被分享

+   通过混搭摄影师的图像，希望他们会分享我的内容（比如在Twitter线程中提到一个大账户，让他们转发给他们的受众）

这几乎是在5年前，这是我如何在Instagram上获得前2500名追随者的方式。

在把自己锁在房间里3个月，忘记吃饭，把心都投入到我的创作中之后，我放弃了。

我喜欢创作，但不知道如何盈利。

我的艺术是有价值的，但还不够有价值以至于要求支付。

这甚至在我脑海中都不是一个可能性。

我意识到我无法维持作为艺术家的生活方式。

我感觉我必须学习一项更有利可图的技能才能进行自由职业。

当然，我是个白痴。

如果我知道我现在知道的事情，我可能还会在Photoshop中创作。

现在我更喜欢写一些东西，所以我计划不再回到数字艺术，但如果我能回到过去，我会这样做：

+   继续成长并意识到我的受众是流量来源

+   创建一个产品或服务，将流量引导到那里（在这种情况下是课程和辅导）

+   学习文案写作、营销和提供创造，以在市场上定位我的产品

+   与个人发展创作者合作，将他们的最佳想法视觉化，并成为该领域的权威

换句话说，我会做我们在单人企业视频中讨论的所有事情。

就像我的所有失败业务一样，它们教会了我宝贵的教训。

这些技能，比如Photoshop，都为我的当前品牌成功做出了贡献（即我制作了所有的设计和缩略图）。

但有一种技能，我将其归功于过去几年中95%的品牌、销售和受众增长。

## 人类心理学的通用模式

<picture decoding="async" class="wp-image-1112">![](../Images/9ba4c95d64151919f06e3d86c43d99ac.png)</picture>

人们告诉你要学习的每一个“赚钱”的技能都与宇宙模式和人类心理学相关联。

如果它们不赚钱，那么它们就不会赚钱。

如果它们不赚钱，那么你需要堆叠那些使它们能够赚钱的技能。

所有人类行为的基础（包括人们拿出信用卡支付给你）有三件事：

**1) 生存**——复制你基因和意识中的信息，无论是物理的还是心理的愿望。

**2) 进化**——通过创造超越破坏的愿望。

**3) 变革**——追求更高品质体验的愿望。

这些都相当相似，可以被认为是这封信余下的同义词。

故事讲述、游戏、音乐和建筑也可以观察到这些模式。每个原则都可以用来增强你的技能堆栈的影响力。

以我能用最简单的方式解释，所有这些都使用相同的框架：目的、路径、优先级。

### 1) 目的

行为改变始于一个问题，并照亮一个目标来克服它。

目标 = 目的。

目的 = 欲望。

欲望 = 问题如何产生。

就像故事通过描绘或暗示问题来开启好奇心循环。

或者，如何让着陆页针对你生活中相关的问题来激励行动。

或者，当你对生活中遇到的问题感到厌倦，最终决定解决它时。

通过照亮某人生活中的一个问题，你创造了目的。

### 2) 路径

当一个问题被识别，并伴随着一个目标揭示出潜在的将来时，清晰就是桥梁。

清晰的反面是混乱，我们不希望我们的客户在想到使用我们的产品时感到焦虑和不知所措。

### 3) 优先级

为了让问题在一个人意识中留下印象，它必须对他们来说很重要。

如果问题对他们来说并不重要，你就无法吸引他们的注意力来展示他们前进的道路。在那个阶段，道路甚至不再重要，因为它无法帮助他们，他们也不会有采取行动的动力。

重要性是主观的。

人类从他们身份的角度感知问题。

如果你试图向一个超重的游戏玩家推销自我提升产品，你必须非常具体地定位该产品。

顺便说一句，这是你可以用来重新掌控你生活的相同框架。

## 价值创造框架

目的 > 路径 > 优先级框架是很好的，但让我们让它变得实用。

在下一节中，我们将讨论我们可以使用这些内容的地方。

将以下所有部分视为“营销火力”，你可以将其用于你的内容、促销，甚至日常生活中的语言，以展示你的产品或你的价值。

这些是将泥土变成黄金的步骤。

这些步骤，当应用到我的任何一项事业中时，使我的企业实现了100万美元的营业额。

这些是将实践最有价值的技能——价值创造——的步骤。

如果你不知道如何提供产品，学习[如何在这里创建一个](https://thedankoe.com/the-best-online-business-model-to-make-1-million-in-2023/)，或者如何打包你头脑中的[知识](https://thedankoe.com/you-have-100000-of-knowledge-trapped-in-your-brain/)在这里。

这些会随着时间的推移而变得更容易。

如果你发现很难填写这些内容，不要担心。

### 1) 意识层次

在线或离线遇到的每个人，都在他们的问题和你的价值（产品、服务或存在）周围有一个意识层次（或意识）。

注意，这些与[自我发展阶段](https://thedankoe.com/a-message-from-a-recovering-atheist-how-to-find-god/)或心理学/哲学中的螺旋动态量表相重叠。

**第1级：无意识**

如果你想要写病毒性的推文或创建病毒性的YouTube视频，让你的内容围绕无意识阶段。

在这个阶段，人们对自己的问题和你的提议都无意识。

你希望通过写作和演讲让他们意识到他们的问题以及它如何影响他们的生活。

用教育、信息和同理心对待你的读者，解决他们面临的问题。

如果你试图帮助某人变得健康，你可能会指出缺乏适当的营养是他们困倦的原因。

我会写的推文：

*“你总是感到疲倦，因为你总是吃热狗、Uber外卖和会导致营养缺乏的食物。”*

确实有点严厉，但那是我的风格。

**第2级：问题意识**

你的读者意识到一个问题，但可能不了解它或不知道解决方案。

我会写的帖子：

*“如果你总是感到疲倦：*

*– 每天早上称体重*

*– 吃你体重的蛋白质*

*– 喝水直到尿清澈*

*– 每天走10000步*

*“等你感觉那么好，以至于你讨厌用垃圾食品破坏这种感觉时，再感谢我吧。”*

在长篇内容中，你可以既教育人们面对的问题，又提供解决方案。

在简短的形式，如推文或短视频中，你通常针对1-3级以获得最大效果。不要过于高深或聪明。

**第3级：解决方案意识**

在这个阶段，人们意识到他们的问题和解决方案的存在。但他们不知道*你的*解决方案存在。

现在你正在提供一个明确的解决方案或鼓励他们采取行动。

**第4级：产品意识**

人们意识到你的解决方案和其他类似的解决方案。

你的工作是提出理由，说明为什么你的更好。

富含益处的语言是关键。

这里是我会写的帖子：

*“为什么你应该开始间歇性禁食：*

*– 早晨的专注力是疯狂的*

*– 你不必计划早餐*

*– 午餐和晚餐可以是丰盛的餐点”*

我可以完成帖子，但你已经明白了，而且这已经够长了。

**第5级：最高意识**

人们会无情地意识到你的解决方案，并需要一点推动才能跨过界限。

你可以继续推广你的产品，直到时机成熟（比如重新定位广告，人们通常不会在第一次看到你的产品时购买）……

或者你可以提供限时折扣或实施另一种稀缺策略。要适度使用。

### 2) 定位

<picture decoding="async" class="wp-image-1113">![](../Images/082f4810ec33030cf6604ae7eabddfac.png)</picture>

现在我们已经了解了意识层次，我们可以开始构建一个有说服力的信息。

顺便说一句，我在[数字经济学](https://digitaleconomics.school)中提供给你Notion模板和深入的教育，教你如何从你大脑中的知识中创建和销售产品。

我们之前已经多次讨论过永恒的市场（健康、财富、关系），但现在我们要深入探讨。

有8个人类欲望是难以忽视的。

你所有的内容、销售页面或写作都应该从这些欲望的角度来写：

+   生存

+   生活享受

+   摆脱恐惧

+   性伴侣

+   舒适与清晰

+   感知地位

+   社区安全

+   社会接受

教育人们了解阻碍这些愿望的问题，并提供解决方案来克服它们。

注意到每个人的“细分市场”都是自我实现。

我的商业理念是：

解决你自己的问题 > 记录解决方案 > 提高他人的集体意识仍然适用。

### 3) 重大问题

从你选择的特定意识和角度出发，你确定一个重大问题开始你的写作、演讲或创作。

从“无意识”阶段和“感知地位”角度，我可以确定一个重大问题，比如：

无法获得六位数的工作。

我可以激化这个问题，并说明没有六位数的工作是如何影响他们的约会、支付账单的能力，以及世界自然缺乏对他们的一定尊重。

你也可以从“生存”等角度看到该问题的影响。

浅薄？

是的。

欢迎来到人类生活。

在你解决生活中（以及他人）的浅层问题之后，深度才会出现。

如果占据他们全部心思的问题是支付账单或性行为，那么没有人会在意正念或存在主义。

我们必须提高人们的发展阶段。

### 4) 独特机制

你需要一个与众不同的解决方案。

再次强调，这就是我建议“自己找出答案”的原因，因为通过实验，你创造了一个独特的解决方案。

就像格雷格·奥·加勒格如何将间歇性禁食和倒金字塔训练结合成一个适合他特定生活方式的健身计划。

然后，他针对他生活方式中的问题，将它们定位在8个人类欲望上，并销售他的独特解决方案。

想想*你自己*会如何一步步解决问题，并为这些步骤起一个吸引人的名字。

我的[2小时内容生态系统](https://2hourwriter.com)就是一个例子。

### 5) 列表喷洒好处

我喜欢在我的文案和推文中用好处来压倒人们。

当你列出你独特解决方案的显著好处时，你给了人们*多个*购买（或参与）的理由。

即使一个观点没有引起共鸣，另一个可能就是他们转变的原因。

### 6) 证据

个人成果、推荐信以及你能收集到的任何统计数据或事实，都将帮助人们相信你能取得成果。

不要低估个人经历的力量。这是我们很少看到的东西。人们渴望人与人之间的联系。

这就是为什么我开始90%的通讯和视频都以一个个人经历开始，这个经历说明了我在遇到的问题。

当然，推荐信是强大的，你应该把它作为你的全职工作在开始时获取它们，但你不应该等到你有了它们才开始销售。

### 7) 大想法

在你完善了其他价值创造点之后，你需要仔细思考它们，以获得大的想法。

大想法是一种总结，用一句话说明你的价值。

最受欢迎的引言是大的想法。

那些留在你脑海中的是大的想法。

大想法最好是通过寻找它们并使它们成为你自己的方式来发现。

比特币是未来的数字货币。

Uber是现代出租车服务。

ChatGPT是**他妈的**一切。

这可以与或*是*你列出的任何点重叠，只要它是有效的。

这对我来说花了一些时间才变得有意义，我仍然无法清晰地表达它。

通过实践和努力去理解它。

或者，你可以通过[上周的信件](https://thedankoe.com/how-i-read-books-for-maximum-intelligence-5-minute-habit/)来寻找大想法。

### 8) 风险反转与最终结果

一个有说服力的风险反转将帮助你在你旅程的开始阶段达成交易。

大多数人选择退款保证，这很酷，但你可以通过将其与明确的结果相结合来使其更加诱人。

当销售约会产品时的例子：

“如果你在60天内没有安排5次约会，我们将退还你的投资。”

谁会不买这个？

它给你施加了压力去交付。

你将不得不对你的风险反转添加一些限制，并声明他们必须对你提供的一切采取行动。

## 营销积木

将上述内容视为你写、说或与他人互动时“营销积木”。

一切。

如果你想变得出色，你必须时刻练习这一点。

阅读这篇文章是不够的。

阅读任何东西都是不够的。

你必须开始一个项目，遇到*现实世界的问题*，让现实告诉你阅读无法教给你的东西。

有成千上万的小细节，没有人能在一篇内容、一本书或一门课程中向你展示。

你必须通过行动有意识地和无意识地拾起它们。

这里有一些你可以开始练习价值创造的地方：

### 社交媒体帖子

吸引人的长篇内容遵循以下格式：

+   提出一个问题并激化它

+   提供背景信息或提升到解决方案意识水平（通常通过证明或一个故事）

+   提供可操作的步骤来克服问题（并且最好将这种方法命名为你自己的）

+   将实施该系统的益处融入其中，让人们采取行动

+   将你内容的大想法作为标题或标题

我经常提到的“一人企业”既是一个经过验证的大想法，也是解决生活中许多大问题的独特机制。

长篇内容可以引导人们了解所有层次的意识。

为了吸引关注，目标为1-3。

对于销售，目标为2-5。

对于强硬的促销，目标为3-5。

这适用于通讯、帖子、轮播图、标题和YouTube视频。

短篇内容一次只能有一个。

你可以将任何你的“营销积木”变成一篇帖子。

这适用于推文、单个Instagram帖子、tiktoks和短视频等。

### 着陆页与销售页

与撰写长篇内容类似，着陆页和销售页面遵循相同的模式。

如果你想练习文案框架，请使用：

+   **PASTOR** – 问题、放大、故事、证词、提供、回应

+   **AIDA** – 注意力、兴趣、欲望、行动

+   **PAS** – 问题、放大、解决方案

并且随着时间的推移变得更好。

对于你的标题和副标题，你试图将大多数，如果不是全部，营销积木放入其中。

这也适用于销售电话、私信或演示。

### 课程大纲

你希望人们对你教授的内容采取行动。

在他们购买你的产品后，你并不会停止具有说服力。

你必须保持他们的注意力，给予他们清晰度，并鼓励他们采取行动。

他们还能通过什么方式获得结果呢？

如果他们没有获得结果，你将如何获得推荐信？

### 每日对话

所有的人类沟通都具有说服力。

当你与人交谈时，你试图从一个共同的角度出发。

当你没有这样做时，争论就开始了。

一个观点有一个共同的目标，所以从一个问题开始对话以吸引他们的注意。

你说：“我最近感觉如此疲倦，我无法完成任何工作。”

他们说：“还是那个总是感到疲倦的我。”

你说：“是的，上次我感到这样的时候，我深入健身，我打算恢复我的常规。”

他们问：“那是什么常规？”

你说：“我遵循了垂直饮食，在健身房进行了兄弟分割。”

对话将继续，希望教育他们你所做的事情，并给他们一些有用的尝试（以及这样做的好处）。

那次对话很有价值（并且可以转化为内容 ;))

就这些。

去实践你做的每一件事。

你的表达、现金流和生活质量将直线上升。
