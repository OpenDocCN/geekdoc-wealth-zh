# 如何成为任何技能的顶尖1%

> 原文：[https://thedankoe.com/letters/how-to-become-the-top-1-percent-in-any-skill/](https://thedankoe.com/letters/how-to-become-the-top-1-percent-in-any-skill/)

每个人都想要取得伟大的成就，但没有人愿意连续多年专注于发展自己的技艺。

（剧情转折：每个人都想专注于自己的技艺，但没有人愿意进行内在的工作，摆脱错误的信念，追求非传统的道路，并看穿朋友、家人和文化的投射）

恐惧正阻碍你的道路。放手吧。

## **唯一值得走的道路**

在你的一生中，你一直被训练去走别人为你设定的道路。你周围的人将他们的恐惧、不安全感和糟糕的选择投射到你身上。无论你是否意识到，他们都不希望你做得比他们好。所以他们说出话语和反应，希望让你保持在他们的同一水平。

如果你没有注意到，大多数人并不快乐。我的意思是，他们怎么能快乐呢？过多的屏幕时间和低俗娱乐导致多巴胺受体受损。不良的营养、缺乏运动和缺乏目标感导致基础能量低下。这些感知让你无法看到眼前无限的机会。这是一场灾难（对于那些让它成为灾难的人来说）。

问题是，成长是自然的。也就是说，当你更接近自然、“宇宙”或你真实的自我时，脱离那些阻碍你的事物，成长和进步是唯一的选择。一旦你摧毁了基于恐惧的世界观——你就向现实敞开了自己。你的直觉显现出来。你的好奇心显现出来。你内在的男性/女性优势驱动力显现出来。去建造、保护、提供、培养、支持等。

个人和职业发展是“放手”那些对你没有帮助（且在你控制之外）的外部事物的自然副产品。抵制、斗争和坚持这些事物是让你陷入枷锁的原因。

这个困境让我想起了詹姆斯·克莱尔的一个伟大的推文：

> 我有一种怀疑，大多数成年人（75%以上）可以挑选任何技能——除了运动——并且通过每天专注于它两年，就能进入世界前10%。
> 
> 但是几乎没有人表现出那种程度的专注，所以我们永远也不会知道。
> 
> — 詹姆斯·克莱尔 (@JamesClear) [2020年8月9日](https://twitter.com/JamesClear/status/1292574538912456707?ref_src=twsrc%5Etfw)

首先，我会争辩说，大多数人可以进入顶尖1%——而不是10%。（那些将这个“统计数据”理性化的人不会达到1%，因为他们的心理杂音阻止了他们采取行动，放手）。现在有太多优质资源可以在短时间内建立你自己和你的业务。然而，人们被我们上面讨论的瞬间干扰所阻碍。外部刺激让你陷入熟悉和舒适的泡沫中。

*问题是，你从哪里开始？*

我已经让你意识到了这些干扰——你的任务是时刻留意它们。呼吸，让它们过去。顺应自然、积极的生活方式。这需要练习。终身的练习。给一些时间，你的好奇心将开始把你拉向你应该去的地方。

当那个时刻到来（或者如果你想强迫这个过程），你就这样做。

## **3-2-1掌握框架**

获得任何技能或在生活中取得进步的第一步是*学习*。学习是好奇心的下一步。学习是任何特定领域成功的第一步。你的父母为什么在青春期教你，然后你继续在教育体系中花费12-24 *年*，这有原因。幸运的是，在线教育已经并且将继续随着任何你想要做的特定知识而增长。

顺便说一句：学习可能会带来比好处更多的伤害——但这将是另一封通讯的主题。教育是必要的邪恶。它只对那些仍然沉睡的人是邪恶的。保持警觉，并开放地放弃那些对你当前情况没有帮助的学习信息。

回到我刚才说的：

> 真是疯狂，你只需看几个YouTube视频就能比一个拥有12年学位的人赚得更多
> 
> — 丹·科伊 (@thedankoe) [2021年10月11日](https://twitter.com/thedankoe/status/1447493938605068289?ref_src=twsrc%5Etfw)

对于一些人来说，自我教育和追求非传统路径的愿望是自然而然的。他们在孩提时代就反叛，拒绝听从权威。他们唯一的选项是跟随那些自信地在线上发言并与他们兴趣相符的人。大多数人都会这样做——但许多人没有从消费者到创造者的内在压力。

对于那些想要摆脱面前传统路径（你被期望出于预期的恐惧而采取）的人来说——这是为你准备的。这是第一步。如果你不愿意或准备好通过你天生的好奇心追求你的人生工作，就停止阅读。

以下3-2-1框架将为“弄清楚我想用我的生活做什么”的困境带来结构。

### **3个资源——兴趣沉浸**

将自己暴露于与你的兴趣相关的信息的目的，是为了建立联系。联系和模式识别是创造动力的源泉。多巴胺信号提供的难以言喻的兴奋感。

到这个阶段，你需要有一个追求的兴趣。可能是一些你看到别人在做而你也想做的事情。可能是一些会为你的生活带来积极改变的事情。可能是一些听起来像有趣爱好的事情。选一个。你总是可以重新开始。

接下来是3个资源。你将选择一本书、一个长篇资源和一篇短篇资源，开始沉浸在那个兴趣中。

这本书应该在该兴趣领域内成为畅销书或获得高度评价。这将为你提供一个广泛的概述，说明该兴趣如何与其他生活领域相交。这将给你提供一些想法去咀嚼，以及一些值得注意的黄金法则。

长篇形式资源将来自播客、博客或YouTube频道。这将为你打开一个机会的兔子洞，让你可以深入探索。这是你必须拥抱你的好奇心的地方。如果你觉得上帝让你花5个小时观看健身YouTube视频——那就去做吧。这本身就可能是终身充满激情收入的开始。

简短形式资源将来自讨论你感兴趣主题的社交媒体账户。你可以查找“最佳X领域账户推荐”的汇编，或者留意出现在你时间线上的推特账户列表。

有趣的事实：那些看似烦人的推特账户列表之一让我接触到了推特上的改进方面。现在，我成为了那里最大的账户之一。有趣的是，事情就是这样运作的。

请注意，简短形式的内容有其作用，但很容易让你分心，从而无法取得进步。你不需要一天之内看到300多条意见（推文），告诉你该做什么，不该做什么。简短形式的内容有助于你保持方向，并带来潜在的机会，但应该适度。如果你愿意，完全可以完全不用简短形式的内容。你的大脑会感谢你。

### **2个出口——激情测试**

模式识别和将信息应用于现实世界问题的能力创造了激情。“将现实世界与头脑中的信息联系起来”会增加多巴胺水平。你将感受到兴奋。

当你通过你的3个资源——学习时，你必须开始产出，以试图理解你的学习成果。根据[常青技能](https://www.modernmastery.co/post/learn-these-skills-if-you-want-to-thrive-in-2022)，写作和演讲是你的出口。在推特上、在你的日记中，或在一个新的博客上写下来，以激发更多的模式识别，帮助你理解你的学习成果。写一本电子书，为应用程序编写代码，或构建一个真实世界的项目。与你可以信赖的朋友交谈，向他们传授你正在学习的内容，让你的潜在热情在你的声音中展现出来。

你可以通过开始与该兴趣相关的一个项目来进一步这个。一个社交媒体品牌、电子商务商店，或者有货币化潜力的东西——这带来了其他常青技能，如营销和销售。这让你能够连接更多的点。这就是让人们痴迷于建立企业的原因。

### **1个月——建立联系与反思**

你必须给你的兴趣探索留出时间，让它在大脑中沉淀。当大多数人两周后就放弃时，你应致力于这个兴趣一个月。向自己证明，你至少可以学习该兴趣中的一些复杂性。

经过一个月的阅读书籍、消费内容、写作和演讲——是时候反思了。这将确定你是否应该现在放弃，还是继续前进。

> 理解：这与“闪亮物体综合症”无关。一切事物都是相互关联的。如果你继续追求你的兴趣，你将找到你想要精通的领域。你在路上学到的其他知识只会补充这一事业。我的7次“失败”的商业尝试，只在我找到我想要追求的东西时帮助我成功。*如果我没有从其他兴趣中获得的背景知识来使之成为现实，我就不会知道我想要追求它。*

把这些问题写下来，反思你的1个月旅程：

+   这是我想要进一步追求的吗？

+   我是否期待深入研究更多关于这个主题的书籍和内容？

+   这项技能或兴趣是否有实用性？其他人是否在从中获利？

+   我可以用它解决什么问题？我能看到自己将此作为终身事业追求吗？

+   这次探索让我意识到我想要探索的新途径了吗？

我不能告诉你你应该选择做什么。这取决于你。这些问题会帮到你。

因此，我鼓励你使用这个3-2-1过程。最坏的情况是，你花了一个月追求积极的进步，并在未知领域开辟了新的探索途径。最好的情况是，你找到了你将致力于一生的东西。这将导致精通、财富和更高品质的生活。
