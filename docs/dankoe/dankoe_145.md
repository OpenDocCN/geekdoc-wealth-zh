# 社会资本：新富的货币

> [https://thedankoe.com/letters/social-capital-the-currency-of-the-new-rich/](https://thedankoe.com/letters/social-capital-the-currency-of-the-new-rich/)

我一直以为建立企业是遥不可及的。

作为孩子，你不知道有什么更好的。

你唯一相信可能的事情就是你的父母能做到的事情，因为那是你唯一能接触到的信息。随着时间的推移，你形成了关于为什么你*不能*拥有更多自由、更多金钱和更多满足感的小而强大的限制性信念。

那就是为什么我要写这封信。

因为每个人都有那些想法，你知道，那些你告诉所有朋友但从未付诸实践的想法。

“兄弟，我们应该开发一个像Tinder但针对餐馆的应用程序。这样人们就不必挣扎着选择吃饭的地方。”

但你无法构建应用程序、企业或你梦想构建的其他任何事物。

为什么？

你没有钱。

你以为你需要投资者。

你缺乏如此多的知识和技能。

你喜欢最初的想法，但随后你的脑海中充满了有限责任公司、S型公司、隐私政策以及所有这些你不想处理的商业垃圾。

我也有同样的想法。

我为此挣扎了相当长一段时间。

直到那时，我才意识到我并不需要启动资金。我需要的是社会资本。

什么是社会资本？

我们很快就会谈到这一点，但简而言之，*那就是在那些愿意为你提供价值的人中拥有良好的声誉*。

一旦你有了这些，你就有了杠杆：

+   你可以成为一位成功的音乐家，即使没有唱片公司。

+   你可以成为一位成功的作家，即使没有出版商。

+   你可以成为一位成功的设计师、程序员或通才，即使没有雇主。

+   你几乎可以开始任何业务，并拥有使它成功的技能和潜在客户。

+   你不必依赖外部资金、投资或类似的东西。你可以快速组建团队或网络。

更进一步，看看那些仅仅通过在品牌上露脸就能在大公司中获得股权的创作者。想想马奎斯·布朗利是如何成为Ridge钱包的创意合作伙伴。或者初创公司是如何给创作者股权，这样他们就可以接触到受众并快速成长，无需付费广告。

换句话说，社会资本是下一代的货币，而不是风险投资。

换句话说，注意力是新的货币，但仅仅注意力可能会导致浅薄而空虚的生活。

另一方面，社会资本在目的、利润和和平方面都符合要求，因为你知道你在为世界提供价值。

## 社会资本：就像黑镜，但积极正面

你看过Netflix上的那部黑镜剧集吗？

那个被称为*“鼻涕虫”？*

不是吗？

这关乎一个使用社会信用体系的社会。

你的声誉和价值基于你作为个人持有的五星级评价。

每次你与他人互动时，他们都有机会根据与你互动的经验，用1到5颗星对你进行“评分”。你只需在手机上选择一个评分，向上滑动，这个人的评分就会上升或下降。

你的评分决定了你可以生活在哪些社区，可以从事哪些工作，人们是否愿意接受你进入他们的社交圈，等等。这非常具有判断性。正如你所看到的，高4星评分的人通常是富有且势利的人。低评分的人通常是贫穷且暴力的人（大多数时候）。

现在，我来问你一个问题：

这让你想起了什么吗？

创作者经济，也许？

你有没有看到有人炫耀自己有很多粉丝？或者那个女孩因为父母几乎没有粉丝而否认了他们的关系？

在我看来，这有点疯狂。

但也有积极的一面。

### 网络上的三种人

对普通人来说，互联网看起来就像《黑镜》中的地方。

人们赞扬那些不在乎他们的影响者和名人。

影响者和名人会得到特殊待遇。品牌合作。可以接触到他们想要的一切。他们拥有所有的关注……他们是其他品牌发声的营销渠道。

但影响者并不是网络上的唯一人。

他们并不是唯一拥有*力量*的人。

网络上的第二种人是消费者。

他们不是为了贡献在线空间而存在的。他们不是为了提供产品或服务而存在的。他们只是来接受教育、娱乐、启发，很多时候……愤怒。通常是由影响者、名人和政治家引起的。

但还有第三种人：

**创作者**。

我知道创作者和影响者某种程度上是相同的，但我想在这里区分他们。

为了解释方便，创作者可以被称为*价值创造者*。

一个不沉迷于消费到精神肥胖的人，一个不专注于虚荣、认可或看到点赞数增加的人。

价值创造者是指那些基于他们创造的价值建立自己价值的人。他们写的文字。他们构建的产品。他们解决的问题。他们创作的音乐。他们敲打的代码。在社会资本的情况下，创作者的故事还有更多内容。

他们关心的是影响力而非点赞数，深度而非外观，真相而非从众。

为了创造价值，需要有一个观众——无论大小——他们把创作者的创造视为有价值的。

如果一棵树在森林里倒下，周围没有人，它会发出声音吗？

如果一个人创造了一个产品，周围没有人，它会赚钱吗？它会影响他们的生活吗？

不。

1 X 0 = 0

那么，创作者有什么是影响者、名人、政治家和消费者所没有的呢？

### 伦理社会资本的三个支柱

社会资本不仅仅是关于建立观众群。

任何人都可以创建病毒性的TikToks和毫无意义的梗来吸引大众，除了偶尔的笑和尴尬之外，对人类没有任何贡献。

如果你是一位作家、设计师、程序员、任何类型的创意人士，或者只是想以他们想做的事情为生的人，那么你必须建立社会资本。

社会资本 = 分销 + 声誉 + 价值。

这是正确做到的方法：

### 分销

有三种类型的分销：

+   建立——就像建立追随者或通讯录。

+   借来的——就像人们引用你的书或分享你的帖子。

+   购买——就像付费广告或通讯录赞助。

大多数人未能从工作中建立收入来源的原因是他们做了一切，除了尝试吸引客户。

他们开发了应用程序，却无法吸引用户。

他们写了书，却无法吸引读者。

他们制作了音乐，却无法吸引听众。

但如果你仅仅专注于**分销**，你就能让你的创作出现在其他人面前。而且，这是你唯一能够判断它是否体面或有用的方法。

如果没有人喜欢你所做的事情，那么你并不像你想象中那么有技能。

不，这不是关于验证。这是关于你需要收入，这样你就可以全职维持你的激情。否则，生活就会强加给你。你会找借口说，你必须工作，因为没有人喜欢你的工作，这样你就不必放下你的自尊，最终帮助其他人。

话虽如此，每个人都应该**建立**分销渠道。

社交媒体是免费的。写作是免费的。对任何有互联网连接的人来说，这两者都是可访问的。

一旦你建立了分销并利用它来赚取收入，那么再考虑**购买**分销。

如果你想知道如何通过写作建立分销，请查看[《作家训练营》课程。](https://bootcamp.kortex.co)

### 声誉

你可以基于外貌和喜好建立庞大的观众群，但这不会带你走得很远。

你在建立一群生活在生存模式下的观众。追逐下一个廉价的多巴胺刺激。他们看到的是你的外表，而不是你的思想。他们将你视为一种商品，当更有趣的东西出现在他们屏幕上时，你就可以被遗弃在尘埃中。

价值创造者专注于深度、创造力、解决复杂问题，并通过传播有助于理解而非混淆和压倒的见解来实现长期满足。

他们用他们的思想赚钱。

不是他们的时间、外貌或劳动。

其中一个是无限的。其他随着时间的推移而消逝。

其中一个促进了中心化。[其他则增加了熵。](https://thedankoe.com/letters/the-dopamine-epidemic-how-society-failed-to-think-long-term/)

其中一个是人类社会的净正面。其他则将人类社会拖向地狱。

你可以在没有声誉的情况下建立分销，但你会陷入更糟糕的境地。

与其没有用户、读者或买家，不如拥有一个庞大的观众群，他们关心你的一部分，这部分必然会发生变化。他们不关心你的工作，而且在未来几年里，他们也不会关心你。

### 价值

你可以整天写作、设计、建造和创造，但这并不意味着它有价值，也不意味着它值得付费。而且如果它不值得付费，那么在保持对你一天中所有事情的些许控制的同时，独立维持你的激情，祝你好运。

价值 = (人们对你所做的事情关心的程度) X (你解决的问题的规模)

价值创造者不仅仅是为了创造而创造。

他们创造是为了解决真实、具体的问题。

他们不会用酷炫的创业想法来解决不存在的想象中的问题。他们解决人类问题。健康、财富、关系和幸福。他们从个人层面消除改进和进化的障碍。

你可以拥有分销和声誉，但如果你不解决有价值的问题，你就是一个商品，不久之后，你将质疑你工作带来的满足感。你将追逐快速现金，牺牲你一生的作品，只是为了回到起点——就像所有疲惫的商人一样—— wonder where your life has ended up.

## 如何从无到有建立社会资本

让我们明确一点。

大多数人没有足够的钱作为生活或商业中的捷径。

他们看不到通往他们想要的生活的道路，因为他们认为他们需要工作，存钱，到处投资一点钱，等等。

但然后他们的梦想最终和其他人的梦想一样，埋葬在墓地中。他们被责任和消费文化所包围。

他们从未建立过业务。他们从未控制过他们的早晨、下午和晚上。他们陷入了一个永无止境的循环，早上醒来感到压力，与不在乎的人一起工作，回家太累，无法做任何有生产力的事情。

基本误解在于世界已经改变。

我们已经不再处于工业时代，那时唯一的选择是上学找工作，实现美国梦。

钱不是唯一的杠杆形式。

另一方面，社会资本可以用作杠杆，让你赚到你想要的那么多钱。

*通过建立受众群体来建立社会资本*。

这就是你要怎么做（以及为什么这很重要）：

### 1) 学习一个相关且具有未来保障的技能组合

当你建立受众群体时，你被迫学习一套几乎任何人都能从中受益的通用技能：

+   写作

+   说服力

+   产品开发

+   市场营销和销售

+   图形设计

+   品牌建设

+   流量和转化

+   更多

你不会浪费时间学习你认为能帮助你变得有价值的一项技能。你建立一个受众群体，让市场告诉你*它是否有价值*。

你不会在孤岛中学习这些技能。你在现实世界中学习它们。你跳过无用的东西，实施能带来结果的方法。如果你没有取得结果，你很可能专注于错误的事情，意识到这一点本身就是个人和职业发展的旅程。

而且，你不需要许可就可以开始。

你只需简单地在网上开始建立自己的个人品牌。你创建横幅、简介、电子邮件列表，你撰写帖子、线程，你构建并销售产品或服务。

你可以做个人和商业都能从中受益的所有事情。

一旦你为自己取得了成果，你可以向其他人或企业提供服务，从中获得收入。

最好的部分：到那时，你已经拥有了一个愿意为你付费的受众。

第二好的部分：你可以在你喜欢的任何其他技能或兴趣周围建立你的受众。这不必是关于商业或营销的。可以是关于哲学、生产力、健身、编程或任何其他你已经在网上看到、关注和消费的东西。

### 2) 从小、易于访问和免费（或非常便宜）开始

我刚才说的可能会让人感到不知所措。

你可能会觉得你*必须*学习所有这些技能来建立社会资本。

如果你足够聪明，情况并非如此。

这是我会做的事情：

+   **从写作开始** – 你不需要学习视频、音频或图形设计来在社交媒体上写作。此外，写作增强了任何视频脚本、音频脚本或你在网上看到的设计中的文本的影响力。

+   **从单一短形式平台开始** – X、LinkedIn或Threads。你需要一种在人们可以*发现*和*关注*你的平台上生成流量的方式。像博客和播客这样的东西发现性很差。

+   **从单一长期平台开始** – 我推荐通讯。这样你可以保留你建立的追随者的一部分。然后，你总是可以将通讯转换为YouTube或播客脚本……如果你真的想这么做的话。Justin Welsh正在做得很好，仅通过LinkedIn和他的通讯就赚了200万+美元。

为什么我推荐这三个步骤？

因为它们充当了（1）快速获得结果和（2）学习真正重要的技能的强制函数。

如果你希望你的写作具有吸引力和影响力，它必须具有说服力。如果你想建立你的电子邮件列表，你需要对营销和销售有基本的了解。

问题是，你不需要正式学习这些技能。你只需尝试、失败并改进。如果你放弃，那是因为你认为自己没有看到唯一可行的选择。你宁愿花那个时间让自己从你放弃后接受的舒适生活中分心。

### 3) 建立声誉，传播价值

大多数人“等待开始销售”是因为他们认为他们的受众不想被推销。

我为个人和业务发展所做的最伟大的一件事就是在开始时就出售产品。

这样，我有了行动背后的动力。我旨在通过一款帮助他人的产品来赚取收入。

我有*方向*。

我知道，如果我没有收入，实际上我就没有价值。而且如果我的收入没有增加，我的声誉也不会增加。

免费内容、写作、设计以及其他东西并不足以影响他人的生活。

他们需要有意与你进行价值交换。他们需要投资于你的一款能够解决他们生活中问题的产品。因为如果你不能解决他们生活中的问题，对他们来说你就是可替代的，他们不会记住你，也不会将他们的成果归功于你。

从我开始到现在，过去5年里我每个季度都会构建并推出一个产品或服务。这些包括电子书、课程、咨询服务、一本书、那本书的限量版、小组、规划者，现在还有一款名为Kortex的软件。

直到你自己这样做，你才会理解这是多么困难，但我将我所有的社会成长和感知成功归因于这一点。

当我开始时，我没有想到会最终达到这里。但我可以直截了当地告诉你，这只能通过用我认为有影响力的产品或服务来换取金钱才有可能实现。

这就是建立声誉的方式。

这就是“给予价值”的方式。

这就是如何打破社会矩阵，并为互联网上的伪自由市场做出贡献。

如果你害怕金钱或者因为你的有限编程而将其妖魔化，认为它更高尚、更精神，那么我无法帮助你，因为你看不到金钱有多个视角，其中许多都是极其积极的。

就是这样。

建立社会资本。
