# 生产力未来（一种日常惯例来组织你不确定的生活）

> [生产力未来的秘密：一种日常惯例来组织你不确定的生活](https://thedankoe.com/letters/the-future-of-productivity-a-daily-routine-to-organize-your-uncertain-life/)
> 
> 他们说，聪明人可能工作得更聪明，而不是更努力，但创意人士根本不工作。 ——亚历克斯·庞

真是遗憾，平均美国人把过度工作等同于一种地位象征。

或者更糟，他们把它作为一种逃避其他责任的方式。

因为如果你对你的工作认真负责，你会优化创造力，而不是生产力。你会吃得好。每天锻炼。散步。保证充足的睡眠。社交。以计算的方式挑战自己的极限。并设计一个优先考虑你的精神输入和输出的生活。你会明白，4小时的专注工作比12小时的分散工作产生更高的结果。

如果你理解了人的思维，你就会明白人类与动物有一点不同。动物试图繁殖它们的物理身体，它们基因代码中的信息。人类也是如此，但我们还试图繁殖我们的精神身体，我们意识中的信息。我们的身份。我们写书，与他人争论，获取知识，用微小的想法去孕育他人的心灵，这些想法生根发芽（或衰败）。

当过度工作成为你的身份，通常是通过社会分配给你的目标的无意识重复，当你“没有在提高生产力”时，你总是感到焦虑。

但大多数人误解了生产力。他们可能没有明确地说出这一点，但他们的行为却表明了这一点。他们认为生产力是关于数量，而不是质量。他们认为一天中的每一小时都是平等的，他们在一天的最后四小时和第一四小时是同一个人。他们认为生产力是关于处理所有事情，而不是带来所有结果的那一件事情。

他们没有区分“任务”和“杠杆”。他们没有意识到A个人可以每天工作12小时，一年赚5万美元，而B个人可以每天工作1小时，一年赚500万美元。区别在于技能、杠杆和认识。不是你工作有多努力、多长时间或多么有组织。

秘诀在于设计一种创意生活方式。在这封信中，我们将讨论以下内容：

+   随着人工智能恐惧感的增加，为什么你的想法质量比以往任何时候都更重要。

+   创意生活的4个模块和规划你一周的模板。

+   著名创意人士的日常惯例，以及查尔斯·达尔文如何通过最少的工作改变世界。

+   我个人的日常惯例，为什么我要这样做，以及如何创建你自己的。

话虽如此，这封通讯是给那些*想要*这种杠杆作用的人。这不是给那些坚持更传统工作风格的人。我并不是来说服你，我只是在陈述你可能不会从这封信中受益。

你工作的质量取决于你想法的质量。因此，生产力的未来是创造力。随着经济继续偏向利用AI和技术的创作者和创始人，我们的工作方式正在改变。

生产力的未来是创造力。

## 理念流动比以往任何时候都更重要

任何人都可以在浏览器中输入ChatGPT并让它输出一本书、通讯、营销资产、艺术品、待办事项应用代码或法律文件。

然而，并不是有上百万本畅销书或生产级别的软件。工作变得更容易，但大多数人缺乏自主性的事实仍然存在。如果人们在过去10年里没有动力去建立自己的东西，那么他们现在也不会，他们可能会被那些有动力的人利用。

我可能大错特错。但我认为AI永远无法取代作家。当我提到“作家”时，我并不是指这个头衔。我并不是指创造的工具。我指的是写作背后的本质。

+   从100个想法中筛选出1个

+   构建一个有创造性的叙事目标

+   吸引一群人的注意力

+   通过有意识的体验与他们深入联系

+   为未来世代记录高信号信息

现在，我一点也不反对AI。

事实上，我们将在下一个月在[Kortex](https://kortex.co)实施它（它非常酷）。

问题是，如果你不是作家或者不从作家那里学习，你的AI输出可能让你失望。如果你不知道自己想要建造什么，而且即使AI让这个过程变得容易（尽管过去10年已经变得容易），你仍然太懒惰去学习，那么祝你建造“任何东西”好运。如果任何生活领域的巨大成功需要10年才能看到，而AI将其缩短到5年（甚至1年），这并不会改变大多数人在两周后放弃的事实。世界会快速发展，这会让那些准备不足的人感到害怕。

人工智能缺乏执行功能。

这是一个需要大师的工具。一个需要通才的专家。当许多人已经被他们的父母、学校和重复的任务导向型工作变成了工具，难怪人们如此害怕被取代。因为他们确实会被取代，除非他们开始像人类一样行动。自主性。有意识的体验。风险。错误。发现。感知。

例如，詹姆斯·帕特森为他的小说使用代笔作家。

作者詹姆斯仍然创作情节、基调、写作风格和最终输出，并对与分发、销售和社区相关的所有事物都有大局观。

代笔作家或AI消除了产生这种写作所需的时间和劳动，以及将之呈现在他人面前所需的辅助任务。大师仍然需要是一个通才。他们仍然需要是一个企业家。他们仍然需要愿景、故事和价值。

但如何让这些事物产生影响？

你就不能让ChatGPT吐出一个愿景、故事和价值吗？

你可以，但除非你不想受默认AI写作方式（内置偏见）的影响，否则你仍然会做大多数其他人正在做的事情。你仍然缺乏自主权。你将被归入有偏见的模型同质化的输出中，在市场上，你将变成一种商品。你将无法*维持*任何有意义的结果。

答案在于想法流动。

罗伯特·格林在写一本书之前会读100本书。

创始人在一个改变一切之前有1000个想法。

创作者在1个让他们一夜成名之前会发布1000次帖子。

更多的想法会导致更多的指数级事件。没有发生任何事情，然后一切事情都发生了。

你不能通过让ChatGPT给你提供一个100个想法的列表来找到这些想法。因为你知道吗，你仍然缺少整体的愿景和故事。

*你缺少经验。你缺少意义。你缺少目的。*

*你缺少一种旨在实现你愿景的前进式生活方式，你可以从中提取和连接想法以创造价值。你缺少上下文。*

人们不是因为你拥有信息而跟随你。他们是因为你拥有*传说*而跟随你。如果我的社交媒体全是那些默认的AI生成声音，我的手机可能已经在湖底了。就像人们不相信系统一样，他们也不会相信AI，而志同道合的部落将是你在其中找到意义的地方。受众仍然是你能构建的最重要的事情之一。你仍然是这个细分市场。特别是如果加密货币允许你验证并拥有你的作品，无论你是否使用AI。

在发展的螺旋上升中，摆锤正在回到我们祖先部落的本质。

## 创意生活方式设计的4Cs

<picture fetchpriority="high" decoding="async" class="wp-image-2428">![](../Images/2a3c68986b10f9883899f116993f274a.png)</picture>

当我说我每天工作4小时时，人们会给我一个奇怪的表情。

表面上看，我好像做了很多事情。每天在所有平台上发布帖子。每周一篇通讯和YouTube视频。管理并构建Kortex。一本书。产品。很多东西。

我的问题是：我做的事情有什么疯狂的地方？

我的意思是，当我开始建立受众时，我会在视频游戏比赛之间写帖子，与人交谈。如果我把实际工作的时间算进去，可能每天只有一小时。我非常擅长拖延，这迫使我专注于重要的事情。

即使现在，我也会把写一本书、写每周通讯和写社交媒体帖子纳入我早晨的前两小时。如果不是在写书，那么我就在构建另一个产品或项目。这些都是我的基本杠杆。如果我完成了这些，那么我一天的工作就完成了。

有时，我想要创造某物的欲望变得非常强烈。我忍不住要工作8-12个小时。问题是，我无法强迫这些强烈的季节。这是清晰的自然积累——通过投资一个愿景——直到我别无选择，只能冲刺。

我们稍后会讨论我的其他常规。

但是的，我必须承认，其中有些是谎言。

我从未真正把阅读、散步、小憩或训练视为工作的一部分。我阅读是为了寻找想法。我散步是为了理解那些想法。我小憩是因为我喜欢。我训练是因为我喜欢能量和健康。

如果我不做那些事情，我的工作和数量质量将不可能实现。我不是为了更多而优化，我是为了优化带来我实现目标所需结果的正确数量。我可以用这些简单的习惯做更少的“工作”，比那些每天工作12小时的人有更大的影响。再次强调，创造力是生产力的燃料。

所以，如果考虑到我工作的时间，那么我每天都在工作，因为我的工作是我的使命。这是我生活中最重要的事情。

话虽如此，你大大高估了开始时你需要做的工作量。理性一点。你知道人们也有像你一样的责任。你知道人们不是从12小时开始的，他们是从早上1小时开始的。你只是喜欢找借口说你的情况很特殊。

现在的问题是：我如何充分利用那1小时的专注工作时间？

通过设计一种创造性的生活方式，并随着你能够全职投入这些活动而逐渐增加在这些活动上花费的时间。

你需要4个习惯。

把这些想象成你一天中的模块。你不必按顺序做它们，你只需要做它们。

如果你愿意，这里有一个[新的每周计划表](https://app.kortex.co/public/document/e930cde4-6021-47e8-b25f-085c8679e9f7)，它每天都会引导你完成这些任务（还有一个地方可以写下日常任务）。

### 习惯1：清理

我最喜欢的清理思绪的方法是将消极的想法和经历转化为5-10个可以用于工作的想法。

我倾向于在早上开始写作之前做这件事（技术上，这就是写作），但我知道很多人在睡前做这件事，这样他们早上就有东西可以写了。

静坐于一张白纸前，探索你的思绪。

思想可能不会立即出现，但它们会出现。

另一种清理思绪的方法是简单的脑图日记。

+   记录下你能从脑海中挤出的每一件事。

+   将它们组织成不同的类别（如工作、烦恼和不重要的事情）

目标不是对这些想法做任何事情。目标是意识，因为意识是自我纠正的。

### 习惯2：消费

简单明了：

如果你不去消费，你就无法创造。

我不是在谈论像无脑僵尸一样在社交媒体上消费无尽的内容。我是在谈论*学习*。寻找你工作的想法。就像你的祖先在物质世界中做的那样，你在精神和数字世界中做。

+   阅读一本长书

+   阅读一篇长文章

+   浏览5分钟的精选社交媒体时间线

+   学习你购买但忘记的课程

+   听播客或讲座

但不要忘记一个绝对关键的步骤：有一个地方可以捕捉那些想法到你所创建的相关项目中。

在Kortex中，你可以在捕获功能中这样做。写下想法，并输入“@文档名称”将其保存到特定的项目中。市场上没有其他具有此功能的捕获功能。[我在这里为您演示，并提供每周内容模板。](https://youtu.be/nktY727HVKQ)

重点：如果你不学习……那么，你就没有学习。因此，你在生活中没有前进，也没有开启新的机会。

### 习惯3：创造

核心的人类欲望是创造、扩展和超越。

其他任何东西要么是干扰，要么是次要的目的，它会把你引向那些欲望。

“创造”是大脑的一个关键功能，大多数人都没有利用到。

每个人都在消费信息（他们获得心理脂肪）。有些人写下信息（他们减轻了心理负担）。少数人将信息综合成对他人有帮助并产生影响的东西（他们有力量在心理车祸中拯救某人）。

换句话说，大多数人没有意识到他们对世界的贡献。他们缺少目的。深层次的目的，而不是肤浅的。

你需要一个项目去构建。

一个可以连接想法的思维结构。

### 习惯4：连接

当你有一个有意义的项目去构建时，不是父母、老师或整个社会分配给你的项目，你将以完全不同的方式看待生活。

当我写一本书时，它与社交媒体帖子显然完全不同。

我的思维扩展以*匹配那本书的框架*。

由于我有一个想法的*锚点*，我在日常体验、我消费的信息和我的对话中注意到更多。

我的思维可以迅速穿越复杂的推理，理解我之前无法理解的想法。我可以表达出AI无法表达的独特思维线路，因为我必须训练它以相同的框架才能接近。

不幸的是，大多数人没有优化对模式识别（这个4C框架能帮助的）的优化。这样做，他们很少变得好奇或着迷。他们没有在鼓励他们的思维形成新形态的项目上投入精力。

## 如何以及为什么以某种方式结构化和平衡我的深度工作块

> 最终目标是拥有一个如此美丽设计的生活，以至于工作、休息和娱乐不再存在。你的一生都融入了一个单一而不断发展的愿景中。

没有正确的工作方式。

显然，Hormozi每天工作12小时表现良好，而世界上许多最著名的创意人士几乎不工作。

目标是实验。为了收集各种观点，体验它们，并丢弃那些不适合你独特的目标、兴趣和价值观的部分。这被称为生活方式的*设计*，是有原因的。你不能简单地复制别人的设计并期望它让你致富。如果你不能将创造性过程应用于影响一切的主要事物，你怎么能称自己为有创造力的人呢？

<picture decoding="async" class="wp-image-2429">![](../Images/7b59d5574a1b7d001cb0917c69ea2e35.png)</picture>

我想关注查尔斯·达尔文的日常作息，因为它反映了我自己的许多原因。达尔文的日常作息非常有序，包括在他的书房中进行多次散步和工作时段。

早上，他在7点开始一天，先进行短暂的散步，然后吃早餐。他会在上午8点到9点30分开始他的第一个——*最富有成效的*——工作时段。在接下来的工作之后，达尔文会休息，阅读信件并与家人共度时光，然后从上午10点30分开始，直到中午，他觉得自己已经“完成了一天的好工作”。

现在，达尔文会在他著名的“沙滩小径”周围进行他第二天的散步，这是一条四分之一英里的砾石小径，达尔文专门为了思考和反思而创造。午餐和阅读报纸后，达尔文会休息并听他的妻子大声朗读。下午4点，他开始了他当天的第三次散步，通常也是围绕沙滩小径，有时会有人陪伴。

我分享这个观点给你的目的是，查尔斯·达尔文，这位所有人中最为人熟知的人，他在生活中出人意料地工作很少，却比今天大多数沉迷于生产力的人在生活中取得了更多的成就。

这一认识至关重要，因为它让你可以将工作时间与产生的结果分开，你可以开始*思考*和*实验*你的日子，而不是采用适用于不属于你自己的目标的模型。

如你所猜，我喜欢达尔文日常生活中的原因是[散步的重要性](https://thedankoe.com/letters/the-power-of-long-walks-daily-walks-will-change-your-life/)。那些深入思考的时刻是他工作的不可或缺的一部分。专注的工作时段与定期的散步相结合，使他能够在紧张的知识工作与休息和沉思的时段之间达到平衡。想象一下，如果他能够接触到人工智能。

达尔文通过这种方式写了*19(!)本书*，并发展了他的进化论理论。

### 我的日常安排 – 释放与约束熵

我喜欢将我的日子分成三个部分：

+   **低熵** – 几乎没有不确定或混乱的空间。清晰的头脑，深入的专注，高意向。

+   **中熵** – 允许自己与他人和手机互动。

+   **高熵** – 受控的混乱，这让我第二天早上可以充分利用众多想法。

我通常在早上5:00左右自然醒来，有时更早。在温暖的月份，我首先进行10-20分钟的散步。在寒冷的月份，我发现直接开始工作对我更好。冬天就像一个建设季节。当我散步时，我只是专注于“我回来后要写什么。”

我的第一个工作块是90分钟，我将其用于我最具杠杆作用的任务。通常，这是一本书、一个产品或一项业务。这类事物只需要构建一次，而不是永久性地系统化和维护。现在，我正在写我的第二本书。这也是我通过头脑风暴5-10个相关想法来“烘焙”我一天中的“清晰”模块的时候，如果需要的话。

在一顿简单方便的早餐后，通常只是水果（以免我的思绪变得混乱），我的第二个工作块持续45-90分钟。这时，我会做高优先级任务，这些任务可以增长和维护我的工作。对我来说，这包括写通讯的一节（20-45分钟），尽可能多地写内容想法（10-30分钟），并在各个平台上交叉发布（大约15分钟）。这高度取决于我的大脑产生想法的能力。在星期二，我会用录制视频来替换这个块。我还在[2小时作家](https://2hourwriter.com)中教授这个过程。

注意：这也就是我在Kortex的主要角色。我负责引流并执行增长策略。

对于那些对此有疑问的人，是的，我每天都写我的通讯和内容。我不喜欢“主题”日。我每天都训练。每天都写作。并且每天都做我认为是我生活中重要的事情。即使在度假期间。即使在节假日。这就是给我带来快乐并保持我理智的原因。

在完成这个工作块并收获更多成果后，就像达尔文一样，我也觉得自己度过了一个充实的工作日。这时，我会进行第二次散步，通常不穿衣服，去晒中午的太阳（这对许多与健康相关的话题都有帮助），并消费某种形式的有声书、视频、讲座，有时甚至直接从我的手机上阅读。这时，我会产生关于我正在写的最佳想法。我将所有想法都捕捉并连接到相关的项目中（这样它们就不会丢失），在Kortex Capture中完成。

在这次散步之后，有时在散步过程中，我会做一些行政任务，比如回复邮件、给合作伙伴发消息，并检查[高级Discord](https://kortex.co/premium)。

然后，我去健身房。由于我每天训练，我的训练量分布在7天中，所以每次训练时间都很短。最多30-40分钟。高强度。训练躯干、手臂、腿部，重复。每组肌肉4次。主要是交替身体部位的超级组。健身后，我吃一顿蛋白质含量较高的午餐。

健身房标志着我一天中的关键区分。这是我从工作过渡到接下来几小时轻松休息的地方。此时我并不真正追踪时间。当我回来时，我会做我的大脑想做的事情。我会散步，阅读，小憩，当有想法时我会写一点，还会发信息给人。这时我会安排大多数会议或电话。当我处于高强度阶段时，这些小时通常都用于专注于那个需要我注意力的项目。

大约在5:00，我会和我的编辑和/或女朋友一起吃饭。周六是约会之夜。除非我有一些非常紧急的事情需要做，否则我不会在这之后工作。如果我无聊，我可能会再出去散步，但大多数时候我会非常懒惰。最初让我自己这样做并*停止*专注于工作是很困难的。

晚上9:00-10:00上床睡觉。

就这些。这就是我的生活。

我希望它在某种程度上给你带来了一些灵感。

至少，如果我能提供至少一件可以实验的东西，我会感到很感激。
