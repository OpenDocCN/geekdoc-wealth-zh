### 如何（一般而言）赚到100万美元

在我们开始之前，让我们分析一些实现每年100万美元的几个现实路径。

$1,000,000 / 12 = 每月$83,333

$83,333 / 30 = 每天收入$2,777

现在，有几种方法可以做到这一点。

+   每天卖出18个150美元的产品

+   每天卖出111个25美元的订阅

+   每隔一天签下一个5000美元的客户

+   每4天签下一个10000美元的客户

+   结合两者。每周1-2个客户和每天几件产品销售。

+   或者任何其他组合和价格点，以达到100万美元。

如果你选择客户服务路线，在一段时间后你可能需要一支团队（或者将服务产品化到你可以自己处理履约）。

如果你选择产品路线或订阅路线，比如软件或会员制，你需要大量的流量。

你如何获得足够的流量来实现那么多产品或服务的销售？

在一个允许人们购买你产品的着陆页上，如果转化率为2.5%，你需要每天有720人访问该页面，才能在价值150美元的产品上实现18次销售。

那么这720人从哪里来？

社交媒体、广告、SEO，或者某种形式的网红营销或口碑传播。

所以，如果我们假设你在社交媒体方面很在行（是的，社交媒体是一项技能，而不是基于算法的运气），你可以：

+   每个YouTube视频获得10-50,000次观看

+   每月获得社交媒体上50k至100万次曝光

这意味着在那么多人中，你只需要720人点击你的着陆页，就能每年赚得100万美元。

在数以万计的人中，要求720人并不算多。

*当然，大多数人会满足于每年低于100万美元的收入。*

对于那些不习惯那种收入或基于薪水的劳动的人来说，认为它能那么简单似乎有点“离谱”。这就是你的雇主如何通过执行他们赚钱的特定过程的一部分来在你支付的基础上获得多倍收益。
