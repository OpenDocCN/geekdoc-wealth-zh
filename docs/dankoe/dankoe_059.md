# 为什么自我帮助正在毁掉你的生活

> [`thedankoe.com/letters/why-self-help-is-ruining-your-life/`](https://thedankoe.com/letters/why-self-help-is-ruining-your-life/)

自我帮助让你痛苦。让我们改变这一点。

这是一个最近一直在我脑海中的话题。我想在开始时就明确说，自我帮助本身并不坏。在很多情况下，它是好的。

从阅读自我帮助书籍中获得能量和信息需要正确引导。

## 自我帮助的问题

你有没有买过一本书，期望你的生活会发生改变？别撒谎。

书籍的标题、副标题和社会赞誉吸引了我们的注意力。我们在脑海中建立了虚假的期望，带着兴奋订购书籍，并开始消费它。

事情是这样的…你和我都知道，仅仅阅读是改变不了你的生活的。

我们最终陷入了一个有良好意图但最终只是为了让自己感觉良好的阅读循环中。

这被称为心理自慰。

仅仅阅读关于自我提升的内容，但并没有真正提升自己。

阅读以让自己感觉良好，感觉自己实际上在进步，也就是说。

实际上，你的大脑喜欢收集那些多巴胺的刺激，并进一步强化心理自慰的循环。

## 自我帮助与自我提升的区别

自我帮助是外在动力——依靠外部因素来激励你。

对于像大卫·戈金（“去他妈的动力，你所需要的只是纪律”）这样的人，你可以开始看到这里的问题。

“你所需要的只是纪律，”这部分是正确的。

我们从自我帮助书籍中获得的外在动力有三个原因是有用的：

+   我们是有动力的（这可以用作纪律的初始动力）

+   我们拥有一个可以改变我们生活的概念

+   我们理解如何实施这个概念

魔法发生在最后一个要点上。它构成了自我提升的基础。

这就是两者的区别。

自我帮助 = 不可持续的动力

自我提升 = 实施自我帮助概念，看到结果，并培养内在动力

## 会让你陷入困境的事情

并非所有的自我提升都是平等的。

许多人未能理解的是，*某些*概念可能对你不起作用。我们是人类。我们每个人都在以自己独特的方式不同。

这就是创造反自我提升心态的原因。

采取这种心态的人实施了自我帮助概念，但它对他们不起作用，然后他们继续告诉每个人它不起作用。

同时，也有人对完全相同的概念深信不疑。

以健身房为例。

有些人喜欢卧推，有些人则不喜欢。

有些人说这对胸部增长是坏练习，而另一些人则说这是最好的。

除了机械、遗传、新手收益和肌肉插入之外…

你怎么能知道卧推对你来说是一个好的练习，除非你一直坚持做？

如果你实施了这项练习，持续增加重量，并且坚持了 6-12 个月——那么你就可以决定它是好是坏了。

这适用于任何其他可以改善你生活的概念。

尝试一天，觉得它很糟糕，然后放弃，这会毁了你。

## 意识的基础与直接经验的房子

所有真正的自我提升都源于意识，并通过直接经验得到巩固。

自我提升信息让你*意识到*你可能存在的问题。它给你解决问题的步骤。

按照那些步骤（直接经验）进行一定时间，这是提升你生活的方式。

如果你已经给了足够的时间，但仍然没有看到结果，意识就会再次发挥作用，循环继续。持续改进。

如果你有一个模糊/焦虑的头脑，而一本自我提升的书告诉你去冥想，你需要持续冥想以看它是否真的对你有帮助。如果没有，那也行。

自我提升的概念充当了提升你意识的“助推器”。

直接经验帮助你理解它对你是否有用。

如果它对你有用，内在动机开始建立。

如果它对你没有用，回到意识步骤。这是大多数人跳过的一步。

在这个阶段，最糟糕的事情就是将自我提升整体否定。

这就是你最终过上平凡生活的方式。

给我找一个你渴望成为的人，他们生活中没有连续改进的基础。你做不到。如果你能做到，这些人不值得你效仿。

如果你随着业务的增长而不提升自己，你的业务将会受到影响。

如果你随着关系的增长而不提升自己，你的关系将会受到影响。

如果你没有意识到它所造成的隐藏问题，你的健康将会受到影响。

## TLDR

如果你只是为了感觉自己在做些什么而阅读自我提升书籍，那么自我提升是糟糕的。

如果你随着时间的推移发展了意识和内在驱动力，自我提升是好的。
