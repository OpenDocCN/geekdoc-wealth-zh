# 这个单一的决定将决定你大部分的生活。

> [https://thedankoe.com/letters/this-single-decision-will-determine-most-of-your-life/](https://thedankoe.com/letters/this-single-decision-will-determine-most-of-your-life/)

你的决策质量决定了你的生活品质。

但有一个决定决定了所有其他决定。

对于我自己来说，我在年轻时做出了这个元决策。我记不清确切的时间——自从我10到12岁起就一直在思考——但有一个明确的时间点，我知道默认的生活不适合我。我全身心地厌恶它。

我做出的最重要的单一决定是，不要整天为了微薄的收入而做一件我讨厌的事情，同时对着电视大喊政治和体育。让我的所有时间都被工作耗尽，能量被我不愿意选择的任务耗尽，钱消失在我甚至不知道需要支付的账单上。

许多人因为无法做出决定而毁掉了自己的生活，更不用说做出任何决定了。普通人犹豫不决，他们没有意识到这一点背后巨大的问题。

### 你正在错误的道路上狂奔。

当你出生时，你就被放在了默认的道路上。

就像一列货运列车沿着你之前几代人铺设的轨道飞驰。在你走出青少年时期时，你已经积累了20年的动力。生存模式已经深深印在你的脑海中。影响你决策的程序。问题是，这是一个非常基础的操作系统。上学，找工作，退休。这没有什么非凡之处。除非你做出大胆而深思熟虑的决定来改变轨道，否则你永远不会达到你的潜力。

所有决定的决定是拒绝默认的道路。

为了回应冒险的召唤。为了最终开始写第一章。为了离开教程，开始第一级。

那就是你的生活开始的时候。

你生活的结果是由一系列10到12个大胆的决定决定的。十二次轨道的改变。如果你不做出这些改变，你将陷入平淡无奇的生活。你停滞不前。这就是现代生活的巨大陷阱。

想象一下你在一份你讨厌的工作中。

你可以选择不再忍受这种情况，或者继续犹豫不决，两年后醒来，却发现你的生活没有任何改变。事实上，在那段时间里，情况可能变得更糟。

犹豫不决的问题在于，不做出决定，你就让别人为你做决定。这就是生活。你必须*做*些什么。你必须每天朝着某种方向前进，但大多数人并不掌握自己的轨迹。

让我们称这个状态为“闲置状态”，当你的大脑开启并运转，但没有处理任何有意义的事情。你在做些什么，但实际上并没有做什么。你讨厌它，但不足以改变它。

### 你默默地珍视平庸。

你的决定揭示了你的价值观。

尽管你可能会认为自己有高尚的价值观，但我希望你深入地看看*你实际上做了什么*。

如果你每个周末都出去，你重视社会地位。如果你每晚都叫外卖，然后看Netflix，你重视舒适。如果你有一份工作，你重视安全感。

价值观表明了对你来说什么是重要的。美好的生活源于力量集中在对的价值观上。如果你重视地位、舒适或安全感，那完全没问题，但请诚实地面对自己。你大部分的能量都花在了哪里？朝着什么价值观？如果你有任何，你为更好的生活投入了哪些资源？

大多数人都在把时间、精力和金钱浪费在下水道里。他们可以用来改善生活的货币一旦出现就消失了。他们将在多年后（那是个谎言……许多人从未自我反思）回首，意识到尽管他们感到疲惫和接近崩溃，但他们却什么有意义的事情都没有完成。

如果你想要改变你生活的轨迹，你需要对这个事实感到厌恶。你需要残酷地意识到你几乎所做的一切都将付诸东流。你需要感受到这种痛苦深入骨髓，同时在你心中保持你的负面和正面潜力，并看着正面占据上风。

### 与这个问题同坐

用接下来的7天时间只思考这个问题：

如果我继续做出同样的决定，我的生活最终会怎样？

很可能，你得到的答案不会很美好。

为了使这一点更加具体，拿出一张纸。现在就做。不要忽略这一点，因为你只是在浪费时间阅读一些互联网上的智者。然后，写下你今天所采取的每一个单独的行动，以及你本周将采取的每一个其他例行和重复的行动。

更进一步，去一个公共场所。一个杂货店。一个快餐店。一个有大量普通人聚集的地方。只是观察。观察他们。虽然我确信他们是好人，但你真的想成为他们那样的人吗？你真的想让你的眼睛里出现那种死气沉沉的表情？那种缓慢而疲惫的步伐？尽管你可能会说服自己你不想，但如果你继续保持犹豫不决，你正在走向的就是这样的生活。

可能需要7天或365天，但如果你观察得足够久，并且真正诚实地面对自己，你将别无选择，只能彻底改变影响你决策的价值观念。

通过简单地观察大众，你会意识到健康、商业和自我实现是不可或缺的。

### 没有错误的生活就是死亡

错误的决定总比没有决定好。

但有那么多决定要做。

你应该专注于你的职业生涯还是精神修行？你应该学习编程还是学习市场营销？你应该寻找你的目标还是寻找伴侣？你应该尝试生酮饮食还是糖分饮食？

我们正处于一个奇怪的历史时刻，生活中有更多的选择可以改变你的生活，以至于你感到不知所措，选择放弃所有选项。

事实上，无论你选择哪个选项，只要选择了，即使结果可能灾难性，都没有关系。选择变得果断意味着你不会做出一个决定就草草了事。相反，你会从更高的意识层面继续做出决定。

我在生活中做出的某些决定，别人可能会认为它们是愚蠢的、疯狂的或者不值得做出。

几乎5年前，我因为不想花钱搬出前女友的地方，所以在客卧的地毯上睡觉。我节俭且过于节俭。当时我正在经营我的副业，我决定“见鬼去吧”，把我拥有的东西都处理掉，除了几包衣服，然后和一个在网上认识的朋友搬到墨西哥。

两个星期后，我感觉自己犯了我一生中最大的错误。我讨厌那里。天堂般的地方也让我感到厌烦。我需要一个根据地。我的事业开始取得进展，但我无法在Playa Del Carmen的干扰下发展它。

因此，我做出了另一个决定。我给在亚利桑那州的朋友打电话，问他是否愿意和我一起租公寓。但不仅仅是一个普通的公寓，这是一个我一生中负担不起的两倍价格的地方。我自信我是一个能够在巨大压力下让我的事业成功的人。

结果证明，那12个月是我一生中成长最快的时期。我变得最瘦、最平静、财务最稳定、最充满活力。通过两个大胆的决定，我的生活发生了彻底的改变。我变成了一个完全不同的人。

到现在为止，我已经看到了犯错误的无尽力量。在弄清楚我不想要的生活中。任何“疯狂”的想法都不再疯狂，它成为了生活中任何好事的催化剂。搬到另一个国家？这是一个很大的风险，但如果你是一个果断的人，你可以随时搬回来，因为你可以决定提升你的技能集并赚更多的钱。果断改变了你对可能性的看法。

当那个公寓租约结束时，我和现在的编辑以及Dakota Robertson和JK Molina——我在推特上认识的朋友搬到了德克萨斯州。当Dakota因为签证问题在6个月后不得不离开时，那些“错误”真正开始发挥作用。

我们搬回了亚利桑那州，租了一个更昂贵的地方。一年后，我买了我的第一套房子，再次，我深感后悔。但这次我亏损了110万美元，而不是几千美元。我不想在这么早的时候就被束缚住。我甚至都不想住在那个地方。两个月后，我把公寓重新放回市场。幸运的是，最终我只损失了大约十万美元。虽然痛苦，但这是值得的，因为我获得了无价的清晰度——*我想要什么*。

这只是我犯错的开始。我现在正处在另一个伟大的“错误”过程中。这次我投入了300万美元，但那是清晰度的代价，而这种清晰度可以在极短的时间内将这一切都赚回来。

你做出的决定越大胆，犯的错误就越多，这会积累更多经验，确保随着时间的推移，每一次决策都能犯下影响较小的错误，就像弹道导弹一样，将你精准导向成功。

没有犯错的生涯即是死亡。
