# 11个我希望知道的洞察，以使我的单人企业成功

> 原文：[https://thedankoe.com/letters/11-insights-i-wish-i-knew-to-grow-my-one-person-business/](https://thedankoe.com/letters/11-insights-i-wish-i-knew-to-grow-my-one-person-business/)

当我第一次开始这段商业之旅时，是出于好奇心。

我很警觉。

我观察了我不想成为的人，也观察了我想要成为的人。

在我年轻的时候，大约15岁左右的时候，我对这两种对比有了非常清晰的认识。

一方面，人们是：

+   不健康

+   不满意

+   不愿意改变

一种与每周工资挂钩的生活方式。

另一方面，人们是：

+   生活在他们的梦想中

+   传播伟大的信息

+   激励人们变得更好

在社交媒体的早期，他们大多数都是创作者。

有些人会给你经典的，“但他们可能都深陷于不快乐之中！”

*立即关闭任何潜在的机会，并回到他们不快乐的生活方式*

你可能是对的，但如果你了解我，我喜欢无偏见地提问（尽可能多，只要我的自负允许）。

意味着，为什么我不能在生活的各个方面都变得富有？

我必须直接体验那种闪亮的生活方式，并与那些人交谈，才能对其进行判断，不是吗？

如果他们很快乐，只是被大众误解了怎么办？

如果他们不满意，为什么不去改变呢？

这导致我尝试他们推荐的一切。

我尝试了健身房，并爱上了它。

我尝试了新的技能，并爱上了学习。

我失败了多次，偏离了道路，但最终达到了一个点，我的知识、技能和经验已经积累到了“成功”的程度，按照我的标准。

对于我未来的小而稳定的行动，我能够建立一个单人企业，让我能够：

+   研究我喜欢的事物

+   用创造性的方式写关于他们

+   完全控制我工作多少

+   完全控制我可以住在哪里

+   建立那些曾经帮助过我的产品和服务（这样它们也能帮助你）

这种生活方式的唯一缺点就是，你完全控制。如果你没有导航这种自由所需的技能，你的心智可能无法处理这种混乱。

在这封信中，我想提供独特且可操作的教训，帮助你实现同样的目标。

任何人都可以做到这一点，我希望有一天能看到一波个人，将他们作为人类所拥有的价值打包起来，并将这份礼物献给世界。有意识的创业。

## 1) すべてトラフィックとオファー

<picture fetchpriority="high" decoding="async" class="wp-image-839">![](../Images/1fafdb077c251ebaeb52d93f97ebd9cf.png)</picture>

商业是一场创造伟大的产品或服务并将其展示给正确的人的游戏。

是的，营销、推广策略和其他事情中有很多细节……但人们太过于沉迷于这些。如果你周六晚上（交通高峰期）去一个繁忙的酒吧，尽可能多地吸引人们（提供机会），你可能会得到性。太多的人从一开始就追求完美，变成了无意识的（财务）禁欲者。

许多人不理解的是，这仅仅是一个游戏。

这不是为任何人保留的。

你可以为之前购买的产品做联盟推广，写一条推文来推广它，付费让人们转发，如果你足够聪明，仅凭这一点就能获得全职收入。由于产品很好，你应该能够从付费推广中获得比支出更多的收益。这就是广告的全部意义。投资回报率。

这是一个初学者方法，但可以帮助你在旅途中建立受众。这样，当你经验足够丰富时，你可以为你自己的产品或服务做同样的事情。但是，随着时间的推移，你将不必为流量付费，因为你将用优质内容建立它。

当然，在建立品牌的同时，你可以暂时推迟货币化，但大多数人之所以这样做，是因为他们被告知要这样做。

## 2) 你需要3个杠杆

一旦你通过持续拉动正确的**杠杆**来建立杠杆作用，你将获得其他人无法想象的自由。

让我们假设你有什么东西要卖。如果你没有，这应该一直萦绕在你的脑海中。你必须长时间思考，沉浸在你这样的人正在销售的东西中，并在灵感迸发时准备好全力以赴地构建。

自由职业、咨询、辅导或小组服务。

或者是一门课程、物理工具、会员资格、付费通讯或如上所述的联盟推广。

如果你不知道如何做，就去学习。

现在，你需要平衡三件事。

成长、培养与支持、货币化。

这些是杠杆。这意味着你需要学习、发现并执行这些领域的基本行动。这些行动几乎每天都在进行，一旦你找到对你业务有效的方法，就应该系统化。

**杠杆 1) 成长**

如果你的关注者或品牌没有持续增长，那就存在问题。解决问题。

大多数人没有成长，因为他们认为他们高于那些导致成长的内容风格。

内容，以及将流量引导到该内容，是你成长的方式。

市场中的90%都是初学者。

这意味着你顶部漏斗社交平台上的内容应该主要是与你的选择兴趣或专业知识相关的基本建议。

你的社交媒体品牌应该足够广泛，以吸引这些初学者，这样你就可以引导他们走向更高级的内容。

旁白：你确实可以在社交媒体上非常细分，我只是个人不喜欢这样做。你把自己放在一个框子里，给自己更少的调整空间，而且无法谈论各种交叉的兴趣（例如，生产力、自我提升和商业重叠得非常好）。

**杠杆 2) 培养与支持**

如果你是一个价值创造者（而不是一个娱乐者、模因账户或评论账户），那么你的**整个工作**就是教育你的受众。

如果你的社交媒体吸引了广泛的人群，你需要教育和培养他们，直到他们购买你的产品。这就是你改变人们的生活、建立权威并展示你不仅仅是外表的方法。

对于价值创造者来说，一份通讯、引诱物或其他长篇内容是必要的。这就是你展示你所知并教授你兴趣的地方（与你所销售的内容相关）。

跳过这一教育步骤的品牌在货币化他们的受众时总会遇到困难。

**杠杆3）货币化**

不要欺骗自己，你在这里是为了赚钱。如果你是刚开始，那正是你考虑做这件事的全部原因。你越诚实地对待自己，你在销售时就越真实。这没什么错，你需要钱。不要让别人让你觉得自己想要控制自己生活的东西是错误的。

最终，你将在你的工作中找到更深的意义和目的，但无论如何，你必须保持这种状态。

你需要测试不同的促销、角度以及你的推广方式。

你会每周通过Twitter帖子推广一次吗？你只会在你的通讯中推广吗？你会在相关帖子下每天推广一次吗？

一旦你知道什么能带来销售，你就可以开始系统化它，让它成为你日常的一部分，这样收入就会更加可持续。

在[数字经济学](https://digitaleconomics.school)中，我们帮助你系统化你的品牌、内容和产品，以遵循这三个社会杠杆的支柱。这是大多数个人企业所忽视的，因此它们变成了商品。

## 3) 质疑一切

很不幸，“质疑一切”已经变得如此主流，以至于它作为建议的影响力已经丧失。

问题是推动我在商业上成功的关键。

*“你必须细分市场！”*

谁说的？

*“你必须在达到10,000名关注者之后才能开始销售！”*

谁说的？

*“你不可能每天工作2小时就能年收入100万美元！”*

我还没有完成这个，但WHO SAID?!

离开你的限制性信念，证明它们是错误的（为了你自己和你的追随者的利益）。

这就是乐趣开始的地方。游戏提出一个挑战，将你带入心流状态。

要固执。

当人们说你不能做你想做的事情时，**你要第一个找到方法**。

这就是你会获得建立你名声和财富的知识。

## **4) 解构一切**

意识和创造力是信息时代成功的关键驱动力。它们是你如何思考、生活和更深层次地产生影响的方式。

从另一方面来看，无意识和思想狭隘是通往平庸的保证途径。

创造力是连接意识之点的。

<picture decoding="async" class="wp-image-840">![](../Images/69dd252439e97ebc8dc7a7742a823884.png)</picture>

通过直接意识到生活中的那个时刻，你经历了某件事。

通过不关闭你对冰山任何一角的思维，你会变得更加有意识。

每件事都是冰山一角。

每件事都是一个由部分组成的整体。

每件事都是一个事物。一个思维单元。意识的产物。部分和整体。

这些部分也构成了其他整体，这意味着它们是相互连接的。

那就是生活的全部。宇宙是一个整体，它将自己分成了无限的部分。这些部分本身也是整体。人类是整体。他们是自我发现的探测器。当我们发现自己时，宇宙也在发现自己。没有什么需要做的，但一切都可以做。享受这个过程吧。

你通过提问深入到冰山之下。

**整体**：一本关于生产力的书

**部分**：生产力的原则，提高生产力的策略，吸引你注意并帮助你理解你所学习的故事和轶事。

通过将这些记录下来，你能用其中一个有趣的故事或轶事来提出另一个观点吗？

在[数字经济学](https://digitaleconomics.school)中，我谈论了电子舞曲音乐如何像创作者一样。

每个电子舞曲音乐制作人都有自己的独特声音，他们混合不同的声音，并引导人们经历一系列的高潮和低谷。

创作者有自己的独特风格，他们混合想法，并引导人们经历他们生活的起伏。

我也可以用电子舞曲现场表演来展示生活的起伏。如果DJ只播放音乐而不休息，那会很无聊。如果你的生活全是高潮，没有低谷，那也会很无聊。

练习分解你看到的下一件事（字面上说任何事）。提出问题。

## **5) 不要放弃，转型**

每个人都在担心闪亮物体综合症。

你需要理解的是，你可以从复合知识的位置“重新开始”。

你可以从更好的位置重新开始，拥有可转移的技能，以及激发你“重新开始”的新技能。

如果你学习网页设计，看到使用这项技能的机会并全力以赴进行文案写作，你为什么不呢？

那不是闪亮物体综合症，那是技能堆叠。从长远来看，它会让你赚更多的钱。

如果你仍在朝着同一个宏伟而广泛的目标努力，比如财务自由，你需要在路上进行转型。

如果你放弃那个宏伟的目标，你就是在放弃你的未来。这不好。

当然，放弃一项技能是因为你看到了机会。你的初始技能不会消失，尤其是如果你会在下一个事业中使用它。

知识会累积，技能会累积，一般的好习惯也会累积。

这就是为什么我喜欢个人品牌和单人企业。你的兴趣会改变。你的技能、知识、经验、意识和网络会复合增长。你是一个人类。人类会进化。如果你遵循这条道路，你的业务也会随着你一起进化。事实上，它是加速进化的载体。

不要放弃，转变方向。

## 6) 复制成功之路

成功是一场智能模仿的游戏。

<picture decoding="async" class="wp-image-841">![](../Images/f58f331c4266c7b1559c64e01a24eee2.png)</picture>

不要表现得好像你从出生以来就没有模仿过周围的一切，只是因为你认为自己“原创”、“聪明”或“机智”。

你很可能就是这样，但自我只能存在于与其他人的关系中。你*认为*你是谁只是你一生中暴露于环境、刺激和训练的心理构造。

模仿是任何新手都能遵循的最重要的技能，就像你是一个刚出生进入商业生活的白板孩子。

+   用你最喜欢的作者的语气练习写作

+   用你最喜欢的账号发布的主题作为灵感

+   滑过你的竞争对手在他们的着陆页上针对的问题

+   分析为什么某些帖子会走红，以及它是否带来了收入

+   看看大家都在卖什么（意味着它能卖出去），然后想想你如何能利用你独特的经验创造出自己的东西

+   了解一下让你兴奋的想法，不要高亮，然后不看文本就写下来。通过回忆你从记忆中学习到的内容来使它成为你自己的东西

在我继续更多的哲学狂言之前，让我们回到正题。

## 7) 最有利可图的利基市场是你自己

传统品牌、营销和销售建议正走向终结。我们已探索了现实的这个缝隙，并区分了好坏。

推销漏斗和机器人写作曾经有效，但现在效果不如以前。

如果你想领先于潮流，*产品化你自己*。

通过自我反思和迈尔斯-布里格斯性格测试来了解自己。

然后，把你自己当作你的客户形象。

追求你的目标，研究你的兴趣，并在线向你的过去、现在和未来的自己写信。

没有人能复制真实性。

你是独一无二的，如果你不是外部环境的奴隶。你所说的就是你，而不是别人所说的你。

## 8) 自我意识是关键

自我意识（或积极的自我意识）将帮助你比其他人更快地学习大多数技能。

如果你是一位创作者、个人品牌或其他形式的独立创业者，你会理解持续高质量内容的重要性。

这意味着你必须掌握人类行为和心理。

在这种情况下，你就是你最好的老师。

为什么那个帖子吸引了我的注意？

为什么我喜欢并转发它？

想想看，你希望别人对你做同样的事情。

## 9) 先行动，再学习

信息是丰富的。

信息时代人们犯的一个巨大错误是让知识超越执行。

是的，你可以学习任何东西，但你真的在学习吗？

处理这个问题最好的方式是制定一个项目规划。

+   制定一个通讯简报

+   制定一个产品规划

+   制定一个服务规划

+   制定你的生活规划

然后，开始。

你的思维将准备好进行模式识别。你的生活质量将飞跃，因为你有一个*项目*来应用你在日常生活中所学、所经历和所消费的知识。

创建一个项目，然后寻找实现该项目的具体知识。不要陷入教程的陷阱。

## 10) 创建你的职业生涯

60%的新工作尚未被创造出来。

意思是，我们不知道它们会是什么样子。

我们所知道的是，它们都在互联网上。

价值创造者正在左右创造新的工作。

互联网允许人们追求新的目标，这意味着生活中有新的道路可以选择，这意味着有更多的问题需要解决。

教育行业无法跟上，因此价值创造者开辟道路，并将他们的学习成果以内容、课程和其他形式的教育媒体的形式提炼出来。

我见过音乐制作教师、治疗草药专家、自信教练，甚至生物黑客教练（针对增强身心和精神的性能提升药物……如迷幻药）。

你如何创建自己的职业生涯？

通过追求你自己的广泛兴趣，将它们转化为自己的类别，并利用它们解决问题。

如果你对心态、生产力、商业感兴趣……这些都可能属于“生活方式设计”、“个人主权”或其他某个领域的“利基”。然后，你*创造*这个新类别背后的意义、理论和哲学。

我的业务是单人企业。我将灵性与你作为神圣的“创造者”联系起来，并根据自己的个人经验形成自己的观点。

## 11) 缺乏经验不是借口

令人难以置信的是，自我会进行怎样的心理体操来说服你你不能做什么。

如果我告诉你开始在线写作以及它带来的20多个生活和商业好处，你的思维会如何反对？

“我不擅长写作。”

“只有影响者才能扩大观众。”

“我在建立观众之前只是自由职业。”

虽然所有这些都是愚蠢的，而且比真实的还要多，但最后一个有点太荒谬了。

你告诉我你不想提供免费信息（因为你缺乏经验），但你却要为某人的业务做免费或付费*工作*，那里有收入在考虑？你怎么向自己解释这一点？

写作是你澄清思想的方式。写作是你澄清你所知道的方式。写作是你开启提出*真正*（不是复制的）解决方案的潜力的方式，你可以用它来赚取更多的钱。

学习技能，运用技能（对自己，因为个人品牌需要企业需要的每一项技能），教授你所学的，然后将其出售。

做与别人相反的事情。

每个人只是遗漏了完整画面中引导他们采取行动的部分。
