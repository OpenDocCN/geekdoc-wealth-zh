# 如何像天才一样思考（所有知识的地图）

> 原文：[https://thedankoe.com/letters/how-to-think-like-a-genius-the-map-of-all-knowledge/](https://thedankoe.com/letters/how-to-think-like-a-genius-the-map-of-all-knowledge/)

自由个体的标志是他们能够得到他们生活中的想要的东西。

这需要他们学习如何学习，如何赚钱，以及如何思考。

最后一个——如何思考——是最重要的，因为你的思想会影响你学习的内容，你的行为方式，以及你是否能够制定一个策略，让你实现大多数人甚至不会想到实现的大目标。

但问题是：你不需要聪明才能像天才一样思考。

事实上，你不想像大多数聪明人一样。聪明人通常并不那么聪明。他们通过记忆事实来获得一份安全和高薪的工作，同时过度分析他们想要做的事情的风险。愚蠢的人经常绕过聪明人，因为他们愿意承担那些改变他们生活轨迹的愚蠢风险。

天才级的思考不是关于书本知识。

这关乎清晰。

那就是为什么大多数人最终过上了他们讨厌的生活，而且找不到出路。他们的思维无法筛选出压倒性、焦虑和压力，所以它继续打击他们，直到他们完全放弃。

好消息是，思考是一种技能，而技能是可以练习的。

我一直钦佩乔丹·彼得森的清晰表达，艾伦·瓦茨背后的魔力，以及丹尼尔·施马赫滕贝格的清晰。我想与你分享一个尚未被构思的最强大的模型。这是所有世界伟大哲学、科学发现和心理模式的综合，如果你花足够的时间去研究它，将彻底改变你的生活。

无论你是在试图弄清楚你想要做什么，成为一位精神觉醒的僧侣，还是建立一个价值十亿美元的公司，这个模型都将极大地缩短你实现这些目标的速度。

## 这张所有知识的地图令人震惊。

> “我不必同意你说的每一件事，但至少我应该尝试去理解它，因为相互理解的相反，简单地说，就是战争。”
> 
> – Ken Wilber

这可能听起来很明显，但你要思考才能解决问题或实现目标。

你需要识别一个问题。

你需要深入理解它。

你需要设定一个合理的目标。

你需要制定并执行一个策略来实现目标。

听起来很简单，但极其复杂。大多数企业失败是因为它们没有建立一个真正解决问题的产品。大多数关系失败是因为问题从未被识别。大多数人失败是因为他们根本看不到摆脱困境的方法。

因此，如果你能提高你的思考能力，你就可以更快地解决问题或实现目标。当你将其映射到10年的时间线上，你很容易就能看到，你可以实现比大多数人一生中实现的多10倍。

让我向你介绍肯·威尔伯的AQAL模型，他是一位美国哲学家，他将其一生的工作都致力于将所有知识和经验综合成一个全面的理论。

[![](../Images/2e437a4f6bfc1cc928bbf474cdb004de.png)](https://substackcdn.com/image/fetch/$s_!QXDl!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Ffd44b6c1-e4d5-4486-9011-11353dbec5d6_1099x736.jpeg)

虽然威尔伯表示他的模型不是终极的，但我可以自信地说，这个模型是那些“隐藏的宝石”之一，大多数人之所以不关心，仅仅是因为他们无法超越自己的世界观。

AQAL代表所有象限，所有层级。

威尔伯通过数十年的研究和三年的休假，发现所有知识和经验都映射到四个相互关联的象限中：

+   **个人内在**（左上角）：你的个人思想、情感、信仰和意识

+   **个人外在**（右上角）：你的行为、行动和物理大脑状态

+   **集体内在**（左下角）：共享文化、价值观和群体意识

+   **集体外在**（右下角）：系统、结构和社交机构

对于成为天才级思考者来说，这个模型的作用在于，当你面临任何形式的挑战时，你可以系统地从所有四个象限或视角来审视它。

更重要的是，大多数人只从一个象限思考，并宣扬这是“唯一正确的道路”。资本家关注经济体系，右下角。科学唯物主义者关注物质和行为，右上角。宗教关注共同的信仰，左下角。斯多葛主义者和佛教徒关注内在世界，左上角。

事实上，所有这些观点都是有效且重叠的。将自己锁定在一个教条的世界观中，会让你陷入困境。某些问题最好从某些角度来解决，如果你无法接触到正确的角度，你将在生活中经历不必要的痛苦。

为了使这一点更清晰，让我们通过一个困难的例子来解释。

我认为可以说，大多数人都在努力弄清楚他们生活中想要什么。这是一个难题，90%的人都会卡在这里。

如果你听说过日本的IKIGAI概念，那么在这个背景下，你可以将AQAL模型看作是那种概念，但更加强大。

现在，思考是与自己的对话。

愚蠢的思维是一种变成争论并迅速结束的对话，因为你只抓住一个答案。

天才思维是涵盖多个视角并准确综合它们最佳部分的对话。

为了开始弄清楚我们想要从生活中得到什么，我们在所有知识和经验地图的每个象限中提出一系列问题。

这里有一些我想出的例子（注意，这需要时间。真正的思考是困难的）。

拿出笔和纸可能是个明智的选择。

**个人内部（左上角）- 你的内心世界**

+   我的核心价值观是什么？对我真正重要的是什么？

+   哪些活动让我感到最真实和充满活力？

+   什么恐惧或限制性信念阻碍了我？

+   当我静下心来，摆脱内心的喧嚣时，我的直觉告诉我什么？

**个人外部（右上角）- 你的行动 & 能力**

+   我的自然才能和培养的技能是什么？

+   我的行为揭示了关于我真实偏好的什么？

+   我小时候最喜欢的东西有哪些？

+   我正在采取哪些具体步骤？这会引导我走向什么样的生活？

**集体内部（左下角）- 文化 & 社会背景**

+   我的父母或宗教领袖对我的成功有何期望？

+   我的朋友们是如何影响我的职业抱负和生活道路的？

+   我所使用的语言和接触到的数字文化是如何影响我的机会的？

+   我被哪些共同价值观所吸引？我找到什么是有意义的？

**集体外部（右下角）- 系统 & 结构**

+   在当今世界，有哪些就业、职业或创业的机会？

+   教育体系或互联网是如何塑造我的道路的？

+   哪些技术或社会趋势正在创造新的机会？

+   我面临哪些系统性障碍或优势？

通过使用这个模型，你将“我该如何度过一生？”这个问题转化为一个全面的地图，揭示了你的真实自我、实际能力、个人召唤、经济现实和文化背景如何相互作用。

而不是最终得出“我不知道”或“我对什么都不感兴趣”的答案，你可能会得出一些真正的认识。

你可能会发现，你更重视创造力而不是财务安全。

你可能会发现，害怕让父母失望阻碍了你的真实愿望。

你可能会注意到，当你无法停止谈论其他有趣的任务时，你可能会推迟某些任务（比如从生活中去除或委托出去）。

你可能会意识到，技术让你能够学到更多，做得更多，并创造出一个在过去不久的过去不可能的职业道路。

这些都是你可能自己没有意识到的深刻、改变生活的认识。

虽然这很强大，但这只是这个模型的一半。

## 为什么聪明人如此愚蠢

> 你可以说，人类所选择的，是遗传学所选择的，是遗传学——我们的遗传学选择了极端的神经可塑性，以及拥有更多重大软件升级的能力，这些升级可以改变我们的能力，而无需硬件升级。
> 
> – 丹尼尔·施马滕贝格

当我第一次遇到这个时，我感到震惊。

这听起来如此合理，以至于我感觉我的思维扩展到了一个全新的思考维度。

我们已经涵盖了AQAL的“所有象限”部分，但没有涵盖“所有层次”部分。

所有层次指的是个体和集体可以经历的意识发展阶段。

当谈到像天才一样思考时，这一点至关重要，因为大多数生活冲突或“不可能”的问题都源于人们处于不同的层次。

我们在讨论螺旋动力学和自我发展九阶段时已经深入讨论过这一点，所以我想要关注这4个关键层次，这将改变你对世界的看法。

[![](../Images/8c026276206fead7528924a51bda35d3.png)](https://substackcdn.com/image/fetch/$s_!Snjt!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F78cd2529-ea13-4820-9c9a-06d2e8131678_781x715.png)

这些发展水平展示了一个人或文化的主导价值观。换句话说，这些水平显示了人们、文化或系统认为的中心和重要的事物，这些影响着所做的决策以及如何感知和解决问题。大多数人居住在前三个阶段，而非常少数——为了这封信的方便，我们称他们为天才——落入第四个阶段。

**1) 前现代**

这些个人、文化或系统遵循既定的权威和传统。他们的思维是黑白分明的。只有“一种正确的方式”，这通常涉及服从上帝或符合正确和真实的事物。

**2) 现代**

他们重视科学、个人成就和竞争。你可以在大学、企业结构和自助教学中最好地识别这一层次。具有现代世界观的人坚信成功应该基于功绩和结果。

**3) 后现代**

后现代主义者重视相对主义思考。每个人的真理都是有效的。包括所有人并确保他们平等。

每个这些阶段都有好的一面和不好的一面，在当今世界，后现代思维已经病态化，导致了像DEI、性别政治和试图摧毁所有等级制度这样的现象，这是愚蠢且不可能的。

这前三个阶段的关键在于它们不能持有多个观点。它们是对的，而你是不对的。这使得它们难以超越自己的阶段思考，甚至更难以解决导致人类有益进步的个人和系统性问题。

这就是为什么聪明人非常愚蠢。

[![](../Images/a259d23d73ba5aa3bd542ef98bee6fa8.png)](https://substackcdn.com/image/fetch/$s_!cXTE!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F93320fe5-881c-41e3-aef8-60e6351f51f8_800x450.jpeg)

**4) 二级**

通过足够的努力和发展，你可以达到二级思考的阶段。

在这个阶段，你能够回顾过去，从所有先前的视角中提炼真理，开启一个充满复杂性、系统思维和意识的世界。

这并不是关于黑白分明的“我是对的，你是错的”，而更多是关于细微的“最佳解决方案来自综合所有观点”。权威很重要，功绩很重要，平等很重要，但许多人可以在心中长时间保持这些矛盾，直到它们得到调和。

天才思想家明白这些是阶段，而不是你余生都拥有的静态身份。他们经常充当不同阶段之间的“翻译者”。

如果问题是与业务相关，他们可以戴上CEO的帽子，比那些永远戴着圣经敲击者帽子的人更快地解决问题，原因很明显。

如果问题是个人关系，对女性心理的mansplaining可能会有所帮助，但根据*情境*，它并不是同理心的最佳替代品。

如果问题是撰写社交媒体内容以吸引和帮助自助听众，你必须了解他们的心理，以及如何最好地引导他们的思维以达到下一个发展阶段。

当涉及到确定你生活中想要什么时，接受你正处于现代发展阶段，可以让你退后一步，注意到你的不足，并在追求更好生活的行动中考虑到它们。

这是一个复杂的话题。

我们可以就此话题谈论数小时。

我对你的挑战是在你自己的时间里，通过你自己的好奇心去追求这种知识，因为它将只会增强你掌控自己生活的能力。
