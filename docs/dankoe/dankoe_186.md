# 自律实际上很容易

> 原文：[https://thedankoe.com/letters/self-discipline-is-easy-actually/](https://thedankoe.com/letters/self-discipline-is-easy-actually/)

如果你必须强迫自己去做，你将会失败。

这就是大多数人关于自律的错误理解。他们听到那个词… *自律…*，然后他们脑海中就会播放出戴维·戈金斯跑步和尖叫的艰苦锻炼的蒙太奇。他们认为他们需要鞭策自己去做那些能带来生活中美好事物的工作。

“自律的痛苦必须超过懒惰的痛苦”这句话有很多真理，正如挣扎是值得讲述的故事的一部分一样，但大多数成功的人并没有讨厌整个旅程。事实上，对他们来说，这是一件有趣的事情。他们*想要*去做那些艰难的事情。这指向了一个巨大的洞察：

*你已经对那些你应该追求的目标有了自律。*

这就是思维的工作方式。

> 目标是由你的环境植入你心中的
> 
> 你的目标成为你现实世界的过滤器，塑造你关注的焦点和你忘记的事物
> 
> 你的大脑帮助你注意到并记住有助于实现那些目标的信息
> 
> 自然地，你会朝着那些目标行动，并且你的身份得到巩固

### 自律是身份的一个特征

一个从小就认同自己是一个好学生的人（因为他们的父母告诉他们应该追求那个目标）不需要强迫自己每天学习12个小时。这只是他们的本性。

一个经历了分手并且厌倦了低自尊的健身者不会在吃健康食物或每天去健身房上感到挣扎。事实上，如果他们*不这样做，那会让他们感到痛苦。因为如果不这样做，他们离目标更近不了，而且由于那个目标与他们的身份紧密相连，他们感觉自己的生存受到威胁，因为确实如此。人类在物理和概念层面上（遗传和迷因，如果对我们重要的事情受到威胁，我们会感到威胁，无论那是一只宠物狗还是一种宠物信仰）。

这就是有趣的地方。

一个游戏玩家不需要强迫自己长时间盯着电脑，一次连续几个小时地表演。一个热爱电视剧的人会躺下来，一口气看完整个系列。一个拖延工作的人——因为那个目标与他们不符——会不停地刷手机，因为他们的*目标是不要实现目标*。

这些人并不是“懒惰”，他们可以说是自律的，只是对那些能带来即时回报的目标。这就是自律，不是吗？对目标的持续努力。而且，普通人对于平庸的生活是如此地坚持不懈。是的，大多数人只是没有意识到，他们对于舒适的渴望超过了他们对于改变的渴望。事实上，改变才是他们的目标。

“但是丹！那些很容易，也很舒服！做起来不需要努力！”

当然，对我来说，每天写作是很舒适的。不去健身房对我来说比去健身房更痛苦。我*尝试*像年轻时一样玩游戏，但我做不到。它们无聊且过度刺激。我有更重要的事情要做。

人类行为是目的论的。控制论的。心智是一个实现目标机器。我们用我们的身份作为透镜来诠释世界，而植入我们心中的目标塑造了塑造我们是谁的行为。

如果我们能理解这个过程，或者逆向工程它，我们就可以利用它来实现几乎不可能的目标。

你之所以阅读这篇文章，是因为你正在努力强迫自己变得有纪律。但那永远不会奏效。无论你是在尝试赚更多的钱还是准备考试，如果你遵循这个过程，你就不必担心分心或拖延。

### 悬崖是实验室。

你想要变得有纪律的原因是你已经耗尽了当前的生活阶段。

你开始注意到你开始停滞不前了。

你开始注意到每一天都和前一天一样。

你开始注意到，如果你继续做同样的事情，你将无法实现你想要的生活。

你处于悬而未决的状态，你不再想在那里了。

这很痛苦。但你误解了这种痛苦，并想到了第一个出现在脑海中的解决方案：*更加有纪律*。这就是大多数人陷入强迫自己做一些他们不想做的事情的循环，两周后放弃，回到他们曾经发誓要反对的生活。

疼痛是变化的信号。

走出去的方式是深入其中。意识到你当前生活的痛苦。因为一旦你所在之处的痛苦超过了你想要到达之处的痛苦，纪律就不再痛苦，因为你不需要强迫自己去做。

第一步是写下：

+   你对你当前生活的一切不满（现在我不想听关于积极思考的事情，你是错的）。

+   如果你不变更这些事情，你将会有什么样的生活。

然后，静下心来一周。深思熟虑。去散步，让你的思绪与那些想法翩翩起舞。过了一段时间，如果你不逃避痛苦，你将开始你人生的下一章。

如果那不起作用，我们还可以做更多的事情。

### 纪律不是建立起来的，而是发现的。

我已经在健身房训练了大约12年了。

我已经连续写作了大约5年。

我每天已经走了大约1万步了。

对于上述所有事情，我第一次尝试养成习惯时，我多次失败。我记得被邀请去散步，觉得那是为老年痴呆症患者准备的。我*证实了那些帮助我实现保持现状目标的思想*。我不想做那些事情，但感觉不得不做。当我最终养成习惯时，它与其他时候我试图强迫自己时的感觉不同。它们并不“难”做。它们是实验和发现的副产品。

所有这些习惯都因为三个具体原因成为了我生活的一部分：

+   我深刻地意识到我生活中一个痛苦的问题

+   我寻找证据来支持我理想中的未来版本

+   我改变了我的物理和数字环境，以巩固我的新身份

+   我在每周中划出时间来做那些事情，就像我打算余生都要这样做一样

在中学时，我的自信心极低。我非常高而瘦。我觉得自己很尴尬，所以我想改变这一点。我利用这种痛苦作为自我教育健身和营养的动力。我记得我在去学校的公交车上读过《巅峰表现的营养时间表》。我开始在房间里做俯卧撑，开始上放学后的体重训练，我的健身房之爱就是从那时开始的。

为了改变我的环境，每天放学回家后，我都会直接去电脑前，观看2-3个我最喜欢的YouTube健身博主。他们会给我提供计划、锻炼和饮食建议。他们是我作为青少年的榜样，他们塑造了我今天所做的大部分事情。不要忽视社交媒体的积极力量。

在整个这段时间里，我没有随意选择一种饮食或训练计划，强迫自己坚持下去，仅仅因为有人告诉我这样做是好的。对生活不满意的人的共同特点是他们允许别人的自信建议塑造他们生活的很大一部分。

我将这段时间视为一个发现和实验的时期。

我在收集愿景。我在试驾我未来自我的各个部分。

我读那本营养书是因为它听起来很有趣。我忍不住好奇。它可能解决我的问题。当我阅读它时，我想*尝试*我正在学习的一些新颖的策略，比如像一些奥运运动员那样，在锻炼前吃50克简单碳水化合物和25克蛋白质。（这就是我读大多数书的方式。我不会强迫自己读任何东西，我也不要求自己持续阅读。我读那些我忍不住要读的书，我也能接受长时间不读书。）

当我看健身博客时，我发现了一些新的锻炼方法，我总是兴奋地想要尝试它们。这并不困难。这不需要强迫。尝试可能改变我生活的事情很有趣。随着时间的推移，我所学到的所有小事都变成了对健身和营养的深入理解。

我成为了自己的老师和教练。

这里的问题是（1）大多数人并没有意识到他们生活中有一个痛苦的问题，所以当他们看到可能的解决方案时，他们的头脑不会释放多巴胺；（2）他们没有通过实验来学习，所以他们从未取得必要的进步，以作为保持他们动力的反馈循环。

### 如何构建一个身份

**第一步是认识自我**。你是谁决定了你认为什么重要。你必须首先认识到，无论你是否意识到，你都在追求一系列目标。这些目标塑造了你如何解释现实，你通常注意到那些证实你当前生活方式的事物。

你的大脑运作的目标源于条件反射，而条件反射源于你的环境。你的父母、老师和文化已经让你走上了一条通向死胡同的道路。

**第二步是策略性不协调**。为了避免陷入一种感觉迷失的状态——悬而未决，你让现实中的你和理想中的你之间的差距变得更大。通过注意到如果你不改变，你将走向何方，你培养了对当前生活方式的不满。

通过意识到你生活中的问题，你为模式识别做好了准备。你的大脑开始确认改变是正确的事情，并帮助你注意到那些有助于实现改变的信息。

**第三步是环境工程**。取消关注那些不提供帮助你过上更好生活信息的人。扔掉你在凌晨3点允许自己吃的食物。拿出一张纸，详细规划你这一周要做什么，精确到小时。每周进行回顾和迭代。你需要那个反馈循环。

沉浸在你提供尝试机会的人、地方和事物中。即使你不同意，也要关注新的账户。在亚马逊上滑动，直到你找到一本听起来有趣的书，即使你最终只读了前几页。意图才是最重要的。意图是你为你的大脑提供方向，使其重新连接自身。

**第四步是自我实验**。因为未通过实验解决的问题并不是真正解决了。与其被分配另一个目标，你尝试一切，直到找到那一件你无法摆脱的事情。然后，你重复这个过程，直到它成为你的一部分。

感谢您的阅读。

– 丹
