# 为什么害怕失败正在毁掉你的生活

> 原文：[https://thedankoe.com/letters/why-fear-of-failure-is-ruining-your-life/](https://thedankoe.com/letters/why-fear-of-failure-is-ruining-your-life/)

学习是人类经验的基础。

但是，99%的人做错了。

不是因为他们不知道如何学习，而是因为他们已经忘记了如何学习。

我们在这里谈论的是真正的学习。

为了这封信，像记忆、知识囤积和鹦鹉学舌这样的行为将不被视为学习。

学习是经验的结果。

人类经验不仅仅包含文字。

它是一系列的感觉、光线、声音、味道和其他区分，形成一个单一的*视角*。就像时间的一瞬间。是瞬息万变的当下意识的一个片段，在不到一毫秒的时间内就过去了。

文字是现实之上的一层。

现实是由符号、隐喻和能量组成的，我们为了理解我们作为人类所关注的对象而对其进行了标记。

学习在哪里发挥作用？

斗争。

失败。

人们都回避而不是接受的事情。

我们为什么回避它？因为我们不理解失败是什么。否则你为什么会害怕它？我们没有完整的画面，没有完整的视角。这只能通过失败本身来找到。你看到我在说什么了吗？

由于学习需要失败，请理解这封信是为了提高意识而写的。你不能对不知道的事情采取行动，如果这封信让你意识到失败的重要性，我的工作就完成了。其余的取决于你。我不能为你失败。

知道人们讨厌他们不理解的东西，对许多人来说，失败是一种自动的、无意识的、预先编程的“讨厌”。

<picture fetchpriority="high" decoding="async" class="wp-image-791">![](../Images/4521605559b0ba9de9e5b5e208ae77f8.png)</picture>

*真正的学习的第一步是改变你对失败的看法*。

即，不要评判、假设或期望它不存在。失败是一个低频能量源。当专注于挖掘（将这种能量带入你的意识中）时，相关的感受就会在你的身体中显现，你将这种能量标记为“好”或“坏”。

失败是存在的。意味着，如果你不修复你与它的潜在编程关系，拖延和缺乏满足感将统治你的生活。

为什么我们必须经历失败、斗争和接受挑战才能真正学习？

因为当你处于从斗争中产生的“不好”的意识状态时，你的大脑就准备好接受教训。它是开放接受解决方案、内化它并将这种学习锁定以备下次使用的。

意味着，你必须直接体验来自斗争（阻力）的视角，以将对立面（突破）带入现实。

这种观点在底部。与之相关的感受不是我们应该回避的东西。

## 失败的档案

<picture decoding="async" class="wp-image-781">![](../Images/27fad4918f1a8cf998f6bf2e184e364d.png)</picture>

每个成功的人似乎都要经历一个失败的阈值。

也就是说，在学会成功之前，一个人在追求特定目标时必须经历一定数量的失败。

我个人在赚得第一笔1000美元之前，失败了7+种不同的商业模式。

“丹，如果你能回到过去，你会改变任何事情吗？”

这总是一个有趣的问题，因为它暗示你想要避免失败并保持舒适的那个“秘密”，而失败本身就是秘密。

现在回想起来，不会。我不会回去改变任何事情。如果我这样做，我就不得不再次失败，经历这个循环，最终回到同一个地方。

如果我没有在创造设计上失败，我就不会学习使设计更好的方法。这导致了在Instagram上创建数字艺术页面。

如果我没有在数字艺术页面上失败，我就不会学习网页设计。[这导致了多次自由职业的尝试。](https://thedankoe.com/i-failed-at-7-online-business-models-and-turned-it-into-a-full-time-creative-income/)

如果我没有在自由职业上失败，我就不会去学习品牌、文案和社交媒体。这最终导致了失败的电商品牌。

如果我没有在电商上失败，我就不会得到网页设计的工作。这导致了自由职业的成功，转向社交媒体，并一直保持一致至今。

到最后，我积累了一整套通过斗争精炼的教训，使我想要追求的任何事情都能成功。

<picture decoding="async" class="wp-image-790">![](../Images/8a01891dbbcbc5eed63d9fa6c10052cb.png)</picture>

这是一种在任何“白手起家”的成功中都可以看到的模式。白手起家意味着他们没有继承财富。去听听你最喜欢的任何人的起源故事播客，他们会告诉你同样的故事。失败是不可避免的，是必要的，也是你找到的“快速致富”的最近似物。

在任何事业中，成功都是一个不断投资于你的失败作品集的过程，直到没有其他可以失败的地方。也就是说，直到你的投资累积起来，你才能负担得起成功。

## 如何快乐地失败：已知与未知交汇之处

人类在故事中找到意义。

我们通过故事来理解世界。

在我们不知道的情况下，我们的潜意识会“填补”我们感知到的一切的空白。

当你接近一个新人时，为什么你会感到焦虑？因为你的大脑已经决定了结果。

当你长时间做一件特定的事情，比如在健身房做同样的锻炼，为什么你会感到无聊？因为你的大脑已经知道该做什么。

生活是一场包含无限游戏的比赛。

你选择玩的游戏将决定你从赢得这些游戏中获得的结果。

有什么陷阱？

游戏意味着有失败的可能性。

更重要的是，当你开始持续玩一款新游戏时，你将“失败”得更频繁。你不可能通过赢得每一场比赛就从1级升到100级。

> 阳刚之性想要感受到生活在边缘的极乐，如果他不敢自己去做，他会在电视上、在体育赛事和警察剧中观看。 —— 大卫·迪达

游戏是令人兴奋的。它们是有结构的、你积极参与的故事。这就是你如何在生活中找到意义：通过沉迷于那些导致满意结果的比赛。

*关于* *这句话，不要把“阳刚”和男性或女性混淆。每个人都有阳刚和阴柔的一面。* *这是一种能量，在现实的不同方面表现出不同的特征，而不仅仅是人类**.*

重点是，追求更多、征服以及从赢得战斗中获得的利益是阳刚之气的体现。

男人和女人一样，尤其是随着技术解决某些问题（反之亦然），女性承担更多阳刚活动，每个人都可以：

+   爬上企业阶梯

+   围绕他们的兴趣建立企业

+   在具有挑战性的体育竞赛中竞争

认为一个人可以是100%阳刚，或者认为女性应该是100%阴柔是愚蠢的。每个人都有一个需要在自身内保持的平衡。大多数男性倾向于阳刚，反之亦然。

我会把这个话题留到另一天，但世界正在融合在一起。有些人可能会预测它将融入一体。我不在乎人们如何自我认同，我只是喜欢观察并将事物拼凑在一起（而不是让它让我愤怒到以摧毁这些事物为基础来建立我的身份）。

当我构建（m）时，我会感到充满活力，但我知道何时该放松并庆祝（f）。秩序与混乱。比喻的，不是字面的。

当天气炎热时，你会感到凉爽。当天气寒冷时，你会感到温暖。有些人喜欢大部分时间都保持温暖，有些人则喜欢寒冷。当男性取得重大商业胜利时，他们会喝啤酒庆祝。平衡。

重点是，为了过上充实的生活，每个个体需要通过参与战斗来保持阳刚的平衡。也就是说，玩游戏。需要有悬念。即使你不认同自己有“阳刚的一面”，忽视你身上的这一方面也是不明智的。

阴阳。

> 当一个人的所有相关技能都用于应对某种情况下的挑战时，那个人的注意力就会完全被活动所吸收。没有剩余的心理能量来处理除了活动提供的信息之外的其他信息。所有的注意力都集中在相关的刺激上。 —— 米哈伊·契克森米哈伊

战斗发生在战场上。

游戏是按照一定的规则进行的。

任何现实生活中的情况都呈现出一个带有某些已知和未知变量的*环境*。

*如果你不知道如何玩游戏，那就不会有趣。***你会感到焦虑。***

健身房的初学者不会尝试举起400磅的杠铃。他接受自己当前力量的挑战，并期待在未来几个月内取得进步。

*如果你在游戏中太擅长，那也不会有趣。**你会感到无聊。***

国际象棋大师可能会从教初学者的挑战中找到乐趣，但与初学者对弈？**无聊**。大师的技能必须得到考验，以便他进入心流状态。[为了使他的注意力完全沉浸在他追求的目标中](https://thedankoe.com/why-you-always-feel-overwhelmed/)。

<picture loading="lazy" decoding="async" class="wp-image-782">![](../Images/11fff48579ac0701a56dc27b6c0c69a2.png)</picture>

*舒适存在于已知之中。*

*未知中存在不适。*

*满足感存在于边缘。*

<picture loading="lazy" decoding="async" class="wp-image-788">![](../Images/7bb7d3964a9908b2d7b65a2d8650d70d.png)</picture>

我相信这是古人解释“无念”或“非二”时提到的一部分。

一种类似于现代心理学家所命名的“心流”的意识状态。

这与失败有什么关系？

通过追求正确的目标，你可以更频繁地享受失败。

*一种既不太遥远以至于你在未知中迷失，也不太接近以至于你感到无聊的东西。*

同时，也因其价值而受到认可。

在我们结束之前的一个问题：

你的愿景对你来说有价值吗？

也就是说，你是否已经*内化、欣赏并深入思考*了实现这一愿景将如何渗透到你的生活中？

财务自由值得10年的*失败*吗？如果不这样做，另一个选择是什么？那是值得10年奋斗的吗？

重复那个问题序列。将“财务自由”替换为任何好事。那些不能外包或给予你的事情，比如：

+   你的体格和能量水平

+   你一生中培养的品牌

+   精通你的技艺以及你在世界上的行动方式

写下来。在散步时思考你的未来。让你的愿景随着*时间*变得清晰。它不会瞬间出现。

与其相处。

**你不需要更多的建议**

寻求建议是将你采取行动和自行解决问题的能力外包出去。

（寻求建议与获取你目前正在采取行动的具体信息不同，那被称为教育，这也是需要几十年才能掌握的事情）。

重新看看那个图表。

你已经*知道*你需要执行什么。

你需要挑战已知的边缘，挑战自己，尝尝未知的感觉，并投资于你对真正不知道的事情的教育（以及它们何时最适用）。

*“我应该创业吗？”*

*“我应该去健身房吗？”*

*“我应该向那个人表白吗？”*

如果我告诉你“是”，你会学到什么？如果我告诉你“不”，又会怎样？

如果你听了其中的任何一个，你不会在未知中找到自己的答案。你可能会浪费2-3年的时间，因为有人告诉你不要做某事，而不是自己去尝试并找出答案。

建立起来，创造，挑战自己，尝试一切，并找出答案。

那是人类所做的事情。

– 丹·科
