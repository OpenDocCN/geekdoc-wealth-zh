# 未来将只有一条职业道路。

> 原文：[https://thedankoe.com/letters/the-future-of-work-will-be-play/](https://thedankoe.com/letters/the-future-of-work-will-be-play/)

在过去的一个月里，我意识到我生活中（以及许多其他人生活中）的一个共同主题。

*痴迷*。

作为一个孩子，我痴迷于找到更好的生活方式。我不知道是什么激发了我。**我所知道的是，我是一个沉默的观察者**。无论我们去哪里，我都会密切关注人们的行为。不是评判性的，而是辨别性的。

那个人是如何到达人生那个阶段的？他们为什么看起来那样？是因为他们每天早上都在麦当劳的快车道上吗？两个香肠蛋麦满分，一大杯冰咖啡（别忘了多加三泵！）？许多事情都不太让我舒服。

其中之一是传统的职业道路。上学。*认为*自己在18岁时就知道自己想要做什么（实际上这是文化让你去做的事情）。*可能*第一次就能找到一份好工作。*可能*在无法控制自己70%的一天的时间里享受生活。*可能*在65岁时拥有自由去做自己想做的事情……我报名参加了！

不。我不喜欢那样。一个“安全”的未来听起来并不那么安全。所以我花时间在健身房，从社交媒体上我渴望成为的人那里学习，学习任何能帮助我的技能，并且失败了7次以上的商业模式。

顺便说一句：我发现这是我品牌中缺失的一块。所以我将在6月7日推出数字经济学。这将是一个现代商业学位（由真正理解现代在线业务的人教授）。技能获取、品牌建设、快速内容创作，以及创建一个无人能竞争的自己的利基市场（30天内）。[这里是早期鸟更新的等待名单。](https://digitaleconomics.school)

### 《媒介与信息》

我有一个理论。一个理论，即我们作为人类正在在线绘制集体意识地图。

创作者、个人品牌、“思想领袖”和在线教育者正在以指数级的速度提高进化的速度。我们通过课程、系统、分步指南和导师制来浓缩信息，以可消化的方式提炼出多年的经验。原本需要处理6000万个信息比特现在只需要1000万个。

我们可以学到更多，行动更快，做出新的发现，并以同样的方式传授我们的经验。有些人认为这就是生活的意义，而有些人则坐在场外，其他人则在为他们的未来建立巨大的杠杆。所有这些都在追求他们独特的兴趣，这将使每个人都能在商业中走向独特的、未饱和的方向。

这正是人类所做的事情。他们为了生存而模仿。我们模仿我们的父母、老师和朋友，以便学会如何在世界上生存。许多人就止步于此。智能模仿是在之后发生的。你从那些正在压缩你好奇的信息的人那里学习，剖析它，理解它，并体验它。如何？通过在线记录你所学、所建和所经历的内容。要正确地做到这一点，你需要理解什么能吸引注意力。也就是说，如何以吸引其他追求好奇心的人的方式提炼你的学习成果。

现在，回到正轨。你可以选择两条路径来追求你的内在好奇心。

**1) 媒介（信息）**

媒介是我们与他人沟通的方式，而不必面对面。互联网的前端是媒介。这就是你如何吸引注意力来教育、娱乐和激励他人。这就是你如何吸引观众，公开学习，并分发你像压缩的zip文件一样的知识，你可以将其出售（同时加速进化并提高集体福祉）。

要掌握媒介，学习营销、说服、销售、心理学、认识论、哲学以及所有其他能让你理解心智的东西。最好的方法是公开学习。也就是说，通过建立一个现实世界的项目来学习。基于你的好奇心创建一个社交媒体账户。制作视频、播客、文章、帖子以及其他内容。把它当作一种爱好，一个游戏，你在其中学习是什么让其他人兴奋。

至少，你创建了一个公开的资料库，并发展了可以让你在没有学位的情况下获得高薪工作的现代技能。媒体工作很热门。你可以为营销和媒体机构、创意人士、其他个人品牌以及许多其他选项工作。然后你可以利用你收入的增加来推动你的创业之旅（并且已经掌握了媒体游戏）。

最多，你为未来建立了杠杆，创建了一个基于系统的项目，并且赚得比你过上幸福生活所需的资金要多得多。

### 赚大钱

到目前为止，我们已经经历了好奇心、媒体、代码和分销的步骤。但你分发什么呢？数字产品或服务。通过加速教育或解决问题影响人类直接体验的东西。

有三种方法可以做到这一点：

**1) 完成你的服务**

完成你的服务包括像自由职业或代理工作这样的服务。有人有问题，你有一个独特的系统来解决它，他们雇佣你，然后你为他们解决问题。

**2) 与你一起完成**

完成你的服务包括咨询、辅导或指导。你并不是为某人做某事，而是引导他们通过你的系统。你教导他们、教育他们，如何改善他们生活的某个方面。你帮助他们克服你之前经历过的障碍。

**3) DIY（自己动手做）**

DIY适用于数字产品。你通过无代码工具将你的系统打包成易于消化的形式，并通过互联网无限期地分发。你可以直接进入数字产品市场，但你需要分销才能有效地盈利。当你没有分销渠道时，自由职业或咨询可以帮助你发展你的系统，取得成果，并在没有大量受众的情况下赚取收入。

这就是如何创造财富。通过攀登阶梯，发展你的系统，将其打包，并以“没有边际复制成本”的方式分发。
