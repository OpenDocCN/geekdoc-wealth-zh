# 如何通过解决自己的问题来获利

> 原文：[https://thedankoe.com/letters/how-to-profit-from-solving-your-own-problems/](https://thedankoe.com/letters/how-to-profit-from-solving-your-own-problems/)

从我记事起，我就着迷于做不可能的事情。

其他人在以他们热爱的事情为生，*为什么我不能？*

这似乎是最能决定大多数领导者、创意人士以及我们渴望成为的人的成功的问题：

***为什么我不能？***

最近，我在社交媒体上看到了很多这样的说法。

“别想着找到你的激情。”

“先赚钱，然后研究灵性。”

“先找份工作，然后用钱来激发你的激情。”

他们都是很好的建议，我对这些领域也有自己的看法，但就我的观点而言：

它们可能会让人们远离有良好意图的追求，走向更好的生活。

而借助互联网的力量，一些营销知识，以及撰写吸引人信息的能力；你几乎可以货币化任何东西。信息、实体工具或体力劳动。

正如我之前提到的，我一直想做一些不可能的事情。

作为一个孩子，我观察着“社交媒体影响者”（我不是很喜欢这个短语，有负面含义。换句话说，就是那些在网上发布他们所知道的东西的人）。

他们谈论着自己的兴趣，并教授他人他们所知道的知识。所以我花了10年时间，一头扎进书籍、课程、项目和任何其他能让他们不可能的现实成为我自己的事情中。

在大多数人眼中，我已经做到了“不可能的事情。”

我每天早上醒来，写几个小时，然后按照自己的意愿度过一天。构建有意义的计划，去健身房，进行长途散步，并与我最关心的少数人保持关系。

我在这里告诉你，你可以将你热爱的事情作为职业。

[关于工作未来的预测将会同意这一点。](Predictions for the future of work would agree.)

## **通过解决自己的问题来找到目标**

*理解这一点：*

问题不是个人的。

*它们是系统性的。*

你的思想是一个包含系统的系统。

你的身体是一个包含系统的系统。

宇宙是一个包含系统的系统。

目标是引导你生活的神圣力量。

目标是使积极选择变得无缝的内生能量。

目标是引导你向内的指南针——不受外部环境的影响。

**你如何找到你的目标？**

通过解决系统性问题，让自然按照其本意流动。

将这些问题带到你的脑海中：

*你想要在世界上解决什么大问题？*

*你想要在生活中解决什么大问题？*

随着经验的积累，你会意识到这些问题的答案其实是一致的。

在这个当下，那是什么？我会从我的生活中举一个例子。

我正在重读亚历克斯·苏钟·金庞的《Rest》一书。

这是一本令人难以置信的书，它让我想起了自己的工作哲学。

通过重新阅读这本书，我提醒了自己我的“为什么”，我以前的一个目的。

> 重新阅读那些对你影响最大的书籍，它们会再次产生影响。
> 
> — 丹·科伊 (@thedankoe) [2022年7月7日](https://twitter.com/thedankoe/status/1544975014499319810?ref_src=twsrc%5Etfw)

我因为花太多时间离开自己而忘记了自我。

很可能，那个向内的拉力——你将找到你的目的——被社会观点、信仰和其他在你多年无意识生活中根深蒂固的投射所覆盖。

对于可能不理解的人来说，这种向内的拉力可以有很多名字。

有些人称它为你的更高声音。有些人称它为你的召唤。有些人称它为宇宙。

现在，我们将称之为***拉力***。

![图片](../Images/9e1bf96c8de0adac7ad42a236a17c2df.png)

通过剥去抑制它的层，你将增强拉力的强度。

这意味着从外部到内部工作。

从表面到深刻。

从表面到核心。

冰山一角到海底。

不要让深度吓到你，它之所以不拥挤，是有原因的。

**你现在生活中最迫切需要解决的问题是什么？**

你有自尊问题吗？去健身房。

你有金钱问题吗？找一份更好的工作或者开始一个低风险的业务。

通过攀登马斯洛的需求层次，*你揭示了通向你最深层次目的的道路。*

![图片](../Images/073ff58f08784565de3bf58b593e9711.png)

你不知道如何做这些？如果你是我的读者，我就不必提这个……但是，**自我教育。**

学习。这是现代干扰所劫持的最基础的人类驱动力。

你面前就是亚历山大图书馆。

当然，这将需要一些挖掘。几天的研究后，你不会找到所有答案。

你的问题会变得更好。

你将看到表面层次的外部追求的无意义。

你将会有一个（好的）愿望去探索你内在的深度。

你将终身采纳和放弃宏观和微观的目的，直到你发现你的人生目的。

## 目的、成长和影响力的载体

到目前为止，我们一石二鸟。

找到目的、个人成长以及随之而来的众多好处。

想要一举三得吗？

***养成写作的习惯。***

为什么？

想想看。

媒体的基础是有影响力的信息。

一个有影响力的信息始于一个想法。

如果这个想法不是为了通过文字理解而 flesh out；它将仅仅是一个想法。就像你一生中堆积起来的数百万个“百万美元的想法”一样，被扔进了垃圾桶。

通过不写作，你阻止了自己发现新的心理路径。

一个字可以改变你的人生方向。就像重新阅读《休息》这本书一样，为我的业务设定了新的轨迹。

（而且，你不必问，我将在未来的信中写关于工作/休息平衡的内容）。

*写作是你尽可能快速执行想法的方式，这样它们就不会在你心中的深处迷失。*

> 一个人能够培养出的最伟大的技能是缩短想法和执行之间的时间。
> 
> — 丹·科伊 (@thedankoe) [2022年5月5日](https://twitter.com/thedankoe/status/1522144576768823296?ref_src=twsrc%5Etfw)

如果你想要参与任何形式的在线业务，**媒体**是你的基础。

+   教育内容

+   通讯和文章

+   YouTube视频脚本

+   销售和产品页面

+   推特、Instagram帖子、TikTok脚本

列表还在继续。媒体是互联网的前端，如果你不打算学习编程……最好是开始写作。

创意、有说服力、个性化的媒体机构（个人品牌）是无法被取代的。

大型公司可以被自动化，但社会正在失去对那些公司的信任。

> 我们正在过渡到一个真实性的时代。

> – 大型公司

> 人们正在对以下事物建立信任：
>

未来属于那些真实的人。

真实性源于通过你的目标产生的能量。

寻找目标源于解决你自己的问题。

写作是你追求好奇心、了解自己、为你的问题创造解决方案的载体。

媒体是你记录过程、教授你所学，并因此获得报酬的方式。

*另一个论点：*

我最近在Instagram上达到了224K的粉丝。

当与Dakota Robertson交谈时，我们想知道为什么我们比模特或健身影响者获得5-10倍的关注和增长。

外观可以吸引关注，但引人入胜的写作可以吸引更多。

好的写作可以吸引更多的注意力（并传递更多的价值），比六块腹肌还要吸引人。

## 单人企业的体验模型

*对于想要从生活中获得更多的人来说，创业是不可避免的。*

如果你正在阅读这封信，我假设你和我有相似的价值和欲望：

+   完全控制你与谁合作、你做什么工作以及何时工作。

+   能够按照自然法则生活，每天工作2-4小时，并将时间投入到你生活的其他重要领域（比如休息！）

+   做你热爱的事情并获得报酬，即使别人说这是不可能的。

+   自给自足、主权，以及没有金钱问题带来的个人自由。

如果不是大多数，那么通过就业实现这一点是不可能的。

我是不是在说就业是坏事？不，它是增加你收入的一个很好的选择。

同时，就业带来的限制和扭曲了他人对金钱的认识。

> 一旦你被所谓的“美好生活”所吸引
> 
> 每周工作40-80小时
> 
> 一个有限且可预测的薪水
> 
> 工作45年后才能休息
> 
> 要摆脱这种状态会变得难上10倍。
> 
> 看到过太多类似的情况。
> 
> 在你还有很少或没有责任的时候就开始建造。
> 
> — 丹·科伊 (@thedankoe) [2022年7月14日](https://twitter.com/thedankoe/status/1547538903162392576?ref_src=twsrc%5Etfw)

当你周围的人限制了你对金钱的认识（那些收入有限的人）时，当你听到大数字时，你会很难相信。

我记得当我从自营职业中每月徘徊在1万美元左右的时候。

我在网上认识的人提到，他们通过做我正在做的事情，每月赚取10万美元。

我意识到我一直在限制自己。自我强加的限制影响了我的收入。大约3个月后，我每天工作3小时，达到了第一个5万美元的月收入…最多。

当你在公共场合写作，解决自己的问题，并帮助他人解决自己的问题时，就会发生这种情况。

那就是商业的本质…解决问题。那仅此而已。金钱是价值交换的结果。*提供*以金钱解决你最擅长的领域内问题的服务。

当你开始认真对待互联网时，与就业相关的限制就不再存在。

我们中的大多数“在线商业人士”避免谈论赚取大量金钱，因为那通常是人们生活围绕的核心。

我认为更多的人需要谈论金钱。使其正常化。让人们相信他们可以通过做自己想做的事情赚取他们想要的任何金额。听起来可能很俗气，但自信几乎是任何事情的第一步。

思考总是在行动之前。

**体验模型**

这可以像你想象的那样简单或困难。

忘记寻找细分市场。我看到成百上千的商业新手因为认为他们选择的细分市场是问题而放弃。

+   选择2-3个有潜力解决你自身问题的好奇心去追求。

+   沉浸在一个有利于解决这些问题的信息和社会环境中。

+   阅读书籍，关注在线人物，并消除所有可能分散你注意力的其他事物。

+   把你的Twitter、Instagram或LinkedIn账户当作一个分享你的发现和以有说服力的方式谈论你的兴趣的地方（研究说服、影响和营销）。

那就是你的细分市场。你，就是你的细分市场。你，在这个时刻，对少数几件事情感兴趣，这些事情交汇在一起。那个交汇点就是你的受众所在。

由于我对书籍《休息》感兴趣，我的细分市场是那些想要减少工作时间、提高工作质量并享受他们生活全部的人。看看这是怎么运作的？

人们对于不止一件事感兴趣。

对健康感兴趣的人也可以表达对灵性或学习编程的兴趣。这就是乐趣所在——这取决于你如何为能帮助他人解决问题的兴趣构建有说服力的论点。

随着你进化，你的内容和品牌也会进化。

这意味着你将离开你现在正在谈论和销售的东西。

你将使你下面的市场失去饱和度。

创作者经济是无法饱和的，否则它就不是创作者经济（个人企业）。

*人类联系无法规模化。*

如果你想要建立一个价值十亿美元的公司，那么是的，坚持一件事，但这不是我所想要的……而且我会争辩说99%的人实际上并不想要那样。

*给我每年100万美元，就能获得完全的自由。我会做得很好。*

目前，创作者经济加上个人商业模式似乎能够可持续地提供这一点。

你如何进化？通过解决你自己的问题。

所有这些都很简单。只是人们需要一点时间来尝试找到复杂的答案，然后他们才会意识到这一点。

我不喜欢自由职业。它占用了我太多的时间，而且我不喜欢从事不属于我的项目。

这是我能解决的问题，可以写下来，并开始教学……看看效果如何？

我必须进化，所以我开始在Twitter上写作，建立了一个产品，并减少了客户工作的时间。

是时候再次进化了，我建立了一个新的品牌咨询服务，收费更高，将其某些方面产品化，并实现了第一个5万美元的月收入。

我想保持这种收入作为基础，并将工作时间减少到早上2-4小时——所以我再次进化，建立了现代精通，并再次减少了客户工作。

对大多数自由职业者和B2B机构来说，这是不可能实现的。

如果你就是其中之一，你能做的最好的事情就是通过在线写作来建立影响力。

如果你决心过一种自给自足、充实和有意义的生活，就离开B2B空间。

加入持续增长的C2C（创作者到创作者）空间。

随着少数人能看到的潮流而动。

随着社会结构和大型企业向个人教育系统（社交媒体和真实产品）以及以媒体（写作）作为成功支柱的个人企业的转变而流动。

**经验悖论**

就像一份糟糕的招聘启事，你觉得自己不够资格：

*你需要经验才能开始，但你通过开始来获得经验。*

大多数你被告知你不能做的事情都是一种错觉。你把它当作法律，并对自己施加限制。

让我为你简化一下。

在你执行并自己解决这个问题之前，屏蔽掉所有关于建立在线品牌的建议：

**将Twitter（一个写作平台）视为一个公开的日记本。**

+   发布你的想法、观点、信仰、喜好和厌恶。

+   与自己对话。给你的当前自我提供它需要的建议，给你的过去自我提供它希望听到的建议。

+   研究你的好奇心，分享你所学到的知识。

+   积极改善你的生活，分享你遇到的教训和洞察。

+   追求你的目标，记录你解决问题的方法，并出钱帮助他人。

这是一条极其艰难的旅程。

你将面临许多心理战斗。通过写作来克服它们。

你将面临许多个人战斗。通过写作来克服它们。

但你还能期待什么？

一个舒适的、有限的、每两周给你发一次工资的工作，以换取你80%的清醒时间？

不谢。听起来比之前更难。

通过追求你的愿景、目标，并在日常生活中寻找生存，你的生存心理将会通过良好的多巴胺来感谢你。

— 丹·科
